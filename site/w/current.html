current

形容詞・名詞米 /ˈkɜːrənt/ ／ 英 /ˈkʌrənt/cur·rent（2音節）CEFR B1

🗣 第1音節に強勢（ˈCUR-rent）。母音は/ɜːr/で、米音特有の巻き舌のr音を伴う「アー」に近い響き。第2音節はあいまい母音əで軽く添える程度に発音する

  1. 【形容詞】現在の、今の（current events 時事問題、current price 現在の価格）
  2. 【形容詞】現行の、最新の、通用している（Please check the current version of the software. 現行版のソフトウェアを確認してください。）
  3. 【名詞】（電気の）電流（A strong electric current can be dangerous. 強い電流は危険なことがある。）
  4. 【名詞】（水・空気の）流れ、海流・気流（The current carried the raft downstream. 流れがいかだを下流へ運んだ。）

語形変化

形種類発音備考
currents複数形（名詞）/ˈkɜːrənts/電流・海流・気流を指す名詞用法の複数形。語末は/nts/と無声子音が連続するため、はっきり破裂させず「レンツ」を短く添えるように発音する。形容詞用法（現在の）には複数形はない

語法マーク

  • 形容詞は限定用法（current price のように名詞の前）が最も多いが、This information is no longer current. のような叙述用法（be動詞の補語）も可能
  • 形容詞は分類形容詞に近く、比較変化は more current / most current のように迂言的に作る（currenter のような語尾変化はしない）
  • 名詞用法（電流・流れ）は可算名詞：an electric current、strong currents のように単数・複数どちらも取る
  • current exchange rate、current events、current situation など「current + 名詞」の定型コロケーションが非常に多い
  • 他品詞での用法: 名詞用法もある。(1) 電気の「電流」: An electric current flows through the wire.（電線に電流が流れる）。(2) 水や空気の「流れ・海流・気流」: The current swept the swimmer out to sea.（その流れは泳いでいた人を沖へ押し流した）。日本語のカタカナ「カレント」はこれらの名詞用法にも、形容詞用法（現在の、現行の）にも由来しうる。

文型・頻出パターン

  • current + N（限定用法）現在の～、現行の～
    current price現在の価格
    current situation現在の状況
    current events時事問題
  • electric / ocean / air current電流・海流・気流（自然科学的な「流れ」を表す名詞用法）
    ocean current海流
    electric current電流
  • S + be + no longer current（情報・制度などが）もはや現行のものではない、通用しなくなっている（叙述用法）
    This map is no longer current.この地図はもう現行のものではない。
  • against / with the current（文字通り）流れに逆らって／流れに乗って、（比喩的に）大勢に逆らって／大勢に従って
    swim against the current流れに逆らって泳ぐ
    go with the current流れに身を任せる
  • stay / keep current (on / with + N)（情報などについて）最新の状態を保つ、遅れずについていく
    stay current with the latest news最新のニュースに遅れずついていく

他のNGSL語への置き換え

current → flow
元: The strong current made the boat drift far from where it had started.
後: The strong flow made the boat drift far from where it had started.
訳: 強い水の流れのせいで、そのボートは出発した場所からずいぶん流された。
📝 flow は「流れ」全般を指すより一般的な語で、専門的なニュアンスは弱い。current は海流・川の流れ・電流など特定の専門的文脈でよく使われるのに対し、flow は日常会話でも幅広く使える中立的な語。
current → latest
元: Please read the current guidelines, which were updated just last week.
後: Please read the latest guidelines, which were updated just last week.
訳: 最新のガイドラインを読んでください。それはつい先週更新されたばかりです。
📝 latest は「一連のものの中で最も新しい」ことを強調し、過去のバージョンとの比較を暗示する。current は単に「今現在有効な」ことを表すだけで、必ずしも複数版の存在を含意しない。
current → today's
元: Always check the current exchange rate before you transfer money abroad.
後: Always check today's exchange rate before you transfer money abroad.
訳: 海外に送金する前は、必ず今日の為替レートを確認しなさい。
📝 today's は「その日」という具体的な時点を口語的に指す言い方。current はより一般的・フォーマルな文脈（ニュースや金融関連の文章）で使われ、「今このとき有効な」という含意が強い。

⚠️ 日本人の典型誤用
✕ I current live in Osaka, but I plan to move soon.
「私は今大阪に住んでいる」と言いたかったが、current は形容詞なので動詞を修飾できず、文法的に破綻した文に聞こえてしまう。
○ I currently live in Osaka, but I plan to move soon.
私は今大阪に住んでいますが、近いうちに引っ越す予定です。
I currently live in Osaka, but I plan to move soon.
current は形容詞であり、動詞 live を修飾する副詞にはなれない。「現在は」という副詞の意味を表すには -ly を付けた currently が必要。
✕ Please check the current of Japan before you travel.
「日本の通貨を確認して」と言いたかったが、current では意味が通じず、「日本の現在（の状態）？」のように聞こえてしまい、通貨の話だと伝わらない。
○ Please check the currency of Japan before you travel.
旅行前に日本の通貨（為替）を確認しておいてください。
Please check the currency of Japan before you travel.
current（現在の／流れ）と currency（通貨）はスペルが似ているため混同しやすいが、current に「通貨」の意味はない。「為替・通貨」を言いたい場合は currency を使う。
✕ The current situation now is very difficult for small businesses.
「今の状況は今、とても厳しい」という重複した発想の直訳になっており、ネイティブにはくどく不自然に響く。
○ The current situation is very difficult for small businesses.
現在の状況は、中小企業にとって非常に厳しい。
The current situation is very difficult for small businesses.
The situation is very difficult for small businesses right now.
current 自体にすでに「現在の」という意味が含まれているため、now を重ねると意味が重複し、くどい表現になる。
🗾 カタカナ語とのギャップ

IT・ビジネス用語で「カレント」（例: カレントディレクトリ、カレントプライス）として定着しているが、英語の強勢は第1音節（CUR-rent）にあり、その母音/ɜːr/は巻き舌のr音を伴う「アー」に近い音。カタカナの「カ」は短く平板になりがちで、このr音性母音の響きが抜け落ちやすい。また日本語の「カレント」は形容詞（現在の）用法で使われがちだが、同じ発音のまま名詞（電流・海流）の意味でも使われる点に注意。

🌱 Word family・語源
currently（副詞）現在は、目下のところNGSL
currency（名詞）通貨；流通、普及NGSL
concurrent（形容詞）同時に起こる、並行する
undercurrent（名詞）底流、（表面には出ない）根底にある感情や動き
語源: ラテン語 currere「走る」の現在分詞 currentem（「走っている、流れている」の意）に由来。古フランス語 corant / curant を経て中英語に入り、「（水や電気が）流れている」→「（時間が）今まさに流れている＝現在の」という意味に広がった。
currently（現在は）— current + -ly（副詞化）で「現在流れている状態で」→「今は」
currency（通貨）— current + -cy（名詞化）で、社会を「流通・循環している」お金を指すようになった
concurrent（同時に起こる）— con-（共に）+ current（走る、流れる）で「共に走る・並行して流れる」
💡 覚え方: ラテン語 currere「走る」のイメージが核。水や電気が「走り流れる」のが current の名詞用法、その流れが「今まさに走っている」ことから形容詞の「現在の」が生まれた。currency（通貨）はお金が社会を「流れ走る」姿、concurrent は「共に走る＝同時進行」と覚えると芋づる式につながる。
⚖️ 類義語の使い分け
present — present も「現在の」を表せるが、より形式的・時間的な位置づけを淡々と述べる語で「変化していく／更新される」含みが薄い。current は「今のところ有効な、最新の」という、移り変わりを前提としたニュアンスが強い。
up-to-date — up-to-date は「最新の情報に更新されている」ことを強調する口語的な言い方。current は単に「今のところ通用・有効である」という事実を述べるだけで、必ずしも「最近更新された」ことを含意しない。
flow — flow は液体・データなどの「流れ」全般を指す一般的な語。current（名詞）は海流・気流・電流のように、より大きな流体の中の特定の方向を持つ力強い流れを指す、より専門的・具体的な語。
stream — stream は連続した細い流れ（小川やデータの連続的な流れ）を指すことが多い。current は海や大気など大きな塊の中で生じる方向性のある流れを指し、より力強く物を押し流すニュアンスがある。
💬 熟語・慣用表現 (2)
swim against the current 大勢に逆らう、周囲とは違うやり方をする中立
She decided to swim against the current and start her own business instead of joining a big company.（彼女は大企業に就職する代わりに、周囲に逆らって自分の会社を興すことにした。）
🔍 「流れに逆らって泳ぐ」という文字通りの意味から転じて、比喩的に「世間の風潮や多数派の意見に逆らって行動する」ことを表す。
📝 文字通り水流の中を泳ぐ意味でも使えるが、比喩的に「主流・多数派に逆らう」という意味で使われることが多い。ほぼ同義の表現に swim against the tide がある。反意表現は swim with the current。
swim with the current 大勢に従う、周囲に流される中立
Instead of arguing with his boss, he just decided to swim with the current and go along with the plan.（彼は上司に反論せず、周りに流されてその計画に従うことにした。）
🔍 「流れに乗って泳ぐ」という文字通りの意味から転じて、比喩的に「抵抗せず周囲の意見や多数派に従う」ことを表す。
📝 楽な方へ流されるという消極的なニュアンスを含むこともある。反意表現は swim against the current。
