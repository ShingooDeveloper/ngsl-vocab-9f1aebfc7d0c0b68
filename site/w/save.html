save

動詞・名詞米 /seɪv/ ／ 英 /seɪv/save（1音節）CEFR A2

🗣 1音節語なので強勢の位置に迷う余地はない。母音は二重母音 /eɪ/ で「エ」から「イ」へなめらかに滑らせて1拍で発音する。語末の /v/ は有声子音なので最後まで声を保ったまま出す（無声化して safe /seɪf/ のように /f/ にならないよう注意）

  1. 【動詞】(お金を)貯める、節約する（save money for a new phone 新しいスマホのためにお金を貯める）
  2. 【動詞】救う、助ける（save someone's life 人の命を救う）
  3. 【動詞】(データ・ファイルを)保存する（save a file ファイルを保存する）
  4. 【動詞】(時間・手間などを)省く、無駄にしない（save time 時間を省く）
  5. 【動詞】(スポーツで)シュート・ゴールを防ぐ、セーブする（save a penalty PKを止める）
  6. 【名詞】(スポーツでの)好セーブ、防いだシュート（The goalkeeper made a fantastic save. ゴールキーパーがすばらしいセーブをした。）

語形変化

形種類発音備考
saves三単現/seɪvz/語末が有声音 /v/ の後なので -s は /z/（「セイヴズ」であって「セイヴス」ではない）
saved過去形・過去分詞/seɪvd/規則変化。語末が有声音 /v/ の後なので -ed は /d/（「セイヴド」）
savinging形/ˈseɪvɪŋ/語末の e を落として -ing を付ける（saveing は誤り）

語法マーク

  • 基本は他動詞。目的語（お金・時間・データ・人など）を必ず伴い、文脈によって『貯める』『節約する』『保存する』『救う』と訳し分ける
  • save + 人 + 物（SVOO）で『(人)のために(時間・お金など)を省いてあげる』の意味になる: This shortcut will save you a lot of time.
  • 『AをBから救う』は save A from B。前置詞は of ではなく from を使う点に注意
  • save up（for ~）で『(～のために)お金を貯める』という句動詞になり、この形では目的語を伴わない自動詞的な使い方になる
  • 『(時間・お金を)省く』の意味で It/This will save ~ の構文を使うときは、恒常的な事実を述べることが多く進行形にはしないのが普通
  • 他品詞での用法: 名詞用法もある。スポーツで守備側がシュート・ゴールを防ぐ「セーブ」: The goalkeeper made a fantastic save.（ゴールキーパーがすばらしいセーブをした）。カタカナ「セーブ」は主にゲームでの「データ保存」の意味で定着しているが、この名詞用法はサッカーやアイスホッケーなど守備プレーの実況で頻出する。

文型・頻出パターン

  • save + O〜を貯める・節約する・保存する・救う（最も基本的な他動詞用法）
    save moneyお金を貯める
    save the fileファイルを保存する
    save her life彼女の命を救う
  • save + 人 + 物（SVOO）（人）のために（時間・お金など）を省いてあげる
    This app saves me a lot of time every day.このアプリのおかげで毎日かなりの時間が省ける。
  • save + O + from + O〜を…（危険や困難）から救う
    The vaccine has saved millions of people from the disease.そのワクチンは何百万もの人々をその病気から救ってきた。
  • save (up) + O + for + 目的〜のために（お金など）を取っておく・貯める
    save some money for a rainy dayいざという時のためにお金を少し取っておく
    save up for a new laptop新しいノートパソコンのためにお金を貯める

他のNGSL語への置き換え

save → earned
元: I've saved enough money to buy a new bike.
後: I've earned enough money to buy a new bike.
訳: この自転車を買うのに十分なお金をもう稼いだ。
📝 earn は「(労働などで)稼ぐ」で、収入をゼロから得る行為そのものを指す。save は既にあるお金を使わずに「貯める」ことを表すのに対し、earn は稼ぐ過程に焦点がある点が異なる。
save → invested
元: She keeps all the money that she has saved in a special account.
後: She keeps all the money that she has invested in a special account.
訳: 彼女は投資したお金を全部、特別な口座で管理している。
📝 invest は「(将来の利益を期待して)投資する」で、お金を積極的に運用に回すニュアンス。save が使わずに貯め込む受動的な行為であるのに対し、invest は増やす目的で能動的にお金を動かす点が異なる。
save → copied
元: The file was saved automatically before the computer crashed.
後: The file was copied automatically before the computer crashed.
訳: コンピューターがクラッシュする前に、ファイルは自動的にコピーされていた。
📝 copy は「複製を作る」ことで、元のファイルは残したまま別に複製ができる。save は保存動作全般（新規保存・上書き含む）を指すのに対し、copy は複製の作成に限定される点が異なる。

詳細
⚠️ 日本人の典型誤用
✕ I don't have time to save today.
「今日は時間の余裕がない」と言いたかったが、save を使うと「今日は貯める時間がない」のような意味の通りにくい文になってしまう。
○ I don't have time to spare today.
今日は時間に余裕がない。
I don't have time to spare today.
「(時間の)余裕がある、割く」という意味の言い方には spare を使う。save は「貯める・節約する」という意味であり、時間に余裕があるかどうかを表す語ではない。
✕ The firefighters saved the family of the fire.
「消防士たちはその家族を火事から救った」と言いたかったが、of を使うと前置詞が合わず、ネイティブには意味が取りにくい不自然な文に聞こえる。
○ The firefighters saved the family from the fire.
消防士たちはその家族を火事から救った。
The firefighters saved the family from the fire.
「AをBから救う」は save A from B。of は所有・構成を表す前置詞であり、「〜から」という起点・脱出元の意味にはならない。
✕ Please save a seat to me.
「席をとっておいて」と言いたかったが、to me にすると前置詞が不自然で、ネイティブには違和感のある文に聞こえる。
○ Please save a seat for me.
席をとっておいてください。
Please save a seat for me.
日本語「私に席をとっておいて」の「に」を直訳して to を選んでしまう誤り。save の SVOO（save + 人 + 物）を前置詞句で言い換えるときは、give型（to）ではなく buy/make型と同じ for を使う。
🗾 カタカナ語とのギャップ

カタカナ「セーブ」はゲームでの「データ保存」の意味に限定されがちだが、実際の save は「貯める」「節約する」「(命を)救う」まで含む使用範囲の広い基本動詞。発音面では、二重母音 /eɪ/ を「エー」と伸ばさずに「エィ」となめらかに滑らせ、語末の /v/ は最後まで有声のまま出す（無声化して safe /seɪf/（セーフ）のように聞こえないよう注意。save と safe は綴りも意味も別の語）。

🌱 Word family・語源
savings（名詞）貯金、貯蓄（額）（常に複数形で使う）
saver（名詞）貯蓄する人；(時間・労力などを)省くもの（energy saver 省エネ機器 等）
savior / saviour（名詞）救世主、救った人
lifesaver（名詞）命の恩人；(転じて)大助かりのもの
語源: 古フランス語 sauver（救う）を経由し、ラテン語 salvare「救う、安全にする」に由来。さらに遡るとラテン語の形容詞 salvus「無事な、安全な」に行き着く。
safe（安全な）— 同じラテン語 salvus に由来する語で、save は salvus を『動詞化』したものと言える
salvation（救済、救い）— ラテン語 salvare の名詞形が古フランス語・英語に入ったもの。religiousな文脈での『救い』を表す
💡 覚え方: save・safe・salvation はすべてラテン語 salvus（無事な）から枝分かれした親戚語。『無事な状態にする』が save の core イメージで、そこから『(お金を)無事に残す＝貯める』『(データを)無事な状態にする＝保存する』『(人を)無事にする＝救う』の3つの意味が生まれたと考えると覚えやすい。
🔗 句動詞
save up — こつこつお金を貯める
They saved up for two years to buy their first house.（彼らは最初の家を買うために2年間こつこつお金を貯めた。）
save on — (費用・資源など)を節約する
Switching off standby mode can save on electricity.（待機電力をオフにすると電気代を節約できる。）
⚖️ 類義語の使い分け
economize — より形式的な語で、支出全般を切り詰めるという行為そのものに焦点がある（economize on electricity）。save はもっと日常的で、貯める・省く対象（お金・時間・データなど）を幅広く取れる点が異なる。
rescue — 危険や緊急事態から助け出すという劇的な状況に限定される。save も『命を救う』の意味では近いが、save はデータ保存や時間の節約など、危険を伴わない日常的な場面でも使える点が大きく異なる。
store — store は情報や物を後で使うために保管しておくことに焦点があり、『失われるのを防ぐ』というニュアンスは弱い。save a file は『今の状態を失わないようにする』という緊急性を含むのに対し、store はより中立的な保管行為を表す。
💬 熟語・慣用表現 (7)
save face 面目を保つ、体面を保つ中立
The company quickly issued an apology to save face after the scandal.（その会社はスキャンダルの後、面目を保つためすぐに謝罪を出した。）
🔍 「顔を救う」という直訳から「恥をかかずに面目・体面を保つ」という意味に転じた表現。個人の評判だけでなく、企業や国家の外交でも使われる。
📝 反対語は lose face（面目を失う）。ビジネスや外交の記事で頻出する決まり文句。
save the day 窮地を救う、土壇場で危機を切り抜ける口語寄り／中立
The backup generator saved the day when the power went out during the concert.（コンサート中に停電が起きたとき、予備の発電機が窮地を救ってくれた。）
🔍 「その日を救う」という比喩から「危機的な状況を土壇場で救う」という意味になった。人だけでなく道具や偶然の出来事が主語になることも多い。
📝 ヒーローが危機的状況を解決する場面の定番表現。ニュースや物語で幅広く使われる。
save your breath 言うだけ無駄だから黙っておく、口をきくだけ損口語
Save your breath—he never listens to anyone's advice.（言っても無駄だよ。彼は誰の忠告も聞かないんだから。）
🔍 「自分の息を節約する」という直訳から「話しても無駄だから話すな」という忠告の意味になった。
📝 命令形 Save your breath! の形で「言うだけ無駄だからやめておけ」と強く忠告する時によく使われる。
save (something) for a rainy day まさかの時に備えて蓄えておく中立
Try to save a little money for a rainy day instead of spending everything right away.（すぐに全部使ってしまわずに、まさかの時のためにいくらか蓄えておくようにしなさい。）
🔍 「雨の日のために取っておく」という比喩から「将来の困難・非常時に備えて(お金などを)蓄えておく」という意味になった。
📝 a rainy day は文字通りの雨天ではなく「将来訪れるかもしれない困難・非常時」を指す比喩表現。for a rainy day 単独でも「万一に備えて」の意味で使われる。
saving grace (欠点だらけの中で)唯一の美点、辛うじて評価できる点中立／やや文語
The acting was terrible, but the special effects were the movie's one saving grace.（演技はひどかったが、特殊効果だけはその映画の唯一の見どころだった。）
🔍 元は「救いの恵み」という宗教的な意味だったが、そこから転じて「全体としては良くないものの、これのおかげで何とか評価できる唯一の良い点」を指す名詞句になった。
📝 "the saving grace of ~" や "one saving grace" の形で使われることが多い名詞表現。動詞 save とは離れた慣用的な意味を持つ。
save your (own) skin (自分の)身を守る、窮地を脱する口語
Calling for help at the last minute saved his skin.（土壇場で助けを呼んだおかげで彼は窮地を脱した。）
🔍 「(人の)皮膚を救う」という直訳から「危険な状況から自分や誰かの身を守る」という意味に転じた。
📝 save one's own skin は「(他人を犠牲にしてでも)自分の身を守る」という利己的なニュアンスで使われることが多い。save someone's neck もほぼ同義。
a penny saved is a penny earned 一銭の節約は一銭の稼ぎ（無駄遣いをしないことは、お金を稼ぐのと同じ価値がある）ことわざ／やや文語
My grandmother always says a penny saved is a penny earned, so she never wastes anything.（祖母はいつも『貯めた一銭は稼いだ一銭』と言っていて、何一つ無駄にしない。）
🔍 「貯めた1ペニーは稼いだ1ペニーに等しい」ということわざ。節約の大切さを説く決まり文句で、構成語からは節約と収入が同等だという教訓は読み取れない。
📝 ベンジャミン・フランクリンの言葉として広く知られることわざ。節約を美徳とする価値観を表す決まり文句として日常会話でも引用される。
