because

接続詞 (conjunction)米 /bɪˈkɔːz/ ／ 英 /bɪˈkɒz/be·cause（2音節）CEFR A1

🗣 第2音節 -cause に強勢（/bɪˈkɔːz/）。第1音節 be- は弱く曖昧母音 /bɪ/ で軽く添える程度、第2音節の /kɔːz/ を長めにはっきり発音する。語末は有声音 /z/ で、「コーズ」であって「コース」ではない

  1. 【接続詞】〜だから、〜なので（理由・原因を導く従属接続詞。because節が原因、主節が結果を表す。I was late because the train was delayed. 電車が遅れたので遅刻した）
  2. 【接続詞】（just because 〜 doesn't mean ... の形で）〜だからといって…というわけではない（表面的な決めつけを否定する頻出構文。Just because it's expensive doesn't mean it's good. 高いからといって良いとは限らない）
  3. 【接続詞】（not because ... but because ... の形で）〜だからではなく…だから（理由を対比・訂正する。I stayed not because I wanted to, but because I had no choice. 残ったのはそうしたかったからではなく、選択肢がなかったからだ）

語法マーク

  • 従属接続詞（subordinating conjunction）。because節は単独では文として成立せず、必ず主節とセットで使う
  • 主節の後ろに because 節を置くのが最も自然な語順（S + V because S + V.）。この場合カンマは不要
  • because 節を文頭に置く倒置も可能（Because S + V, S + V.）。ややフォーマル・書き言葉的で、この場合は節の終わりにカンマを打つ
  • 否定文で because 節が続くと、否定が主節にかかるか理由節にかかるか曖昧になることがある（He didn't go because he was afraid. は「怖かったから行かなかった」とも「怖かったからという理由だけで行かなかったわけではない」とも取れる）。曖昧さを避けたい場合はカンマやnot because ... but because ...の対比構文で明確化する
  • as/since と異なり、because は時間の意味を一切持たない純粋な理由専用の接続詞（as/sinceは「〜する時に」「〜して以来」の意味も持ちうる）
  • 他品詞での用法: 従属接続詞が基本用法だが、くだけた話し言葉・ネット英語では because の直後に名詞（句）を直接置き、前置詞のように使う用法がある（because of を省略した形。例: I'm late because traffic.（渋滞のせいで遅れた）、She left early because reasons.（理由があって早退した、の意でわざとおどけて理由をぼかす言い方））。この用法はくだけた口語・SNS特有でフォーマルな文章では使わない。なお発話では because がしばしば 'cause や cos（/kəz/, /kʌz/）と短縮される（コーパスの語族データにある cos はこの口語短縮形で、独立した見出し語ではない）。

文型・頻出パターン

  • S + V because + S + V.結果を先に述べ、理由を後ろに添える最も基本的で自然な語順（話し言葉の第一選択）
    She smiled because the news was good.知らせが良かったので、彼女は微笑んだ。
    He stayed home because he had a cold.風邪をひいていたので、彼は家にいた。
  • Because + S + V, S + V.理由を先に述べて強調する倒置の語順。フォーマルな文章・スピーチで好まれ、文頭に来た場合は節の終わりにカンマが必須
    Because the bridge was closed, we had to take a detour.橋が閉鎖されていたので、私たちは迂回しなければならなかった。
  • It's (simply/just) because + S + V.疑問（Why ...?）への答えや強調構文として、理由だけを取り出して独立させて述べる言い方
    Why did you quit? — It's simply because I needed a change.なぜ辞めたの?— 単に変化が欲しかったからだよ。
  • not because S1 but because S2理由を対比・訂正する構文。「S1だからではなく、S2だから」と誤解されやすい理由を正す
    I apologized not because I was wrong but because I wanted peace.謝ったのは、自分が間違っていたからではなく、平穏を望んだからだ。

他のNGSL語への置き換え

because → since
元: I haven't spoken to my brother in months because we had a big argument last year.
後: I haven't spoken to my brother in months since we had a big argument last year.
訳: 去年大喧嘩をして以来、何か月も兄と話していない。
📝 sinceは「〜して以来」という時間の起点も表せるため、この文では理由というより「その出来事から現在まで」という時間的な意味合いが強まる。becauseは純粋に理由・原因のみを表すのに対し、sinceは理由（ややフォーマル）にも時間にも使える多義語で、文脈次第で意味がぶれることがある。
because → as
元: The outdoor concert was cancelled because heavy rain had been forecast for the entire evening.
後: The outdoor concert was cancelled as heavy rain had been forecast for the entire evening.
訳: 屋外コンサートは中止になった。一晩中大雨が予想されていたためだ。
📝 asもbecauseと同様に「〜のため」と理由を表せるが、ややフォーマルで書き言葉的。また、asは「〜している最中に」という同時進行の意味も持つため、文脈によっては時間的な意味と混同されることがある。becauseの方が理由であることが曖昧さなく明確に伝わる。
because → given that
元: I built my house on top of a mountain because zombies can't climb up that high.
後: I built my house on top of a mountain, given that zombies can't climb up that high.
訳: ゾンビがそんなに高くまで登ってこられないことを考慮して、山の上に家を建てた。
📝 given thatは「〜という事実を考慮すると」という含みがあり、既知の条件を踏まえた判断であることを強調するため、becauseよりもフォーマルで論理的な響きになる。日常会話のbecauseに比べ、レポートや議論など書き言葉で好まれる。

頻出コロケーション

  • because of（〜が原因で、〜のせいで：後ろに名詞・動名詞が続く前置詞句）
  • just because（〜だからといって：doesn't mean/isn't 等と共に使われ、短絡的な決めつけを否定する）
  • simply because（単に〜という理由だけで：理由を弱める・限定するニュアンス）
  • not because ... but because（〜だからではなく…だから：理由の対比）
詳細
⚠️ 日本人の典型誤用
✕ Because it was raining, so we stayed home.
「雨が降っていたから、それで家にいた」と言いたかったのだろうが、ネイティブにはbecauseとsoが理由と結果を二重に示していて冗長・非文法的に響く。
○ Because it was raining, we stayed home. / It was raining, so we stayed home.
雨が降っていたので、私たちは家にいた。
Because it was raining, we stayed home.
It was raining, so we stayed home.
日本語の「〜だから、それで…」を直訳してbecauseとsoを両方使ってしまう誤り。英語では1つの因果関係に接続詞は1つが原則で、becauseを使ったら主節にsoは不要（逆にsoを使うならbecauseを消す）。
✕ I like this city. Because it's very safe.
「この街が好きだ。とても安全だから」と2文に分けて言いたかったのだろうが、ネイティブにはBecause以下が主節を欠いた尻切れの文に聞こえ、書き言葉としては不自然。
○ I like this city because it's very safe.
この街は、とても安全なので好きだ。
I like this city because it's very safe.
becauseが導くのは従属節であり、単独ではピリオドで区切れる独立した文にならない。話し言葉では許容されることもあるが、フォーマルな文章では主節と1文につなげるのが原則。
✕ The reason I was late is because there was an accident.
「遅れた理由は事故があったからだ」と言いたかったのだろうが、the reason isとbecauseで「理由」を二重に述べる形になり、母語話者にはくどく非標準的に響く（ネイティブでも使う人はいるが、フォーマルな文章では避けられる）。
○ The reason I was late is that there was an accident. / I was late because there was an accident.
私が遅れた理由は、事故があったことだ。 / 事故があったので、私は遅れた。
The reason I was late is that there was an accident.
I was late because there was an accident.
the reason is（理由は〜だ）とbecause（〜だから）は同じ「理由」の意味を重ねてしまう典型的な冗長表現（reason is because）。the reasonを主語にするならis thatで受け、becauseを使うならthe reasonを使わない単純な文にする。
🗾 カタカナ語とのギャップ

カタカナ「ビコーズ」は4拍で読みがちだが、実際は2音節（/bɪˈkɔːz/）。第1音節 be- は曖昧母音で弱く、力を入れるのは第2音節。また語末は有声音 /z/ で、日本語の「ス」につられて無声の [s] にすると because ではなく別の音に聞こえる。母音 /ɔː/ も「オー」より口を開けた低めの音で、caught や law と同じ音であることを意識する

🌱 Word family・語源
cause（名詞・動詞）原因；引き起こすNGSL
'cause / cos（接続詞（口語の短縮形））〜だから（becauseのくだけた発音・綴り）
語源: 中英語の by cause（「その原因によって」の意）が1語に結合してできた語。cause はさらに古フランス語 cause を経て、ラテン語 causa「原因、理由、訴訟」に遡る。前置詞 by + 名詞 cause という句がまるごと接続詞化した珍しい成り立ちを持つ。
cause（原因；引き起こす）— because の core となる語そのもの。by + cause がそのまま because になった
excuse（言い訳、許す）— ラテン語 causa 系。ex-（〜から外す）+ causa（原因・申し立て）で「責め・原因から外す」が原義
accuse（非難する、告発する）— ラテン語 causa 系。ad-（〜に対して）+ causari（申し立てる）で「原因・罪を人のせいにする」が原義
💡 覚え方: because = by + cause（その cause＝原因によって）。cause の仲間 excuse（原因から外す＝言い訳）、accuse（原因を人に向ける＝非難する）と一緒に、ラテン語 causa「原因・申し立て」ファミリーとして覚えると綴りも定着しやすい。
⚖️ 類義語の使い分け
since — 理由も表せるが、時間の起点（〜して以来）の意味も併せ持つ多義語。becauseは純粋に理由のみを表し曖昧さがない。sinceによる理由づけはやや軽く、話し手・聞き手が既に知っている理由を確認的に添える響きがある。
as — becauseと同様に理由を表せるが、書き言葉的でよりフォーマル。また「〜している最中に」という同時進行の意味も持つため、文脈によっては時間的な意味と混同されうる。becauseの方が理由であることが明確。
for — 接続詞としてのforは「というのも〜だから」と補足的・説明的な理由を後付けで添える非常にフォーマル・文語的な用法（等位接続詞なので前にカンマを置く）。becauseほど直接的な因果関係を強く主張せず、日常会話ではほぼ使われない。
seeing that / seeing as — 「〜であることを踏まえると」という、話し手・聞き手の双方に明らかな事実を理由として持ち出すくだけた口語表現。becauseより主観的な判断の根拠を示すニュアンスが強い。
