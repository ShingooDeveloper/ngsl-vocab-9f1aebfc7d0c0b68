test

名詞・動詞米 /test/ ／ 英 /test/test（1音節）CEFR A2

🗣 1音節語なので強勢の位置に迷いはない。/test/ は子音クラスタで始まり子音で終わる典型的な英語の形（tests になるとさらに /sts/ と子音が3つ連続する）。カタカナの「テ・ス・ト」のように母音を挟まず、s と t をつなげて一気に発音する

  1. 【名詞】試験、テスト（She passed the test on her first try. 彼女は一回目でその試験に合格した。）
  2. 【名詞】検査、検定（The doctor ordered a blood test to check for infection. 医者は感染がないか調べるために血液検査を指示した。）
  3. 【動詞】(性能・品質などを)試す、試験する（We need to test this software before release. リリース前にこのソフトウェアをテストする必要がある。）
  4. 【動詞】(人・物の限界などを)試す、試練にかける（The long delay really tested everyone's patience. その長い遅延はみんなの忍耐力を本当に試した。）

語形変化

形種類発音備考
tests三単現・名詞複数形/tests/無声子音の後なので -s は /s/。動詞の三単現 tests（試す）と名詞の複数形 tests（試験）は綴り・発音とも同一で、文脈でしか区別できない
tested過去形・過去分詞/ˈtestɪd/規則変化。語末が t のため -ed は音節を増やして /ɪd/ となる（「テステッド」であって「テストゥ」ではない）
testinging形/ˈtestɪŋ/規則変化。e で終わらない語なのでそのまま -ing を付ける

語法マーク

  • 名詞は可算（a test / two tests）。「試験」「検査」いずれの意味でも数えられる
  • 動詞は他動詞が基本で目的語を必ず伴う（test the software）。前置詞 on/for を伴い「AをBに対して試す／Aについて検査する」を表す（test A on B, test A for B）
  • test positive/negative のように形容詞を直接後ろに置く連結動詞的用法がある（不規則な自動詞的働き）
  • 進行形にできる動作動詞（We are testing the app now.）
  • 他品詞での用法: 動詞用法では他動詞（test the water「水を検査する」）だけでなく、testの後ろに形容詞を置いて「（検査結果が）〜と出る」を表す連結動詞的な用法が頻出する: test positive/negative (for ~)（〜の検査で陽性／陰性反応が出る）。ニュース・医療・スポーツのドーピング検査などで非常によく見る言い方で、be動詞に近い感覚で形容詞補語を取る点が特徴。

文型・頻出パターン

  • test + O〜を試す・検査する（最も基本的な他動詞用法）
    test a theory理論を検証する
    test the brakesブレーキを点検する
  • test + O + on/for + NOをNに対して試す／Nの有無についてOを検査する
    test the drug on animalsその薬を動物に対して試験する
    test her blood for allergiesアレルギーの有無を調べるため彼女の血液を検査する
  • test positive/negative (for + N)（Nの）検査結果が陽性／陰性と出る
    He tested positive for the virus.彼はそのウイルスの検査で陽性だった。
  • take / pass / fail / study for + a test試験に関する定番の軽動詞構文（名詞用法）
    study for a testテストのために勉強する
    fail a testテストに落ちる
  • test + 名詞（複合名詞の限定用法）「試験用の〜」を表す名詞修飾
    a test tube試験管
    a test drive試乗

他のNGSL語への置き換え

test → trialed
元: The new vaccine was tested on thousands of volunteers before it was approved.
後: The new vaccine was trialed on thousands of volunteers before it was approved.
訳: その新しいワクチンは、承認される前に何千人ものボランティアで臨床試験が行われた。
📝 trial(led) は医薬品や新製品の「臨床試験・実地試験」を指す専門的で公式な語。test はより一般的で日常的な「試す」全般をカバーする中立語。
test → check
元: The manager had the interns test the new software before the launch.
後: The manager had the interns check the new software before the launch.
訳: マネージャーはインターンたちに、発売前に新しいソフトウェアを確認させた。
📝 check は「(問題がないか)ざっと確認する」という軽いニュアンス。test は実際に動かして機能や性能を検証するという、より踏み込んだ行為を含意する。
test → examined
元: If they tested the water more carefully, they would find the pollution source faster.
後: If they examined the water more carefully, they would find the pollution source faster.
訳: 水をもっと注意深く調べれば、彼らはもっと早く汚染源を見つけられるだろうに。
📝 examine は「詳しく調べる、検査する」で、観察・分析にじっくり時間をかけるニュアンスが強い。test は実験や測定によって性質・機能を確かめる行為に重点を置く。

⚠️ 日本人の典型誤用
✕ Let's test this new restaurant tonight.
「今夜この新しいレストランを試してみよう」と言いたかったが、testは性能や品質を検証するイメージが強く、カジュアルに食事を楽しむ文脈にはそぐわない。
○ Let's try this new restaurant tonight.
今夜この新しいレストランに行ってみよう（試してみよう）。
Let's try this new restaurant tonight.
日本語の「試す」は幅広く test にも try にも対応するが、食事や体験を気軽に楽しむ文脈では try が自然。test は機器・薬・性能などを検証するニュアンスが強く、レストランで食事を楽しむ場面に使うと、まるで料理を分析・検査しているような不自然な響きになる。
✕ I received a difficult test yesterday.
「昨日、難しい試験を受けた」と言いたかったが、receiveでは「試験問題を受け取った」という意味にしかならず、実際に受験したことが伝わらない。
○ I took a difficult test yesterday.
昨日、難しい試験を受けた。
I took a difficult test yesterday.
日本語「試験を受ける」の「受ける」を直訳して receive を選んでしまう誤り。試験を受験する意味では take a test が決まった言い方で、receive a test は「試験問題を手渡された」というだけの意味に聞こえてしまう。
✕ I'm not good at test.
「私はテストが苦手だ」と言いたかったが、可算名詞のtestを単数無冠詞のまま使うと英語として不完全に聞こえる。
○ I'm not good at tests.
私はテストが苦手だ（テストを受けるのが苦手だ）。
I'm not good at tests.
I'm not good at taking tests.
test は可算名詞なので、一般論として「テストが苦手」と言う場合は無冠詞単数では使えない。複数形 tests にするか、taking tests のように動名詞句にする必要がある。日本語には名詞の数・冠詞の区別がないため、この種の脱落が起きやすい。
🗾 カタカナ語とのギャップ

カタカナ「テスト」は「テ・ス・ト」の3モーラだが、英語は /test/ の1音節。特に語末の t は母音を付けずに軽く止める（破裂させないことも多い）。三単現・複数形の tests は /tests/ とさらに子音が連続し、「テストゥス」のように母音を足して読まないこと。

🌱 Word family・語源
tester（名詞）検査係、テスター（測定器・検査担当者）
retest（動詞）再検査する、再試験する
untested（形容詞）未検証の、試されていない
testable（形容詞）検証可能な、試験できる
語源: 中英語 test は古フランス語 test（貴金属の純度を確かめるために使われた素焼きの壺）を経て、ラテン語 testum（素焼きの容器）に由来する。もともと金細工師が貴金属を壺で溶かし、その純度を確かめる「検定」の道具そのものを指す語で、16世紀頃から道具ではなく「検証・試験する行為」そのものを指すよう意味が広がった。
tester（検査係、テスター）— test + -er（人・道具を表す接尾辞）で「試す人・試す道具」
retest（再検査する）— re-（再び）+ test で「もう一度試す」
💡 覚え方: 中世の金細工師は、貴金属の純度を確かめるために素焼きの壺（ラテン語 testum）で金属を溶かして分析した。この「壺で確かめる」イメージがそのまま「試す・検証する」という意味に広がったのが test。
🔗 句動詞
test out — （実際に使って／動かして）試してみる
We tested out the new recipe before serving it to guests.（お客さんに出す前に、その新しいレシピを試してみた。）
⚖️ 類義語の使い分け
exam / examination — 学校の試験など、評価を目的とした公式な筆記・口述試験を指すことが多く、testよりも改まった響き。testは製品・薬・水質検査など対象を問わず「試す・検証する」全般に使える中立語。
quiz — quizは口頭・小テストなど、気軽で採点重視度の低い小規模な確認テストを指す。testはより正式・本格的な検証・試験一般を指す。
trial — trialは新薬・新製品などを実地で試す「試用・治験」に焦点を当てた、より専門的でプロセス志向の語。testは結果の合否・良否の判定に焦点がある。
check — checkは「問題がないかざっと確認する」軽いニュアンス。testは実際に動かす・数値を測るなど、より踏み込んだ検証行為を含意する。
💬 熟語・慣用表現 (5)
put ~ to the test 〜を実際に試す、〜の真価を試す中立
The marathon really put his endurance to the test.（そのマラソンは彼の持久力を本当の意味で試すものだった。）
🔍 「〜を試験の場に置く」というイメージから、能力・理論・主張などを実地で検証する意味に転じた定型表現。
📝 put A to the test の形で使い、Aには能力・理論・忍耐力・製品などが入る。単に『試す』だけならtestで足りるが、この表現は『実際に通用するかどうかを証明する』という緊張感を伴う点が特徴。
stand the test of time 時の試練に耐える、長い年月を経ても価値を失わない中立／ややフォーマル
Shakespeare's plays have stood the test of time.（シェイクスピアの戯曲は時代を超えて今なお高く評価されている。）
🔍 「時間という試験に立ち向かって耐える」というイメージから、長期間にわたって価値・有効性を保ち続けることを表す。
📝 作品・関係・製品など、長期間にわたる評価に耐えたものを褒める文脈で使われる。現在完了形（has/have stood）で使われることが多い。
test the waters 様子を見る、本格的に始める前に反応を探る中立／ビジネス
The company launched a small pilot program to test the waters before a nationwide rollout.（その会社は全国展開する前に、様子を見るため小規模な試験導入を行った。）
🔍 「水に足先を入れて温度を確かめる」イメージから、本格的に取り組む前に反応や状況を慎重に探ることを表す比喩表現。
📝 waters と複数形になる点が比喩表現の目印。文字通り水質を調べる意味なら単数形 test the water が使われ、この慣用句とは区別される（entry.examples の仮定法の例文を参照）。
acid test 決定的な試金石、真価を見極める最終的な試練中立／ややフォーマル
The acid test of a good leader is how they handle a crisis.（優れたリーダーかどうかの真の試金石は、危機にどう対処するかだ。）
🔍 貴金属の純度を強酸にかけて確かめた歴史的な検査法に由来し、ある物事の真の価値・真偽を見極める決定的な判断基準を指す比喩表現。
📝 the acid test of/for ~ の形で使う名詞句。単なる検査ではなく『真贋・真価を判定する最終的な決め手』というニュアンスを持つ。
litmus test （物事の性質を判定する）決定的な指標、踏み絵中立／ビジネス・政治
For many voters, the candidate's stance on taxes is a litmus test.（多くの有権者にとって、その候補者の税制に対する立場が踏み絵となる。）
🔍 リトマス試験紙が酸性・アルカリ性を一発で判定することに由来し、ある人の立場や物事の性質をはっきり判定する分かりやすい基準を指す比喩表現。
📝 政治・社会的立場を判定する文脈で特によく使われる。『これ一つで本質や賛否が分かる基準』という含意がある。
