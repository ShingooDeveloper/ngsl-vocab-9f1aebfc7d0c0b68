already

副詞 (adverb)米 /ɔːlˈredi/ ／ 英 /ɔːlˈredi/al·read·y（3音節）CEFR A2

🗣 強勢は2番目の音節 /ˈredi/ に置く。1番目の al- は長母音 /ɔːl/ だが強勢は無く、最後の -y は弱い /i/。「オルレディ」のように平板に読まず、「オール・レ́ディ」と真ん中を強く出す

  1. 【副詞】すでに、もう（ある時点で動作・状態がすでに完了・成立していることを表す。現在完了・過去完了とよく使われる）
  2. 【副詞】（疑問文で）もう〜した?（予想より早いという驚き・確認のニュアンスを込める。Have you finished already? もう終わったの?）
  3. 【副詞】（文中で）すでに、もはや（ある状態や変化がすでに始まっている、進行していることを強調する。It's already dark outside. 外はもう暗い）

語法マーク

  • 比較変化はしない（more already や already-er のような形は存在しない）
  • 肯定文・肯定的なニュアンスの疑問文で使うのが基本。否定文では原則 yet を使う（× I haven't finished already. ○ I haven't finished yet.）
  • 疑問文で使うと「（予想より早く）もう〜した?」という驚き・意外性のニュアンスが加わる。中立に進捗を尋ねたいだけなら yet を使う方が自然なことが多い
  • 現在完了・過去完了と組み合わせる頻度が非常に高いが、be動詞＋形容詞／現在分詞（It's already dark.）でも使える
  • 他品詞での用法: 品詞は副詞のみ（pos_all省略）。位置に自由度があり、(1) have/has/had の直後・過去分詞の直前という中間位置が最も標準的（I have already seen it.）。(2) くだけたアメリカ英語では文末に置いて完了を強調することもある（I saw it already!）。(3) 文頭に置く用法はまれで、置くと不自然かフォーマルな強調に響く。

文型・頻出パターン

  • have/has + already + 過去分詞現在完了で、ある時点ですでに完了していることを表す最頻出パターン
    I have already seen that movie.その映画はもう見た。
    She has already left the office.彼女はもうオフィスを出た。
  • had + already + 過去分詞過去完了で、過去のある時点よりも前にすでに完了していたことを表す
    The movie had already started when we arrived.私たちが着いたときには映画はもう始まっていた。
  • Have/Has + 主語 + already + 過去分詞 ...?疑問文で、予想より早いという驚き・確認のニュアンスを込めて「もう〜した?」と尋ねる
    Have you already paid the bill?もう請求書は払ったの?
  • be動詞 + already + 形容詞／現在分詞状態や変化がすでに始まっている・成立していることを強調する
    It's already too late.もう手遅れだ。
    He's already asleep.彼はもう眠っている。
  • 文末の already（インフォーマル）口語で、動作がすでに完了していることを念押しするように文末に置く
    I finished it already!もう終わらせたってば!

他のNGSL語への置き換え

already → just
元: She has already finished her homework, so she can watch TV now.
後: She has just finished her homework, so she can watch TV now.
訳: 彼女はちょうど宿題を終えたところなので、今はテレビを見てもいい。
📝 already は「（予想より）早くもう完了している」という時間的な早さを強調するのに対し、just は「ほんの少し前に」という直近性を強調する。どちらも現在完了とセットで使われる頻出表現だが、含意する時間のニュアンスが異なる。
already → even
元: Have you already eaten lunch? It's only eleven o'clock!
後: Have you even eaten lunch? It's only eleven o'clock!
訳: そもそもお昼は食べたの? まだ11時だよ!
📝 already は「もう（予想より早く）」と早さへの驚きを表すのに対し、even は「そもそも〜したのか」と行為の有無自体を疑うニュアンスになる。文法的にはどちらも過去分詞の直前という同じ位置に置ける。
already → apparently
元: The tickets were already sold out by the time we got online.
後: The tickets were apparently sold out by the time we got online.
訳: 私たちがオンラインにアクセスした頃には、どうやらチケットは売り切れていたらしい。
📝 already は「（予想より）早くに完了していた」という時間的な早さを表すのに対し、apparently は伝聞・推測に基づく「どうやら〜らしい」という確信度の弱さを表す。どちらも be動詞と過去分詞の間という同じ位置に置ける。

⚠️ 日本人の典型誤用
✕ I haven't finished my homework already.
「宿題をまだ終えていない」と言いたかったが、already を否定文に使うと英語として不自然になり、ネイティブには違和感のある文に聞こえる。
○ I haven't finished my homework yet.
宿題はまだ終えていない。
I haven't finished my homework yet.
already は基本的に肯定文や驚きの疑問文で使う語。「まだ〜していない」と否定するときは yet を使うのが基本ルールで、否定文に already を使うと座りが悪い。
✕ Already, I have eaten breakfast.
「もう朝食を食べた」と言いたかっただけなのに、already を文頭に置くと不自然に強調された、こなれていない英語に聞こえる。
○ I have already eaten breakfast.
私はもう朝食を食べた。
I have already eaten breakfast.
already は通例、have/has/had の直後・過去分詞の直前という中間位置に置く副詞。文頭に置くのはまれで、不自然かフォーマルな強調表現のように響いてしまう。
✕ Are you already finished with the report?
ただ進捗を尋ねたかっただけなのに、already を使うと「もう終わったの?（そんなに早く?）」という驚きや急かしているようなニュアンスが加わってしまう。
○ Are you finished with the report yet?
レポートはもう終わった?
Are you finished with the report yet?
中立に「もう〜終わった?」と進捗を尋ねたいだけなら yet を使うのが基本。already を疑問文で使うと「（予想より早くて）もう終わったの?」という驚き・急かすニュアンスが加わる。
🗾 カタカナ語とのギャップ

「オールレディ」として定着したカタカナ語は特に無い。ただしローマ字読みの感覚で平板に「オルレディ」と読むと誤り。正しくは2番目の音節 /ˈredi/ に強勢があり、「レ」の部分を短い母音 /e/ ではっきり強く出す（「レディー」と伸ばさない）。

🌱 Word family・語源
ready（形容詞）準備ができた、用意ができているNGSL
readily（副詞）たやすく、快く、すぐに
readiness（名詞）準備ができていること、即応性
語源: 中英語の all readi（= all ready「すっかり準備ができた」）が1語に融合してできた語。もとは「完全に用意が整っている」という形容詞的な意味だったが、時間の経過とともに「（ある時点で）すでに」という副詞的な意味へと転じた。
ready（準備ができた）— already の後半部分そのもの。「all（すっかり）+ ready（用意ができた）」が語源
all（すべて、まったく）— already の前半部分。「完全に」を強調する要素として ready にくっついた
💡 覚え方: "all + ready" がくっついて1語になったのが already。もともとは「すっかり用意万端」という意味で、そこから「（時間的に）もう、すでに」へ意味がスライドしたとイメージすると覚えやすい。
⚖️ 類義語の使い分け
yet — already と対になる語。yet は主に否定文・疑問文で「（まだ）〜していない／もう〜した?」と中立に確認するのに使い、already は主に肯定文・驚きの疑問文で「（予想より早く）もう」という完了の早さを強調する。
still — still は動作・状態がまだ続いている「継続」を表すのに対し、already は動作・状態がすでに完了・成立している「完了」を表す。時間的に逆方向を指す語同士。
by now — 「今頃までには」を表す副詞句で、already とほぼ同じ現在完了の文脈で使えることが多いが、by now はやや口語的で「当然そうなっているはず」という推量・期待のニュアンスを伴いやすい。
💬 熟語・慣用表現 (1)
enough already いい加減にして、もう十分だ口語（インフォーマル）
Enough already! I've heard that excuse a hundred times.（もういい加減にして! その言い訳はもう百回も聞いたよ。）
🔍 already 本来の「すでに・もう（時間的完了）」という意味から離れ、文末に添えて苛立ち・うんざりした気持ちを強調する間投詞的な用法。この用法自体は元々イディッシュ語の話し言葉のリズムが英語に取り入れられたものとされ、意味は already 単体の定義からは予測しにくい。
📝 同じ用法で Come on already!（早くしてよ)、I get it already!（もう分かったから）のように文末に already を置くと「もう十分」「いい加減に」という苛立ちの強調になる。時間の早さを表す通常の already（Have you finished already?）とは区別する。
