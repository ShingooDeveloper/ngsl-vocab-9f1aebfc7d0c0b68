class

名詞・動詞米 /klæs/ ／ 英 /klɑːs/class（1音節）CEFR A1

🗣 1音節語で強勢の位置に迷いはない。子音連結 kl を素早く発音し、母音は米音では口を大きく開けた /æ/（「ア」と「エ」の中間）。英音では /ɑː/（口を大きく開けた長い「アー」）になる点に注意

  1. 【名詞】（学校の）クラス、授業（What class do you have next? 次は何の授業?）
  2. 【名詞】（社会的な）階級、階層（the middle class 中流階級）
  3. 【名詞】種類、部類（a new class of smartphone 新しい種類のスマートフォン）
  4. 【動詞】～を分類する、位置づける（The play is classed as a comedy. その劇は喜劇に分類されている。）

語形変化

形種類発音備考
classes複数形（名詞）／三人称単数現在形（動詞）/ˈklæsɪz/語末が /s/ の音で終わるため -es を付けて /ɪz/ を加えて発音する（「クラスィズ」であって「クラスズ」ではない）
classed過去形・過去分詞（規則変化）/klæst/語末の s は無声音なので -ed は /t/ で発音する（「クラスト」であって「クラスド」ではない）
classinging形（規則変化）/ˈklæsɪŋ/そのまま -ing を付ける。ただし実際には be classed as ~ の受動態で使われることがほとんどで、能動の進行形として使われる頻度は低い

語法マーク

  • 「授業・クラス」「（社会的な）階級」「種類」のいずれの意味でも可算名詞（a class, classes）
  • 動詞用法は他動詞。class A as B の形を取るが、実際にはほぼ常に受動態 be classed as ~ で使われる
  • in class（授業中に）、out of class（授業外で）は無冠詞の決まった言い方
  • 「気品・上品さ」を表す口語的な不可算用法もある（have class, no class）。この意味では複数形にならない
  • 他品詞での用法: 動詞用法は「～を分類する」だが、目的語＋as＋補語（class A as B）の形はほぼ常に受動態 be classed as ~ で使われ、能動態で「（人が）AをBに分類する」と言いたい場合は classify の方が自然に響く。また口語ではまれに不可算名詞として「気品、上品さ、貫禄」を表すことがある（She's got real class. 彼女には本物の気品がある／This place has no class. この店には品がない）。この意味では冠詞をつけず、複数形にもならず、much・real・no などで程度を修飾する。

文型・頻出パターン

  • be classed as + 名詞/形容詞～に分類される（受動態が基本の言い方）
    Tomatoes are classed as fruit in botany.植物学では、トマトは果物に分類される。
    This drug is classed as highly addictive.この薬物は依存性が非常に高いものに分類されている。
  • take a class (in/on ~)（～の）授業・講座を受ける
    take a class in pottery陶芸のクラスを受ける
    take a class on nutrition栄養学の講座を受ける
  • a class of + 複数名詞～という種類・部類
    a new class of antibiotics新しい種類の抗生物質
    a whole class of problems問題のひとつの大きな部類
  • in class / out of class授業中に／授業外で（無冠詞の慣用表現）
    Please don't use your phone in class.授業中は携帯電話を使わないでください。
    She rarely talks to teachers out of class.彼女は授業外では先生とほとんど話さない。
  • the working/middle/upper class(es)労働者・中流・上流階級（社会階級を表す決まった言い方）
    the growing middle class拡大する中流階級
    policies that favor the upper classes上流階級を優遇する政策

他のNGSL語への置き換え

class → course
元: I've never taken a cooking class before, so I'm a little nervous.
後: I've never taken a cooking course before, so I'm a little nervous.
訳: 料理の講座を受けたことが一度もないから、ちょっと緊張しているんだ。
📝 course は体系立てられた一連の学習課程全体を指すことが多く、class は個々の1回の授業や比較的短い期間のクラスを指すことが多い。ここでは意味はほぼ同じだが、course の方がより長期的・体系的な学びを連想させる。
class → backgrounds
元: People from different social classes often have very different opinions about money.
後: People from different social backgrounds often have very different opinions about money.
訳: 社会的な背景が異なる人々は、お金に対する考え方も大きく異なることが多い。
📝 background は出身・生い立ちなど個人の背景全般を指す柔らかい言い方で、class ほどはっきりと経済的・社会的な階層を区別しない。social class が地位の分類を明示するのに対し、background はより婉曲的で日常会話に好まれる表現。
class → lessons
元: If you miss more than three classes, you won't pass the course.
後: If you miss more than three lessons, you won't pass the course.
訳: 3回以上レッスンを休んだら、その講座は単位を落とすことになるよ。
📝 lesson は語学・音楽など個別指導的な1回分の授業内容を指すニュアンスが強く、class は時間割で決まった授業の回そのものを指すことが多い。学校の一般的な授業の文脈では class の方が自然。

⚠️ 日本人の典型誤用
✕ I have a class of math every Tuesday.
「毎週火曜に数学の授業がある」と言いたかったが、class of math では「数学という種類（の何か）」のように聞こえ、教科の授業を表す言い方としては不自然に響く。
○ I have a math class every Tuesday.
毎週火曜に数学の授業がある。
I have a math class every Tuesday.
日本語「数学のクラス」の「の」を直訳して class of math としてしまう誤り。教科名は class の前に置いて複合名詞にする（math class）か、in を使う（a class in math）。class of ~ は「～という種類」（a new class of drugs）や「～年度の卒業生」（the class of 2020）の意味になり、教科を表す言い方としては不自然。
✕ This restaurant has a very high class.
「このレストランはとても高級だ」と言いたかったが、a very high class という形は英語として不自然で、ネイティブには違和感を与える。
○ This restaurant is very classy.
このレストランはとても上品で高級だ。
This restaurant is very classy.
This restaurant has real class.
「クラスが高い」という日本語の直訳で、class を可算・程度修飾可能な名詞として使ってしまう誤り。「品格・上質さ」を表すこの class は不可算で、much・real・no などで修飾するのが普通（a very high class のように a や比較級を伴う言い方はしない）。程度を言うなら形容詞 classy を使う方が自然。
✕ I'm in class 3 at high school.
「高校3年生です」と言いたかったが、class 3 では「3組」や「（何らかの）分類3」のように聞こえてしまい、学年の意味では伝わらない。
○ I'm in my third year of high school.
高校3年生です。
I'm in my third year of high school.
I'm a third-year student in high school.
日本語の「高校3年生です」を class 3 と直訳してしまう誤り。英語の class は学年ではなく、授業や学級のまとまりを指すことが多い。学年を言うときは year（英）や grade（米）を使う（my third year, the 9th grade など）。
🗾 カタカナ語とのギャップ

カタカナ「クラス」は「ク・ラ・ス」の3モーラだが、英語は1音節 /klæs/。子音連結 kl を母音を挟まず一気に発音し、語末の s にも母音を付けない（「クラスゥ」と読まない）。母音も米音では口を大きく開けた /æ/、英音では長めの /ɑː/ で、日本語の「ア」より口の開きが大きい。

🔗 句動詞

⚖️ 類義語の使い分け
course — course は体系立てられた一連の学習課程全体を指すことが多い。class は個々の1回の授業や比較的短い期間のクラスそのものを指す。
lesson — lesson は語学・音楽など個別指導的な1回分の授業内容を指すニュアンスが強い。class は時間割で決まった授業の回そのものを指すことが多い。
category — category は「種類・部類」の意味で class と近いが、より一般的・中立的な分類を指す。class（a class of drugs など）にはしばしば品質や特性による序列のニュアンスが伴う。
rank — rank は主に序列・順位（軍隊の階級など）を指す。class は経済的・社会的な集団としてのまとまり（階級）を指し、個々の順位よりも所属する層に焦点がある。
💬 熟語・慣用表現 (5)
in a class of your own / in a class of its own ずば抜けている、群を抜いている、他に類を見ない中立
As a goalkeeper, she is in a class of her own—no one else comes close.（ゴールキーパーとして、彼女はずば抜けた存在だ。他の誰も彼女には及ばない。）
🔍 「自分だけの階級にいる」という比喩から、比較対象がいないほど優れている、唯一無二であることを表す。単なる「分類」ではなく「卓越の度合い」を意味する点が構成語からは予測しにくい。
📝 人には in a class of your/her/his own、物・作品には in a class of its own を使う。比較を絶するほど優れている、という肯定的な評価にのみ用いる。
class act 一流の人（もの）、洗練された立派な人物中立
The way he handled the criticism with such grace made him a real class act.（あれほど優雅に批判に対応した彼は、本当に立派な人物だった。）
🔍 「クラス（品格）のある行い・人」から転じて、能力・品格・振る舞いのすべてにおいて称賛に値する人や物を指す名詞句。「授業」の意味とは無関係。
📝 a class act の形で名詞として使うのが基本。人だけでなくチームやパフォーマンスを称える時にも使える。
cut class 授業をサボる、無断で授業を休む口語
He got in trouble for cutting class three times last month.（彼は先月、3回も授業をサボって怒られた。）
🔍 「クラスを切る」という比喩表現で、正当な理由なく授業を欠席することを指す。ここでの cut は「切る」ではなく「（義務を）サボる」という予測しにくい意味になる。
📝 主にアメリカ英語のカジュアルな言い方。イギリス英語では同義で skive off や bunk off (class/school) がよく使われる。
second-class citizen 二流市民、不当に軽んじられる人中立／やや批判的
Immigrants in that era were often treated as second-class citizens.（その時代、移民はしばしば二流市民として扱われた。）
🔍 「二等級の市民」という表現が比喩的に広がり、法的・社会的に不当に低い扱いを受ける人を指す決まり文句になった。文字通りの「等級2の国民」という意味では使わない。
📝 差別や不平等な扱いを批判する文脈で使われることが多い。形容詞単独でも second-class treatment（二流の扱い）のように使う。
have no class 品がない、下品だ口語
Talking loudly on your phone during the movie? That's just no class.（映画中に大声で電話するなんて、まったく品がないね。）
🔍 不可算名詞としての class（気品・上品さ）を使い、have no class で「品格のかけらもない」という酷評の決まり文句になる。単に「分類が無い」という意味ではない。
📝 主に人の振る舞いを非難する時に使う。反対に「品がある」は have class / show real class という。
