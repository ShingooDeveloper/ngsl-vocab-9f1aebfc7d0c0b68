memory

名詞 (noun)米 /ˈmeməri/ ／ 英 /ˈmeməri/mem·o·ry（3音節）CEFR A2

🗣 第1音節 mem に強勢がある（/ˈmeməri/）。第2音節の o は曖昧母音 /ə/ に弱まり、「モ」とはっきり発音しない。語末の -ry も軽く短い /ri/ で、カタカナ「メモリー」のように長く伸ばさない（伸ばすと別語 memoir や memo と紛れやすい）

  1. 【名詞】記憶、記憶力（She has an excellent memory for faces. 彼女は人の顔を覚えるのがとても得意だ。）
  2. 【名詞】思い出、記憶に残っていること（happy memories of childhood 子供時代の楽しい思い出）
  3. 【名詞】追悼、記念（a statue built in memory of the soldiers 兵士たちを偲んで建てられた像）
  4. 【名詞】（コンピュータの）メモリー、記憶容量（This laptop doesn't have enough memory to run the game. このノートパソコンはそのゲームを動かすのに十分なメモリーがない。）

語形変化

形種類発音備考
memories複数形（可算用法のみ）/ˈmemərɪz/「個々の思い出・記憶」を指す可算用法でのみ使う複数形（happy memories）。「記憶力」「(コンピュータの)メモリー容量」「追悼」の意味では不可算のままで複数形にしない。語末は有声音の後なので /z/

語法マーク

  • 「記憶力」の意味では不可算（have a good/bad memory。× have a good memories は誤り）
  • 「個々の思い出」の意味では可算で、単数・複数どちらも使う（a vivid memory / vivid memories）
  • in memory of ~（~を偲んで）は不可算の固定表現。in memories of とは言わない
  • from memory（記憶を頼りに、暗記で）、within (living) memory（記憶にある限り）など、前置詞句として副詞的に使われる表現が多い
  • コンピュータ用語の memory も不可算。個別の部品を数えたいときは a memory chip / module のように別の可算名詞を添える
  • 他品詞での用法: 名詞1語のみだが、意味によって可算・不可算が入れ替わる点に注意。(1) 不可算：「記憶力」（have a good memory 記憶力がいい）。(2) 可算：「個々の思い出・記憶」（a memory / memories of my childhood）。(3) 不可算の熟語的用法：「追悼・記念」（in memory of ~ ~を偲んで）。(4) 不可算：「(コンピュータの)メモリー、記憶容量」（run out of memory メモリー不足になる）。カタカナ「メモリー」は日本ではPCパーツの(4)の意味で使われることが多いが、英語では(1)(2)の人間の記憶を指す用法の方が圧倒的に高頻度。

文型・頻出パターン

  • have a memory of / for + 名詞〜の記憶がある／〜の記憶力がある（能力・記憶両方の中心構文）
    have a memory for names人の名前を覚えるのが得意だ
    I have a clear memory of that day.その日のことはくっきりと覚えている。
  • in memory of + 人〜を偲んで、〜の記念に（追悼を表す固定表現）
    a scholarship established in memory of the professorその教授を偲んで設立された奨学金
    They planted a tree in memory of him.彼らは彼を偲んで木を植えた。
  • from memory記憶を頼りに、（資料を見ずに）暗記で
    play the piece from memory楽譜を見ずに（暗記で）その曲を演奏する
    I can only quote the poem from memory.その詩は記憶を頼りにしか引用できない。
  • within (living) memory（人々の）記憶にある限り（の期間）
    the worst storm within living memory人々の記憶にある限り最悪の嵐
  • run out of / free up memory（コンピュータの）メモリーが足りなくなる／メモリーを空ける（IT用語としての不可算用法）
    The app crashed because the phone ran out of memory.スマホのメモリーが足りなくなってそのアプリは落ちた。
    Close some tabs to free up memory.タブをいくつか閉じてメモリーを空けて。

他のNGSL語への置き換え

memory → images
元: I still have vivid memories of the summer we spent at the lake.
後: I still have vivid images of the summer we spent at the lake.
訳: 湖で過ごしたあの夏の光景は、今でも鮮明に目に浮かぶ。
📝 memories は五感や感情も含む「思い出」全体を指すのに対し、images は頭の中の「映像・視覚的な光景」に焦点を絞った語。memory の方が体験の意味づけまで含む広い概念。
memory → honor
元: This old photograph was kept in memory of my grandmother.
後: This old photograph was kept in honor of my grandmother.
訳: この古い写真は、祖母に敬意を表して大切に保管されていた。
📝 in memory of は亡くなった人を「偲んで」追悼する意味合いが強いのに対し、in honor of は故人・存命の人を問わず「称えて、敬意を表して」という祝意寄りのニュアンス。文脈が追悼なら memory、称賛なら honor が自然。
memory → mind
元: If I had a better memory, I wouldn't forget people's names so often.
後: If I had a better mind, I wouldn't forget people's names so often.
訳: もっと頭が良かったら、人の名前をこんなに忘れたりしないのに。
📝 memory は「記憶力」という特定の能力を指すのに対し、mind は思考力・知性全般を指すより広い語。名前を忘れる原因を記憶力に限定したいなら memory の方が的確。

⚠️ 日本人の典型誤用
✕ I don't have memory that.
「それを覚えていない」と言いたかったが、名詞 memory をそのまま動詞のように使ってしまい、文法的に破綻していてネイティブには意味が通じにくい。
○ I don't remember that.
それは覚えていない。
I don't remember that.
名詞 memory を動詞のように使ってしまう誤り。「覚えている・思い出す」という動作を言いたいときは動詞 remember を使う。memory はあくまで名詞で、have a memory of ~ の形で使うか、動詞なら remember に切り替える必要がある。
✕ I have many memory of my childhood in this town.
「この町で過ごした子供時代の思い出がたくさんある」と言いたかったが、manyの後を単数形のままにすると数の一致が崩れ、ネイティブには基本的な文法ミスとして響いてしまう。
○ I have many memories of my childhood in this town.
この町で過ごした子供時代の思い出がたくさんある。
I have many memories of my childhood in this town.
「個々の思い出」の意味の memory は可算名詞。many の後は複数形 memories が必要だが、不可算用法（記憶力）と混同して単数形のままにしてしまう誤り。
✕ This mug is a memory of my trip to Kyoto.
「これは京都旅行の思い出の品だ」と言いたかったが、memory は物ではなく記憶そのものを指すため、マグカップを memory と呼ぶと意味が通らず不自然に聞こえる。
○ This mug is a souvenir of my trip to Kyoto.
これは京都旅行のお土産だ。
This mug is a souvenir of my trip to Kyoto.
日本語の「思い出の品」を直訳して memory を「記念品・お土産」の意味で使ってしまう誤り。memory はあくまで頭の中の「記憶・思い出」という抽象概念を指し、手に取れる「品物」そのものは指せない。物としての記念品は souvenir や keepsake を使う。
🗾 カタカナ語とのギャップ

カタカナ「メモリー」は語末を長く伸ばすが、英語 /ˈmeməri/ の語末は短い /ri/ で伸ばさない（伸ばすと memoir や memo と紛れやすい）。また日本ではPCパーツの「メモリ増設」のようにコンピュータ用語として使われる頻度が高く、その意味が真っ先に浮かびがちだが、英語の memory はまず人間の「記憶・思い出」を指す語で、コンピュータの意味はその比喩的な拡張用法にすぎない点に注意。強勢は第1音節 mem のみで、2・3音節は曖昧母音に弱まる。

🌱 Word family・語源
remember（動詞）覚えている、思い出すNGSL
memorize（動詞）暗記する、記憶する
memorial（名詞・形容詞）記念碑、追悼の（式典など）
memorable（形容詞）忘れがたい、記憶に残る
commemorate（動詞）（人・出来事を）記念する、追悼する
語源: 古フランス語 memoire を経て、ラテン語 memoria「記憶、思い出」に由来する。memoria はさらに形容詞 memor「心に留めている、忘れない」から派生し、その語根はインド・ヨーロッパ祖語の *(s)mer-「心に留める、気にかける」に遡る。remember もこの memor と同じ語根を共有する語。
remember（覚えている）— ラテン語 re-（再び）+ memor（心に留めている）が原義。memory と同じ memor- の語根を共有する最も身近な語
memorize（暗記する）— memoria + -ize（〜化する）で「記憶の状態にする」が原義
memorial（記念碑）— memoria + -al（〜に関する）で「記憶に関わるもの」が原義。人の記憶にとどめるための碑や式典を指す
💡 覚え方: ラテン語の語根 memor-（心に留めている）から一本につながる: remember（re- + memor で繰り返し心に留める＝覚えている）→ memory（memor の状態＝記憶）→ memorize（記憶の状態にする＝暗記する）→ memorial（記憶にとどめるもの＝記念碑）。すべて『心から離さない』というイメージでつながる。
⚖️ 類義語の使い分け
recollection — recollection はやや形式的な語で、意識的に思い起こす『(具体的な)想起・記憶』を指すことが多い。memory はより日常的で、能力・思い出の両方を指せる汎用的な語。
remembrance — remembrance は追悼・記念の文脈で使われることが多いフォーマルな語（in remembrance of ~）。memory も同じ文脈で使えるが、remembrance の方がより儀礼的・文語的な響きを持つ。
reminiscence — reminiscence は懐かしさを伴って昔を振り返る『回想』を指し、しばしば複数形 reminiscences で「昔語り」の意味になる。memory はそうしたノスタルジーの有無に関わらず使える中立的な語。
💬 熟語・慣用表現 (6)
jog someone's memory （人）に思い出させる、記憶を呼び覚ます口語
This old photo might jog your memory about that summer trip.（この古い写真を見れば、あの夏の旅行のことを思い出すきっかけになるかもしれないよ。）
🔍 jog は本来「軽くつつく、揺さぶる」の意。記憶を軽く「揺さぶって」思い出すきっかけを与える、という比喩表現。
📝 主語は人ではなく、きっかけとなる物・出来事であることが多い（写真、匂い、音楽など）。似た意味の表現に refresh someone's memory がある。
if memory serves (me right) 私の記憶が正しければ中立／ややフォーマル
If memory serves me right, this restaurant used to be a bookstore.（私の記憶が正しければ、このレストランは昔本屋だったはずだ。）
🔍 「記憶が（私に）役立つなら」が直訳。memory を人であるかのように「仕える（serve）」存在として擬人化した表現。
📝 確信が完全ではないことを示す前置き表現。if my memory serves とも言う。話し言葉・書き言葉どちらでも使われる。
commit something to memory 〜を記憶する、暗記する中立／ややフォーマル
Actors have to commit their lines to memory before rehearsals begin.（俳優はリハーサルが始まる前にセリフを覚えなければならない。）
🔍 「〜を記憶（という保管場所）に委ねる」が直訳。commit の「（資金・任務などを）委託する」という意味を記憶に対して使った比喩表現。
📝 memorize とほぼ同義だが、意図的に努力して覚えるニュアンスが強く、書き言葉でもよく使われる。
a trip down memory lane 思い出巡り、懐かしい思い出に浸ること口語
Looking through these old yearbooks was a real trip down memory lane.（この古い卒業アルバムを見返していたら、すっかり懐かしい気分に浸ってしまった。）
🔍 memory lane（思い出の小道）という架空の道を「散歩する」イメージの比喩表現。実在する地名ではない。
📝 動詞として使う場合は walk/go down memory lane の形になる（Let's walk down memory lane. 昔を懐かしもう）。
have a memory like a sieve 物忘れがひどい、ざるのような記憶力だ口語
I have a memory like a sieve—I forgot my own anniversary again.（私は本当に物忘れがひどくて、また自分の結婚記念日を忘れてしまった。）
🔍 sieve（ふるい・ざる）は水も何もかもすぐに漏れ出てしまう道具。記憶が「ざる」のようにすぐ抜け落ちる、という比喩。
📝 自虐的・ユーモラスな文脈で使われることが多い表現。
have a memory like an elephant 抜群の記憶力を持っている、忘れっぽくない口語
Don't try to fool her—she has a memory like an elephant and remembers every promise.（彼女をだまそうとしても無駄だよ、記憶力が抜群で約束事は全部覚えているから。）
🔍 象は物事を決して忘れないという俗説に基づく比喩表現で、have a memory like a sieve と対をなす表現。
📝 elephants never forget（象は決して忘れない）という俗説に由来する慣用句。肯定的な意味で使われる。
