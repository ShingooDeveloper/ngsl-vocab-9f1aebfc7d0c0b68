network

名詞・動詞米 /ˈnetwɜːrk/ ／ 英 /ˈnetwɜːk/net·work（2音節）CEFR B1

🗣 第1音節 NET に強勢が置かれる（ˈnetwɜːrk）。カタカナ「ネットワーク」のように全体を均等に伸ばさず、NET を強くはっきり、-work は短めに添える。work部分の母音 /ɜːr/ は舌をそり上げて響かせるr性母音で、単純な「アー」とは音質が異なる

  1. 【名詞】（コンピューター・道路・人などをつなぐ）ネットワーク、網状の組織・システム（The company uses a secure network to share files. その会社はファイル共有に安全なネットワークを使っている。）
  2. 【名詞】（人と人との）人脈、交友関係（Networking events help you build a strong professional network. 交流会は強力な人脈を築くのに役立つ。）
  3. 【名詞】（テレビ・ラジオの）放送網、系列局（The show will air on a major TV network next month. その番組は来月、大手テレビ局系列で放送される。）
  4. 【動詞】人脈を作る、（仕事上の）交流をする（She spent the evening networking with other professionals. 彼女はその夜、他の専門家たちと交流して過ごした。）
  5. 【動詞】（コンピューターを）ネットワークでつなぐ、接続する（These computers are networked together in the office. これらのコンピューターはオフィス内でネットワーク接続されている。）

語法マーク

  • 名詞は可算名詞。a network / networks のように冠詞・複数形を伴う（× I have network. のような無冠詞単数は誤り）
  • 「人脈」の意味でも可算で、通常は a network（単数のまとまり）として扱う（many networks とは言わず a large network / a wide network とする）
  • 動詞「交流する・人脈を作る」の意味では自動詞的で、相手は前置詞 with で導く（network with colleagues）。目的語を直接取らない
  • 動詞「接続する」の意味では他動詞（network the computers）だが、実際には受動態 be networked（ネットワーク接続されている）の形で使われることが多い
  • networking は動詞のing形であると同時に独立した不可算名詞としても頻出する（Networking is essential in business.）
  • 他品詞での用法: networkはもともと「網目状に編まれた構造物」を指す具体的な名詞だったが、19〜20世紀に鉄道網・放送網、さらにコンピューターネットワークへと意味が拡張し、1980年代以降は『人脈を作る』という動詞用法も定着した。動詞は自動詞用法（She was networking at the party. パーティーで人脈作りをしていた。前置詞withで相手を導く）と他動詞用法（The IT team networked all the office computers. ITチームがオフィスの全パソコンをネットワーク接続した）の両方で使われる点に注意。

文型・頻出パターン

  • a network of + 名詞（複数形）〜からなるネットワーク・網（構成要素を列挙する定番の型）

    a network of friends友人のネットワーク
  • be connected to / joined to + a networkネットワークに接続されている（受動態でよく使われる型）
    This printer is connected to the office network.このプリンターはオフィスのネットワークに接続されている。
  • network with + 人（仕事上のつながりを作るために）〜と交流する（動詞・自動詞用法）
    He networked with several investors at the conference.彼はその会議で何人かの投資家と交流した。
  • build / set up a networkネットワークを構築する
    set up a private network専用ネットワークを構築する
    build a network of contacts人脈のネットワークを築く
  • social network / networking event人脈・SNSに関する定番の複合表現
    She rarely posts on social networks.彼女はSNSにはめったに投稿しない。
    I met him at a networking event.その交流会で彼と知り合った。

他のNGSL語への置き換え

network → linked
元: All the devices in this building are connected to the same wireless network.
後: All the devices in this building are linked to the same wireless network.
訳: この建物内のすべての機器は同じ無線ネットワークにリンクされている。
📝 linked は connected よりやや技術文書・仕様書寄りで簡潔な響きを持つ。意味はほぼ同じだが、日常会話では connected の方が一般的で自然に聞こえる。
network → build
元: The company had its engineers set up a private network before the product launch.
後: The company had its engineers build a private network before the product launch.
訳: その会社は製品発売前に、エンジニアたちに専用ネットワークを構築させた。
📝 build はゼロから作り上げる労力・規模感を強調するのに対し、set up は設置・準備という実務的なニュアンスが強い。使う場面はほぼ同じだが、build の方がプロジェクトの大変さを感じさせる。
network → community
元: He joined an online network that connects freelance designers with small businesses.
後: He joined an online community that connects freelance designers with small businesses.
訳: 彼はフリーランスデザイナーと中小企業をつなぐオンラインコミュニティに参加した。
📝 community は人のつながりや所属感を強調し、温かみのある印象を与える。network は接続の仕組み・構造そのものを指すニュアンスが強く、より実務的・技術的に響く。

⚠️ 日本人の典型誤用
✕ I want to network more people at the conference.
「その会議でもっと多くの人と交流したい」と言いたかったが、network を他動詞のように使ってしまい、前置詞が欠けた不完全な文に聞こえる。
○ I want to network with more people at the conference.
その会議でもっと多くの人と交流したい。
I want to network with more people at the conference.
動詞 network（人脈を作る・交流する）は自動詞的で、相手は前置詞 with で導く。目的語をそのまま置く他動詞のように使うのは誤り。
✕ This computer is network to the server.
「このパソコンはサーバーに接続されている」と言いたかったが、過去分詞にせず原形のまま使ったため、文法的に不完全な受動態に聞こえてしまう。
○ This computer is networked to the server.
このパソコンはサーバーに接続されている。
This computer is networked to the server.
受動態は be + 過去分詞 の型が必要。動詞の原形 network をそのまま be動詞の後に置くのは誤りで、過去分詞形 networked にしなければならない。
✕ She is doing a networking now.
「彼女は今、人脈作りをしている」と言いたかったが、a networking は不可算名詞に不定冠詞をつけた誤りで、ネイティブには文法的な違和感として響く。
○ She is doing some networking now.
彼女は今、人脈作りをしているところだ。
She is doing some networking now.
networking は不可算名詞として定着しており、不定冠詞 a を伴って可算名詞のようには使わない。some / a lot of をつけるか無冠詞で使う。
🗾 カタカナ語とのギャップ

カタカナ「ネ・ッ・ト・ワ・ー・ク」は多モーラで全体が平坦に伸びがちだが、英語は2音節 /ˈnetwɜːrk/ で強勢は net にのみ置かれる。work部分は単純な「ワーク」の母音ではなく、舌をそり上げるr性母音 /ɜːr/ で発音し、伸ばしすぎたり平板にしたりしない。語末の k にも母音を足さない

🌱 Word family・語源
framework（名詞）枠組み、骨組みNGSL
homework（名詞）宿題NGSL
teamwork（名詞）チームワーク、協力NGSL
internet（名詞）インターネットNGSL
語源: net（網、古英語 net に由来）+ work（作られたもの・仕事、古英語 weorc に由来）の複合語。16世紀には文字通り「糸が交差して編まれた網状の構造物」を指す語だったが、19世紀に鉄道網・河川網などの比喩的用法が生まれ、20世紀に放送網、さらにコンピューターネットワークへと意味が拡張した。今日の「Internet」も internetwork（相互接続されたネットワーク網）の短縮に由来し、network と語源的につながっている。
framework（枠組み）— frame（枠）+ work（作られたもの）で「骨組みとなる構造」。networkと同じ「-work」型複合語の作り方
teamwork（チームワーク）— team（チーム）+ work（働き）で「チームとしての協働」。networkと同じ「-work」複合パターン
internet（インターネット）— inter（相互の）+ net(work) の短縮形。「複数のネットワークが相互接続したもの」が原義
💡 覚え方: net（網）+ work（作られたもの）→「網目状に編まれた仕組み」。糸を編んだ網のように、道路・人・コンピュータが編み目状につながっている姿をイメージすると覚えやすい。framework「枠組み」、teamwork「チームの働き」と同じ『-work』型複合語の仲間。
⚖️ 類義語の使い分け
system — systemは複数の要素が連携して機能する「仕組み全体」に焦点を当てる、より広い語。networkは要素同士の「つながり・接続」という構造そのものに焦点がある。
web — webはクモの巣のように複雑に絡み合ったイメージを伴い、比喩的・やや口語的（a web of lies、the World Wide Web）。networkはより中立的・技術的な語感を持つ。
connections — connectionsは個々の人間関係・つてに焦点を当てる語（She has good connections.）。networkはそうしたつながり全体を1つの構造・集合体として捉える言い方。
💬 熟語・慣用表現 (1)
the old boy network （男性中心の）閥、コネ社会、旧知の間柄を通じた便宜の融通し合いビジネス／やや批判的・口語
Some critics say the old boy network still decides who gets promoted at the firm.（その会社では今でも旧知のコネ社会が誰を昇進させるかを左右していると批判する人もいる。）
🔍 「古い少年たちの人脈」ではなく、同じ学校や社会階層の出身者同士が非公式に助け合い、仕事や機会で便宜を図り合う仕組みを指す。networkという語の字面からは、こうした閉鎖的な相互扶助の文化的含みまでは予測できない定型表現。
📝 主にイギリス英語で、パブリックスクールや名門大学出身の男性同士の非公式な相互扶助を指し、しばしば機会の不公平さを批判する文脈で使われる。old boys' network とアポストロフィs付きで綴られることもある。
