today

副詞・名詞米 /təˈdeɪ/ ／ 英 /təˈdeɪ/
  1. 【副詞】今日、本日（現在のこの1日を指す。文頭・文末に置かれることが多い。I have a doctor's appointment today. 今日は病院の予約がある）
  2. 【副詞】今日では、現代では、今どき（現在の時代・風潮を表す。文頭でコンマを伴って使われることが多い。Today, more people work from home than ever before. 今日では、以前よりも多くの人が在宅勤務をしている）
  3. 【名詞】今日（という日）、本日（文の主語や所有格として使われる。Today is a public holiday, so the office is closed. 今日は祝日なので、オフィスは休みだ）

例文 — 6つの文法・構文で学ぶ

1I've already finished all of my homework today, so I can relax now.

今日はもう宿題を全部終わらせたので、もうくつろげる。

現在完了（have + 過去分詞）：already「すでに」を伴い完了を強調

現在完了（have + 過去分詞）：already「すでに」を伴い完了を強調。all of + 名詞「〜の全部」という頻出コロケーションも含む。

/aɪv ɔːlˈredi ˈfɪnɪʃt ˈɔːl əv maɪ ˈhoʊmwɜːrk təˈdeɪ | soʊ aɪ kən rɪˈlæks naʊ/
  • I've already → 語末子音vと母音ɔːが連結 [aɪvɔːlˈredi] アイヴォールレディ
  • all of → 語末子音lと弱形of(/əv/)の母音əが連結 [ˈɔːləv] オーラヴ
  • can → 助動詞なので弱形 /kən/ に弱化
アイヴォールレディ フィニシュト オーラヴ マイ ホウムワーク トゥデイ、ソウ アイ クン リラックス ナウ

2The trip that was originally planned for today has been postponed until next week.

今日に予定されていた旅行は、来週まで延期された。

関係代名詞 that（主格、the trip を修飾）＋受動態 was ... planned：もともとの予定を説明

関係代名詞 that（主格、the trip を修飾）＋受動態 was ... planned：もともとの予定を説明。has been postponed（受動態の現在完了）が延期という現在の結果状態を表す。

/ðə trɪp ðət wəz əˈrɪdʒənəli plænd fər təˈdeɪ həz bɪn poʊstˈpoʊnd ənˈtɪl nekst wiːk/
  • that → 関係代名詞は弱形 /ðət/ に弱化
  • was → 弱形 /wəz/ に弱化
  • for → 前置詞なので弱形 /fər/ に弱化
  • has → 助動詞は弱形 /həz/ に弱化
  • postponed until → 語末子音dと弱形untilの母音əが連結 [poʊstˈpoʊndənˈtɪl] ポウストポウンドゥンティル
  • next week → 語末のtは次の子音wの前で未開放になる [nekst̚ wiːk] ネクストゥウィーク
ザ トリップ ザット ワズ オリジナリ プランド ファ トゥデイ ハズ ビン ポウストポウンドゥンティル ネクストゥウィーク

3Today's weather is much colder than it was yesterday.

今日の天気は昨日よりもずっと寒い。

所有格として使われる名詞 today's（今日の）＋比較級 (much + 比較級 + than)：muchで比較級を強調する「ずっと〜」

所有格として使われる名詞 today's（今日の）＋比較級 (much + 比較級 + than)：muchで比較級を強調する「ずっと〜」。

/təˈdeɪz ˈweðər ɪz mʌtʃ ˈkoʊldər ðən ɪt wəz ˈjestərdeɪ/
  • weather is → 語末のrと母音ɪが連結（米音の連結r） [ˈweðərɪz] ウェザリズ
  • than it → than は弱形 /ðən/、語末nと母音ɪが連結 [ðənɪt] ザニット
トゥデイズ ウェザリズ マッチ コウルダー ザニット ワズ イエスタデイ

4Today is the day that we've all been waiting for since January.

今日は、私たちが1月からずっと待ち望んでいた日だ。

関係代名詞 that（主格、the day を修飾）＋現在完了進行形 (have been waiting)：過去から続く動作を強調

関係代名詞 that（主格、the day を修飾）＋現在完了進行形 (have been waiting)：過去から続く動作を強調。today がここでは名詞として文の主語になっている点に注目。

/təˈdeɪ ɪz ðə deɪ ðət wiv ɔːl bɪn ˈweɪɾɪŋ fər sɪns ˈdʒænjuˌeri/
  • Today is → 語末母音eɪと母音ɪの間にわたり音[j]を挟んで連結 [təˈdeɪjɪz] トゥデイイズ
  • that → 関係代名詞は弱形 /ðət/ に弱化
  • we've → we have の短縮形 /wiv/
  • waiting → 母音間のtがフラップ化 [ˈweɪɾɪŋ] ウェイリング（米音）
  • for → 前置詞なので弱形 /fər/ に弱化
トゥデイイズ ダ デイ ザット ウィヴ オール ビン ウェイリング ファ シンス ヂャニュエリ

5If it weren't so cold today, we would go for a walk in the park.

もし今日がこんなに寒くなければ、公園へ散歩に行くのに。

仮定法過去（If + were not, would + 原形）：現在の事実に反する仮定

仮定法過去（If + were not, would + 原形）：現在の事実に反する仮定。weren't は were not の縮約形で、仮定法では主語に関わらず were/weren't を使う。

/ɪf ɪt wɜːrnt soʊ koʊld təˈdeɪ | wi wəd ɡoʊ fər ə wɔːk ɪn ðə pɑːrk/
  • If it → 語末子音fと母音ɪが連結 [ɪfɪt] イフィット
  • would → 弱形 /wəd/ に弱化
  • for a → for は弱形 /fər/、語末rと母音əが連結 [fərə] ファラ
  • walk in → 語末子音kと母音ɪが連結 [wɔːkɪn] ウォーキン
イフィット ワーント ソウ コウルド トゥデイ、ウィ ワド ゴウ ファラ ウォーキン ダ パーク

6Please finish this report today; don't leave it until tomorrow.

今日中にこの報告書を仕上げてください。明日まで残しておかないように。

命令文（Please + 動詞の原形）：肯定の依頼

命令文（Please + 動詞の原形）：肯定の依頼。don't + 動詞の原形（否定命令文）：leave it until tomorrow「明日まで残しておく」を禁じる。

/pliːz ˈfɪnɪʃ ðɪs rɪˈpɔːrt təˈdeɪ | doʊnt liːv ɪt ənˈtɪl təˈmɔːroʊ/
  • leave it → 語末子音vと母音ɪが連結 [liːvɪt] リーヴィット
  • it until → 語末のtが母音に挟まれてフラップ化し連結 [ɪɾənˈtɪl] イロンティル
プリーズ フィニシュ ディス リポート トゥデイ、ドウント リーヴィット イロンティル トゥモロウ

⛏️ マインクラフト例文

⛏I finally found a diamond ore today after searching for hours.

今日、何時間も探した末についにダイヤモンド鉱石を見つけた。

単純過去形（found）＋副詞 finally「ついに」

単純過去形（found）＋副詞 finally「ついに」。after + 動名詞 (after searching)：〜した後で。

/aɪ ˈfaɪnəli faʊnd ə ˈdaɪmənd ɔːr təˈdeɪ ˈæftər ˈsɜːrtʃɪŋ fər ˈaʊərz/
  • found a → 語末子音dと母音əが連結 [faʊndə] ファウンダ
  • diamond ore → 語末子音dと母音ɔːが連結 [ˈdaɪməndɔːr] ダイモンドー
  • searching for → for は前置詞なので弱形 /fər/
アイ ファイナリ ファウンダ ダイモンドー トゥデイ アフター サーチング ファ アワーズ

⛏Let's build a new base today instead of repairing the old one.

古い拠点を修理する代わりに、今日は新しい拠点を建てよう。

勧誘表現 Let's + 動詞の原形

勧誘表現 Let's + 動詞の原形。instead of + 動名詞 (repairing)：〜する代わりに。

/lets bɪld ə nuː beɪs təˈdeɪ ɪnˈsted əv rɪˈperɪŋ ði oʊld wʌn/
  • build a → 語末子音dと母音əが連結 [bɪldə] ビルダ
  • instead of → 語末子音dと弱形of(/əv/)の母音əが連結 [ɪnˈstedəv] インステドヴ
  • the old → 母音の前なので the は /ði/ と発音
レッツ ビルダ ニュー ベイス トゥデイ インステドヴ リペアリング ディ オウルド ワン

他のNGSL語への置き換え

today → completed
元: I've already finished all of my homework today, so I can relax now.
後: I've already completed all of my homework today, so I can relax now.
訳: 今日はもう宿題を全部やり終えたので、くつろげる。
📝 finish は日常会話で使う中立的な動詞だが、complete はより形式張った語で、作業を隅々まで完全にやり遂げるニュアンスを強調し、書き言葉や仕事の文脈でよく使われる。
today → delayed
元: The trip that was originally planned for today has been postponed until next week.
後: The trip that was originally planned for today has been delayed until next week.
訳: 今日に予定されていた旅行は、来週まで延期された。
📝 postpone は新しい日時を決めて意図的に予定を組み直すニュアンスが強いのに対し、delay は明確な新しい日時がなくても「遅れる・長引く」ことに焦点がある。日常会話では delay の方が頻度が高い。
today → submit
元: Please finish this report today; don't leave it until tomorrow.
後: Please submit this report today; don't leave it until tomorrow.
訳: 今日中にこの報告書を提出してください。明日まで残しておかないように。
📝 finish は作業そのものを完成させることに焦点があるのに対し、submit は完成した物を正式に提出・送付する行為に焦点がある。書類が finish されていても submit されていない状態はあり得る点が使い分けの鍵。
