type

名詞・動詞米 /taɪp/ ／ 英 /taɪp/type（1音節）CEFR A2

🗣 1音節語。母音は二重母音 /aɪ/ で「ア」から「イ」へ滑らせて1拍で発音する。語末の /p/ は多くの場合開放せず飲み込むように止める

  1. 【名詞】種類、タイプ（What type of music do you like? どんな種類の音楽が好き？）
  2. 【名詞】（人・性格などの）タイプ、典型（He's not really my type. 彼は私のタイプじゃない。）
  3. 【動詞】（キーボードなどで文字を）入力する、タイプする（She typed her report quickly. 彼女はレポートを素早くタイプした。）
  4. 【動詞】（データ・人などを）分類する、位置づける（Doctors type blood before a transfusion. 医者は輸血前に血液型を判定する。）

語形変化

形種類発音備考
types複数形（名詞）／三単現（動詞）/taɪps/語末は無声音なので /s/（「タイプス」であって「タイプズ」ではない）。名詞なら『種類』の複数、動詞なら三人称単数現在で綴りは同じ
typed過去形・過去分詞/taɪpt/規則変化。無声音 /p/ の後なので -ed は /t/ と発音（「タイプド」ではなく「タイプト」）
typinging形/ˈtaɪpɪŋ/語末の e を落として -ing（typeing は誤り）。動名詞として『タイピング（技能）』の意味でも名詞的に使われる（Her typing is very fast.）

語法マーク

  • 名詞は可算。『種類』の意味では a type of ~ / this type of ~ のように単数でも冠詞や this/that を伴うのが基本（Type of computer is old. のように無冠詞単独では使わない）
  • 『~の種類』は type of + 名詞（of を落とさない。What type music は誤りで What type of music が正しい）
  • 動詞は他動詞が基本（type a message）だが、対象を言わずに『（人が）タイプが打てる／速い』という能力を表す自動詞的用法もある（She can type very fast.）
  • practice, enjoy, finish など動名詞を目的語に取る動詞の後では typing（動名詞）にする（practice typing であって practice to type ではない）
  • 他品詞での用法: 元は印刷用語で、活字・活字体を指した名詞から発展した語。typewriter（タイプライター）や typeface（書体）はこの『活字』の意味を引き継いでいる。現代では主にキーボード入力の動詞用法と、『種類』を表す名詞用法の2系統で使われる。血液型は blood type、性格分類は Type A / Type B personality のように無冠詞の固定表現になることが多い。

文型・頻出パターン

  • type of + 名詞〜の種類、〜というタイプの（分類を表す）
    What type of music do you like?どんな種類の音楽が好きですか。
    This type of software is easy to use.この種類のソフトウェアは使いやすい。
  • type + O〜をキーボードで入力する、タイプする（最も基本的な他動詞用法）
    type a messageメッセージを打つ
    type your name名前を入力する
  • type + O + into/in + 場所〜を（画面・欄など）に入力する
    Type the command into the chat.コマンドをチャットに入力してください。
  • have + O + type（使役）（人）に〜をタイプさせる
    The teacher had us type our essays.先生は私たちにエッセイをタイプさせた。
  • practice/enjoy + typing（動名詞）タイプすることを練習する・楽しむ
    I practice typing every morning.私は毎朝タイピングの練習をしている。

他のNGSL語への置き換え

type → written
元: The whole contract was typed by a professional secretary before lunch.
後: The whole contract was written by a professional secretary before lunch.
訳: その契約書は昼までにプロの秘書によって書かれた。
📝 type は「キーボードで打つ」という動作に限定されるが、write はより一般的に「書く」全般（手書きも含む）を指す。作成手段を明示したいなら typed、単に文書が作成された事実を言うなら written が自然。
type → kinds
元: There are several types of rock that form deep underground.
後: There are several kinds of rock that form deep underground.
訳: 地下深くでできる岩石にはいくつかの種類がある。
📝 type と kind はどちらも「種類」を意味しほぼ置き換え可能。type は blood type や body type のように科学的・専門的な分類でよく使われ、kind はより日常的・口語的な響きを持つ。
type → input
元: Please type your password carefully before you press enter.
後: Please input your password carefully before you press enter.
訳: エンターを押す前に、パスワードを注意深く入力してください。
📝 input はよりフォーマルで技術的な響きを持ち、マニュアルやシステムメッセージでよく使われる。type はキーボードで打つという物理的動作を強く連想させる中立的な語で、日常会話にも使える。

⚠️ 日本人の典型誤用
✕ What type music do you like?
言いたかったのは『どんな種類の音楽が好き？』だが、of が抜けると文法的に不完全で、ネイティブには言い間違い・書き間違いに聞こえる。
○ What type of music do you like?
どんな種類の音楽が好きですか。
What type of music do you like?
『〜の種類』は type of + 名詞という決まった形で、of を省略できない。日本語の『どんな種類の』を直訳する際に of が抜け落ちやすい。
✕ I want to practice type every day.
言いたかったのは『毎日タイピングを練習したい』。practice の後に原形 type を置くと文法的に誤りで、不自然に聞こえる。
○ I want to practice typing every day.
毎日タイピングを練習したい。
I want to practice typing every day.
practice は動名詞（-ing）を目的語に取る動詞。『タイプすることを練習する』を直訳して原形 type や to type を続けてしまう誤り。
✕ This is type of computer I use at work.
言いたかったのは『これは職場で使っている種類のコンピューターだ』。a が抜けると英語として不完全に聞こえ、ネイティブには違和感を与える。
○ This is a type of computer I use at work.
これは職場で使っている種類のコンピューターだ。
This is a type of computer I use at work.
type of ~ が単数名詞の補語として使われる場合、a type of ~ のように冠詞 a を伴うのが基本。日本語には冠詞に当たる語がないため抜け落ちやすい。
🗾 カタカナ語とのギャップ

カタカナ「タイプ」は「タ・イ・プ」の3モーラだが、英語は1音節。母音は二重母音 /aɪ/ として1拍でつなげ、語末の /p/ には母音を付けない（「プゥ」と読まない）。また日本語の『彼、タイプ』のように名詞単独で好みの相手を指す口語用法は、英語では he's not my type のように所有格＋type の形が必要で、type だけを名詞的に切り出して使うことはできない。

🌱 Word family・語源
typical（形容詞）典型的な、いかにものNGSL
typewriter（名詞）タイプライター
typist（名詞）タイピスト、入力担当者
prototype（名詞）試作品、原型
stereotype（名詞・動詞）固定観念、ステレオタイプ；型にはめて見る
typo（名詞）タイプミス、誤字（typographical error の略）
語源: ギリシャ語 typos（τύπος「打刻された跡、型、模型」）に由来し、ラテン語 typus を経て中世フランス語 type から英語に入った。もとは『型に押して刻んだ跡』を意味し、そこから活版印刷の『活字』、さらに『分類の型・種類』という抽象的な意味へ広がった。
typical（典型的な）— 『型』にぴったり当てはまる様子から『典型的な』の意味に
prototype（試作品・原型）— proto-（最初の）+ type（型）で『最初の型』が原義
stereotype（固定観念）— もとは印刷用語で『同じ型（stereo-）を繰り返し使う鋳型版』を指し、そこから『紋切り型の見方』の意味に転じた
💡 覚え方: ギリシャ語で『打ち込んで刻んだ跡』が原義。ハンコを型に押し込むイメージで覚えると、活字（typewriter）も『種類・型』（type of ~）も『あらかじめ決まった型に当てはめる』という一本の線でつながる。

⚖️ 類義語の使い分け
kind — type と kind はどちらも『種類』を意味しほぼ置き換え可能。type は blood type や body type のように科学的・専門的な分類でよく使われ、kind はより日常的・口語的な響きを持つ。
sort — sort も『種類』を表すが、type よりくだけた口語表現で、境界のはっきりしない曖昧な分類（sort of a joke など）にもよく使われる。
enter — 動詞としての type（打ち込む）と近い意味だが、enter はキーボードに限らずタッチパネルやバーコードなど、あらゆる手段でデータを『入力する』ことを指す広い語。type はキーボードで文字を打つという物理的動作に限定される。
💬 熟語・慣用表現 (2)
not my type （恋愛対象として）好みのタイプではない口語
He's a nice guy, but honestly he's just not my type.（彼はいい人だけど、正直言って全然タイプじゃないんだよね。）
🔍 「私のタイプではない」という言い方だが、単なる『分類上の種類』ではなく暗黙のうちに『恋愛対象としての好み』を指す決まった言い方。type の後に of person などを補わず単独で使う点が非構成的。
📝 肯定形 She's exactly my type. のように褒め言葉としてもよく使う。my/your/his/her など所有格＋type の形が基本で、type だけを単独の名詞として切り出すことはできない。
true to type （人や物事が）いつも通り、案の定その種類らしく振る舞う中立／やや文語
True to type, the weather forecaster predicted rain right before our picnic.（案の定というべきか、天気予報士はピクニック直前に雨を予想した。）
🔍 「型に忠実」という直訳から転じて、『その人・ものの性質やパターン通りに事が運ぶ』という意味の定型表現。予想通り、あるいは皮肉を込めて『やっぱりね』というニュアンスで使われる。
📝 文頭に副詞句として置かれることが多い（True to type, he arrived late again. 「相変わらず、彼はまた遅刻した」）。良くも悪くも『予想通りの展開』を指摘する際に使われ、ネガティブな文脈での皮肉としてよく登場する。
