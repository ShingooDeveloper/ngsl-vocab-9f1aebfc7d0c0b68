second

形容詞・名詞・動詞米 /ˈsekənd/ ／ 英 /ˈsekənd/
  1. 【形容詞】2番目の、第二の（This is the second time I've visited Kyoto. 京都を訪れるのはこれで2回目だ。）
  2. 【名詞】秒（時間の単位）（Wait a few seconds. 数秒待ってください。）
  3. 【名詞】（一連の中で）2番目のもの・人、2位（She came in a close second. 彼女は僅差の2位でゴールした。）
  4. 【動詞】（動議・提案に）賛成する、支持する（No one seconded the motion, so it failed. 誰も動議に賛成しなかったので、それは否決された。）

他のNGSL語への置き換え

second → first
元: This is the second time I've been to Japan.
後: This is the first time I've been to Japan.
訳: 日本に来たのはこれが初めてです。
📝 second を first に置き換えると「初めて」という意味になり、経験がゼロ回であることを示す。second は「2回目」で、既に1回経験済みであることを含意する点が対照的。
second → support
元: If you propose the new schedule, I'll second it at the meeting.
後: If you propose the new schedule, I'll support it at the meeting.
訳: もし新しいスケジュールを提案してくれたら、会議でそれを支持するよ。
📝 動詞 second は「（動議に対して公式に）賛成する」という会議・議事進行に限定的なフォーマルな語。support はより一般的で日常会話でも使える。second を使うと「正式な賛成の意思表示」というニュアンスが加わる。
second → minute
元: It only takes a second to send a text message.
後: It only takes a minute to send a text message.
訳: テキストメッセージを送るのに1分もかからない。
📝 second を minute に置き換えると、より長い（が依然として短い）時間を表す。a second は誇張表現で「一瞬」、a minute はやや現実的な「ちょっとの時間」を示す口語表現。
