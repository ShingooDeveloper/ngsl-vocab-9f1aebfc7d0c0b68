function

名詞・動詞米 /ˈfʌŋkʃən/ ／ 英 /ˈfʌŋkʃən/func·tion（2音節）CEFR B1

🗣 第1音節に強勢（FUNC-tion）。強勢母音は /ʌ/ で「ア」に近いが口をあまり開けない。第2音節 -tion は無強勢の /ʃən/ で、母音は曖昧母音（シュワ）に弱まる

  1. 【名詞】機能、働き（The function of the heart is to pump blood. 心臓の機能は血液を送り出すことです）
  2. 【名詞】（数学の）関数
  3. 【名詞】公式行事、式典（a formal function 公式行事）
  4. 【動詞】機能する、作動する（The device functions well even in cold weather. その装置は寒い天候でもよく機能する）
  5. 【動詞】（as ~として）役割を果たす（It can function as a backup power source. それは予備電源として機能しうる）

語法マーク

  • 動詞は基本的に自動詞。目的語を直接取らない（× function a machine のようには使えない。「動かす」なら operate や run を使う）
  • 「〜として機能する」は必ず function as + 補語 の形（as を省略できない）
  • 名詞は可算：機能の意味でも行事の意味でも a function / functions と数えられる
  • 動詞は状態的にも動作的にも使え、進行形も可能（The engine is functioning normally now.）
  • 他品詞での用法: 数学・プログラミングの「関数」の意味では可算名詞として a function, functions のように使う: This spreadsheet has a function that calculates the average automatically.（この表計算ソフトには平均を自動計算する関数がある）。また「〜に応じて変化するもの」という比喩的な意味で a function of ~ という言い方もよく使われる: Success is often a function of effort and luck.（成功はしばしば努力と運次第で決まる）。これは entry.json の【名詞】機能・数学の関数の用法をさらに広げたもの。

文型・頻出パターン

  • function as + 補語〜として機能する・役割を果たす
    The old barn now functions as a small museum.その古い納屋は今、小さな博物館として機能している。
    Vitamin D functions as a hormone in the body.ビタミンDは体内でホルモンとして機能する。
  • function + 副詞（properly / normally / well 等）正常に・きちんと機能する
    The smoke alarm didn't function properly during the test.そのテストでは、火災報知器がきちんと作動しなかった。
  • the function of A is to doAの機能は〜することだ
    The function of a valve is to control the flow of a liquid or gas.バルブの機能は、液体や気体の流れを制御することだ。
  • a function of XXの関数、Xに応じて決まるもの
    The rent you pay is largely a function of location.支払う家賃は主に立地次第で決まる。
  • perform / serve a function機能を果たす（動詞perform/serve+名詞functionの軽動詞的な組み合わせ）
    Each organ performs a specific function in the body.それぞれの臓器は体内で特定の機能を果たしている。

他のNGSL語への置き換え

function → role
元: The main function of the immune system is to protect the body from disease.
後: The main role of the immune system is to protect the body from disease.
訳: 免疫系の主な役割は、体を病気から守ることです。
📝 function は「（機械的・実務的に）何をするか」という働きそのものに焦点を当てるのに対し、role はより大きな仕組みの中で「どんな立場・役割を担うか」という関係性に焦点を当てる。この文脈ではどちらも自然だが、function の方がやや専門的・客観的に聞こえる。
function → worked
元: This old refrigerator has functioned reliably for over fifteen years.
後: This old refrigerator has worked reliably for over fifteen years.
訳: この古い冷蔵庫は15年以上、信頼できる形でずっと動いてきた。
📝 work は日常会話で機械が「動く・機能する」ことを表す最も普通の語。function はやや形式的・技術的な響きがあり、説明書や専門的な文章でよく使われる。話し言葉では work の方が自然。
function → served
元: During the power outage, the emergency generator functioned as the building's only source of electricity.
後: During the power outage, the emergency generator served as the building's only source of electricity.
訳: 停電の間、非常用発電機がそのビルの唯一の電力源としての役目を果たした。
📝 serve as と function as はどちらも「~として機能する・役目を果たす」という意味で近いが、serve as は人や物が果たす「用途・貢献」に重点があり、function as はより機械的・システム的な「働き方」に重点がある。この文ではほぼ同じ意味で使える。

⚠️ 日本人の典型誤用
✕ This software functions many useful tools.
「このソフトには便利な機能がたくさんある」と言いたかったが、function を他動詞のように目的語付きで使っており、英語として成立していない。
○ This software has many useful functions.
このソフトには便利な機能がたくさんある。
This software has many useful functions.
動詞 function は自動詞で、直接目的語を取れない。「〜という機能を持つ」と言いたい場合は名詞 function(s) を使い、have/offer などの他動詞と組み合わせる。
✕ It can function a backup power source.
「それは予備電源として機能しうる」と言いたかったが、as が抜けているため「それは予備電源を機能させる」のような不自然な文に聞こえてしまう。
○ It can function as a backup power source.
それは予備電源として機能しうる。
It can function as a backup power source.
「〜として機能する」は function as + 補語 の形が必須。日本語の「〜として」につられて as を落としてしまう誤りが多い。
✕ I need to function this machine before the demonstration.
「発表の前にこの機械を操作しておく必要がある」と言いたかったが、function にはそもそも「〜を操作する」という他動詞の意味が無いため、意味が通らない。
○ I need to operate this machine before the demonstration.
発表の前にこの機械を操作しておく必要がある。
I need to operate this machine before the demonstration.
function は「（機械などが）自分で動く・働く」という自動詞であり、「（人が）機械を操作する」という他動詞の意味は持たない。その意味では operate や use を使う。
🗾 カタカナ語とのギャップ

カタカナ「ファンクション」は「ファ・ン・ク・ショ・ン」の5モーラで発音されがちだが、英語は func·tion のわずか2音節。強勢は第1音節にのみ置き（FUNC-tion）、fun-の母音 /ʌ/ を「ア」で強く短く、-tionは弱く /ʃən/ とシュワ化させる。カタカナ発音のように各拍を均等な強さで区切ると不自然な棒読みになる。

🌱 Word family・語源
functional（形容詞）機能的な、実用本位の；正常に作動するNGSL
functionality（名詞）機能性、（機器・ソフトの）機能一式
malfunction（名詞・動詞）誤作動、故障；誤作動する
dysfunction（名詞）機能不全、（組織・関係の）機能障害
functionary（名詞）（やや否定的に）お役所的な役人、末端の職員
語源: ラテン語 functio「実行、遂行」に由来し、動詞 fungi「（任務などを）果たす、遂行する」から派生。「本来果たすべき役目・働き」という核となるイメージが今も残る。
defunct（機能を失った、廃止された）— de-（〜し終えた）+ funct（果たす）＝「役目を果たし終えた」→もう機能していない
perfunctory（おざなりな、義務的な）— per-（徹底して、の意だが皮肉に）+ funct（果たす）＝「ただ義務だけを果たす」→形だけの、心のこもらない
functionary（（お役所的な）役人）— function に -ary（人）が付き、「決められた役目を果たすだけの人」を指す
💡 覚え方: ラテン語 fungi「任務を果たす」から来た一族。function は「果たすべき働き」、defunct は「もう役目を終えて機能しない」、perfunctory は「義務だけ形式的に果たす」— どれも『役目を果たす／果たさない』のイメージでつながる。
⚖️ 類義語の使い分け
work — work は機械やシステムが「動く・機能する」ことを表す最も普通で口語的な語。function はより形式的・技術的な響きがあり、説明書や専門的文脈で好まれる。
operate — operate は人が機械を操作する意味（他動詞）でも、機械やシステムが動く意味（自動詞）でも使えるが、function は基本的に自動詞のみ。また operate は「（一定の方式・時間で）運転・稼働する」というやや管理された動きのニュアンスを持つ（The store operates 24 hours a day.）。
serve — serve as は function as と同様「〜として役目を果たす」を表すが、serve は人や物が誰かの役に立つという貢献・恩恵のニュアンスが強く、function as はより中立的・機械的な「働き方」を表す。
💬 熟語・慣用表現 (2)
form follows function 形は機能に従う（見た目より働きを優先してデザインすべきという設計思想）書き言葉／専門的
The architect believed that form follows function, so every room was designed around its practical use.（その建築家は「形は機能に従うべきだ」という信念を持っていたので、すべての部屋は実用性を軸に設計されていた。）
🔍 建築家ルイス・サリヴァンが唱えた設計哲学の格言。「物の形は、それが果たすべき役割・働きに基づいて決まるべきだ」という考え方を表す定型句で、単語を足し合わせただけでは伝わらない専門的な含意を持つ。
📝 元は1896年の建築家ルイス・サリヴァンの言葉で、後にモダニズム建築・工業デザイン全般のスローガンとなった。近年ではソフトウェアやプロダクトデザインの議論でも比喩的に引用される。
bodily functions 生理現象、生理的な機能（排尿・排便など人目をはばかる体の働きを指す婉曲表現）中立／婉曲表現
As people age, they can sometimes lose control of their bodily functions.（人は年を取ると、時に生理的な機能（排泄など）をコントロールできなくなることがある。）
🔍 文字通りには「体の機能全般」を指すが、実際には排尿・排便のような、口に出しにくい生理現象を遠回しに指す決まり文句として使われることが多い。
📝 医療・介護の場面で「排泄」を直接的に言うのを避けるための婉曲表現として頻繁に使われる。文脈によっては単に「体の働き全般」という文字通りの意味で使われることもある。
