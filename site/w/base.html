base

名詞・動詞・形容詞米 /beɪs/ ／ 英 /beɪs/base（1音節）CEFR A2

🗣 1音節語。母音は二重母音 /eɪ/ で、「エ」から「イ」へなめらかに滑らせて1拍で発音する。カタカナ「ベース」につられて母音を平坦に伸ばしすぎないこと

  1. 【名詞】土台、基礎、（物の）底部（the base of the mountain 山のふもと／the base of a triangle 三角形の底辺）
  2. 【名詞】基地、本拠地、拠点（a military base 軍事基地／my home base 私の本拠地）
  3. 【名詞】（化学）塩基；（数学）底、基数（an acid and a base 酸と塩基）
  4. 【動詞】（be based on/in の形で）～に基づく、～を拠点とする（The company is based in Osaka. その会社は大阪に拠点を置いている。）
  5. 【形容詞】卑しい、下劣な；（貴金属に対して）質の劣る、卑金属の（base motives 卑劣な動機／base metal 卑金属）

語法マーク

  • 名詞は可算名詞。単数 a base、複数 bases（-es）
  • 動詞はほぼ受動態限定で使われる：be based in + 場所（拠点は〜にある）／be based on + 根拠・情報源（〜に基づく）。前置詞 in と on を混同しないこと
  • 動詞を能動態で使う場合は base A on B「AをBに基づかせる」の形（The design team has based the new logo on the company's first emblem. のように使う）
  • 形容詞用法（卑しい、質の劣る）は名詞の前に置く限定用法が多く、日常会話では稀。比較級・最上級を持つ点で他の基本的な形容詞と同様に変化する
  • 他品詞での用法: 動詞用法はほぼ受動態 be based in/on/at の形で使われ、「（場所）を拠点とする」「（情報・根拠）に基づく」を表す。能動態で base A on B「AをBに基づかせる」（You should base your decision on facts.）と使うこともある。形容詞用法は「卑しい、下劣な」（base motives 卑劣な動機）や、貴金属に対する「質の劣る、卑金属の」（base metal 卑金属）を意味し、日常会話よりも文語的・やや古風な響きを持つ。この形容詞は比較級 baser・最上級 basest を持つ点も名詞・動詞用法とは異なる特徴。

文型・頻出パターン

  • be based in + 場所（組織・人が）〜を拠点とする
    Our headquarters is based in New York.私たちの本社はニューヨークに拠点を置いている。
  • be based on + 情報源・根拠〜に基づいている
    The film is based on a true story.その映画は実話に基づいている。
  • base A on BAをBに基づかせる（能動態）
    You should base your decision on facts, not rumors.噂ではなく事実に基づいて判断すべきだ。
  • the base of + 名詞〜の底部、ふもと（物理的な位置を表す）
    They set up camp at the base of the cliff.彼らは崖のふもとにキャンプを張った。
  • base + 名詞（形容詞用法）卑しい〜、質の低い〜（文語的な形容詞用法）
    He was driven by base motives.彼は卑しい動機に駆り立てられていた。
    The ring is made of base metal, not gold.その指輪は金ではなく卑金属でできている。

🗾 カタカナ語とのギャップ

カタカナ「ベース」は由来の異なる複数の語をまとめて指すため意味がぼやけやすい：①この語base（拠点・土台）、②音楽の低音楽器bass（発音は同じ/beɪs/だが別の単語）、③化粧下地の「ベースメイク」。発音面では、二重母音/eɪ/を「ベーース」のように単なる長音として伸ばしてしまいがちだが、実際は「エ」から「イ」へなめらかに移行する二重母音である点に注意。

⚖️ 類義語の使い分け
foundation — foundationは建物の基礎工事や、組織・信頼関係などの具体的・比喩的な「土台」を指し、baseよりも「支える構造物」というニュアンスが強い。the base of the mountainのような単純な位置を表す用法ではfoundationは使いにくい。
basis — basisは議論・判断・契約などの抽象的な「根拠」を表す語で、物理的な土台や建物の底部にはbasisを使わない（on the basis of the agreementのように使う）。baseはこの意味も含むより広い語。
headquarters — headquartersは組織の「本部・本社」という公式な所在地を指し、be based inよりも硬い表現。軍事的な「基地」の意味ではbaseの方が自然（military headquartersよりmilitary baseの方が一般的）。
💬 熟語・慣用表現 (4)
touch base (with someone) （人と）連絡を取る、近況を報告し合うビジネス／中立
Let's touch base again next week to check on the project's progress.（来週またプロジェクトの進捗を確認するために連絡を取り合いましょう。）
🔍 野球で走者がベース（塁）に触れることから転じた表現。実際に塁に触れるのではなく、「短く連絡を取り合う」という比喩的な意味で使う。
📝 with を伴って touch base with him のように使うことが多い。深い話し合いではなく、短時間の確認・近況報告を表す軽いニュアンスの表現。
off base 見当違いで、間違って口語
You're way off base if you think the plan will finish on time.（その計画が予定通りに終わると思っているなら、それは大きな見当違いだよ。）
🔍 野球で走者がベースから離れてアウトになりやすい危険な状態にあることから。「的外れ」「見当違い」を表す比喩。
📝 way off base のように way で強調されることが多い。動詞 base の be based on（〜に基づく）とは意味的なつながりのない別表現。
cover all the bases あらゆる可能性に備える、抜かりなく準備する中立
We double-checked every detail to cover all the bases before the presentation.（発表前にあらゆる事態に備えるため、細部まで二重に確認した。）
🔍 野球の守備で全ての塁を守り切ることから。「あらゆる事態に対応できるよう手を打っておく」という比喩表現。
📝 アメリカ英語では cover your bases とも言う。リスク管理や事前準備の文脈でビジネス・日常どちらでもよく使われる。
get to first base (with someone) （特に恋愛関係で）第一段階まで進む、最初の一歩を踏み出す口語
He couldn't even get to first base with her before she lost interest.（彼女に興味を失われる前に、彼は最初の一歩すら踏み出せなかった。）
🔍 野球で一塁に到達することから。「物事の最初の段階に到達する」比喩で、特に口説きや交渉の初期段階を指す口語表現。
📝 否定文（can't get to first base）で使われることが多く、「まったく進展しない」という失敗のニュアンスを伴う。恋愛以外の交渉・営業の場面にも使われる。
