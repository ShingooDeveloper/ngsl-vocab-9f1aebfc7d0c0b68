structure

名詞・動詞米 /ˈstrʌktʃɚ/ ／ 英 /ˈstrʌktʃə/struc·ture（2音節）CEFR B1

🗣 第1音節に強勢。語頭の子音連結 /str/ は日本語のように母音を挟まず一息で発音する（「ストラクチャー」と5拍にしない）。語末は r音化母音 /ɚ/ で、舌をそり上げるだけで母音を足さない

  1. 【名詞】（物事の）構造、組織、仕組み（the structure of the human body 人体の構造）
  2. 【名詞】建造物、建築物、構築物（an ancient stone structure 古代の石造建築物）
  3. 【動詞】（～を）組み立てる、構成する、体系立てる（structure your time effectively 時間を効果的に組み立てる）

語形変化

形種類発音備考
structures複数形（名詞）／三単現（動詞）/ˈstrʌktʃɚz/名詞なら「複数の構造・建造物」、動詞なら三人称単数現在（She structures her essays carefully. のように）。語末は有声のzで終わる
structured過去形・過去分詞（動詞）／形容詞（well-structured など）/ˈstrʌktʃɚd/規則変化。語幹が有声の r音化母音 /ɚ/ で終わるため -ed は有声の /d/ で発音される（/ˈstrʌktʃɚd/。「ストラクチャード」と各拍を伸ばさず、語末は軽い /ɚd/）。受動態 be structured や a well-structured essay のような過去分詞の形容詞的用法で特によく使う

語法マーク

  • 名詞：抽象的な「仕組み・組織」の意味では原則不可算（the structure of society）。具体的な「建造物」の意味では可算（an ancient structure / structures around the world）
  • 動詞：他動詞。目的語を必ず伴う（× I need to structure. だけでは不自然。何をstructureするか明示する）
  • 受動態でよく使われる：be structured（well / poorly / logically などの副詞を伴うことが多い）
  • 主語が structure（単数）のときは動詞も単数で受ける（The structure of the sentences is complex. であって are ではない。近くの複数名詞につられやすいので注意）
  • 他品詞での用法: 名詞は2つの顔を持つ。(1) 抽象的な「構造・仕組み」の意味では不可算的に使われることが多く、無冠詞や the をつけて the structure of ~ の形で使う（This poem has no clear structure. この詩には明確な構造がない）。(2) 具体的な「建造物」の意味では可算名詞（an ancient structure, several structures）。動詞用法はどちらかというとフォーマル・学術的な語で、time / essay / course / argument のような抽象的な対象を目的語に取り「組み立てる、体系立てる」を表す。build/construct（物理的に建てる）とは意味の対象が異なる点に注意。過去分詞 structured は well-structured（よく組み立てられた）、highly structured（きっちり体系化された）のように形容詞としても定着している。

文型・頻出パターン

  • the structure of + N〜の構造。物事がどのように組み立てられているかを表す最も基本的な形
    the structure of DNADNAの構造
    the structure of a sentence文の構造
  • structure + O（他動詞）〜を組み立てる・体系立てて構成する
    structure your essayエッセイを組み立てる
    structure a lesson授業を組み立てる
  • structure + O + around/into + N〜をNを中心に／Nという形に組み立てる
    structure the course around student projects生徒のプロジェクトを中心にコースを組み立てる
    structure the report into three sections報告書を3つの節に組み立てる
  • well-structured / poorly structured + N過去分詞が形容詞化し「よく組み立てられた／お粗末に組み立てられた」を表す
    a well-structured argumentしっかり組み立てられた論理
    a poorly structured essay構成のまずいエッセイ

他のNGSL語への置き換え

structure → composition
元: Scientists have studied the structure of DNA for decades.
後: Scientists have studied the composition of DNA for decades.
訳: 科学者たちは何十年もの間、DNAの組成を研究してきた。
📝 structure は部分同士の組み立てられ方・立体的な形を指すのに対し、composition は「何から成るか」という成分・構成要素そのものに焦点を当てる。DNAの二重らせんの形を指すなら structure、含まれる塩基などの成分を指すなら composition が自然。
structure → frame
元: This building's structure was severely damaged by the earthquake last year.
後: This building's frame was severely damaged by the earthquake last year.
訳: この建物の骨組みは、昨年の地震でひどく損傷を受けた。
📝 frame は建物の骨組み・フレームという具体的な物理的部材を指す語。structure はより抽象的・全体的に「建物がどう組み立てられているか」という設計・構造全体を指すため、意味の範囲が広い。
structure → pattern
元: The molecules that make up this crystal have a very regular structure.
後: The molecules that make up this crystal have a very regular pattern.
訳: この結晶を構成する分子は、とても規則正しいパターンをしている。
📝 pattern は繰り返しのある模様・型に着目し、視覚的・反復的な規則性を強調する語。structure は分子の立体的な組み立てられ方・骨組みそのものを指す点で、より構造的なニュアンスを持つ。

⚠️ 日本人の典型誤用
✕ I want to structure a new building next year.
「来年、新しい建物を建てたい」と言いたかったが、structure を使うと「建物という抽象的な概念を組み立てたい」のような不自然な響きになり、実際の建築工事の話には聞こえない。
○ I want to build a new building next year.
来年、新しい建物を建てたい。
I want to build a new building next year.
動詞 structure は「（考え・時間・組織などを）組み立てる、体系立てる」という抽象的な意味であり、レンガや木材で物理的に「建てる」ことは指さない。物理的な建築物を建てる場合は build や construct を使う。
✕ The structure of this building are very complex.
「この建物の構造はとても複雑だ」と言いたかったが、動詞を are にすると文法的に崩れており、ネイティブには単純な単複の間違いとして目立ってしまう。
○ The structure of this building is very complex.
この建物の構造はとても複雑だ。
The structure of this building is very complex.
「the structure of + 名詞」では文の主語はあくまで structure であり、of 以下の名詞（this building）は修飾語にすぎない。近くにある of 以下の名詞につられて動詞を複数扱いにしてしまう典型的な一致の誤り（近接一致の誤り）。
✕ We need to make the report more structure.
「報告書をもっと構造立ったものにしよう」と言いたかったが、structure を名詞のまま使ってしまい、英語として不完全な文になる。
○ We need to make the report more structured.
報告書をもっと構造立ったものにする必要がある。
We need to make the report more structured.
more の後ろには形容詞（過去分詞由来の structured）が必要。名詞のまま structure を置くと文法的に成立しない。
🗾 カタカナ語とのギャップ

カタカナ「ストラクチャー」は4〜5拍で均等に伸ばして読まれがちだが、英語は2音節で第1音節 struc に強勢が集中し、-ture は弱く短い /tʃɚ/ につぶれる。また過去形 structured を「ストラクチャード」と各拍を伸ばして読むのも英語の2音節 /ˈstrʌktʃɚd/ から離れてしまう（語末は軽い /ɚd/ で終わる）。

🌱 Word family・語源
structural（形容詞）構造上の、構造的な（structural damage 構造的な損傷）
structurally（副詞）構造的に、構造上
restructure（動詞）（組織などを）再編する、立て直す
infrastructure（名詞）インフラ、基盤設備
unstructured（形容詞）構造化されていない、まとまりのない
語源: ラテン語 structura「建築、組み立てられたもの」に由来。動詞 struere「積み上げる、組み立てる」の過去分詞 structus から派生した名詞で、「積み重ねて組み立てられたもの」が原義。
construct（建設する、組み立てる）— con-（共に）+ struere（積み上げる）＝「一緒に積み上げて造る」
instruct（教える、指示する）— in-（中に）+ struere（積み上げる）＝「知識を中に積み上げる」→教え込む
destroy（破壊する）— de-（下へ、逆に）+ struere（積み上げる）＝「積み上げたものを崩す」で structure の反対の動き
💡 覚え方: struere「積み上げる」の同族語でまとめる：construct は「一緒に積み上げて造る」、instruct は「頭の中に積み上げて教える」、destroy は「積み上げたものを崩す」— structure はその積み上げの結果できた「組み立てられたもの」。
⚖️ 類義語の使い分け
framework — framework は物事を支える骨組み・下地となる仕組みを指し、その上に他の要素が乗ることを含意する（a theoretical framework）。structure はより広く、物理的・抽象的を問わず「全体がどう組み立てられているか」自体を指す。
system — system は部分同士が機能的・動的に連動して働く仕組みに焦点がある（transport system）。structure はどちらかというと静的な組み立てられ方・配置に焦点があり、機能していることを必ずしも含意しない。
organization — organization は要素が秩序立って配置・運営されているという過程やまとまり方に焦点があり、しばしば人や活動の管理を含む。structure は組み立てられた形そのもの、骨組みに焦点がある。
