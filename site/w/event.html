event

名詞 (noun)米 /ɪˈvent/ ／ 英 /ɪˈvent/e·vent（2音節）CEFR A2

🗣 第2音節 vent に強勢（/ɪˈvent/）。語頭の e- は曖昧母音 /ɪ/ で弱く短く、はっきり「イー」と伸ばさない。カタカナ「イベント」のように4拍を均等に読まず、2拍目だけを強く長めに発音する。語末の t は次に母音が続くと（event is のように）フラップ化して弱い「ラ行」に近い音になる

  1. 【名詞】出来事、事件（Several unusual events have occurred in this town recently. 最近この町ではいくつか奇妙な出来事が起きている）
  2. 【名詞】（計画された）行事、催し、イベント（host a charity event 慈善イベントを主催する）
  3. 【名詞】（スポーツ大会の）種目、競技（the final event of the tournament 大会最後の種目）

語形変化

形種類発音備考
events複数形/ɪˈvents/語末が無声音 /t/ の直後なので、複数の -s は無声の /s/ になる（「イヴェンツ」であって「イヴェンズ」ではない）。/ts/ の連続はカタカナのように母音を挟まず、破裂を伴わない一息の音でつなげる

語法マーク

  • 常に可算名詞。単数では必ず a/an か the を伴う（an event, the event）。area や space のような不可算用法は存在しない
  • in the event of + 名詞 / in the event that + 節 で『〜の場合には』というフォーマルな条件表現になる（案内・注意書きに頻出）
  • current events は『時事問題、最近のニュース』を表す決まった複数形表現。単数の current event はほぼ使わない
  • 形容詞＋event（rare, historic, major, tragic, sporting など）の形で幅広い性質の出来事を修飾できる中立語。良い意味にも悪い意味にも使える点が incident との違い
  • 他品詞での用法: event に動詞用法はなく、名詞のみの語（pos_all は不要）。entry.meanings の3つの意味（①出来事・事件、②行事・催し、③競技種目）はいずれも可算名詞としてふるまう。加えて in the event of ~ / in the event that ~ という定型の前置詞句・接続詞的表現があり、これは『②催し』ではなく『①何かが起きるという想定』の意味から派生した、フォーマルな条件表現（『〜の場合には』）として使われる。この用法はほぼ書き言葉・アナウンス・契約文書に限られる硬い表現。

文型・頻出パターン

  • in the event of / in the event that〜の場合には（フォーマルな条件表現）
    In the event of an emergency, please use the stairs.緊急時には、階段をご利用ください。
    In the event that the match is canceled, tickets will be refunded.試合が中止になった場合、チケットは払い戻されます。
  • current events時事問題、最近の出来事（決まった複数形）
    stay informed about current events時事問題に通じておく
  • 形容詞 + event催し・出来事の性質を表す（rare, historic, major, tragic など）
    a historic event歴史的な出来事
    a tragic event痛ましい出来事
  • organize / attend / cancel / cover an eventイベントを企画する／参加する／中止する／報道する
    attend a networking event交流イベントに参加する
    The media covered the event live.メディアはその出来事を生中継で報じた。
  • the event of + 名詞（フォーマル）〜という出来事（硬い言い方。しばしば of one's death などと共起）
    in the event of his death彼が死亡した場合には（万一のことがあった場合には）

他のNGSL語への置き換え

event → competition
元: The annual sports event is organized by volunteers from the local community every summer.
後: The annual sports competition is organized by volunteers from the local community every summer.
訳: 毎年恒例のスポーツ大会は、地元コミュニティのボランティアによって毎夏運営されている。
📝 event は「行事・催し」全般を指す広い語だが、competition は勝敗を争う「競技会」であることを明確に示す。単なる交流の場なら event、順位や勝者を決める大会なら competition がより的確。
event → campaign
元: The charity event that we organized last month raised more money than anyone expected.
後: The charity campaign that we organized last month raised more money than anyone expected.
訳: 先月私たちが企画したチャリティキャンペーンは、誰もが予想した以上の資金を集めた。
📝 event は特定の日時に行われる単発の「催し」を指すが、campaign は募金活動などが一定期間にわたって継続的に行われる組織的な「取り組み」を指す。一日限りの催しなら event、数週間~数ヶ月続く活動なら campaign が自然。
event → incidents
元: Several unusual events have occurred in this town since the new mayor took office.
後: Several unusual incidents have occurred in this town since the new mayor took office.
訳: 新しい市長が就任してから、この町では不可解な出来事がいくつも起きている。
📝 event は良い意味にも悪い意味にも使える中立的な語だが、incident はしばしばトラブル・事故など好ましくない出来事のニュアンスを帯びる。ここでは「不可解な」と合わせ、やや不穏な響きの incident の方がしっくりくる文脈もある。

⚠️ 日本人の典型誤用
✕ There was much event in the news last week.
「先週ニュースでたくさんの出来事があった」と言いたかったが、event を数えられないもののように扱うと文法が崩れ、ネイティブには基本的な数の一致ミスとして響いてしまう。
○ There were many events in the news last week.
先週ニュースでたくさんの出来事があった。
There were many events in the news last week.
event は常に可算名詞。不可算扱いの much や単数の was を使ってしまう誤り。数量表現には many を、動詞は複数の were を使う。
✕ In the event fire, please leave the building immediately.
「火事の場合には、すぐに建物から出てください」と言いたかったが、of が抜けると定型表現として成立せず、指示・注意書きとして不自然な英語に聞こえてしまう。
○ In the event of a fire, please leave the building immediately.
火事の場合には、すぐに建物から出てください。
In the event of a fire, please leave the building immediately.
in the event of ~「〜の場合には」は前置詞 of を省略できない定型表現。日本語の「〜の場合には」につられて of を落としてしまう誤り。
✕ What's the event for tonight?
「今夜は何するの?（今夜の予定は?）」と聞きたかったが、event を使うと『今夜、何か特別な催し物があるの?』という限定的な意味に聞こえ、意図がずれて伝わってしまう。
○ What are we doing tonight? / What's the plan for tonight?
今夜は何するの?
What are we doing tonight?
What's the plan for tonight?
日本語の口語「今日のイベント」は『予定・やること』全般を指せる和製英語的な拡張表現だが、英語の event はあくまで特定の催し・出来事を指し、日常のルーティンや単なる予定全般には使わない。
🗾 カタカナ語とのギャップ

カタカナ「イベント」は「コンサート」「バーゲンセール」のような楽しい催し・行事のみを指すよう意味が狭まっているが、英語の event ははるかに広く、事故・事件・災害のような望ましくない出来事（a tragic event）や、単なる『出来事』一般（Several unusual events have occurred ~）まで中立的にカバーする語。また日本語の口語で『今日のイベント』を『予定・やること』の意味で使うのは和製英語的な拡張で、英語の event にはこの意味はない。発音面では、カタカナ「イ・ベ・ン・ト」の4拍均等リズムに対し、英語は2音節 /ɪˈvent/ で2拍目にのみ強勢が置かれ、語末の t には母音を付けない（「イベントゥ」としない）。

🌱 Word family・語源
eventually（副詞）結局は、いずれ、そのうちNGSL
eventual（形容詞）最終的な、結果として起こる
eventful（形容詞）出来事の多い、波乱に満ちた
語源: ラテン語 ēventus「結果、出来事」（動詞 ēvenīre「外に出てくる、結果として起こる」の名詞形）に由来する。ēvenīre は ē-（外へ）+ venīre（来る）という構成で、直訳すると『外へ出てくる』。何かの成り行きの末に『表に出てきたこと・結果として起きたこと』が原義。
prevent（防ぐ、妨げる）— 同じ venīre（来る）の系列。pre-（前もって）+ venīre で『先回りして来て行く手を阻む』が原義。つづりの上でも pr + event という形で偶然 event を含む
invent（発明する）— in-（〜の中へ）+ venīre で『中に入り込んで出会う、思いつく』が原義
venue（開催地、会場）— venīre の名詞形。『人々が来る場所』が原義で、event（催し）が行われる『場所』を表す語として意味的にも密接に結びつく
💡 覚え方: 「来る」を意味するラテン語 venīre が隠れている一族。e-（外へ）+ venīre → event（外に出てきた結果＝出来事）、in-（中へ）+ venīre → invent（中に入り込んで思いつく＝発明する）、pre-（前もって）+ venīre → prevent（先に来て塞ぐ＝防ぐ）。『来る』という動きのイメージで event・invent・prevent がひとつながりになる。
⚖️ 類義語の使い分け
occurrence — occurrence はよりフォーマル・中立的で、単に『何かが起きたという事実』に焦点を当てる語。event は起きたこと自体に加え、ある程度注目に値する・重要だというニュアンスを持ちやすい。
incident — incident はしばしば事故・衝突・トラブルなど望ましくない『出来事』を指す。event は良い意味にも悪い意味にも使える中立語である点が異なる。
occasion — occasion は『特別な機会・場面』に焦点を当てた語で、儀式的・記念的な意味合いを伴うことが多い（a special occasion）。event は催し物そのもの（式典・大会など）を指すのに対し、occasion はその『機会・場面』という抽象的な側面を指すことが多い。
💬 熟語・慣用表現 (3)
in the event of / in the event that 〜の場合には、もし〜が起きたらフォーマル／文書・アナウンス
In the event of a power outage, please contact the front desk immediately.（停電が発生した場合は、直ちにフロントデスクにご連絡ください。）
🔍 「その出来事の中で」が原義。実際に何かが起こる状況を指して「〜という事態になったら」というフォーマルな条件表現に転じた。
📝 契約書や案内表示などの硬い書き言葉に頻出。日常会話では if を使う方が自然。of の後には名詞（句）、that の後には節が続く。
in any event とにかく、いずれにせよ、何があっても中立／ややフォーマル
The flight might be delayed, but in any event, we should arrive before the ceremony starts.（便が遅れるかもしれないが、いずれにせよ式典が始まる前には到着できるはずだ。）
🔍 「どんな出来事の中でも」が原義。状況にかかわらず結論は変わらないと述べる際の締めくくりの言葉。
📝 anyway や in any case とほぼ同義。話や議論を切り上げて結論に移る際の合図としても使われる。
in the event 実際には、ふたを開けてみると（予想と違う結果になったことを表す）ややフォーマル／主にイギリス英語
We had prepared for the worst, but in the event, the storm passed without causing any damage.（最悪の事態に備えていたが、実際には嵐は何の被害も出さずに過ぎ去った。）
🔍 「その出来事の中では」が原義。予想していたことと実際の結果が異なったときに使う言い回し。
📝 in the event of と混同しやすいが別の意味。予想と現実のギャップを述べる際に文頭や文中に置く。イギリス英語でよく使われ、アメリカ英語では in fact や as it turned out の方が一般的。
