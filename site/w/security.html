security

名詞 (noun)米 /sɪˈkjʊrəti/ ／ 英 /sɪˈkjʊərəti/se·cu·ri·ty（4音節）CEFR B1

🗣 第2音節 -cu- に強勢がある（/sɪˈkjʊrəti/）。語頭の se- は曖昧母音 /sɪ/ で弱く短く、日本語の「セ」のようにはっきり発音しない。-ri- の t は前後を母音に挟まれてフラップ化しやすく、「リディ」寄りに聞こえる

  1. 【名詞】安全、安心（危険や脅威がない状態。a sense of security 安心感）
  2. 【名詞】防犯、警備（不正侵入や犯罪を防ぐための対策・体制。airport security 空港の警備；security guard 警備員）
  3. 【名詞】(national security) 安全保障（国家・社会全体を守ること。a threat to national security 国家安全保障への脅威）
  4. 【名詞】担保、保証（借金などの見返りとして提供するもの。put up one's house as security 家を担保に入れる）
  5. 【名詞】証券、有価証券（複数形 securities で用いる。株式・国債など。government securities 国債などの政府証券）

語形変化

形種類発音備考
securities複数形/sɪˈkjʊrəɾiz/「安全・警備」の意味では通常不可算で複数形にしないが、『証券・有価証券』の意味に限り可算名詞として複数形 securities を用いる（yをiesに変える規則的な綴り変化）

語法マーク

  • 「安全・安心・警備」の意味では通常不可算名詞。数量を言う時は a lot of security、not much security のように扱い、a security のように可算扱いしない
  • 複数形 securities は『証券・有価証券』の意味に限られる、意味が変わる可算名詞
  • 名詞を形容詞的に用いて複合語を作りやすい（security guard, security camera, security check, security clearance など）
  • for/as security の形で「担保として」を表す（put up ~ as security, give ~ as security for a loan）
  • 他品詞での用法: 他の名詞の前に置いて複合語を作る用法が非常に生産的（security guard 警備員、security camera 防犯カメラ、security clearance 機密情報取扱資格、security deposit 敷金・保証金）。『安全・警備』の意味では不可算だが、『証券』の意味の securities は可算名詞として振る舞う点で品詞的に独特（entryのmeaningsにある複数形の用法を参照）。さらに for/as security という形で「担保として」という金融・法律分野の前置詞句を作る用法もある（hold shares as security for a loan 株式をローンの担保として保有する）。

文型・頻出パターン

  • a sense of security安心感（securityが不可算なのでa securityとは言えず、a sense ofを介して数える）
    Owning a home gives many people a sense of security.家を持つことは、多くの人に安心感を与える。
  • security against/from ~〜からの安全・防御
    security against theft盗難への備え
    security from outside threats外部の脅威からの防御
  • security + 名詞（複合名詞）防犯・警備に関わる具体的な人・物・制度を表す
    security guard警備員
    security camera防犯カメラ
    security clearance機密情報取扱資格
  • put up/give O as security (for ~)〜（の担保）としてOを差し出す
    He put up his house as security for the loan.彼はローンの担保として家を差し出した。
  • national/job/social security特定の分野の安全保障・保障を表す決まった複合語
    job security雇用の安定
    Social Security（米国の）社会保障制度

他のNGSL語への置き換え

security → safety
元: The company has improved its online security significantly after last year's data breach.
後: The company has improved its online safety significantly after last year's data breach.
訳: その会社は昨年のデータ漏えいの後、オンラインの安全性を大幅に改善した。
📝 security は不正アクセスや攻撃からの「防御」を指す専門的な語で、オンラインの文脈では最も自然。safety に替えると事故や誤操作からの「安全」というニュアンスに寄り、ハッキング対策という原意がぼやける。
security → protection
元: If the airport had tighter security, this kind of incident wouldn't happen so often.
後: If the airport had tighter protection, this kind of incident wouldn't happen so often.
訳: 空港がもっと厳重な保護態勢を敷いていれば、この種の事件はそう頻繁には起こらないだろうに。
📝 security は監視・検査など制度化された防犯体制全体を指すのに対し、protection は「守る」という行為・手段そのものに焦点がある、より広い語。空港・国家・データなど組織的な防犯の文脈では security の方が自然。
security → bonds
元: Government securities are often considered a safer investment than individual stocks.
後: Government bonds are often considered a safer investment than individual stocks.
訳: 国債は個別株よりも安全な投資先だと考えられることが多い。
📝 securities は株式・債券などの有価証券全般を指す包括的な語であるのに対し、bonds は利子付きの「債券」に限定される具体的な語。政府発行の有価証券の多くは実際には bonds（国債）なのでほぼ同じ意味になるが、securities の方が株式なども含む広いカテゴリーを表す点が異なる。

🗾 カタカナ語とのギャップ

カタカナ「セキュリティ」は各拍が均等に強く聞こえがちだが、英語では第2音節 -cu- (/kjʊ/) に強勢が集中する（/sɪˈkjʊrəti/）。語頭の se- は曖昧母音/sɪ/で弱く短い「シ」寄りの音で、「セ」とはっきり言わない。また -ri- の t は母音に挟まれてフラップ化し、「ティ」を強く発音すると不自然に聞こえる。

🌱 Word family・語源
secure（形容詞・動詞）安全な、確実な；～を確保する、守るNGSL
securely（副詞）しっかりと、安全に
insecure（形容詞）不安な、不安定な、自信のない
insecurity（名詞）不安、不安定さ
語源: ラテン語 securus（se-「〜から離れて」+ cura「心配、気遣い」）に由来し、『心配から解放された状態』が原義。同じラテン語は古フランス語 seur を経て英語 sure にもなっており、security/secure と sure は同じ語源を共有する兄弟語である。
secure（安全な、確保する）— 同じラテン語 securus に由来する形容詞・動詞形
sure（確信して、確かな）— 同じ securus が古フランス語 seur を経て英語に入った、より口語化した兄弟語
cure（治療、治す）— 語源の cura（気遣い、世話）を共有。『世話をする』から『治療する』へ意味が発展した
💡 覚え方: se-（〜から離れて）+ cura（心配）＝『心配のない状態』が security の原義。sure も同じ語源から生まれた兄弟語で、secure は『心配を取り除いて安全にする』、sure は『（心配なく）確信している』という意味に枝分かれしたと覚える。
⚖️ 類義語の使い分け
safety — safety は事故・怪我など偶発的な危険からの『安全』に重点があるのに対し、security は盗難・攻撃・侵入など意図的な脅威に対する『防御・警備』の体制を指すことが多い（road safety vs. airport security）。
protection — protection は『守る』という行為・手段そのものを表す一般的な語。security は監視カメラや警備員など制度化・組織化された防犯体制を指す、より具体的で専門的なニュアンスを持つ。
safeguard — safeguard は具体的な『予防策・保護措置』を指す語で、しばしば複数形 safeguards で用いられる。security は体制・状態全体を指す抽象度の高い語である点が異なる。
💬 熟語・慣用表現 (2)
security blanket 心のよりどころ、心理的な支えとなるもの（元は子どもが安心のために手放さない毛布）中立（やや口語的）
Ever since he was little, he has carried that old teddy bear around like a security blanket.（彼は小さい頃からずっと、あの古いテディベアを心のよりどころのように持ち歩いている。）
🔍 直訳は「安全毛布」。幼児が不安なときに肌身離さず持つ毛布を指す言葉から転じて、大人にとっての精神的な支え・安心材料（人・物・習慣など）全般を比喩的に表す。
📝 具体的な毛布に限らず、スマートフォンや決まった習慣など「手放せない安心材料」全般を指して比喩的に使われることが多い。

🔍 直訳は「偽りの安全の感覚」。実際にはリスクが残っているにもかかわらず、大丈夫だと錯覚してしまう心理状態を指す。
📝 しばしば give/create/lull someone into a false sense of security の形で使われ、「油断を招く」という警告的なニュアンスを伴う。
