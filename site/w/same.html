same

形容詞・代名詞・副詞米 /seɪm/ ／ 英 /seɪm/same（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。母音は二重母音 /eɪ/ で、「セ」から「イ」へなめらかに滑らせて1拍で発音し、語末の /m/ は最後まで唇をしっかり閉じて終える（母音を足して「セイムゥ」としない）

  1. 【形容詞】同じ、同一の（ふつう the/this/that などの限定詞を伴い、"the same + 名詞（+ as ~）"の形で使う）（We stayed at the same hotel as last year. 私たちは去年泊まったのと同じホテルに泊まった。）
  2. 【形容詞】（at the same time で）同時に（We can't watch TV and study at the same time. テレビを見ながら同時に勉強することはできない。）
  3. 【代名詞】（the same で）同じもの、同じこと（前に述べた内容を指す）（A: I'll have a coffee. B: I'll have the same. A: コーヒーをください。B: 私も同じものを。）
  4. 【副詞】（the same で）同じように、同様に（Twins often think and act the same. 双子はしばしば同じように考え、行動する。）

語法マーク

  • 原則として the/this/that/one's などの限定詞を伴う（× I have same bag. は誤り。○ I have the same bag.）
  • 比較変化を持たない（同一性には程度差がないため more same / samer のような形は存在しない。程度を表したい場合は similar を使う）
  • 比較対象を導くときは as を使う（the same ... as ~）。than は使わない点に注意（different from/than とは異なる）
  • 代名詞用法の the same は前に述べた内容を受ける（I'll have the same.）
  • 副詞用法の the same は動詞を修飾し「同じように」を表す（look the same, taste the same, act the same）
  • all the same は「それでも、やっぱり」という熟語的意味になり、「同一」の意味からは離れる（別途idiomsを参照）
  • 他品詞での用法: 形容詞用法（the same + 名詞）が中心だが、代名詞・副詞としても頻繁に使われる。(1) 代名詞: the same だけで「同じもの・同じこと」を指し、前に述べた内容を受ける: A: I'll have the soup. B: I'll have the same.（私も同じものにします）。(2) 副詞: the same の形で動詞を修飾し「同じように」の意味になる: The twins always dress the same.（その双子はいつも同じ格好をする）。いずれも same 単独ではなく the same の形で使われる点が共通する。

文型・頻出パターン

  • the same + 名詞（+ as ~）〜と同じ…（限定詞 the を伴う最も基本的な形）
    the same color as yoursあなたのと同じ色
    the same size同じサイズ
  • the same as + 名詞／節〜と同じである（be動詞の補語として使う述語的な形）
    Your bag is the same as mine.君のかばんは私のと同じだ。
  • the same + 名詞 + 関係詞節関係代名詞節で修飾された「同じ〜」（先行詞を限定する用法）
    the same book that I read last year去年読んだのと同じ本
  • S + and + S + be + the same2つのものが同じであると述べる叙述用法
    These two photos are the same.この2枚の写真は同じだ。
  • 動詞 + the same（副詞用法）「同じように〜する」という様態を表す
    look the same同じに見える
    sound the same同じように聞こえる

他のNGSL語への置き換え

same → similar
元: The rules have stayed the same since the club was founded twenty years ago.
後: The rules have stayed similar since the club was founded twenty years ago.
訳: そのクラブが設立された20年前から、規則はずっと似たようなものであり続けている。
📝 same は「完全に同一」であることを示すのに対し、similar は「似ているが完全には同じでない」ことを示す。細部の違いをどこまで許容するかが決め手。
same → similar
元: The same mistake was made twice before anyone noticed the pattern.
後: A similar mistake was made twice before anyone noticed the pattern.
訳: 似たような間違いが2度起きて、ようやく誰かがそのパターンに気づいた。
📝 the same mistake は全く同一の間違いが繰り返されたことを示すが、a similar mistake にすると細部は異なるものの似た種類の間違いだったというニュアンスになる。same は基本的に the/this/that 等の限定詞を伴うのに対し、similar は不定冠詞 a とも自然に使える点にも注意。
same → different
元: We ordered the same dish that we had last time, and it turned out just as good.
後: We ordered a different dish than we had last time, and it turned out just as good.
訳: 前回とは違う料理を注文したが、それでも同じくらいおいしかった。
📝 same と different は対義語ペア。比較対象を導く語も that → than に変わる点、また the same（限定詞つき）から a different（不定冠詞）へと冠詞の使い方が変わる点に注意。

⚠️ 日本人の典型誤用
✕ I bought same shoes as my sister.
「姉と同じ靴を買った」と言いたかったが、same の前に the が抜けているため文法的に不完全で、ネイティブには書きかけの文のように響いてしまう。
○ I bought the same shoes as my sister.
私は姉（妹）と同じ靴を買った。
I bought the same shoes as my sister.
same は必ず the などの限定詞を伴って使う語であり、裸で名詞の前に置くことはできない。「the same ~ as …」の型を丸ごと覚えるとよい。
✕ This bag is the same than mine.
「このかばんは私のと同じだ」と言いたかったが、same に than を続けるのは英語として成立しておらず、different と same を混同しているとネイティブに伝わってしまう。
○ This bag is the same as mine.
このかばんは私のと同じだ。
This bag is the same as mine.
same が比較対象を導くときは常に as を使い、than は使わない。different from/than との混同から than を選んでしまう誤り。
✕ Their opinions became more same over time.
「彼らの意見は時間とともにだんだん似てきた」と言いたかったが、same には程度の変化がないため more same は英語として不自然で、意味が理解しづらい表現になってしまう。
○ Their opinions became more similar over time.
彼らの意見は時間とともにだんだん似通ってきた。
Their opinions became more similar over time.
same は「完全に同一」という絶対的な状態を表すため、程度を表す比較級 more same は存在しない。徐々に近づいていく様子を表したいときは gradable な similar を使う。
🗾 カタカナ語とのギャップ

カタカナ「セイム」は「セ・イ・ム」の3モーラで捉えられがちだが、英語は1音節。母音 /eɪ/ は「エ」から「イ」へ滑らかに移る二重母音として1拍でつなげ、語末の /m/ は母音を足さずに唇を閉じたまま止める（「ムー」と伸ばさない）

🌱 Word family・語源
sameness（名詞）同一であること、同様であること
語源: 古ノルド語（ヴァイキングの言語）の sami「同じ」に由来し、中英語期に same の形で定着した北欧系借入語。究極的にはインド・ヨーロッパ祖語の語根 *somo-（同一の）に遡り、ラテン語 similis（似た）やギリシャ語 homos（同じ）と同じ根から枝分かれしている。
similar（似ている）— 同じ印欧祖語の語根 *somo- に遡る遠い親戚（ラテン語 similis 経由で英語に入った）
homogeneous（均質な）— ギリシャ語 homos「同じ」を含み、same と同根の語根を共有する
💡 覚え方: same はヴァイキングが持ち込んだ「まるごと同じ」を表す語。ラテン語経由の similar（似ている程度）とは同じ祖先から分かれた双子のような関係で、same=完全一致、similar=近い、と覚えるとよい。
⚖️ 類義語の使い分け
identical — same より意味が強く、細部まで寸分違わず一致することを強調するややフォーマルな語（identical twins のように使う）。日常会話では the same の方が圧倒的に多用される。
similar — same が完全な同一性を表すのに対し、similar は「似ているが完全には同じでない」という部分的な一致を表す。程度を表せる（more similar）点も same との大きな違い。
alike — alike は叙述用法（be alike）でしか使えず、名詞の前には置けない（× an alike bag は不可）。same はふつう the same + 名詞のように名詞の前でも使える点が異なる。
💬 熟語・慣用表現 (6)
all the same それでも、やはり、それにもかかわらず中立
It's raining outside, but I'm going for a run all the same.（外は雨が降っているが、それでも私は走りに行くつもりだ。）
🔍 「全てが同じ」ではなく、前の内容と矛盾するように見えても事態や気持ちは変わらないと述べる副詞句。nevertheless とほぼ同義。
📝 文末に置かれることが多い。断った申し出に対する「Thanks all the same.」（それでもありがとう）という決まり文句もよく使われる。
it's all the same to me 私にとってはどちらでも構わない口語
Coffee or tea? It's all the same to me.（コーヒーと紅茶、どっちでもいいよ。）
🔍 「私にとっては全て同じだ」という文字通りの意味から転じて、選択肢のどちらでもよい・無関心であることを表す。上の all the same（それでも）とは意味が異なる点に注意。
📝 me の部分を入れ替えて It's all the same to him. のようにも使う。無関心のニュアンスがあるため、場面によっては素っ気なく響くこともある。
same here 私も同じです、同感です口語
A: I'm exhausted after that hike. B: Same here.（A: あのハイキングでへとへとだよ。 B: 私も。）
🔍 「ここも同じ」という字面から転じて、相手の発言（感情・状態・意見）に軽く同意する際に使う相づち表現。
📝 I feel the same way. を短く砕けた形にした表現で、フォーマルな場では使わない。
one and the same まさに同一のもの、全く同じもの中立
The mysterious author and our teacher turned out to be one and the same.（その謎の作家と私たちの先生は、実は同一人物だとわかった。）
🔍 one と the same を重ねて意味を強調する常套句。別々のものに見える2つが実は同一であることを念押しする。
📝 ふつう be動詞の補語として使われ、単に the same と言うより強調度が高い言い方。
same to you そちらこそ、お互い様です口語
A: Merry Christmas! B: Same to you!（A: メリークリスマス！ B: そちらこそ！）
🔍 相手からの挨拶・祝福の言葉、時には皮肉や悪態をそのまま相手に返すときの決まり文句。
📝 好意的な言葉だけでなく、相手の暴言への皮肉交じりの切り返し（"Get lost!" "Same to you!"）としても使われる。
same difference （口語）結局同じこと、大差ない口語（くだけた）
A: It's a cottage, not a house. B: Same difference.（A: それはコテージであって家じゃないよ。 B: 同じようなもんでしょ。）
🔍 文字どおりには矛盾する表現（同じなのに違いがある）だが、相手の細かい訂正・区別を「どうでもいい、実質同じだ」と軽くいなすときに使う。
📝 相手の訂正を軽視するニュアンスを含むため、使う相手や場面に注意。フォーマルな文章では使わない。
