key

名詞・形容詞・動詞米 /kiː/ ／ 英 /kiː/key（1音節）CEFR A1

🗣 1音節語。長母音 /iː/ を「キー」のように伸ばして発音する。文字K（/keɪ/、二重母音で「ケイ」に近い）とは音もつづりも別物なので混同しないこと

  1. 【名詞】鍵、キー（ドア・南京錠などを開ける道具）
  2. 【名詞】（問題解決の）鍵、手がかり（The key to success is hard work. 成功の鍵は努力だ）
  3. 【名詞】（ピアノ・パソコンなどの）キー、鍵盤（Press the space key. スペースキーを押して）
  4. 【名詞】（音楽の）調、キー（in the key of C Cの調で）
  5. 【形容詞】重要な、鍵となる（a key issue 重要な問題）
  6. 【動詞】（データを）キーボードで打ち込む、入力する（key in the numbers 数字を入力する）

語形変化

形種類発音備考
keys複数形（名詞）／三人称単数現在形（動詞）/kiːz/語末は有声音なので -s は /z/ と発音する（「キーズ」であって「キース」ではない）
keyed過去形・過去分詞（動詞）/kiːd/規則変化。key は母音字+yで終わるため、study→studied のように y を i に変えず、そのまま -ed を付ける（keyed が正しく、keied は誤り）。語末は有声音なので /d/
keyinging形（動詞）/ˈkiːɪŋ/規則変化。同様に y をそのまま残して -ing を付ける（keying）

語法マーク

  • 名詞としては可算名詞。複数形は keys
  • 形容詞の key は限定用法（a key issue）が基本だが、近年は叙述用法（Success is key.）も一般的
  • 形容詞の key は比較級・最上級を作らない。程度を強めたい場合は crucial, vital, essential など別の語に言い換える
  • 動詞用法は他動詞。key in（打ち込む）、be keyed to（〜に連動する）の形で使われることが多く、単独で『鍵を使って開ける』の意味では使わない
  • 他品詞での用法: 形容詞用法もある。名詞の直前に置いて「重要な、鍵となる」を表すのが基本（限定用法）だが、Timing is key.（タイミングが重要だ）のように be動詞の後に置く叙述用法でも使われる。比較級・最上級は作らない（more key や keyest は誤り）。動詞用法は「（データなどを）キーボードで入力する」（key in the numbers 数字を入力する）や、受動態 be keyed to 〜「〜に連動している」の形でも使われる（Salaries are keyed to inflation. 給与は物価に連動している）。

文型・頻出パターン

  • key + 名詞（形容詞用法）重要な〜、鍵となる〜
    a key role重要な役割
    a key factor重要な要因
    a key player中心人物、キーパーソン
  • the key to + 名詞／動名詞〜の鍵、〜するための重要な要素
    the key to success成功の鍵
    the key to solving this problemこの問題を解決する鍵
  • key + O + in（句動詞の分離型）〜をキーボードで打ち込む
    key the number inその番号を打ち込む
    key it inそれを打ち込む
  • be keyed to + 名詞〜に連動している、〜と結び付けられている
    Salaries are keyed to the cost of living.給与は生活費に連動している。
  • in the key of + 音名〜調で（音楽の調性）
    a song in the key of Cハ長調の曲

他のNGSL語への置き換え

key → hidden
元: She has kept a spare key under the doormat for years.
後: She has hidden a spare key under the doormat for years.
訳: 彼女は何年も予備の鍵をドアマットの下に隠している。
📝 kept は「（そのまま）保管している」という中立的なニュアンス。hidden は「人目につかないよう意図的に隠している」という秘匿性の強いニュアンスを加える。
key → misplace
元: If you lose your key, you'll have to call a locksmith.
後: If you misplace your key, you'll have to call a locksmith.
訳: もし鍵をどこかに置き忘れたら、鍵屋を呼ばないといけない。
📝 lose は完全に失ってしまう深刻なニュアンス。misplace は「うっかりどこかに置いてしまい、場所が分からなくなる」という一時的で軽いニュアンスで、後で見つかる可能性を含意する。
key → type
元: Please key in your password before the screen locks automatically.
後: Please type in your password before the screen locks automatically.
訳: 画面が自動的にロックされる前に、パスワードを入力してください。
📝 key in はテンキーやキーパッドでの入力を連想させるやや事務的・機械的な語感。type in はキーボードでの一般的なタイピングを指し、より日常的でカジュアルな言い方。

⚠️ 日本人の典型誤用
✕ I keyed the door to enter the room.
「鍵でドアを開けて部屋に入った」と言いたかったが、動詞 key を使うと、ネイティブには『（データを）キーボードで打ち込んだ』、あるいは俗語で『（車などを）鍵で引っ掻いた』というまったく違う意味に聞こえてしまう。
○ I unlocked the door with the key to enter the room.
私は鍵でドアを開けて部屋に入った。
I unlocked the door with the key to enter the room.
「鍵でドアを開けた」を直訳して key を『開ける』の動詞として使ってしまう誤り。動詞 key は主に『（データを）打ち込む』(key in) や、俗語で『（車などを）鍵で引っ掻く』を意味し、『鍵で開錠する』という意味では使わない。開錠は unlock、開ける動作は open ~ with a key で表す。
✕ I have two key's in my bag.
「かばんの中に鍵を2本持っている」と言いたかったが、key's と書くとネイティブには『鍵の（何かの所有物）』や『鍵は〜だ』の省略のように読め、単純な複数形として不自然に見える。
○ I have two keys in my bag.
私はかばんの中に鍵を2本持っている。
I have two keys in my bag.
アポストロフィs（key's）は所有格や is/has の短縮形に使うもので、単純な複数形にはつけない。複数形は keys（アポストロフィなし）。日本人学習者に非常に多いスペリングの誤り。
✕ This point is very key for our success.
「この点は我々の成功にとってとても重要だ」と言いたかったが、very key は英語として不自然に響き、また key for という組み合わせもネイティブには馴染みが薄く違和感を与える。
○ This point is key to our success.
この点は我々の成功にとって鍵となる。
This point is key to our success.
This is a very important point for our success.
形容詞 key は程度を強める very と一緒には使わないのが一般的（crucial や important と違い、very key は不自然）。また『〜にとって重要』の意味では key to〜が定型で、key for〜は不自然に響く。
🗾 カタカナ語とのギャップ

カタカナ「キー」は「キ・ー」の2モーラだが、英語の key は1音節の長母音 /kiː/。伸ばして発音する感覚自体は正しいが、アルファベットの文字名『K』（/keɪ/、二重母音で『ケイ』に近い）と混同しないこと。key と K は綴りも音も別物である。

🌱 Word family・語源
keyboard（名詞）キーボード、鍵盤
keyword（名詞）キーワード、重要語
keyhole（名詞）鍵穴
keynote（名詞・形容詞）基調（の）（keynote speech 基調講演）
keystone（名詞）要石、（比喩的に）要となるもの
low-key（形容詞）控えめな、地味な
語源: 古英語 cæg（鍵）に由来するが、これより前の語源は不詳（西フリジア語 kaai などとの関連が指摘されるのみで確証はない）。ラテン語・ギリシャ語由来ではなく、ゲルマン語派内で完結する固有語。
keyboard（キーボード）— key（鍵）+ board（板）で『鍵（キー）が並んだ板』が原義
keynote（基調）— key（音楽の調）+ note（音）で『曲全体を決定づける音・調』
keyhole（鍵穴）— key + hole（穴）で『鍵を差し込む穴』という直接的な合成語
💡 覚え方: 語源不詳の古い固有語 key だが、『差し込んで開ける・要となる』というイメージが keyboard（鍵が並んだ板）、keynote（曲の要となる音）、keyhole（鍵を差し込む穴）に一貫して受け継がれている。
🔗 句動詞
key in — （データなどを）キーボードで打ち込む、入力する
She keyed in her PIN at the ATM.（彼女はATMで暗証番号を打ち込んだ。）
key someone in (on something) — （人に重要な情報を）伝える、教える
Can you key me in on what happened at the meeting?（会議で何があったか教えてくれる?）
⚖️ 類義語の使い分け
crucial — crucial は述語としても限定用法としても自然で、very crucial のように程度強調もできる。key はやや口語的・実務的な響きで、very key とは言わない。
essential — essential は『それなしでは成り立たない、絶対に必要な』という必須性に焦点を当てる。key は『多くの要素の中で特に重要な・中心的な』という相対的な重要性を表し、必須性まで含意しない場合もある。
main — main は数ある中の『主な・最大の』ものを客観的に指す中立的な語。key はその要素が結果を左右するという含意が強く、より評価的・強調的なニュアンスを持つ。
💬 熟語・慣用表現 (3)
under lock and key 厳重に施錠されて、厳重に保管されて中立（ややフォーマル）
The company's most valuable documents are kept under lock and key.（その会社の最も貴重な書類は厳重に施錠保管されている。）
🔍 「錠（lock）と鍵（key）の下に」という並び表現で、単に「鍵がかかっている」以上に「簡単には触れられないほど厳重に管理・保管されている」ことを強調する。
📝 be kept/be held under lock and key のように受動的な文で使われることが多い。物理的な保管だけでなく、比喩的に情報や機密が厳重に管理されている状況にも使う。
keyed up 緊張して、興奮して、気が張り詰めて口語
She was so keyed up before the interview that she could barely eat breakfast.（彼女は面接前にとても緊張していて、朝食もほとんど喉を通らなかった。）
🔍 楽器の弦を締め上げるイメージから転じて、人の神経が張り詰めている・興奮している状態を表す。
📝 良い意味の高揚感にも、悪い意味の不安・緊張にも使われる。get keyed up（だんだん興奮・緊張してくる）の形でもよく使われる。
the keys to the kingdom （組織・システムなどへの）絶大な権限、最重要機密への完全アクセス権口語／ビジネス
Giving him admin access is like handing him the keys to the kingdom.（彼に管理者権限を与えるのは、王国の鍵を渡すようなものだ。）
🔍 「王国の鍵」という比喩から、最も重要かつ機密性の高い領域への完全な支配権・アクセス権を表す。
📝 主に情報セキュリティやビジネスの文脈で「システムやデータへの絶対的なアクセス権」を指す際に使われる比喩表現。
