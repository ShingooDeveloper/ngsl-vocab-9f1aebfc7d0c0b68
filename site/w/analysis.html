analysis

名詞 (noun)米 /əˈnæləsɪs/ ／ 英 /əˈnæləsɪs/a·nal·y·sis（4音節）CEFR B2

🗣 強勢は2音節目の nal に落ちる（/əˈnæləsɪs/）。1音節目は曖昧母音 /ə/ で弱く、3音節目 -ly- も /lə/ と弱化する。強弱がはっきりしているので「アナリシス」と均等に読まないこと

  1. 【名詞】分析、解析（物事を要素に分けて詳しく調べること）
  2. 【名詞】（データ・化学物質などの）分析結果、分析内容（The analysis showed no traces of the chemical. その分析では、その化学物質の痕跡は見つからなかった。）
  3. 【名詞】（口語）精神分析（を受けること）（She's been in analysis for two years. 彼女は2年間精神分析を受けている。）

語形変化

形種類発音備考
analyses複数形（不規則）/əˈnæləsiːz/ギリシャ語源の学術語に共通する -is → -es の不規則複数（basis→bases, crisis→crises, thesis→theses と同じパターン）。「アナリシーズ」ではなく語末は長母音 /iːz/ で「アナラスィーズ」。単複同形の analysises は誤り

語法マーク

  • 可算・不可算の両用語：抽象的な行為・分野を指すときは無冠詞の不可算名詞（analysis is time-consuming）、具体的な1件を指すときは可算（an analysis, two analyses）
  • 不規則複数形 analyses（発音は /əˈnæləsiːz/、綴りの -es に引きずられて /əˈnæləsɪz/ と読まない）
  • 動詞形は analyze/analyse。analysis 自体を動詞として活用させることはできない
  • 対象を示すときは analysis of + 名詞の形を取ることが最も多い（analysis of the data）
  • 軽動詞構文で使われることが多い：do/conduct/carry out an analysis（analysis 自体には動作性がなく、動詞と組み合わせて「分析する」という動作を表す）
  • 他品詞での用法: analysis は名詞のみで、動詞化した形（分析する）は analyze（米）/ analyse（英）という別語になる（× to analysis）。可算・不可算の両方で使われる点に注意：一般的・抽象的な「分析という行為・手法」を指すときは不可算（Data analysis is essential in this field.）。一方、具体的な「1件の分析（作業・報告書）」を指すときは可算で an analysis / analyses と冠詞・複数形を伴う（We did a thorough analysis of the results. / Several analyses were carried out.）。

文型・頻出パターン

  • analysis of + N〜についての分析（分析の対象を示す前置詞 of）
    an analysis of the dataデータの分析
    an analysis of the problem問題の分析
  • do / conduct / carry out + an analysis分析を行う（analysis 自体には動作性がないため、軽動詞と組み合わせて動作を表す）
    carry out a detailed analysis詳細な分析を行う
    do a quick analysisさっと分析する
  • 形容詞 + analysis分析の深さ・種類を表す形容詞との組み合わせ
    an in-depth analysis詳細な分析
    a comparative analysis比較分析
  • on analysis / under analysis前置詞+無冠詞の analysis で「分析してみると」「分析中で」を表す定型表現
    On analysis, the results were inconclusive.分析してみると、その結果は決定的なものではなかった。
    The samples are still under analysis.サンプルはまだ分析中だ。

他のNGSL語への置き換え

analysis → finish
元: You need to complete the analysis before the meeting on Friday.
後: You need to finish the analysis before the meeting on Friday.
訳: 金曜日の会議の前に、その分析を終わらせる必要があります。
📝 complete は「抜け漏れなく完全に仕上げる」という達成度を強調するフォーマルな響きがあり、ビジネス文書でよく好まれる。finish は単に「終える」という日常的な言い方で、仕上がりの完成度までは含意しない。
analysis → brief
元: My professor had us write a full analysis of the novel for homework.
後: My professor had us write a brief analysis of the novel for homework.
訳: 教授は私たちに、宿題としてその小説の簡単な分析を書かせた。
📝 full analysis は網羅的・詳細な分析を指すのに対し、brief analysis は要点だけをまとめた簡潔な分析。求められる分量や深さのニュアンスが変わる。
analysis → further
元: If we had more time, we could do a deeper analysis of the survey results.
後: If we had more time, we could do a further analysis of the survey results.
訳: もっと時間があれば、アンケート結果についてさらに追加の分析ができるんだけど。
📝 deeper analysis は同じデータをより深く掘り下げる分析、further analysis はすでにある分析にもう一段階、追加で手を加えるニュアンス。「これ以上何をするか」の焦点が異なる。

⚠️ 日本人の典型誤用
✕ I will analysis the data tomorrow.
「明日データを分析します」と言いたかったが、analysis は名詞なので動詞の位置に置くと英語として成立しない誤文になる。
○ I will analyze the data tomorrow.
明日データを分析します。
I will analyze the data tomorrow.
analysis は名詞のみで動詞として使えない。「分析する」という動詞が必要な場合は analyze（米）/ analyse（英）を使う。
✕ We did three analysises of the samples.
「サンプルを3回分析した」と言いたかったが、analysises という綴り・発音は英語に存在せず、不自然な形として耳障りに響く。
○ We did three analyses of the samples.
サンプルを3回分析した。
We did three analyses of the samples.
analysis の複数形はギリシャ語源に由来する不規則変化で analyses（-is → -es）。英語の規則複数のように -es を足して analysises とするのは誤り。
✕ We need to do analysis of this problem quickly.
「この問題をすぐに分析する必要がある」と言いたかったが、冠詞が抜けているため「分析というものを（抽象的に）する必要がある」という座りの悪い文に聞こえてしまう。
○ We need to do an analysis of this problem quickly.
この問題をすぐに分析する必要がある。
We need to do an analysis of this problem quickly.
「この問題について1件の具体的な分析を行う」という意味なら可算名詞として冠詞 an が必要。無冠詞の analysis は「分析という行為・分野」一般を指す抽象的な言い方になり、of this problem のように対象を特定する文脈とはかみ合わない。
🗾 カタカナ語とのギャップ

カタカナ「アナリシス」は5モーラを均等に読みがちだが、英語は4音節で強勢は2音節目の nal に集中し（/əˈnæləsɪs/）、1音節目 a- と3音節目 -ly- は弱い曖昧母音になる。また動詞形「アナライズ」（analyze）と名詞形「アナリシス」（analysis）は語形が似ているため混同しやすく、日本語話者は analysis を動詞のつもりで使ってしまう誤りが起きやすい。複数形 analyses も「アナリシーズ」ではなく語末が長母音 /iːz/ の「アナラスィーズ」に近い点に注意。

🌱 Word family・語源
analyze（動詞）分析する（米国つづり。英国式は analyse）NGSL
analyst（名詞）分析者、アナリスト（分析を専門に行う人）
analytical（形容詞）分析的な、分析による
psychoanalysis（名詞）精神分析（学）
語源: ギリシャ語 analysis「分解すること」に由来。ana-（上へ、徹底的に）+ lysis（緩めること、ばらすこと。動詞 lyein「緩める」から）が語源で、ラテン語を経て16世紀後半に英語へ入った。もともとは論理学・数学で「複雑なものを単純な要素に分解して考える」手法を指す語だった。
paralysis（麻痺）— para-（傍らに）+ lysis（緩み）＝「体の働きが緩んで利かなくなること」が原義。analysis と同じ -lysis「緩める」を共有する
catalyst（触媒）— cata-（下へ）+ lyst（< lysis「緩める」）＝反応を「緩めて」進める物質。analysis と同語根
dialysis（透析）— dia-（〜を通して）+ lysis（分離）＝血液を通して不要物を分離する処置。analysis と同じ -lysis の仲間
💡 覚え方: -lysis は「緩める、ばらばらにする」という意味の共通パーツ。ana-lysis は物事を「上から徹底的にばらして」調べること、para-lysis は体の働きが「緩んで」麻痺すること、dia-lysis は血液を「通して分離する」透析——どれも「lysis＝ばらす・緩める」でつながっている。
⚖️ 類義語の使い分け
examination — examination は「じっくり調べて見ること」全般を指す広い語で、要素に分解するニュアンスは薄い。analysis は対象を構成要素に分けて体系的に調べる、より方法論的な行為を指す。
breakdown — breakdown は analysis の結果を項目ごとに分けて示したもの（費用の内訳など）を指すことが多く、よりくだけた日常的な語。analysis は分析という行為そのものや学術的な文脈でも使われるフォーマルな語。
assessment — assessment は価値・質・程度を「評価・判定する」ことに重点があるのに対し、analysis は評価を下す前段階として対象を分解し仕組みを明らかにすることに重点がある。
💬 熟語・慣用表現 (2)
in the final analysis 突き詰めて考えると、結局のところ文語／ビジネス
In the final analysis, what matters most is whether the customers are satisfied.（突き詰めて考えると、最も重要なのは顧客が満足しているかどうかだ。）
🔍 文字通りは「最終的な分析においては」。あらゆる要素を検討し尽くした末にたどり着く結論を導入する副詞句として使われ、単なる「最後の分析」という意味では使わない。
📝 文頭に置き、それまでの議論を締めくくって結論を述べる際に使う。同義の言い方に in the last analysis がある。
analysis paralysis 分析しすぎて身動きが取れなくなること、考えすぎによる決断麻痺口語／ビジネス
The team suffered from analysis paralysis, spending weeks comparing options without ever choosing one.（そのチームは分析麻痺に陥り、何週間も選択肢を比較するばかりで何も決められなかった。）
🔍 analysis と paralysis の脚韻を利用した口語表現。選択肢や情報を分析しすぎるあまり、かえって決断できず行動が止まってしまう状態を指す比喩。
📝 意思決定論やビジネスの文脈でよく使われる。suffer from analysis paralysis の形で使われることが多い。
