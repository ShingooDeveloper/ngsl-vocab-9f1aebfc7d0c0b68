name

名詞・動詞米 /neɪm/ ／ 英 /neɪm/name（1音節）CEFR A1

🗣 1音節語。母音は二重母音 /eɪ/ で、「エ」から「イ」へなめらかに滑らせて1拍で発音する。語末の /m/ を発音し忘れず、「ネー」で止めないこと

  1. 【名詞】名前、氏名
  2. 【名詞】名声、評判（She's built quite a name for herself in the fashion industry. 彼女はファッション業界でかなりの名声を築いた。）
  3. 【動詞】（人・物に）名前をつける、名付ける（They named their daughter Emma. 彼らは娘をエマと名付けた。）
  4. 【動詞】（人・物・事を）名指しする、具体的に挙げる（Can you name three countries in South America? 南米の国を3つ挙げられますか?）

語形変化

形種類発音備考

語法マーク

  • 可算名詞。単数 a name／複数 names で使う。「名声」の意味（make a name for oneself）でも a name のように可算的に使うことが多い
  • 動詞としては他動詞。第5文型（name + O + C）で使うのが基本形（× I named. のように目的語なしでは使えない）
  • name A after B「AをBにちなんで名付ける」の after を for に置き換えるのは主にアメリカ英語の口語に限られる用法なので注意
  • first name（名）と last name／family name（姓）の対応は、日本語の「姓・名」の語順とは逆になる点に注意
  • 他品詞での用法: 動詞nameには大きく2つの用法がある。(1) 命名: name A B「AをBと名付ける」（They named the boat Lucky Star.）。(2) 列挙・指名: 「〜を(具体的に)挙げる、名指しする」（Name five countries in Europe.）。さらに「(人)を…に任命する」の意味でも使われる（She was named CEO of the company. 彼女はその会社のCEOに任命された）。この任命の意味は名詞の「名声・評判」の用法とも意味的につながっている。

文型・頻出パターン

  • name + O + C（第5文型）OをCと名付ける／指名する
    They named their son David.彼らは息子をデイビッドと名付けた。
    The committee named her chairperson.委員会は彼女を議長に指名した。
  • by the name of ~〜という名前の（やや堅い言い方で、初出の人物を紹介する時に使う）
    A woman by the name of Sarah answered the phone.サラという名の女性が電話に出た。
  • name + wh節／数量（列挙）〜を(具体的に)挙げる、名指しする
    Name any five countries in Africa.アフリカの国を5つ挙げなさい。
    She refused to name her source.彼女は情報源の名前を明かすことを拒んだ。
  • What's your name? / My name is ~名前を尋ねる・名乗る基本表現
    What's your name, please?お名前は何ですか?

他のNGSL語への置き換え

name → called
元: She has already named the puppy Max, even though she just got him yesterday.
後: She has already called the puppy Max, even though she just got him yesterday.
訳: 彼女は昨日もらったばかりなのに、その子犬をもうマックスと呼んでいる。
📝 name は正式に「名前を付ける」行為そのものを表すのに対し、call は「そう呼んでいる・呼称している」という日常的な呼び方を表す。子犬に公式な名前を与えたことを強調するなら name、単に呼んでいる習慣を述べるなら call が自然。
name → admit
元: If I had to name my biggest weakness, it would be that I talk too fast.
後: If I had to admit my biggest weakness, it would be that I talk too fast.
訳: 自分の最大の弱点を認めるとしたら、話すのが速すぎることだ。
📝 name「～を挙げる、指摘する」は客観的に列挙するニュアンスなのに対し、admit「～を認める」は本人が渋々でも事実を認めるという心理的な動きを含む。話し手の態度の違いが表れる。
name → address
元: Please print your full name clearly at the top of the form.
後: Please print your full address clearly at the top of the form.
訳: 用紙の一番上に、あなたの住所をはっきりと書いてください。
📝 name と address はどちらも書類でよく求められる個人情報だが、指す情報の種類が異なる。full name は「氏名」、full address は「住所全体」を指し、書類記入の場面でどちらを尋ねているか明確に区別する必要がある。

⚠️ 日本人の典型誤用
✕ My name is called Yamada.
「私の名前はヤマダと呼ばれています」という直訳から生まれた誤り。意味が重複しており、ネイティブには回りくどく不自然な言い方に聞こえる。
○ My name is Yamada.
私の名前はヤマダです。
My name is Yamada.
I'm called Yamada.
name という語自体に「呼び名」の意味が含まれるため、is called を続けると意味が重複してしまう。名前を名乗るときは My name is X か I'm called X のどちらか一方を使う。
✕ I was named as my grandfather.
「祖父にちなんで名付けられた」と言いたかったが、as では意味が成立せず、ネイティブには何を伝えたいのか分からない文になってしまう。
○ I was named after my grandfather.
私は祖父にちなんで名付けられた。
I was named after my grandfather.
「〜にちなんで名付けられる」は name A after B の形を取り、after を使うのが正しい。as を使うと意味が通らず、「ちなんで」というニュアンスが伝わらない。
✕ My first name is Yamada, and my last name is Taro.
日本語の感覚で「姓が先」と考え、first name に姓（ヤマダ）を、last name に名（タロウ）を入れてしまった誤り。書類上、姓名が逆に記録されてしまう。
○ My first name is Taro, and my last name is Yamada.
私の first name（名）はタロウで、last name（姓）はヤマダです。
My first name is Taro, and my last name is Yamada.
日本語は「姓・名」の順で名乗るが、英語の first name（名）と last name（姓）はその対応が逆になる。日本語の語順のまま first/last に当てはめると、姓と名を取り違えてしまう。
🗾 カタカナ語とのギャップ

カタカナ「ネーム」は「ネ・エ・ム」または長音を含む2〜3モーラで発音されがちだが、英語は1音節 /neɪm/。二重母音 eɪ を1拍でなめらかに滑らせ、伸ばさない。また日本語の「ネーム」は「名札・銘板そのもの」を指すことがあるが、英語の name はそのような実体（タグ自体）は指さず、あくまで「名前」という情報を表す語である点にも注意。

🌱 Word family・語源
nickname（名詞・動詞）あだ名；あだ名をつける
rename（動詞）改名する、名前を変える
namely（副詞）すなわち、つまり（具体例を挙げて説明する際の接続語）
surname（名詞）姓、苗字
unnamed（形容詞）名前が明かされていない、匿名の
語源: 古英語 nama に由来し、さらに遡るとゲルマン祖語 *namon、インド・ヨーロッパ祖語 *h₁nómn̥ に行き着く。ラテン語 nomen、ギリシャ語 onoma、サンスクリット語 nāman もすべて同じ祖語から枝分かれした遠い親戚で、英語の noun（名詞）や nominate（指名する）もこの系統に連なる、非常に古く広範な語族の一員。
noun（名詞）— ラテン語 nomen（名前）に由来し、name と同じ語根を共有する『名前を表す語』
nominate（指名する）— ラテン語 nominare「名前をつける」から。name の動詞用法と意味が重なる
anonymous（匿名の）— ギリシャ語 an-（無）+ onyma（名前）で「名前が無い」。onyma は name と同根
synonym（同義語）— syn-（共に）+ onyma（名前）で「同じ名前（意味）を持つ語」
💡 覚え方: 英語の name、ラテン語の nomen、ギリシャ語の onoma は全部同じ祖先から枝分かれした「名前」ファミリー。nominate（指名する）は『名前を挙げる』、anonymous（匿名の）は『名前が無い(an-)』、synonym（同義語）は『同じ名前(syn-)を持つ語』— どれもこの nomen/onyma の血筋でつながっている。
🔗 句動詞
name after — 〜にちなんで名付ける
They named the park after a former mayor.（彼らはその公園を元市長にちなんで名付けた。）
name for — 〜にちなんで名付ける（アメリカ英語で after と同義に使われる）
The hospital was named for a generous donor.（その病院は気前の良い寄付者にちなんで名付けられた。）
⚖️ 類義語の使い分け
title — title は肩書き・呼称（社会的地位や役職に伴う名称）を指すことが多く、name のような個人固有の名前とは異なる。書籍・映画の「題名」の意味でも title を使う。
label — label は物に貼る『ラベル・分類名』のニュアンスで、単純化・カテゴリー化して呼ぶ場合に使う。動詞としては『レッテルを貼る』というやや否定的な含みを持つこともある点が name と異なる。
call — call は日常的に『〜と呼ぶ』ことを表す動詞で、正式に名前を与える name の行為よりも軽く、単なる呼称・呼び方を指すことが多い（例: We call him Tom, though his name is Thomas.）。
💬 熟語・慣用表現 (6)
in the name of 〜の名において、〜を名目として中立／ややフォーマル
Countless wars have been fought in the name of freedom.（自由の名の下に、数え切れないほどの戦争が行われてきた。）
🔍 「〜の名前の中で」から「その大義・権威を掲げて」。良い目的の看板を掲げて行動する意にも、正当化の口実にする皮肉な意にも使う。
📝 in the name of God/science/progress のように大義名分を掲げる時に使う。しばしば「〜を口実にして」という皮肉な含みを伴う。
the name of the game 肝心な点、最も重要なこと口語／ビジネス
In this business, customer trust is the name of the game.（このビジネスでは、顧客からの信頼こそが最も肝心な点だ。）
🔍 「ゲームの名前」からゲームの本質・目的そのものを指す比喩。転じて「その分野で一番大事なこと」を表す。
📝 The name of the game is X. の形で「重要なのはXだ」と断言する時に使う定型表現。
make a name for oneself 名を上げる、有名になる中立
He made a name for himself as one of the best young chefs in the city.（彼はその街で最も優れた若手シェフの一人として名を上げた。）
🔍 「自分のために名前を作る」から「実力や功績によって評判・知名度を築く」。
📝 make a name for oneself as ~「〜として名を上げる」の形でよく使われる。良い意味での評判獲得を表す。
call someone names 人を(暴言・悪口で)罵る口語
The teacher told the boys to stop calling each other names.（先生は男の子たちにお互いを罵り合うのをやめるよう言った。）
🔍 文字通りには「人をいろいろな名前で呼ぶ」だが、そこから「侮辱的な言葉で相手を呼ぶ→罵る」という意味に転じた。
📝 name-calling（名詞形）は「悪口の応酬」。子ども同士のけんかだけでなく、大人の口論・中傷合戦にも使われる。
in name only 名ばかりの、名目上だけの中立
He's the manager in name only—his assistant actually runs the department.（彼は名ばかりのマネージャーで、実際は部下がその部署を仕切っている。）
🔍 「名前の中だけで」から「肩書き・名称はあっても実質は伴わない」ことを表す。
📝 肩書きと実態のギャップを指摘する時に使う。husband/wife in name only（名ばかりの夫婦）のようにも使われる。
you name it 何でも(言ってみて)、あらゆるもの口語
This store sells clothes, electronics, toys—you name it.（この店は服から電化製品、おもちゃまで、何でも売っている。）
🔍 「あなたがそれの名前を言う」から「思いつくものは何でも」という網羅性を表す決まり文句。
📝 列挙の後に添えて「その他何でも当てはまる」ことを示す。単独で相手の要望を何でも受け入れる返答としても使われる（"What do you want to eat?" "You name it."）。
