each

限定詞・代名詞・副詞米 /iːtʃ/ ／ 英 /iːtʃ/each（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。母音は長母音 /iː/ で、日本語の「イ」より明確に長く伸ばす。語末は無声破擦音 /tʃ/ で、母音を付けずに「チ」で息を止めるように終える

  1. 【限定詞】それぞれの、各〜（単数名詞の前に置き、集団の一つひとつを個別に捉える）（Each player chooses a character before the game starts. 各プレイヤーはゲーム開始前にキャラクターを選ぶ）
  2. 【代名詞】（each of ～ の形で、または単独で）それぞれ、各自（Each of the rooms has its own bathroom. 各部屋にそれぞれ専用の浴室がある）
  3. 【副詞】（数量表現の後に置いて）1つにつき、それぞれに（These pens cost a dollar each. これらのペンは1本1ドルだ）

語法マーク

  • each は文法上つねに単数扱い：each + 単数名詞 + 単数動詞（Each student has a locker. × Each students have）
  • each of + the/所有格/代名詞 + 複数名詞 の形でも、動詞は each に一致して単数のまま（Each of the boxes was checked.）
  • each other は「お互い」を表す相互代名詞句。所有格は each other's（They read each other's essays.）。one another とほぼ同義だが each other の方が口語で一般的
  • each と every の違い：each は要素を一つひとつ個別に見る視点で2つ以上に使えるが、every は3つ以上の集合を漏れなく全部という総体的視点で、原則2つには使わない
  • every とは異なり、each は of 句を取ったり単独の代名詞になったりできる（each of them / each is fine）が、every は直接複数名詞の前にしか置けず単独では使えない
  • 副詞の each は数量表現の直後や文末に置かれ、「1つにつき」を表す（two dollars each）。one each のように単独でも使う
  • 他品詞での用法: 限定詞としての each（Each student answered a different question. 生徒はそれぞれ違う質問に答えた）が基本用法だが、each of ~ の形、または文脈から明らかな場合は単独でも代名詞として使われる（Each has its own charm. それぞれに独自の魅力がある）。さらに数量表現の後に置かれる副詞用法もあり、「1つにつき」を表す（The tickets are ten dollars each. チケットは1枚10ドルだ）。相互代名詞 each other（お互い）も each の重要な派生用法で、2者・多数どちらにも使え、所有格は each other's になる（They borrowed each other's books. 彼らはお互いの本を借り合った）。

文型・頻出パターン

  • each + 単数名詞〜のそれぞれ（限定詞。集団の一つひとつを個別に捉える）
    each player各プレイヤー
    each morning毎朝
  • each of + (the/所有格/代名詞) + 複数名詞〜のそれぞれ（代名詞的用法。集団から一つひとつを取り出して述べる）
    each of us私たちのそれぞれ
    each of the rooms部屋のそれぞれ
  • each otherお互いに（相互代名詞。2者・多数どちらにも使う）
    look at each otherお互いを見る
    help each other助け合う
  • 数量表現 + each（副詞）1つにつき、それぞれ
    two dollars each1つ2ドル
    one each1人1個ずつ

他のNGSL語への置き換え

each → every
元: Each of the students has already submitted the assignment.
後: Every student has already submitted the assignment.
訳: 生徒全員がもう課題を提出済みだ。
📝 each は「一人ひとり個別に」に焦点を当てるのに対し、every は「例外なく全員」という総体的な網羅性を強調する。every は直接単数名詞を取るため、each ofの構文からstudentが単数形に変わる点にも注意。
each → the
元: Each package is carefully checked before it leaves the warehouse.
後: The package is carefully checked before it leaves the warehouse.
訳: その荷物は倉庫を出る前に念入りに検品される。
📝 each は「1つ1つすべてを個別に」検品するという全数検査のニュアンスを持つが、the に替えると特定のその1個の荷物だけを指す意味になり、網羅性が消える。
each → every
元: If each member paid a small monthly fee, the club could easily afford new equipment.
後: If every member paid a small monthly fee, the club could easily afford new equipment.
訳: 会員全員が少額の月会費を払えば、クラブは新しい備品を楽々買えるだろう。
📝 each は会員を一人ひとり個別に捉える視点だが、every に替えると「例外なく全員が」という集団全体としての網羅性が前面に出る。文構造はほぼ同じだが視点が個別から全体へ変わる。

⚠️ 日本人の典型誤用
✕ Each students have their own locker.
「生徒はそれぞれ自分のロッカーを持っている」と言いたいが、each の後に複数形を続けるのは英語として崩れており、ネイティブには文法ミスとしてすぐに違和感を持たれる。
○ Each student has their own locker.
生徒はそれぞれ自分のロッカーを持っている。
Each student has their own locker.
each の後は必ず単数名詞＋単数動詞になるルールを忘れ、日本語の「生徒たち」につられて複数形の名詞と複数扱いの動詞を使ってしまう誤り。
✕ Each of the boxes were empty.
「箱はどれも空だった」と言いたいが、of の直前の複数名詞に動詞を合わせてしまっており、each が主語であるという文法関係が崩れて聞こえる。
○ Each of the boxes was empty.
箱はどれも空だった。
Each of the boxes was empty.
each of + 複数名詞の形でも、主語はあくまで each（単数扱い）なので動詞は単数形の was を使う。直前の boxes（複数形）につられて were を選んでしまう誤り。
✕ Every of the two doors was locked.
「2つのドアはどちらも施錠されていた」と言いたいが、every は2つの物には使わず、しかも every of という形自体が英語として存在しないため、文法的に破綻して聞こえる。
○ Each of the two doors was locked.
2つのドアはどちらも施錠されていた。
Each of the two doors was locked.
every は3つ以上の集合に使い、2つのものには使わないのが原則。さらに every は of 句を直接取れず、単独の代名詞にもなれない（× every of ~ は不可）。2つのものを個別に述べる場合は each が正しい。
🗾 カタカナ語とのギャップ

カタカナ「イーチ」は音としてはかなり近いが、母音 /iː/ を日本語の短い「イ」のように短く読んでしまいがちな点に注意。しっかり長く伸ばす。また語末の /tʃ/ は「チ」という母音付きの音節ではなく、舌先で息を止めて閉じる無声の破擦音であることを意識するとより自然になる。

⚖️ 類義語の使い分け
every — each は集合の要素を一つひとつ個別に見る視点で2つ以上に使え、each of のように of 句も取れる。every は3つ以上の集合を例外なく全部という総体的視点で、of 句を直接取らず単独の代名詞にもなれない（× every of them は不可）。
either — either は2つのうち「どちらか一方」、または否定文で「どちらも（〜ない）」を表し、二者択一・選択のニュアンスが強い。each は2つ以上の要素それぞれ全部を個別に指す点で異なる。
all — all は集合をひとまとまりの全体として総体的に捉え、複数扱いになる。each は集合の中の個々の要素に一つひとつ焦点を当てて捉え、単数扱いになる点が対照的。
💬 熟語・慣用表現 (2)
each and every 一人残らず、どれも全部（強調表現）中立（強調表現）
The manager personally thanked each and every employee for their hard work.（マネージャーは社員一人ひとりに、直接その働きに感謝の言葉をかけた。）
🔍 each（それぞれ）と every（すべて）を並べた強調のための定型ペア。単純に2つの限定詞を足した意味ではなく、「例外なく全員・全部」であることを強く印象づけるための決まり文句。
📝 後ろには必ず単数名詞が続く（each and every student、× each and every students）。every だけより語気が強く、スピーチや感謝の言葉などで漏れなく全員に触れたいときによく使われる。
to each his own 人それぞれ、たで食う虫も好き好き口語
She loves country music while I prefer jazz—but to each their own.（彼女はカントリーミュージックが好きで、私はジャズ派。まあ、人それぞれだよね。）
🔍 元は There is/should be to each his own のような文から動詞が省略されて定着した諺的表現。「それぞれの人にはそれぞれのもの（好み）がある」という意味で、直訳から意味を組み立てるのは難しい定型句。
📝 his を性別中立な their に置き換えた to each their own も現代では一般的。他人の好み・趣味・意見の違いを、口出しせず寛容に受け止める場面で使う。
