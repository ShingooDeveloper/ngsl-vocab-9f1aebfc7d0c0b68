return

動詞・名詞米 /rɪˈtɜːrn/ ／ 英 /rɪˈtɜːn/re·turn（2音節）CEFR A2

🗣 第2音節 turn にアクセントが置かれる（リターン↑ではなく リ・ターン）。turn の母音 /ɜːr/ はrの音色を伴う長めの母音で、口を横に引かず舌を軽く丸める。第1音節 re は弱母音 /ɪ/ で軽く添えるだけ

  1. 【動詞】(元の場所に)戻る、帰る
  2. 【動詞】(人・物を元の持ち主・場所に)返す、返却する
  3. 【動詞】(利益・結果として)もたらす、生み出す（The stock returned a 5% profit last year. その株は昨年5%の利益を生んだ。）
  4. 【名詞】戻ること、帰還（her return to the office after maternity leave 産休明けの職場復帰）
  5. 【名詞】返却、返品（Returns must be made within 30 days. 返品は30日以内にしてください。）
  6. 【名詞】見返り、お返し（in return for your kindness ご親切のお返しに）

語形変化

形種類発音備考
returns三単現/rɪˈtɜːrnz/語末 n は有声音なので -s は /z/（「リターンズ」であって「リターンス」ではない）

returninging形/rɪˈtɜːrnɪŋ/e で終わらないので綴りはそのまま -ing を付加するだけ

語法マーク

  • 自動詞（戻る: return home, return from a trip）と他動詞（返す: return the book）の両方で使われる
  • 他動詞での基本語順は return + 物 + to + 人（× return + 人 + 物 の二重目的語は不自然。return the book to her であって return her the book とは言わない）
  • 自動詞で場所を伴うときは前置詞 to が必須（return to Tokyo であって return Tokyo ではない）
  • back の意味をすでに含むため、return back は冗長な誤用とされることが多い（common_errors参照）
  • 名詞は可算（a return, two returns）。tax return や return ticket のように複合名詞を作りやすい
  • 他品詞での用法: 名詞用法も高頻度。(1)「戻ること・帰還」: her return to work after the holidays（休暇明けの職場復帰）。(2)「返却・返品」: Returns must be made within 30 days.（entry参照）。(3) 形容詞的に名詞を修飾し「往復の」を表す: a return ticket / a return flight（片道は a one-way ticket）。(4) tax return「確定申告書」という定型表現も日常的に頻出（file a tax return 確定申告をする）。いずれも動詞の「戻る・返す」イメージから派生している。

文型・頻出パターン

  • return to + 場所／活動〜に戻る（自動詞。場所や活動の再開を表す）
    return to work仕事に復帰する
    return to normal正常な状態に戻る
  • return + 物 + to + 人〜を…に返す（他動詞の基本語順）
    return the book to the library本を図書館に返す
    return the favor恩返しをする
  • return from + 場所〜から戻る（出発点を明示する自動詞用法）
    return from vacation休暇から戻る
  • in return (for ~)（〜の）お返しに、見返りとして
    in return for your help手伝ってくれたお礼に
  • a return + 名詞（複合名詞）「往復の〜」など、名詞を修飾する形容詞的用法
    a return ticket往復切符
    a tax return確定申告書

他のNGSL語への置き換え

return → come
元: I've just returned from a business trip to Osaka.
後: I've just come from a business trip to Osaka.
訳: 出張先の大阪から今来たところです。
📝 come は単に「（話者のいる場所へ）来た」という移動そのものを述べるのに対し、return は「元いた場所（＝家・職場）へ戻る」という起点への回帰を強調する。日常会話では return の方が「帰ってきた」という含みが強い。
return → deliver
元: The rented car was returned to the agency a day late.
後: The rented car was delivered to the agency a day late.
訳: レンタカーは1日遅れて業者に届けられた。
📝 return は「借りていたもの・元々あった場所に戻す」という含みを持つのに対し、deliver は単に「運んで届ける」という中立的な移動動詞で、借りた物を戻したという文脈は失われる。
return → answer
元: Please return my call as soon as possible.
後: Please answer my call as soon as possible.
訳: できるだけ早く私の電話に出てください。
📝 return my call は「(受けた電話・伝言に対して)折り返し電話をかけ直す」という応答のコロケーションだが、answer my call は「(かかってきた電話に)出る」という意味になり、折り返しのニュアンスが消える。

  • return home（家に帰る）
  • return a favor（恩返しをする、お返しをする）

⚠️ 日本人の典型誤用
✕ I returned back home late last night.
「昨夜遅くに家に戻った」と言いたかったが、return back は英語として冗長で、ネイティブには「戻り戻った」のような回りくどい言い方に聞こえる。
○ I returned home late last night.
昨夜遅くに家に戻った。
I returned home late last night.
return にはすでに「元の場所へ戻る」という back の意味が含まれているため、return back とすると意味が重複する。come back や go back なら back が必要だが、return には不要。
✕ She will return Tokyo next Monday.
「彼女は来週の月曜に東京に戻る」と言いたかったが、to が抜けると場所を表す前置詞句として成立せず、文が不完全に聞こえてしまう。
○ She will return to Tokyo next Monday.
彼女は来週の月曜に東京に戻る予定だ。
She will return to Tokyo next Monday.
return が自動詞で「（場所）に戻る」を表すとき、visit のように目的語を直接取ることはできず、前置詞 to が必須。reach や visit との混同で to を落としてしまう誤りが多い。
✕ Please return me the documents by Friday.
「金曜までに書類を私に返してください」と言いたかったが、return me the documents は英語として不自然で、give me the documents のような別の動詞の型と混同しているように聞こえる。
○ Please return the documents to me by Friday.
金曜までに書類を私に返してください。
Please return the documents to me by Friday.
give のような二重目的語（SVOO: give + 人 + 物）の型を return にも適用してしまう誤り。return は return + 物 + to + 人 の語順を取り、人を直接目的語にはしない。
🗾 カタカナ語とのギャップ

カタカナ「リターン」は「リ・タ・ー・ン」の4モーラで、しばしば頭の「リ」に力が入るが、英語は2音節で強勢は後ろの turn 側（リターン↗ではなく リ・ターーン↘）。turn の母音 /ɜːr/ は「アー」とは異なり、舌を丸めるように後方に引いた曖昧母音で、日本語の「ア」よりこもった音になる。また語末の n はカタカナの「ン」のように母音を伴わず、舌先を歯茎につけたまま止める

🌱 Word family・語源
turn（動詞・名詞）曲がる、回る；順番、回転NGSL
overturn（動詞）（判決・決定などを）覆す、ひっくり返す
turnover（名詞）（企業の）総売上高、離職率
unreturned（形容詞）返却されていない、未返信の
語源: 古フランス語 retourner（re-「再び・戻って」+ tourner「回る、向きを変える」）に由来し、さらにラテン語 tornare「（ろくろで）回す」、その元は tornus「ろくろ」（ギリシャ語 tornos が語源）に遡る。「向きを変えて元の位置へ回り戻る」が原義。
turn（回る、曲がる）— return の後半部分そのもの。「回る・向きを変える」という核となる意味を共有する
tour（旅行、周遊）— 同じ tourner から派生。ぐるっと回って戻ってくる「周遊」のイメージ
tournament（トーナメント、勝ち抜き戦）— 元は「馬上槍試合で馬をぐるりと回す」動作を表した語で、tourner と同根
💡 覚え方: re-（再び・戻って）+ turn（回す）で「向きを変えて元の場所へ回り戻る」がreturnの核イメージ。同じ turn の仲間の tour（ぐるっと回る旅）、tournament（回り合う勝ち抜き戦）と一緒に覚えると芋づる式に定着する。
⚖️ 類義語の使い分け
go back — return より口語的で日常会話に馴染む言い方。return はやや硬く、書き言葉やアナウンス・ビジネス文脈でよく使われる。
come back — 話し手・聞き手のいる場所へ「戻ってくる」ことを強調する（視点が固定される）。return は視点に関係なく、起点への回帰全般を表せるより中立的な語。
give back — 「(物を)返す」の意味では give back の方がカジュアルで口語的。return は返品・返却手続きなど、より正式・事務的な場面で好まれる。
revert — 「元の状態・設定に戻る」という意味に特化したフォーマル・専門的な語（revert to default settings）。人が場所へ戻る意味では使えない。
💬 熟語・慣用表現 (4)
point of no return 後戻りできない地点、引き返せない段階中立
Once you sign the contract, you'll have passed the point of no return.（契約書にサインしてしまえば、もう後戻りできない地点を越えたことになる。）
🔍 直訳は「戻り(return)がない(no)地点(point)」。文字通りの意味を超えて、決断や行動が後戻り不可能になる決定的な段階を比喩的に表す定型句。
📝 元は航空用語で、燃料切れのため引き返せなくなる地点を指した。転じて人生やビジネスの重大な決断の場面でも比喩的に使われる。
many happy returns （誕生日に）おめでとう、これからも良い一年を口語（やや古風・主に英）
Many happy returns! I hope you have a wonderful year ahead.（お誕生日おめでとう！素敵な一年になりますように。）
🔍 直訳は「多くの幸せな戻り」で意味不明だが、"many happy returns of the day" の略で「この日が何度も幸せに巡ってきますように」という誕生日の祝いの決まり文句。
📝 主にイギリス英語の誕生日の挨拶表現。アメリカ英語では単に Happy birthday! が一般的で、この形はやや古風・儀礼的に響く。
law of diminishing returns 収穫逓減の法則ビジネス／中立（ややフォーマル）
After a certain point, studying more hours a day hits the law of diminishing returns.（ある程度を過ぎると、1日の勉強時間を増やしても効果は徐々に小さくなっていく。）
🔍 直訳は「減っていく戻り（利益）の法則」。ある一定点を過ぎると、投入（労力・資金など）を増やしても得られる成果の増加分が徐々に小さくなっていく、という経済学由来の定型表現。
📝 元は経済学の専門用語だが、日常会話やビジネスでも「頑張りすぎても効果が頭打ちになる」の意で比喩的に広く使われる。
return to sender 差出人へ返送（する）中立（郵便用語）／比喩的にはやや口語
The letter was stamped 'return to sender' because the address no longer existed.（その手紙は、住所がもう存在しないため「差出人へ返送」のスタンプを押された。）
🔍 文字通りには「送り主に戻す」だが、配達不能な郵便物に押すスタンプの文言として定型化しており、比喩的に「受け取りを拒否して送り返す」ことも表す。
📝 エルヴィス・プレスリーの曲名としても有名。比喩的に「（贈り物・申し出などを）突き返す」の意味で使われることもある。
