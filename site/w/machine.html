machine

名詞・動詞米 /məˈʃiːn/ ／ 英 /məˈʃiːn/ma·chine（2音節）CEFR A1

🗣 アクセントは第2音節 chine に置かれ、母音は長い /iː/ を伸ばして発音する。第1音節 ma は曖昧母音 /ə/ になり、強く「マ」と読まない。カタカナ「マシン」のように平板に読まず、後半「シーン」を強く長く発音するのがコツ

  1. 【名詞】機械、機械装置（This washing machine is over ten years old. この洗濯機は10年以上使っている）
  2. 【名詞】（コンピュータの）マシン、本体（Restart the machine and try again. マシンを再起動してもう一度試してください）
  3. 【名詞】（比喩的に）機構、組織（the party machine のように、個人の意思よりも仕組み・組織力で動く体制を指すことがある）
  4. 【動詞】機械で加工する、機械にかける（These parts are machined to a precise shape. これらの部品は精密な形状に機械加工される）

語形変化

形種類発音備考
machines複数形／三人称単数現在形/məˈʃiːnz/名詞の複数形にも動詞の三単現にも使われる同形。語末が有声音 /n/ の後なので -s は有声の /z/ で発音する
machined過去形・過去分詞/məˈʃiːnd/規則動詞。語末が有声音の後なので -ed は /d/（マシーンドであってマシーンティッドではない）。工業英語で be machined の形（受動態）で頻出
machininging形/məˈʃiːnɪŋ/語末の e を落として -ing を付ける（machineing は誤り）。動名詞として machining（機械加工）という工程名詞としても使われる

語法マーク

  • 可算名詞。単数は a/the machine、複数は machines と規則変化する
  • 動詞用法は主に受動態（be machined）で使われ、能動態（machine something）は工業英語でも比較的まれ
  • washing machine, vending machine, sewing machine, machine gun など「名詞＋machine」の複合名詞を作りやすい生産的な語
  • 比喩的に「機構・組織」を指す場合は the machine という形で使われることが多い（例: the machine of government 統治機構）
  • 他品詞での用法: 動詞用法「機械で加工する」は製造業・工業の専門的文脈に限定され、日常会話ではほぼ使われない。多くの場合 be machined という受動態で登場する（These parts are machined to size. 部品は寸法通りに機械加工される）。また名詞には比喩的用法もあり、a well-oiled machine（よく油を差した機械→よく機能する組織・体制）や the party machine（個人の意思よりも仕組み・組織力で動く政治マシーン）のように、人間味のない機械的な体制・組織を指すことがある。

文型・頻出パターン

  • operate / run a machine機械を操作する・動かす
    She operates the machine skillfully.彼女はその機械を巧みに操作する。
  • a machine breaks down機械が故障する（最頻出のコロケーション）
    The machine broke down twice last week.その機械は先週2回故障した。
  • turn on/off a machine機械の電源を入れる／切る
    Remember to turn off the machine at night.夜は機械の電源を切るのを忘れないで。
  • be machined to + 精度〜の精度・公差まで機械で加工される（工業英語の受動態パターン）
    The parts are machined to a tight tolerance.その部品は厳しい公差まで機械加工される。
  • N + machine（複合名詞）「〜機」を表す生産的な複合名詞パターン
    a sewing machineミシン
    a vending machine自動販売機

他のNGSL語への置き換え

machine → device
元: This machine was invented by a Japanese engineer in the 1980s.
後: This device was invented by a Japanese engineer in the 1980s.
訳: この装置は日本人のエンジニアによって発明された。
📝 device は machine よりも小型・単純な仕組みのものを指すことが多く、電子機器やガジェットにもよく使われる。machine は歯車やモーターなど機械的な仕組みを持つ、比較的大きなものを連想させる。
machine → appliance
元: You should turn off the machine before you clean it.
後: You should turn off the appliance before you clean it.
訳: 掃除する前に、その電化製品の電源を切っておいたほうがいいよ。
📝 appliance は家庭用電化製品（洗濯機・電子レンジなど）を指す語で、machine より対象が家庭用に限定される。machine は家庭用から工業用の大型機械まで幅広くカバーする、より一般的な語。
machine → unit
元: There's a vending machine that sells cold drinks near the station entrance.
後: There's a vending unit that sells cold drinks near the station entrance.
訳: 駅の入り口の近くに、冷たい飲み物を売る販売ユニットがある。
📝 unit は技術文書やビジネスの文脈で使われる、やや事務的・無機質な言い換え。日常会話では vending machine が圧倒的に自然で、unit を使うと不自然に響くことがある。

⚠️ 日本人の典型誤用
✕ I bought a new machine yesterday.
「昨日新しい車を買った」と言いたかったが、machine を使うと工業機械か何か大掛かりな装置を買ったように聞こえてしまう。
○ I bought a new car yesterday.
昨日、新しい車を買った。
I bought a new car yesterday.
日本語のカタカナ「マシン」は俗にレーシングカーやバイクなど「車」を指すことがあるが、英語の machine に日常会話でその意味はない。乗り物の話なら car や bike など具体的な語を使う。
✕ This car has a very powerful machine.
「この車はとてもパワフルなエンジンを積んでいる」と言いたかったが、machine を使うと「パワフルな機械（装置全般）」という漠然とした響きになり、エンジンの話だと伝わりにくい。
○ This car has a very powerful engine.
この車はとてもパワフルなエンジンを積んでいる。
This car has a very powerful engine.
engine は車を動かす原動機そのものを指す語で、machine は装置・機械全般を指す一般語。エンジンの性能について話すときは engine を使うのが自然。
✕ I need machine to finish this job faster.
「この仕事を早く終わらせるのに機械が必要だ」と言いたかったが、冠詞が抜けていて不自然かつ非文法的に響く。
○ I need a machine to finish this job faster.
この仕事を早く終わらせるのに機械が必要だ。
I need a machine to finish this job faster.
machine は可算名詞なので単数で使うときは必ず冠詞（a/the）を伴う。日本語には冠詞という概念がないため、単数可算名詞の前で抜け落ちやすい典型的な誤り。
🗾 カタカナ語とのギャップ

カタカナ「マシン」は「マ・シ・ン」の3モーラだが、英語は2音節 /məˈʃiːn/。アクセントは後ろの chine に置き、母音 /iː/ を長く伸ばす（前の ma は弱く曖昧母音）。また日本語の俗語「マシン」＝車・バイクという用法は英語の machine には無いので、乗り物を指すつもりで machine を使うと誤解を招く。

🌱 Word family・語源
machinery（名詞）機械類、機械設備（集合的に）
machinist（名詞）機械工、旋盤工など工作機械を操作する職人
machine gun（名詞）機関銃
machine-readable（形容詞）（コンピュータが）機械可読の、読み取り可能な形式の
mechanism（名詞）機構、仕組み、メカニズム
語源: 16世紀にフランス語 machine を経て英語に入った語で、そのフランス語はラテン語 machina「装置、工夫された仕掛け」に由来する。ラテン語 machina はさらにギリシャ語 mēkhanē（μηχανή）「工夫、仕掛け、からくり」に遡り、これは mēkhos「手段、方策」と同根。「知恵を働かせて作った仕掛け」というのが根っこの意味。
mechanism（機構、仕組み）— machine と同じギリシャ語 mēkhanē を語源に持つ姉妹語。ラテン語経由（machina）か、より直接ギリシャ語経由（mēkhanē→mechanism）かの経路の違いだけで、根は同じ
mechanic（整備士、機械工）— 同じギリシャ語 mēkhanē 由来。「機械を扱う人」の意味で machine と語源上つながる
mechanical（機械的な、機械の）— mechanic の形容詞形で、根は machine と共通のギリシャ語 mēkhanē
💡 覚え方: machine も mechanism も mechanic も、たどれば同じギリシャ語 mēkhanē「工夫された仕掛け」にたどり着く三兄弟。「機械（machine）を扱う仕組み（mechanism）を知っている人が整備士（mechanic）」と一続きに覚えると根っこの意味でつながる。
⚖️ 類義語の使い分け
device — device は machine よりも小型・単純な仕組みを指すことが多く、電子機器やガジェットにも広く使われる。machine は歯車やモーターなど機械的な駆動部を持つ、比較的大きな装置を連想させる。
equipment — equipment は不可算名詞で、道具・装置一式を集合的に指す。1台を数えるときは a piece of equipment のようにする必要があり、machine のように a machine と直接数えられない点が大きく異なる。
apparatus — apparatus は実験・体操・消火など特定の目的のために組み合わされた器具一式を指すことが多く、machine よりも専門的・フォーマルな響きを持つ語。
💬 熟語・慣用表現 (3)
a well-oiled machine よく機能する組織、円滑に回る仕組み中立
After months of training, the sales team now runs like a well-oiled machine.（数か月の研修を経て、営業チームは今やよく機能する組織として回っている。）
🔍 「よく油を差した機械」が転じて、各部門・メンバーが摩擦なく連携し、滑らかに機能する組織やチームを指す比喩表現。
📝 run/work like a well-oiled machine の形で動詞的に使うことが多い。組織やチームだけでなく、イベント運営やプロジェクト全体の円滑さを表すのにも使う。
a cog in the machine 大きな組織の中の歯車、取るに足らない一部品中立／やや否定的
In such a huge corporation, I sometimes feel like just a cog in the machine.（こんなに巨大な会社にいると、自分はただの歯車の一つに過ぎないと感じることがある。）
🔍 「機械の中の一つの歯車」が転じて、大きな組織やシステムの中で個人が果たす、代わりの利く小さな役割を指す。
📝 自分や他人の存在の重要性の低さを嘆く文脈でよく使われる。近い表現に a cog in the wheel がある。
work like a machine 機械のように黙々と（休みなく）働く中立／口語
He worked like a machine all week to finish the project on time.（彼はプロジェクトを期限内に終わらせるため、一週間ずっと機械のように黙々と働いた。）
🔍 「機械のように働く」から、感情を交えず、疲れも見せず、正確かつ休みなく作業をこなす様子を表す比喩表現。
📝 効率的で疲れ知らずという肯定的な意味にも、人間味がなく感情を出さないという否定的な意味にも使われる。文脈で判断する。
