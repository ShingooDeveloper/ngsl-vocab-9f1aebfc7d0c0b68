available

形容詞 (adjective)米 /əˈveɪləbəl/ ／ 英 /əˈveɪləbəl/a·vail·a·ble（4音節）CEFR A2

🗣 第2音節 vail に第一強勢（/əˈveɪləbəl/）。第1音節は弱いあいまい母音 /ə/ で、はっきり「ア」と言わないのがコツ。

  1. 利用できる、入手できる
  2. （人が）手が空いている、都合がつく
  3. （部屋・席・時間などが）空いている
  4. 販売されている、発売中で

語法マーク

  • 比較変化は more available / most available（-er/-est 型ではない）
  • 叙述用法（The room is available.）・限定用法（available seats）ともに可
  • 限定用法では名詞の後置修飾が非常に多い：rooms available / tickets available now / the best option available（『今その状態にある』という一時的な意味のときは後置が基本）
  • 人を主語にして『手が空いている・都合がつく』の意味で使える（He is available at 3.）

文型・頻出パターン

  • available to + 人誰に対して利用可能・提供されているかを示す
    The service is available to all members.そのサービスは全会員が利用できます。
  • available for + 目的・用途何のために使える・利用できるかを示す
    The room is available for meetings.その部屋は会議用に利用できます。
  • available from / at / in + 入手先どこで入手・利用できるか（入手元・販売場所）を示す
    The report is available from our website.そのレポートは当社のウェブサイトから入手できます。
    The product is available at any store.その商品はどの店舗でも購入できます。
  • make O available (to 人)O を（人に）利用できる状態にする、提供する
    We'll make the data available to the public.私たちはそのデータを一般に公開します。
  • 副詞 + available（高頻度コロケーション：readily / freely / widely / easily available）程度や入手のしやすさを表す副詞と結びつく定番の組み合わせ
    This medicine is readily available.この薬は簡単に手に入ります。
    The software is freely available.そのソフトウェアは無料で自由に入手できます。
    Smartphones are widely available.スマートフォンは広く出回っています。
    The information is easily available online.その情報はオンラインで簡単に入手できます。
  • available + to do「〜するために利用できる」と、名詞を後置修飾して用途・可能な動作を示す
    Tickets are available to purchase online.チケットはオンラインで購入できます。

他のNGSL語への置き換え

available → popular
元: The new app has been available in Japan since April.
後: The new app has been popular in Japan since April.
訳: その新しいアプリは4月から日本で人気です。
📝 available は「入手・利用できる状態」、popular は「人気がある」という評価を表す。どちらも be + 形容詞の形で現在完了の継続（has been ~ since）とそのまま相性が良い。
available → free
元: Could you tell me when the doctor will be available again?
後: Could you tell me when the doctor will be free again?
訳: 先生の手がいつまた空くか、教えていただけますか。
📝 free は「手が空いている」の口語的な言い方で、available の方がビジネスや受付ではフォーマルで丁寧。free には「無料の」の意味もあるため、料金の話と紛らわしい文脈では available が安全。
available → useful
元: We're working hard to make this data available to everyone on the team.
後: We're working hard to make this data useful to everyone on the team.
訳: このデータがチーム全員の役に立つように、頑張って作り込んでいます。
📝 make + O + 形容詞（SVOC）の構文はそのまま使える。available は「アクセスできる状態」、useful は「役立つ性質」を表すので意味が変わる。useful は to / for のどちらも取れる。

⚠️ 日本人の典型誤用
✕ Are you convenient tomorrow?
「あなたは明日、便利ですか？」のように聞こえ、ネイティブには意味が通じない不自然な文。学習者が言いたかったのは「明日はご都合いかがですか？」。
○ Are you available tomorrow?
明日はご都合いかがですか？（お手すきですか？）
Are you available tomorrow?
「ご都合はいかがですか」の「都合がいい」を convenient と直訳する最頻出の誤り。convenient は物・時間・場所にしか使えず、人を主語にできない。人の都合を尋ねるときは available（または free）を使う。
✕ Wi-Fi can be used in this hotel.
「このホテルでは Wi-Fi を使うことが可能だ」という意味は伝わるが、直訳的で硬く、案内表示としては不自然に響く。学習者が言いたかったのは「当ホテルでは Wi-Fi をご利用いただけます」。
○ Wi-Fi is available in this hotel.
当ホテルでは Wi-Fi をご利用いただけます。
Wi-Fi is available in this hotel.
「利用できる」を can be used と受動態で直訳しがちだが、英語では『提供されていて使える状態にある』を1語で表す available が自然。案内表示・アナウンスでは available が定番。
✕ Please check the available now rooms.
語順が崩れており、ネイティブには「available now rooms」の部分がぎこちなく感じられる。学習者が言いたかったのは「今空いている部屋を確認してください」。
○ Please check the rooms available now.
今空いている部屋を確認してください。
Please check the rooms available now.
日本語の「今空いている部屋」の語順につられて修飾語句ごと名詞の前に置いてしまう誤り。available が now や for 句などを伴うとき、また『一時的にその状態』を表すときは名詞の後ろに置く。
🗾 カタカナ語とのギャップ

カタカナ「アベイラブル」は ア・ベ・イ・ラ・ブ・ル の6拍で平板に発音されがちだが、英語は /əˈveɪləbəl/ の4音節。強勢は第2音節 vail に一点集中し、語頭の a- は弱い /ə/、語尾の -ble は母音をほぼ入れず /bl̩/ と発音する。「ア・ベイラボゥ」のイメージで vail だけ強く長く言うと通じやすい。

🌱 Word family・語源
availability（名詞）利用可能性、（人の）都合、空き状況
unavailable（形容詞）利用できない、手が空いていない
avail（動詞・名詞）役立つ／効用（to no avail「無駄に」）
語源: avail（役立つ）+ -able（〜できる）。avail は古フランス語 valoir を経てラテン語 valere（力がある、価値がある）に由来。つまり「役立てることができる」→「利用できる」。なお名詞 availability では強勢が /əˌveɪləˈbɪləti/ と -bil- に移動する点に注意（-ity 接尾辞の直前の音節に強勢が来る規則）。
value（価値）— 同じラテン語 valere（価値がある）から。「価値がある」→「役立つ」
valid（有効な）— valere から。「力がある」→「効力がある」
prevail（優勢である、普及する）— pre-（前に）+ vail（力がある）→「勝る」
💡 覚え方: -able は「〜できる」を作る超生産的な接尾辞（usable, drinkable, washable...）。available は「avail（役立てる）+ able」＝「役立てられる状態」と分解すれば一発。さらに availability / unavailable と接辞を付け替えるだけで語彙が3倍になる、接尾辞学習の代表例。
⚖️ 類義語の使い分け
free — 人の予定について「空いている」を表すカジュアルな言い方。Are you free tonight? は友人同士向きで、available はビジネスでも使える丁寧寄りの語。
accessible — 物理的・制度的に「アクセスできる、近づける」ことに焦点。available は「存在していて手に入る・使える」こと自体を指す。
in stock — 商品の在庫がある場合限定の表現。available は在庫に限らずサービス・人・時間にも使える上位語。
💬 熟語・慣用表現 (6)
become available 入手可能になる、（人が）手が空く中立（一般・ビジネス両用）
Tickets will become available on the website next Monday.（チケットは来週の月曜日にウェブサイトで購入できるようになります。）
🔍 「available（利用可能）」の状態に「become（なる）」＝状態変化を表す
📝 モノにも人（手が空く）にも使える。become available で「新たに手に入る状態になる」変化に焦点。
no longer available もう手に入らない、現在は利用できない中立（接客・案内でよく使う）
I'm sorry, but that model is no longer available in stores.（申し訳ございませんが、そのモデルは現在店頭ではお取り扱いがございません。）
🔍 「no longer（もはや〜ない）」＋「available」で、以前は可能だったものが今は不可の意
📝 商品・サービス・人など幅広く使える。過去との対比を含むのが特徴。
available upon request ご要望に応じて提供可能ビジネス定型（履歴書・カタログ・書面）
References are available upon request.（照会先（推薦者）はご要望に応じてお知らせいたします。）
🔍 「upon request（要求があり次第）」＝依頼があればすぐに、の定型ビジネス表現
📝 on request とも言う。履歴書末尾の決まり文句としても頻出。
make yourself available （人と会う・対応する）時間を空けておくややフォーマル（ビジネス）
Please make yourself available for a short interview on Friday.（金曜日の短い面談のために時間を空けておいてください。）
🔍 再帰代名詞で「自分を available な状態にする」＝都合をつける
📝 make myself / make himself / make ourselves available のように主語に一致した再帰代名詞（-self/-selves）を使う。him など目的格ではなく himself とする点に注意（make him available は「彼を（第三者として）都合させる」で別構文）。for + 用事 / to + 相手を続けることが多い。
commercially available 市販されている、商業的に入手可能なややフォーマル（技術・医療・製品説明）
This drug is not yet commercially available in Japan.（この薬は日本ではまだ市販されていません。）
🔍 「commercially（商業的に）」が available を修飾し「一般の市場で買える」の意

All the data used in this study is publicly available online.（本研究で用いたデータはすべてオンラインで一般に公開されています。）
