issue

名詞・動詞米 /ˈɪʃuː/ ／ 英 /ˈɪsjuː/
  1. 【名詞】（社会的・政治的な）問題、論点（a controversial issue 論争を呼んでいる問題）
  2. 【名詞】（雑誌・新聞などの）号、版（the latest issue of the magazine その雑誌の最新号）
  3. 【名詞】（切手・株式・パスポートなどの公式の）発行、発給（the issue of new passports 新しいパスポートの発行）
  4. 【動詞】（声明・警告などを）発表する、出す（issue a warning 警告を発する）
  5. 【動詞】（切符・証明書などを）発行する、交付する（issue a ticket 切符を発行する）

他のNGSL語への置き換え

issue → released
元: The company has issued a statement about the delay.
後: The company has released a statement about the delay.
訳: その会社は遅延についての声明を公開した。
📝 issue は公的機関・企業などが正式な立場から声明・警告を「発する」というフォーマルな響きを持つ。release は「（情報などを）公開する・出す」という意味で issue よりカジュアルかつ汎用的に使え、必ずしも公式性を含意しない。
issue → problem
元: The main issue that we need to discuss is the budget for next year.
後: The main problem that we need to discuss is the budget for next year.
訳: 話し合う必要がある一番のトラブルは来年度の予算だ。
📝 issue は中立的な「論点・懸案事項」を指し、必ずしも悪い事態を含意しない。problem は「困った事態・トラブル」という否定的なニュアンスが強く、issue よりも深刻さや不都合を強調する。ビジネスの議題では issue の方がフォーマルで角が立たない。
issue → process
元: Please issue the refund before the customer complains.
後: Please process the refund before the customer complains.
訳: お客様がクレームを言う前に返金の手続きをしてください。
📝 issue は「（返金・証明書などを公式に）発行する・出す」という結果に焦点を当てる語。process は「（手続き・事務処理を）進める」という作業そのものに焦点があり、実務的・事務的な響きが強い。
