instance

名詞・動詞米 /ˈɪnstəns/ ／ 英 /ˈɪnstəns/in·stance（2音節）CEFR B2

🗣 第1音節に強勢（ˈɪn）。第2音節は子音連続 -stance で、母音は曖昧母音 /ə/。「イン」を強く短く切り、「スタンス」は弱く添えるイメージ。stan-ce の an を「アン」と強く読まないこと

  1. 【名詞】実例、具体例（This is a typical instance of the problem. これはその問題の典型的な一例だ。）
  2. 【名詞】場合、事例（In this instance, we decided to wait. この場合は、待つことにした。）
  3. 【名詞】（IT・ゲーム用語）インスタンス、実行中の一つの実体（プログラムやサーバーが個別に動いている一つの実体を指す）
  4. 【動詞】（ややかたい語）実例として挙げる、例証する（Two cases can be instanced here. ここで二つの事例を挙げることができる。）

語形変化

形種類発音備考
instances複数形（名詞）／三単現（動詞）/ˈɪnstənsɪz/語末が -ce で終わるため -es を付けて /ɪz/ の音節が増える（「インスタンスィズ」）。名詞の複数形と、まれな動詞用法の三人称単数現在形は同形
instanced過去形・過去分詞（動詞）/ˈɪnstənst/規則変化。語幹が無声音 /s/ で終わるため -ed は /t/ で発音（「インスタンスト」。「インスタンスド」ではない）。動詞用法自体が学術・フォーマルな文脈に限られ、日常会話ではまれ
instancinging形（動詞）/ˈɪnstənsɪŋ/規則変化。IT・ゲーム開発分野では動名詞由来の専門用語「インスタンシング」（同一オブジェクトを複製・効率描画する技術）として名詞的にも定着している

語法マーク

  • 可算名詞。単数 an instance・複数 instances の両方で使う
  • for instance は独立した副詞句として文頭・文中に挿入され「たとえば」を表す（for example とほぼ同義だが、instance の方がやや書き言葉的でかたい）
  • in this instance は「今回・この場合に限っては」という限定的なニュアンスを持つ決まった言い回し
  • 動詞用法（〜を例証する）は非常にフォーマルで、日常会話ではほとんど使われず、学術・法律文書に限られる
  • IT・ゲーム分野では「実行中の一つの実体」を指す専門用語として使われ、multiple instances of the server のように可算名詞として複数形を取る
  • 他品詞での用法: 動詞用法（〜を例証する、例として挙げる）は非常にかたい語で、報告書・論文・法律文書などに限られる：The study instances several cases where the policy failed.（その研究はその政策が失敗した事例をいくつか挙げている）。注意点として、プログラミングで「クラスからオブジェクトを生成する」という意味の動詞は instance ではなく instantiate。instance を動詞として「インスタンス化する」の意味で使うのは英語としては誤りで、これは日本語話者・エンジニアに特有の混同（この誤用は common_errors に別途記載）。

文型・頻出パターン

  • for instance文中に挿入句として使う「例えば」（= for example）
    Some fruits, cherries for instance, are hard to grow here.果物の中には、例えばサクランボのように、ここで育てにくいものもある。
  • in this / that instance「この／その場合に限っては」という限定を表す
    In that instance, the manager made an exception.その場合に限っては、マネージャーは例外を認めた。
  • an instance of + 名詞／動名詞「〜の一例・一事例」
    an instance of poor planning計画の甘さを示す一例
    an instance of the system crashingシステムがクラッシュした一事例
  • instance + O（動詞用法・フォーマル）「〜を例として挙げる」
    The report instances several cases of fraud.その報告書はいくつかの不正の事例を挙げている。

他のNGSL語への置き換え

instance → cases
元: There have been several instances of the printer running out of paper this week.
後: There have been several cases of the printer running out of paper this week.
訳: 今週、プリンターの用紙が切れることが何度かあった。
📝 case はより一般的・中立的な「事例」で、instance よりくだけた印象。instance はややフォーマルで、報告書や説明文でよく使われる。日常会話では case の方が自然なことが多い。
instance → case
元: This is a rare instance where the two teams actually agreed on something.
後: This is a rare case where the two teams actually agreed on something.
訳: これは、両チームが実際に何かで合意した珍しいケースだ。
📝 case は instance よりも口語的で汎用性が高い。instance は「具体的な一例」というニュアンスがやや強く、フォーマルな文脈や統計的な報告で好まれる。
instance → occurrence
元: Every instance of the error is automatically logged by the system.
後: Every occurrence of the error is automatically logged by the system.
訳: エラーが発生するたびに、システムが自動的にそれを記録する。
📝 occurrence は「（同じことが）繰り返し起こること」に焦点があり、頻度・回数を強調する。instance は個々の「事例」そのものに焦点があり、必ずしも繰り返しを含意しない。技術文書ではバグの発生回数を数える文脈で occurrence がやや多く使われる。

⚠️ 日本人の典型誤用
✕ Please give me an instance for your opinion.
「あなたの意見の実例をひとつ挙げて」と言いたかったが、instance for という組み合わせは英語として不自然で、ネイティブには前置詞の選び方がおかしいと感じられる。
○ Please give me an instance of your opinion.
あなたの意見の実例をひとつ挙げてください。
Please give me an instance of your opinion.
instance は「〜の一例」という意味で使うとき、前置詞は of を取る。for と結び付けてしまうのは日本語「〜のための例」からの類推による誤り。
✕ In these instance, we need more data.
「この場合は、もっとデータが必要だ」と言いたかったが、these（複数）と instance（単数形のまま）が食い違っており、ネイティブには単純な文法ミスとして聞こえる。
○ In this instance, we need more data.
この場合は、もっとデータが必要だ。
In this instance, we need more data.
instance は可算名詞で、指示形容詞は単数なら this instance、複数なら these instances と数を一致させる必要がある。this と instance の単複を混同した誤り。
✕ I will instance a new object in the game engine.
「ゲームエンジンで新しいオブジェクトをインスタンス化する」と言いたかったが、instance を動詞に使うと英語では「例として挙げる」という別の意味になってしまい、意図が正しく伝わらない。
○ I will instantiate a new object in the game engine.
ゲームエンジンで新しいオブジェクトをインスタンス化するつもりだ。
I will instantiate a new object in the game engine.
プログラミングで「クラスからオブジェクトを生成する」の意味の動詞は instantiate。instance にも動詞用法はあるが、それは「〜を例として挙げる」という意味であり、オブジェクト生成の意味は持たない。名詞 instance から安易に動詞を作ってしまう典型的な誤り。
🗾 カタカナ語とのギャップ

カタカナ「インスタンス」は「イ・ン・ス・タ・ン・ス」の6モーラで発音されがちだが、英語は2音節 /ˈɪnstəns/ しかない。特に子音連続 nst をひとつの塊として一気に発音し、母音を挟まないこと（「インスタンス」のように各子音の後に母音を入れると通じにくい）。また第2音節の a は強く「ア」と読まず、曖昧母音 /ə/ で弱く添える

🌱 Word family・語源
instant（形容詞・名詞）即座の、即席の；瞬間
instantly（副詞）即座に、すぐに
distance（名詞）距離、隔たりNGSL
substance（名詞）物質、実体NGSL
circumstance（名詞）状況、事情NGSL
instantiate（動詞（IT用語））（プログラミングで、クラスなどから）実体を生成する、インスタンス化する
語源: ラテン語 instantia「差し迫っていること、目の前にあること」に由来。これはさらに instare（in-「上に」+ stare「立つ」）から来ており、原義は「目の前に立ちはだかっていること」。そこから「差し迫った状況」を経て、「具体的に示された一例」という現在の意味に発展した。
distance（距離）— dis-（離れて）+ stare（立つ）で「離れて立っていること」→ 隔たり
substance（実体、物質）— sub-（下に）+ stare（立つ）で「下に立って支えているもの」→ 実体
circumstance（状況）— circum-（周りに）+ stare（立つ）で「周りに立っているもの」→ 取り巻く状況
💡 覚え方: ラテン語 stare「立つ」から来た仲間たち：instance は「目の前に立つ具体例」、distance は「離れて立つ→隔たり」、substance は「下に立って支える→実体」、circumstance は「周りに立つ→状況」。どれも『立つ位置』のイメージでつながっている。
⚖️ 類義語の使い分け
case — case は日常会話でも使うより口語的で汎用的な語。instance はややフォーマルで、報告書・説明文など分析的な文脈で「具体的な一例」を示すときに好まれる。
example — example は最も一般的で使用範囲が広い基本語。instance は example よりかたく、instance of ~ happening のように「実際に起きた個別の事例」を指す構造で使われることが多い。
occurrence — occurrence は「（同じことが）起こること」自体、特に頻度や回数に焦点がある。instance は個々の「事例」そのものに焦点があり、繰り返しを必ずしも含意しない。
💬 熟語・慣用表現 (2)
in the first instance まず最初に、当面の第一段階としてビジネス／中立（主にイギリス英語のフォーマルな事務文脈）
In the first instance, employees should raise the issue with their direct manager.（まずは最初に、従業員は直属の上司にその問題を伝えるべきだ。）
🔍 「最初の事例において」ではなく、「（複数ある手順・選択肢のうち）まず最初の措置として」を表す定型句。first instance は個々の「事件」ではなく手続き上の「第一段階」を指す。
📝 「これでだめなら次の手段もあるが、まずは」というニュアンスを含み、firstly や initially に言い換えられる。日常会話よりも規則書・案内文でよく使われる。
court of first instance 第一審裁判所（最初に事件を審理する裁判所）法律用語／フォーマル
The case was first heard in a court of first instance before it was appealed.（その事件はまず第一審裁判所で審理された後、上訴された。）
🔍 instance がここでは「例」ではなく「審級（裁判の段階）」という特殊な意味で使われており、字面から意味を推測するのは難しい。上訴審（court of appeal）と対になる法律専門用語。
📝 上訴裁判所（appellate court）と対比される概念。日常会話ではまず使わず、法律文書や裁判関連のニュースに登場する専門的な熟語。in the first instance の「まず最初に」という意味が、法律分野で「第一審」という具体的な制度用語に特化した派生表現。
