argument

名詞 (noun)米 /ˈɑːrɡjəmənt/ ／ 英 /ˈɑːɡjəmənt/ar·gu·ment（3音節）CEFR A2

🗣 第1音節 /ˈɑːr/ に強勢がある。第2音節 /gjə/ は半母音 /j/ を伴う曖昧母音、第3音節 /mənt/ も曖昧母音 /ə/ で弱く短く発音する（「アーギュメント」と均等に強く読まない）

  1. 【名詞】(意見の食い違いによる)言い争い、口論（have/get into an argument with someone about something）
  2. 【名詞】(理由を挙げて示す)主張、論拠（make an argument for/against something）
  3. 【名詞】(結論に至るまでの)論の筋道、論証（follow the argument of an essay 論文の論の流れを追う）

語形変化

形種類発音備考
arguments複数形/ˈɑːrɡjəmənts/規則変化。語末が無声破裂音 /t/ で終わるため /s/ が付き、tsのクラスターとして「メンツ」に近い響きになる（「メンズ」ではない）

語法マーク

  • 可算名詞。単数は an argument、複数は arguments で、不可算的に無冠詞で使うことは基本的にない
  • argument for/against ~「~に賛成／反対する論拠」、argument about/over ~「~についての言い争い」、argument with 人「(人)との言い争い」と前置詞で意味が変わる
  • the argument that S+V の形で、that節が argument の内容を同格的に説明する（this/that 節の省略は不可）
  • have, get into, make, win, lose などの軽動詞（light verb）と組み合わさるのが典型的な使われ方
  • 他品詞での用法: argument は名詞専用の語で動詞用法はない（対応する動詞は argue）。可算名詞として単数 an argument・複数 arguments の両方で使う。「言い争い」の意味では have/get into an argument のように具体的な1回の出来事を数え、「論拠」の意味でも make an argument（1つの論拠を述べる）のように数えられる。

文型・頻出パターン

  • have/get into an argument (with 人) about/over ~(人と)~について言い争いになる
    get into an argument about moneyお金のことで言い争いになる
    have an argument with a colleague同僚と言い争いをする
  • make an argument for/against ~~に賛成/反対する論拠を示す、主張する
    make a strong argument for change変化を求める強い論拠を示す
    make an argument against the planその計画に反対する主張をする
  • the argument that S+V~という主張、論拠（同格のthat節）
    the argument that the plan is too costlyその計画は費用がかかりすぎるという主張
  • win/lose an argument言い争いに勝つ/負ける
    You'll never win this argument.この言い争いに勝つことは絶対にないよ。
  • follow the argument (of ~)(~の)論の筋道を追う、理解する
    follow the argument of the essayその論文の論の流れを追う

他のNGSL語への置き換え

argument → fight
元: My brother and I got into an argument about who should walk the dog.
後: My brother and I got into a fight about who should walk the dog.
訳: 私は兄と、犬の散歩を誰が行くかでけんかになった。
📝 argument は主に言葉によるやり取り「言い争い」を指すのに対し、fight は口論だけでなく肉体的な喧嘩も含み得るより広い語で、深刻さや激しさのニュアンスが強まる。
argument → case
元: A convincing argument was made by the defense team during the trial.
後: A convincing case was made by the defense team during the trial.
訳: 裁判の間、弁護側チームによって説得力のある主張が展開された。
📝 argument も case も「主張・論拠」を意味するが、case は複数の理由・証拠を積み重ねた「一連の主張全体」を指すことが多く、特に法律や議論の文脈で make a case for/against ~ の形が頻出する。argument はその中の個々の論点を指すことが多い。
argument → point
元: The argument that she made during the meeting eventually changed everyone's mind.
後: The point that she made during the meeting eventually changed everyone's mind.
訳: 彼女が会議中に述べた指摘は、最終的に全員の考えを変えた。
📝 argument は複数の理由を積み重ねた一連の論証を指すのに対し、point はその中の一つの「論点・指摘」を指すことが多い。make a point「要点を述べる、指摘する」は日常会話でも非常によく使われる。

⚠️ 日本人の典型誤用
✕ We had a good argument about our trip plans.
「旅行の計画についていい話し合いをした」と言いたかったが、argument は対立・言い争いを含意するため、まるでケンカをしたかのように聞こえてしまう。
○ We had a good discussion about our trip plans.
私たちは旅行の計画についていい話し合いをした。
We had a good discussion about our trip plans.
argument は本質的に対立・不一致を含意する語で、「楽しい話し合い」のような中立的な意味では使えない。日本語の「議論した」を直訳すると誤って argument を選びがちだが、ネイティブには「ケンカした」と聞こえる。
✕ I argument with my sister every day.
「毎日妹と言い争う」と言いたかったが、argumentを動詞として使ってしまっており、英語として存在しない形になっている。argumentは名詞専用で、動詞は argue。
○ I argue with my sister every day.
私は毎日妹と言い争いをする。
I argue with my sister every day.
I have an argument with my sister every day.
argument は名詞専用で、そのままの形で動詞として使うことはできない。「言い争う」という動作を言いたい場合は動詞 argue を使うか、have an argument の形にする必要がある。
✕ There is a strong argument to reduce working hours.
「労働時間を減らすことへの強い論拠がある」と言いたかったが、argument to doは英語として不自然で、目的を表すto不定詞と誤解されかねない。正しくは argument for -ing の形。
○ There is a strong argument for reducing working hours.
労働時間を減らすことには強い論拠がある。
There is a strong argument for reducing working hours.
argument の後ろに続く前置詞は for/against であり、to不定詞を続ける形は使わない。argument to do とすると「目的」を表すto不定詞と紛らわしく、不自然に響く。
🗾 カタカナ語とのギャップ

カタカナ「アーギュメント」は各音を均等な強さで読みがちだが、実際は第1音節 /ˈɑːr/ にだけ強勢が来て、r も必ず発音される(米音はrhotic)。また語尾 -ment の母音は曖昧母音 /ə/ なので、「メント」とはっきり読まず「マント」寄りの弱く短い音になる。

🌱 Word family・語源
argue（動詞）主張する、言い争うNGSL
arguable（形容詞）議論の余地がある
arguably（副詞）おそらく、十中八九（議論の余地はあるにせよ）
argumentative（形容詞）議論好きな、けんか腰の
語源: ラテン語 argumentum「証拠、論拠」に由来。動詞 arguere「明らかにする、証明する、非難する」の派生名詞で、さらに遡ると「輝く、白い」を意味する印欧語根に行き着くとされ、ラテン語 argentum「銀」と同語源。14世紀に古フランス語を経て英語に入った。
argue（主張する、言い争う）— argument と同じラテン語 arguere に由来する動詞形
argumentative（議論好きな）— argument + -ative（形容詞化）で「議論を好む性質の」
arguably（おそらく）— argue の形容詞形 arguable に -ly が付いた副詞。「議論の余地はあるが（おそらく）」の意味で使われる
💡 覚え方: argument はラテン語 arguere「明らかにする」から。銀 argentum と同じ語根で「曇りなく明るく示す」イメージ。argue（主張する）からargument（主張・論拠）が生まれ、そこから「主張と主張のぶつかり合い」→「言い争い」へ意味が広がったと覚える。
⚖️ 類義語の使い分け
quarrel — quarrel は感情的な対立・仲たがいに重点があり、しばしば個人的な関係のもつれを指す口語的な語。argument より怒りや気まずさのニュアンスが強い。
dispute — dispute はより公式で長期化する対立を指し、法律・労使関係・国家間の争いなど公的な文脈で使われることが多い。argument は個人間の一時的な言い争いにも使える点でより日常的。
case — case は複数の論拠・証拠を積み重ねた「一連の主張」全体を指し、特にフォーマルな議論や裁判の文脈で make a case for/against の形が典型。argument はその中の個々の論点にも、主張全体にも使える点でより幅広い。
💬 熟語・慣用表現 (2)
for the sake of argument 議論のために(仮に)、話を進めるために中立
Let's say, for the sake of argument, that the new policy actually works — what would happen next?（仮の話として、その新しい政策がうまくいったとしよう。次はどうなるだろうか?）
🔍 「argument（議論）の sake（ため）に」だが、実際の議論を有利にするためではなく、「本気で信じているわけではないが、話を先に進めるために仮にそう仮定する」という前提の導入に使う定型句。sake の通常の「利益・目的」という意味からは仮定法的な用法は予測しにくい。
📝 自分の本当の意見でなくても、議論を進めるために仮の前提を置くときに使う。文頭や let's say の後に挿入されることが多い。
no argument here (それには)異論はない、その通り口語
"This place has the best coffee in town." "No argument here."（「この店のコーヒーは街で一番だよ」「それは異論なし」）
🔍 字義通りには「ここには議論がない」だが、相手の発言に対して即座に全面同意を示す口語的な決まり文句として使われる。「議論するまでもなく賛成」というニュアンスは字面だけでは予測しにくい。
📝 単独の相槌として使われる。you'll get no argument from me や no argument there もほぼ同義の言い回し。
