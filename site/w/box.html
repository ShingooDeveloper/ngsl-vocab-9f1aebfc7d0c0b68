box

名詞・動詞米 /bɑːks/ ／ 英 /bɒks/box（1音節）CEFR A1

🗣 1音節語。母音は米音では /ɑː/ で「アー」に近い低めの開いた音（英音の /ɒ/ とは異なり唇を丸めすぎない）。語末は無声の子音連続 /ks/ で、母音を添えて「クス」と伸ばさない

  1. 【名詞】箱、ボックス（an empty cardboard box 空の段ボール箱）
  2. 【名詞】(用紙などの)四角い欄、枠（Tick the box that applies to you. 当てはまる欄にチェックを入れてください。）
  3. 【動詞】～を箱に詰める、梱包する（box up the dishes for the move 引っ越しのために食器を箱に詰める）
  4. 【動詞】ボクシングをする、拳で戦う（He has boxed professionally for ten years. 彼はプロとして10年間ボクシングをしている。）

語形変化

形種類発音備考
boxes複数形・三単現/ˈbɑːksɪz/語末が -x なので -es を付け、1音節増える（「バックス」ではなく「バックスィズ」）
boxed過去形・過去分詞（規則変化）/bɑːkst/語末 /ks/ は無声音なので -ed は /t/ で発音（「バックスト」。「バックスド」は誤り）
boxinging形/ˈbɑːksɪŋ/x で終わる語なので子音字を重ねずそのまま -ing を付ける。名詞として「ボクシング（競技）」の意味でも使われる

語法マーク

  • 名詞のboxは常に可算名詞。単数では必ず a/the などの限定詞を伴う（× I need box. ○ I need a box.）
  • 動詞box（梱包する）は他動詞。目的語を伴い、実際にはbox upの形で使われることが多い（box the dishes / box up the dishes）
  • 動詞box（ボクシングをする）は自動詞的に使われ、対戦相手はbox against/with 〜のように前置詞を介して示す。box someoneのように直接目的語を取って「殴る」の意味にはならない
  • 分離可能句動詞box upは代名詞が目的語のとき必ず動詞と副詞辞の間に挟む（○ box it up × box up it）
  • 他品詞での用法: 口語では the box が「テレビ」を指すことがある（主にイギリス英語: What's on the box tonight? 今夜テレビで何をやってる?）。またIT分野の俗語で「box」は「コンピュータ・サーバー1台」を指す（a Linux box サーバー機1台）。いずれもカタカナ「ボックス」の日常的な「箱」のイメージとはずれるので注意。

文型・頻出パターン

  • box + O〜を箱に入れる・梱包する（他動詞の基本用法）
    box the dishes食器を箱に詰める
    box a gift贈り物を箱に入れる
  • box up + O ／ box + O + up（分離可能句動詞）〜をきちんと箱詰めする
    box up the old clothes古着を箱に詰める
    box them upそれらを箱に詰める
  • box + against/with + 人（スポーツ用法）〜を相手にボクシングをする
    box against a former champion元チャンピオンを相手にボクシングをする
  • a box of + 名詞箱一杯の〜、ひと箱の〜
    a box of chocolatesチョコレート1箱
    a box of tissuesティッシュ1箱
  • check/tick the box（用紙などの）欄にチェックを入れる
    tick the box next to your name名前の横の欄にチェックを入れる

他のNGSL語への置き換え

box → package
元: The box that arrived this morning contains all the cables we ordered.
後: The package that arrived this morning contains all the cables we ordered.
訳: 今朝届いた荷物には、注文したケーブルが全部入っている。
📝 package は中身の入れ物の形を問わない「荷物・小包」全般を指す包括的な語。box は直方体の容器という具体的な形状を含意するのに対し、package は袋や封筒でも使える点が異なる。
box → wrap
元: Please box up the leftovers before you leave the restaurant.
後: Please wrap up the leftovers before you leave the restaurant.
訳: お店を出る前に、残り物を包んでください。
📝 wrap up は紙やホイルなどで「包む」ことを表し、必ずしも箱という容器を使うとは限らない。box up が容器に詰める動作を指すのに対し、wrap up はより軽い包装一般を表す。
box → fight
元: The coach had him box against a much stronger opponent to test his defense.
後: The coach had him fight against a much stronger opponent to test his defense.
訳: コーチは彼の守りを試すため、もっと強い相手と戦わせた。
📝 fight は殴り合い・格闘全般を指す一般的な語で、ボクシングという特定の競技ルールは含意しない。box がルールに基づく「ボクシングをする」動作に限定されるのに対し、fight はより広く「戦う」を意味する。

⚠️ 日本人の典型誤用
✕ Please box up it before you leave.
「出る前にそれを箱に詰めてください」と言いたかったが、box up it は英語として不自然で、ネイティブには語順が崩れて聞こえる。
○ Please box it up before you leave.
出る前にそれを箱に詰めてください。
Please box it up before you leave.
box upは分離可能句動詞で、目的語が代名詞（it, them など）のときは必ず動詞と副詞辞の間に挟まなければならない。名詞の場合はbox up the leftoversのように後ろに置いても間に挟んでも良いが、代名詞では間に挟む形しか許されない。
✕ Put the old magazines in a case.
「古い雑誌を箱に入れて」と言いたかったが、caseだと専用の型に合わせた入れ物を思い浮かべさせてしまい、単純な段ボール箱のイメージとはずれて伝わる。
○ Put the old magazines in a box.
古い雑誌を箱に入れてください。
Put the old magazines in a box.
日本語の「ケース」につられて何にでもcaseを使ってしまう誤り。caseは中身の形に合わせて作られた保護用の入れ物（メガネケース、ギターケースなど）を指し、段ボールのような単純な直方体の容器にはboxを使う。
✕ He boxed me in the face.
「彼に顔を殴られた」と言いたかったが、boxを使うと（ルールのある）ボクシングの試合をしたかのように聞こえてしまい、日常の暴力の描写としては不自然。
○ He punched me in the face.
彼に顔を殴られた。
He punched me in the face.
動詞boxは競技としての「ボクシングをする」を意味し、日常的な「一発殴る」という意味では使わない。日常の一撃にはpunchやhitを使うのが自然。
🗾 カタカナ語とのギャップ

カタカナ「ボックス」は「ボ・ッ・ク・ス」の4モーラだが、英語boxは1音節。母音は丸めの弱い /ɑː/（米音）で、カタカナの「オ」より口を横に開いた「ア」に近い。語末の /ks/ にも母音を足さず、「クス」と伸ばさずに短く止める。

🌱 Word family・語源
mailbox（名詞）郵便受け、メールボックス
toolbox（名詞）工具箱；(比喩的に)手段一式
sandbox（名詞）砂場；(IT)隔離された試験環境
boxer（名詞）ボクサー；(複数形で)トランクス型の下着
unboxing（名詞）（新品を）箱から開封すること（動画コンテンツの一種として近年定着）
語源: 容器のboxは古英語 box を経て、後期ラテン語 buxis（つげの木で作った小箱）、さらにギリシャ語 pyxis（つげの木 pyxos 製の箱）に遡る。一方、「ボクシングをする」の動詞boxは中英語期に現れた別語源で、由来ははっきりせず「殴る」を意味する語（中世低地ドイツ語などとの関連が指摘される）から来たとされ、容器のboxとは元来無関係の同形異義語。
boxwood（つげの木、ツゲ材）— boxはもともと「つげの木（buxus）」を指す語で、その硬い木材で小箱を作ったことから「箱」の意味が生まれた
pyx（聖体容器（聖餐用の小箱））— ギリシャ語 pyxis に遡る同語源の語で、boxと同じ「つげの木の箱」という語根を共有する
💡 覚え方: 「箱」のboxは元々「つげの木」を指し、その木で作った小箱から意味が広がった語。一方「ボクシングをする」のboxはたまたま綴りが同じになっただけの別語源の同形異義語—つげの木の箱とグローブの拳は語源的には無関係、と覚えておくと混同しにくい。

⚖️ 類義語の使い分け
carton — carton は紙・厚紙製の軽い箱（牛乳パックや卵パックなど）を指すことが多く、boxより小型・軽量な入れ物を連想させる。
case — case は中身の形に合わせて作られた保護用の入れ物（メガネケース、ギターケースなど）を指すことが多く、単純な直方体の容器を指すboxとはニュアンスが異なる。
crate — crate は木製やプラスチック製の頑丈なすのこ状の運搬用の箱（果物や瓶の輸送に使う枠組み箱）を指し、boxより業務用・大型のイメージが強い。
💬 熟語・慣用表現 (7)
think outside the box 既成概念にとらわれずに考える、型にはまらない発想をするビジネス／中立
To solve this problem, we need someone who can think outside the box.（この問題を解決するには、既成概念にとらわれずに考えられる人が必要だ。）
🔍 「箱の外で考える」という比喩表現。ここでのboxは慣習・固定観念という見えない枠組みを指し、その枠を出て自由に発想することを意味する。
📝 形容詞的にハイフンでつないでan outside-the-box solution（型破りな解決策）のようにも使う。ビジネス・教育の文脈で頻出。
box office (映画・演劇の)興行収入、興行成績；切符売り場中立
The movie was a huge box office success, earning millions in its first weekend.（その映画は大ヒットとなり、公開初週で数百万ドルを稼いだ。）
🔍 劇場の「箱型の窓口」がチケット売り場を指すようになり、そこから「興行収入・興行成績」全体を指す意味に広がった。
📝 box-office hit（大ヒット作）のように形容詞的にハイフンでつないで使うことが多い。単なる「切符売り場」よりも「興行収入・人気」の意味で使われる頻度の方が高い。
black box 内部の仕組みが分からない装置・システム、ブラックボックス中立／専門
For most users, the algorithm is a black box—they see the results but not how it works.（たいていの利用者にとって、そのアルゴリズムはブラックボックスだ。結果は見えても、仕組みは分からない。）
🔍 「黒い箱」から転じて、内部構造を見ることも理解することもできず、入力と出力の関係だけが分かる装置やシステムを指す比喩。航空機の「フライトレコーダー」の意味でも使われる。
📝 実際のフライトレコーダーはオレンジ色に塗られていることが多いが、慣用的にblack boxと呼ばれ続けている。ITや工学分野で「内部が不透明なシステム」の比喩として非常によく使われる。
box someone's ears (人)の耳のあたりを平手で叩く、げんこつを食らわせる口語（やや古風）
His grandmother threatened to box his ears if he didn't behave.（行儀よくしないと耳をひっぱたくよ、と祖母は彼を脅した。）
🔍 「箱に入れる」ではなく、boxが古くから持つ「(拳や平手で)打つ」という動詞の意味を残した表現。現代では耳を叩くという体罰的行為を表す定型句としてのみ生き残っている。
📝 現代英語ではやや古風・ユーモラスな響きを持ち、実際の体罰よりも冗談めかした脅し文句として使われることが多い。
tick all the boxes すべての条件・要件を満たす、申し分ないビジネス／中立
This candidate ticks all the boxes we were looking for in a new manager.（この候補者は、新しいマネージャーに求めていた条件をすべて満たしている。）
🔍 採用や購入の判断基準を並べた用紙のboxに一つずつチェックを入れていく様子から、「求められる条件をすべて満たしている」ことを比喩的に表す。
📝 主に英式で使われ、アメリカ英語では check all the boxes が同義でよく使われる。単に用紙の欄にチェックを入れる文字通りの行為（check a box）ではなく、比喩的に「要件を満たす」ことを指す点が異なる。
out of the box (製品などが)購入したそのままの状態で、特別な設定なしにすぐ使える中立／技術的
This printer works out of the box—you don't need to install any extra software.（このプリンターは箱から出してすぐに使え、追加のソフトウェアをインストールする必要はない。）
🔍 「箱から出したそのままの状態」から転じて、追加の調整や設定をしなくてもすぐに機能する様子を表す。think outside the boxの「型にはまらない発想」とは無関係の別の慣用表現。
📝 綴りがthink outside the boxと似ているため混同しやすいが意味は別。ハイフンでつないだout-of-the-box solution（すぐに使える解決策）の形でもよく使われる。
box clever 抜け目なく立ち回る、うまく立ち回って有利になる口語／イギリス英語
If you want to win this negotiation, you'll need to box clever.（この交渉に勝ちたいなら、抜け目なく立ち回る必要がある。）
🔍 「賢く箱に入る」ではなく、状況をうまく利用して賢く立ち回ることを表すイギリス英語のくだけた表現。
📝 主にイギリス英語で使われるくだけた表現で、アメリカ英語ではあまり用いられない。相手を出し抜くようなニュアンスを含むこともある。
