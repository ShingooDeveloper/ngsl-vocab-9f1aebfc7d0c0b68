main

形容詞・名詞米 /meɪn/ ／ 英 /meɪn/main（1音節）CEFR A2

🗣 1音節語。母音は二重母音 /eɪ/ で、「メ」から「イ」へなめらかに滑らせて1拍で発音する。語末の n は鼻音でしっかり閉じ、伸ばして2拍にしない

  1. 【形容詞】主な、最も重要な、中心となる（What's the main reason for the change? その変更の主な理由は何ですか?）
  2. 【形容詞】（食事・行事などの）メインの、中心的な（For the main course, I'll have the salmon. メインディッシュはサーモンにします。）
  3. 【名詞】（水道・ガス・電気などの）本管、幹線（A water main burst under the street. 道路の下で水道の本管が破裂した。）

語形変化

形種類発音備考
mains複数形（名詞用法）/meɪnz/名詞「本管・幹線」の複数形。語末が有声音nの後なので/z/（「メインズ」であって「メインス」ではない）。形容詞のmainには比較変化がない（non-gradable。more main / the mainest という形は存在しない）

語法マーク

  • 限定用法（attributive）が基本。名詞の直前に置く（the main reason）。叙述用法（be動詞の直後に単独で置く形）は不自然（× The reason is main. とは言わず、This is the main reason. のように名詞を伴わせる）
  • 比較変化なし（non-gradable）。more main や the mainest という形は存在しない。「より主要な」は more important などで代用する
  • 副詞ではない。「主に」は mainly を使う（main を副詞的に使うのは誤り）
  • 名詞用法では可算名詞（a main, the mains）。水道・ガス・電気の「本管」を指し、複数形 mains は元栓・供給網を集合的に表す（turn off the mains）
  • 他品詞での用法: 名詞用法もある。水道・ガス・電気などの「本管、幹線」を指す（a water main burst = 水道の本管が破裂した）。複数形 the mains は建物やエリア全体に供給される配管・配線網を集合的に指すことが多く、インフラ関連のニュースでよく使われる。

文型・頻出パターン

  • the main + 名詞最も重要な〜、主な〜（名詞を限定する基本パターン）
    the main problem主な問題
    the main goal主な目標
    the main character主人公
  • 主語 + be動詞 + the main + 名詞〜が主な…である、と述べる基本文型
    Cost is the main concern.コストが主な懸念事項だ。
  • one of the main + 複数名詞主な〜の1つ（複数ある主要な事柄の中の1つ）
    This is one of the main reasons for the delay.これが遅延の主な理由の1つだ。
  • a/the main(s)（水道・ガス・電気の）本管、供給網（名詞用法）
    turn off the mains元栓・ブレーカーを切る
    a burst water main破裂した水道の本管

他のNGSL語への置き換え

main → primary
元: The main reason for the delay has finally been explained to the passengers.
後: The primary reason for the delay has finally been explained to the passengers.
訳: 遅延の第一の理由が、ようやく乗客に説明された。
📝 primary は「一番目の、最優先の」という意味合いが強く、フォーマルで硬い響きを持つ。main はより日常的で中立的な語で、口頭でも文書でも使いやすい。
main → key
元: Focus on the main points of the report before the meeting starts.
後: Focus on the key points of the report before the meeting starts.
訳: 会議が始まる前に、報告書のキーポイントに注目してください。
📝 key は「特に重要な、決定的な」という意味合いが強く、main よりも「他の点より際立って重要」というニュアンスを持つ。ビジネス資料やプレゼンでよく使われる。
main → central
元: Heavy snow made the main highway impossible to drive on.
後: Heavy snow made the central highway impossible to drive on.
訳: 大雪で中心部を通る幹線道路が運転できない状態になった。
📝 central は「地理的に中心に位置する」という意味合いで、main の持つ「最も重要な・主要な」という重要度のニュアンスとは異なり、位置関係を表す点が使い分けのポイント。

⚠️ 日本人の典型誤用
✕ She main works from home now.
「彼女は今、主に家で仕事をしている」と言いたかったが、main を副詞のように動詞の前に置いてしまうと英語として成立せず、ネイティブには文法が崩れているように聞こえる。
○ She mainly works from home now.
彼女は今、主に家で仕事をしている。
She mainly works from home now.
main は形容詞・名詞であり副詞ではない。「主に」という副詞的な意味を表したいときは mainly を使う必要がある。
✕ The reason is main.
「それが主な理由だ」と言いたかったが、main を be動詞の後に単独で置くと不自然で、ネイティブには言い切れていない未完成の文のように聞こえる。
○ That is the main reason.
それが主な理由だ。
That is the main reason.
main は限定用法（名詞の前）が基本の形容詞で、be動詞の直後に単独で置く叙述用法では使いにくい。名詞を伴わせて the main + 名詞 の形にする必要がある。
✕ This is a very main point.
「これはとても主要な点だ」と強調したかったが、very main という組み合わせはネイティブには使わない不自然な英語に聞こえる。
○ This is one of the main points.
これは主な点の1つだ。
This is one of the main points.
main は比較変化しない非段階的（non-gradable）な形容詞で、very のような程度副詞では修飾できない。程度を強めたい場合は important などの段階的な形容詞を使うか、one of the main points のように言い換える。
🗾 カタカナ語とのギャップ

カタカナ「メイン」は「メ・イン」の2モーラで2音節のように感じられるが、英語の main は1音節。二重母音 /eɪ/ を素早く滑らせ、語末の n をしっかり鼻音で止める（「メーン」と伸ばして2拍にしない）。また、綴りの異なる mane（たてがみ）と発音がまったく同じ /meɪn/ の同音異義語なので、リスニングでは文脈で判断する必要がある。

🌱 Word family・語源
mainly（副詞）主に、大部分はNGSL
mainland（名詞）本土
mainstream（形容詞・名詞）主流（の）
mainstay（名詞）主力、大黒柱
語源: 古英語 mægen（力、強さ）に由来し、さらにゲルマン祖語の *maganą（〜する力がある）に遡る。「力を持つ→重要な、中心となる」という意味に発展して形容詞 main が生まれた。同じ語根から助動詞 may・might も生まれている。
might（力、権力）— main と同じ語根 *magan-（力がある）を共有。might は「力・権力」を表す名詞として意味が分化した。
may（〜してもよい、〜かもしれない）— 同語根の助動詞。「〜する力・可能性がある」という原義から、許可・推量を表す助動詞へ発展した。
mainly（主に）— main に副詞化の -ly を付けた直接の派生語。
💡 覚え方: main はもともと「力」を意味した語で、can/may/might と根っこが同じ。「力を持つもの→一番大事なもの」というイメージで main（主な）につながったと覚える。
⚖️ 類義語の使い分け
primary — 「一番目に来る、最優先の」という順序・優先度に焦点を当てる語。main より formal で、primary school（小学校）や primary care（プライマリケア）など決まった複合語で使われることが多い。
principal — 「最も重要な、主たる」という点で main に近いが、より formal で書き言葉的。名詞としては「校長」の意味も持つ点に注意。
major — 「重大な、規模の大きい」という重要性・規模の大きさに焦点を当てる語。main が「複数ある中で中心となるもの」を指すのに対し、major は絶対的な重大さ・大きさを表すことが多い（a major problem）。
💬 熟語・慣用表現 (4)
in the main 概して、大体においてややフォーマル／書き言葉
In the main, the new policy has been well received by employees.（概して、この新しい方針は社員に好意的に受け止められている。）
🔍 「主要な部分において」から「全体としては、大部分は」という意味の副詞的表現。for the most part とほぼ同義。
📝 文頭に置いて文全体を修飾することが多い。日常会話よりも報告書や論説文でよく使われるやや硬い表現。
with might and main 全力を尽くして、力の限り文語／やや古風
The rowers pulled with might and main to win the final race.（漕ぎ手たちは全力を尽くして最後のレースに勝とうとした。）
🔍 might（力）と main（古英語で「力」を意味した語）を並べた頭韻的な畳語表現。「持てる力すべてを使って」という強調。
📝 main はここでは古英語 mægen（力）に由来する古い名詞用法で、現代英語の形容詞 main（主な）とは別に生き残った化石表現。日常会話ではあまり使われず、文学的・修辞的な文脈で使われる。
have an eye to the main chance 自分の利益になる機会を常にうかがっている、抜け目なく損得勘定をする文語／やや古い言い方（主に英）
He's always polite to his boss, but everyone knows he has an eye to the main chance.（彼は上司にはいつも礼儀正しいが、実は抜け目なく自分の得ばかり考えているのは誰もが知っている。）
🔍 「主要な好機（the main chance）に目を向けている」から「常に自分の得になるチャンスを狙っている」という意味に転じた。main chance はもともとサイコロ賭博用語で「最も有利な目」を指した。
📝 やや否定的なニュアンスを含み、打算的・抜け目のない人物を評する際に使われる。現代口語ではやや古風に響く。
main squeeze （恋人としての）本命、彼氏・彼女口語／スラング
She introduced her main squeeze to the whole family at the barbecue.（彼女はバーベキューで、本命の彼氏を家族全員に紹介した。）
🔍 squeeze は口語で「恋人」を意味し、main squeeze で「一番大切な恋人、本命」を表すインフォーマルなスラング。
📝 アメリカ英語のくだけたスラングで、フォーマルな場面では使わない。恋人以外にも「一番頼りにしている人」の意味で使われることもある。
