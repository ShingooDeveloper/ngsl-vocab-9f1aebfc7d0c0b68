there

副詞 (adverb)米 /ðer/ ／ 英 /ðeə/there（1音節）CEFR A1

🗣 1音節語なので強勢に迷いはない。母音はrが後ろに続くr音化母音 /er/（hair, careと同じ）で、日本語の「エア」のように2拍に分けず1拍でつなげて発音する。th は有声の /ð/（舌先を上下の歯で軽く挟んで震わせる）で、無声の think の /θ/ と混同しないこと

  1. 【副詞】そこに、そこで、あそこに（場所を指す。here「ここ」との対比）
  2. 【副詞】（There is/are ～ の構文で）～がある、～がいる（存在を導入する。この用法では弱形 /ðər/ になりやすい）
  3. 【副詞】（文頭に置いて注意を引く）そこに、ほら（There goes the bus! ほら、バスが行っちゃうよ）
  4. 【副詞】（間投詞的に、慰め・宥めで）そら、ほら（There, there. よしよし、大丈夫だよ）

語法マーク

  • 存在のthere構文では文法上の主語はbe動詞の後ろの名詞で、単数・複数の一致もその名詞に従う（There is a car. / There are cars.）。thereそのものは真の主語ではなく「形式主語」
  • 場所を指す内容語のthereは強勢を持ち、文末に置かれることが多く、前置詞を伴わずそのまま副詞として使う（× live in there ではなく live there）
  • thereは前置詞の後ろで名詞のように使われることもある（from there, near there, around there, over there）が、その場合も冠詞は付けない
  • 存在構文の疑問文ではbe動詞がthereの前に出る（Is there a problem?）。これはthere自体が動くのではなく、be動詞が主語相当の位置thereより前に倒置される現象
  • 他品詞での用法: 品詞は副詞のみだが、用法によって性質の異なる3つの顔を持つ。(1) 場所を表す内容語としての there（I live there.）—— 強く発音され、ふつうの副詞として動詞を修飾する。(2) 存在文 There is/are ～ における there —— それ自体は意味を持たない「形式主語」で、文法上の主語の位置を埋めるだけの機能語（弱形 /ðər/ になりやすく、be動詞の数は there ではなく後ろに続く名詞に一致する）。(3) get there / be there for someone のようなイディオムでの there —— 具体的な場所というより「目標地点・達成すべき状態やその場面」を漠然と指す代名詞的な用法（Are we there yet? まだ着かないの？／I'll always be there for you. いつでもあなたのそばにいるよ）

文型・頻出パターン

  • There is/are + 名詞〜がある、〜がいる（新情報として存在を導入する最も基本的な構文）
    There's a message for you.あなたにメッセージが来ているよ。
    There are two ways to solve this.これを解決する方法は2つある。
  • There used to be + 名詞以前は〜があった（今はもう無い、という含み）
    There used to be a small shop on this corner.この角にはかつて小さな店があったんだけどね。
  • 副詞 + there（over there / up there / out there / down there）方向・位置を強調して組み合わせる言い方
    Look over there!あそこを見て！
    Put it down there, please.それをそこに（下に）置いてください。
  • There + goes/comes + 名詞（倒置）目の前で起きている出来事に注意を引く、実況するような言い方
    There goes my bus!あ、バスが行っちゃう！
    There comes the train.ほら、電車が来るよ。
  • There, there.相手を慰める間投詞的な決まり文句（子供や落ち込んでいる人に対して）
    There, there, it's going to be okay.よしよし、大丈夫だよ。

他のNGSL語への置き換え

there → several
元: There are a few emails that I still need to reply to before the weekend.
後: There are several emails that I still need to reply to before the weekend.
訳: 週末までに返信しないといけないメールが、まだ数通残っている。
📝 a few は「少数だがゼロではない」という控えめなニュアンス、several はそれよりやや多めの複数を指し、件数の多さを軽く強調する。存在のthere構文の主語部分の数量表現を入れ替える練習。
there → laws
元: If there were no rules, everything would fall into chaos.
後: If there were no laws, everything would fall into chaos.
訳: もし法律が何もなかったら、すべてが混乱に陥ってしまうだろう。
📝 rules は家庭・学校・ゲームなど身近な規則全般を指すのに対し、laws は国家・社会レベルの法律を指し、より公的で重大な話に聞こえる。仮定法過去の主語だけを入れ替えて話のスケールを変える例。
there → sit
元: Don't just stand there — come and help me with these boxes!
後: Don't just sit there — come and help me with these boxes!
訳: そこにただ座ってないで、来てこの箱を運ぶの手伝ってよ！
📝 stand there は「突っ立ったまま何もしない」、sit there は「座ったまま何もしない」で、怠けている・見て見ぬふりをしているニュアンスがより強い。Don't just sit there は誰かを行動へ促す決まり文句として非常によく使われる。

詳細
⚠️ 日本人の典型誤用
✕ There is three cats in the garden.
「庭に猫が3匹いる」と言いたかったが、単数・複数の一致が崩れており、ネイティブにはうっかりミスとしてかなり目立って聞こえる。
○ There are three cats in the garden.
庭に猫が3匹いる。
There are three cats in the garden.
存在のthere構文でのbe動詞は、thereではなく後ろに続く名詞の単数・複数に一致させる。threeで複数だとわかっているのにisを使ってしまう誤り。
✕ It is a nice park near my house.
「家の近くに素敵な公園があるんだ」と新情報として紹介したかったが、itを使うと「それ（既に話に出た何か）は家の近くの素敵な公園だ」のように聞こえ、話の流れとかみ合わなくなる。
○ There is a nice park near my house.
家の近くに素敵な公園があるんだ。
There is a nice park near my house.
初めて話題に出す新しい情報の存在を導入するときはit isではなくthere isを使う。itは既に話題に出ている特定のものを指すため、この文脈では不自然。
✕ I left my bag at there house.
「彼らの家に鞄を置いてきた」と言いたかったが、thereは場所を表す副詞なので「そこの家に」のような意味不明な文になってしまう。
○ I left my bag at their house.
彼らの家に鞄を置いてきた。
I left my bag at their house.
there（場所を表す副詞）とtheir（theyの所有格、〜たちの）は発音がほぼ同じため綴りを混同しやすい。所有格が必要な場面でthereと書いてしまう典型的な誤り。
🗾 カタカナ語とのギャップ

カタカナ「ゼア」は th を「ゼ」と有声のz音のように発音してしまいがちだが、実際の /ð/ は舌先を上下の歯で軽く挟んで摩擦させる音でz音とは別物。また母音部分も「ア」単独ではなく、r音化した /er/（hairやcareと同じ響き）で1拍にまとめる。さらに存在構文のthereは弱形 /ðər/ となり、カタカナの強い「ゼア」とはかなり違う軽い音になる点にも注意

🌱 Word family・語源
therefore（副詞）それゆえに、したがってNGSL
thereby（副詞）それによって
thereafter（副詞）それ以降、その後
therein（副詞）その中に、そこに（やや形式ばった言い方）
thereof（副詞）それの、その（契約書などで使う形式的な言い方）
thereupon（副詞）そこで、それに応じてすぐに
語源: 古英語の指示副詞 þǣr（「その場所に」）に由来。英語には th- で始まる「指し示す」系の語群（there, then, that, this, though, thus）と、wh- で始まる「尋ねる」系の語群（where, when, what, why）という対応関係があり、there はその th- 系の代表格。同じ -ere という語尾を共有する here（ここ）・there（そこ）・where（どこ）は「場所の三点セット」として古英語の時代からセットで発達してきた。
that（あれ、それ）— 同じth-指示詞グループ。thatは「もの」、thereは「場所」を指す指示語版
then（それから、その時）— thereの時間版。thereが場所、thenが時間を指す対の関係
where（どこ）— 疑問詞版のwh-グループ。here/there/whereは同じ-ere語尾を持つ「場所の三点セット」
💡 覚え方: th-で始まる語（there, then, that, this）は「指し示す」仲間、wh-で始まる語（where, when, what）は「尋ねる」仲間、と覚えると芋づる式に思い出せる。特にhere・there・whereは語尾-ereを共有する「場所ワード三兄弟」
⚖️ 類義語の使い分け
over there — 単独のthereよりも口語で自然に使われることが多く、少し離れた場所を漠然と指すときの決まり文句。thereだけより指し示す身振りが伴うニュアンスが強い。
yonder — 非常に古風・方言的（文学作品や一部の地域の話し言葉に残る程度）で、日常会話ではまず使わない。thereよりもさらに遠い場所を指す響きがある。
thither — 完全に廃れた古語で、「そこへ（向かって）」という移動の方向のみを表す。thereが静止位置にも移動先にも使えるのに対し、thitherは方向専用だった点が異なる。
💬 熟語・慣用表現 (6)
neither here nor there 本題とは関係ない、取るに足らない中立
Whether you liked the movie or not is neither here nor there — we still have to write the review.（その映画が好きだったかどうかは本題とは関係ない。とにかくレビューは書かないといけないんだから。）
🔍 「ここでもそこでもない」から転じて「（議論の本筋にとって）どうでもいい、無関係だ」という意味になる決まり文句。
📝 hereとthereが必ずセットで使われる固定表現で、単独では成立しない。話の脱線や無関係な意見を指摘する時に使う。
not all there 頭が少しおかしい、ぼんやりしている、正気でない口語（やや失礼になりうるため使う相手に注意）
Ever since the accident, he's seemed a bit not all there.（その事故以来、彼はどこかぼんやりしているように見える。）
🔍 「（頭の中身が）全部その場にあるわけではない」から転じて「正気を欠いている、少し変だ」という婉曲表現。
📝 本人に直接言うと侮辱的に響きやすいため、第三者について婉曲的に話す時に使われることが多いカジュアルな表現。
there you go （物を渡す時）はいどうぞ／（期待通りになった時）ほらね、その調子口語
Here's your coffee — there you go.（はい、コーヒーどうぞ。）
🔍 文字通りには「そこへあなたは行く」だが、実際には文脈依存の決まり文句で場所の移動とは無関係。
📝 物を手渡す時の定番フレーズのほか、相手が何かをうまくやり遂げた時の励まし（There you go, you got it! そう、その調子！）としても使われる。
so there だ、そうなんだ（挑発的・子供っぽく言い放つ）口語（やや子供っぽい・挑発的）
I'm not going to apologize, so there!（謝るつもりなんてないから、だ！）
🔍 文字通りの意味はなく、直前の主張を強調して「文句あるか」という反抗的な態度を示す間投詞的表現。
📝 大人が使うと幼稚・喧嘩腰に聞こえるため、親しい間柄の冗談や子供同士の言い争いでよく使われる。
be there for someone （困っている時に）そばにいる、支えになる中立
No matter what happens, I'll always be there for you.（何があっても、私はいつでもあなたのそばにいるよ。）
🔍 「その場にいる」から転じて「必要な時に精神的な支えとなる」という意味に発展した表現。
📝 物理的に居合わせるという意味よりも、感情的な支えを指すことが多い。友情や恋愛関係でよく使われる決まり文句。
get there （目標などに）たどり着く、うまくいく、最終的に成功する口語寄り／中立
It's a difficult project, but if we keep working together, we'll get there.（難しいプロジェクトだけど、力を合わせ続ければ、きっとやり遂げられるよ。）
🔍 文字通りは「そこに着く」だが、比喩的に「時間はかかっても最終的に目標を達成する」ことを表す。
📝 具体的な場所ではなく抽象的な目標やゴールを指して使うことが多く、励ましの文脈で頻出する。
