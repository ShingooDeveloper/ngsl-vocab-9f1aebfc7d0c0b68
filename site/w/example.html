example

名詞 (noun)米 /ɪɡˈzæmpl̩/ ／ 英 /ɪɡˈzɑːmpl̩/ex·am·ple（3音節）CEFR A1

🗣 強勢は2音節目 ex-AM-ple。1音節目 ex- は /ɪɡz/ と有声化する（exam, exact と同様、ex- の後に強勢母音が続くと /ks/ ではなく /ɡz/ になる）。語末の -ple は母音を伴わず、成節子音 l̩ として「プル」ではなく息を止めた p のあと舌先で l を響かせる1拍になる

  1. 【名詞】例、実例（具体的に示すもの）（Can you give me an example? 例を挙げてもらえますか?）
  2. 【名詞】手本、模範（見習うべき人や行動）（follow his example 彼を見習う）
  3. 【名詞】練習問題、例題（教材で）（Look at example 2 on page 10. 10ページの例題2を見なさい。）

語形変化

形種類発音備考
examples複数形/ɪɡˈzæmpl̩z/規則変化。語末の l̩ は有声音なので複数の -s は濁って /z/（「イグザンポーズ」であって「イグザンポース」ではない）

語法マーク

  • 可算名詞。単数では必ず冠詞を伴う（an example）。母音で始まる語なので a ではなく an
  • 「〜の例」は example of ~ が基本の型（× example for ~、× example about ~ は誤り）
  • for example は文修飾の副詞句として挿入的に使う。文中で具体例を並べる場合は such as も使われるが、for example は独立した挿入句、such as は名詞句の直後に続ける点が異なる
  • set an example（模範を示す）、follow someone's example（人を見習う）は example を目的語にとる頻出コロケーション動詞
  • 名詞+名詞で example sentence, example code のように形容詞的に使える
  • 他品詞での用法: example は名詞のみで動詞用法はない（「例証する」は別語 exemplify を使う）。特徴的なのは名詞を修飾する形容詞的用法で、example sentence（例文）、example code（サンプルコード）のように名詞の直前に置いて「見本の〜、例の〜」という意味を作れる点。また for example は文中に挿入句として使う副詞的定型表現で、単独の example とは働きが異なる。

文型・頻出パターン

  • an example of + N〜の例（最も基本的な型。前置詞は of）
    an example of good teamwork優れたチームワークの一例
    This is a good example of that.これはそのよい例だ。
  • for example例えば（文中に挿入して具体例を示す副詞句）
    Some fruits, for example apples and oranges, are rich in vitamin C.果物の中には、例えばリンゴやオレンジのようにビタミンCが豊富なものがある。
  • set an example (for ~)（〜の）手本を示す、模範となる
    Parents should set a good example for their children.親は子どもに良い手本を示すべきだ。
  • follow + 所有格 + example人を見習う、人の前例に倣う
    I decided to follow her example.私は彼女を見習うことにした。
  • N + example（名詞修飾）例の〜、見本の〜（名詞の前に置いて形容詞的に使う）
    example sentence例文
    example codeサンプルコード

他のNGSL語への置き換え

example → model
元: This method is often used as an example in textbooks.
後: This method is often used as a model in textbooks.
訳: この方法は教科書でよく模範として使われる。
📝 example は「具体例・イラストとして示すもの」を指すのに対し、model は「見習うべき理想形・基準となる手本」というニュアンスが強い。model の方が「模範性・完成度の高さ」を強調する。
example → advice
元: If I were you, I would follow her example and start saving money early.
後: If I were you, I would follow her advice and start saving money early.
訳: もし私があなたの立場なら、彼女のアドバイスに従って早くから貯金を始めるだろう。
📝 example は「行動・振る舞いを見て真似ること」を表すのに対し、advice は「言葉で与えられた助言に従うこと」を表す。follow her example は行動の模倣、follow her advice は言葉への服従という違い。
example → sample
元: The teacher had the students write an example sentence for each new word.
後: The teacher had the students write a sample sentence for each new word.
訳: 先生は生徒たちに、新しい単語ごとにサンプル文を書かせた。
📝 example sentence は「その語の典型的な使い方を示す模範的な例文」を指すのに対し、sample sentence は「一つの見本としての文」というやや中立的なニュアンス。example の方が教育的・模範性を強調する。

○ Please give me an example.

example だけを置くと「例えば」という意味にならず、文として成立していないように聞こえてしまう。

🗾 カタカナ語とのギャップ

カタカナ「イグザンプル」は各モーラに母音がついた4〜5拍に聞こえるが、実際は3音節 /ɪɡˈzæmpl̩/。強勢は2音節目「ザン」にあり、1音節目 ex- は弱く曖昧な /ɪɡ/。語末の -ple は母音を足さず、成節子音 l̩ として p の後に舌で l を響かせるだけの1拍にする（「プル」と読まない）。

🌱 Word family・語源
exemplify（動詞）例証する、〜の好例となる
exemplary（形容詞）模範的な、非常に優れた
語源: ラテン語 exemplum「取り出されたもの、見本」に由来。これは動詞 eximere（ex-「外へ」+ emere「取る、買う」）から派生した名詞で、「全体から取り出して示すもの」が原義。古フランス語 example を経て中英語に入った。
exemplify（例証する）— example + -fy（〜化する）で「例にする、実例で示す」
exemplary（模範的な）— example + -ary（〜の性質を持つ）で「例となるべき、模範の」
exempt（免除された）— 同じ eximere（外へ取る）由来。「（義務から）取り除かれた」→「免除された」
💡 覚え方: ex-（外へ）+ emere（取る・買う）が語源。「全体の中から外へ取り出したもの」が example。exemplify は「例として取り出して示す」、exemplary は「取り出して見せられるほど優れた」、exempt は「（義務の外へ）取り出されて免除された」— どれも「取り出す」イメージでつながる。
⚖️ 類義語の使い分け
instance — instance は example よりやや形式的で、特定の1件のケースを指すことが多い。for instance は for example とほぼ同義だが硬い文体で好まれる。
illustration — illustration は「理解を助けるために示す例」というニュアンスが強く、説明・図解の文脈で使われる。example より説明目的が明確。
case — case は「個別の事例・ケース」を指し、医療・法律・統計などの文脈で使われることが多い。example のように模範性は含意しない。
💬 熟語・慣用表現 (2)
make an example of someone 〜を見せしめにする、見せしめとして罰する中立
The teacher decided to make an example of the student who was caught cheating.（先生はカンニングが見つかった生徒を見せしめに罰することにした。）
🔍 「人を例に作る」ではなく、他の人への警告となるようにその人を厳しく罰するという意味。本人の罪の重さそのものより、周囲への抑止効果を狙う点が構成語からは予測しにくい。
📝 受動態にもなる（He was made an example of.）。実際に罰する重さより「他の人への警告」という意図が意味の核心。
lead by example 自ら手本を示して導く、有言実行で人を率いるビジネス／中立
A good manager leads by example rather than just giving orders.（優れたマネージャーは、ただ命令するのではなく自ら手本を示して人を導く。）
🔍 「例によって導く」ではなく、言葉で指示するのではなく自分の行動そのもので模範を示しながら人を導くという意味の定型句。
📝 collocations の set an example（模範を示す）と近いが、lead by example は「導く・率いる立場である」ことを強調する定型句として使われる。
