game

名詞・動詞米 /ɡeɪm/ ／ 英 /ɡeɪm/game（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。母音は二重母音 /eɪ/ で、「エ」から「イ」へなめらかに滑らせて1拍で発音する。語末の /m/ は口を閉じたまま鳴らす有声鼻音で、日本語の「ム」のように母音を足さない

  1. 【名詞】ゲーム、遊び（play a video game テレビゲームをする）
  2. 【名詞】試合、勝負（win the game 試合に勝つ）
  3. 【名詞】(狩猟の対象となる)獲物（hunt big game 大型の獲物を狩る）
  4. 【名詞】策略、駆け引き、魂胆（What's your game? 一体何を企んでいるんだ?）
  5. 【動詞】(制度・ルールを)不正に操作する、うまく利用する（game the system 制度の抜け穴を悪用する）
  6. 【動詞】(口語)テレビゲーム・ビデオゲームで遊ぶ（He games every night. 彼は毎晩ゲームをする。）

語法マーク

  • 『試合・勝負』『遊び』の意味では可算名詞（a game, two games）
  • 『獲物』の意味では通例不可算・集合的に使う（big game, game meat）。この用法では複数形にしない
  • 動詞 game は他動詞用法（game the system）が基本で、目的語を必ず伴う
  • 口語の自動詞用法 games/gaming（テレビゲームで遊ぶ）は informal な響きが強く、フォーマルな文章では play video games を使う方が無難
  • video game, board game, card game, game show のように名詞を前に置いた複合語を作りやすい
  • 他品詞での用法: 名詞の『獲物』の意味では、狩猟対象の動物を集合的に指す不可算用法が中心（big game「大型の獲物」、game meat「ジビエ、狩猟肉」）。この意味は数えられる『ゲーム・試合』とは切り離して覚えるとよい。動詞 game（〜the systemを不正利用する）は、ルールや制度の抜け穴をずる賢く突く、というやや否定的なニュアンスを常に伴う点に注意（単に『ゲームで遊ぶ』の意味ではない。その口語義は entry.json 記載の He games every night. のような自動詞用法）。おまけとして、口語では形容詞的に be game (for something) で『乗り気だ、いつでもやる気がある』という意味にもなる（I'm game if you are.＝あなたがやるなら私もやるよ）。これはNGSLの品詞分類（名詞・動詞）には含まれない周辺的な口語用法なので、まず名詞・動詞の基本義を優先して覚えること。

文型・頻出パターン

  • play + a/the + gameゲーム・試合をする（最頻出のコロケーション）
    play a board gameボードゲームをする
    play the away gameアウェーの試合をする
  • game + O（不正操作の動詞用法）〜（制度・ルール）を不正に操作する、うまく利用する
    game the system制度の抜け穴を悪用する
    game the rankingsランキング（の仕組み）を不正操作する
  • big/fair game大型の獲物／（比喩的に）標的にしてもよい対象
    hunt big game in Africaアフリカで大型の獲物を狩る
    Politicians are fair game for journalists.政治家は記者にとって格好の標的だ。
  • 名詞 + game（複合語）ゲームの種類・ジャンルを表す複合名詞
    video gameテレビゲーム
    card gameカードゲーム
    game showクイズ番組
  • It's a game of + 抽象名詞〜（運・忍耐など）が物を言う勝負である、という比喩表現
    Chess is a game of patience.チェスは忍耐が物を言うゲームだ。
    Poker is partly a game of luck.ポーカーはある程度は運のゲームだ。

他のNGSL語への置き換え

game → beaten
元: I've played this game so many times that I know every level by heart.
後: I've beaten this game so many times that I know every level by heart.
訳: このゲームは何度もクリアしたから、どのステージも隅々まで覚えている。
📝 played は単に「プレイした」ことを表すが、beaten は「クリアした・攻略しきった」という達成のニュアンスを加える。単なる経験ではなく、勝利・完了を強調する場合に使う。
game → postponed
元: The game was cancelled because of the storm.
後: The game was postponed because of the storm.
訳: 嵐のせいで試合は延期になった。
📝 cancelled は「中止（二度と行われない）」を意味するのに対し、postponed は「延期（後日改めて行われる）」を意味する。試合が別日に開催される見込みがあるかどうかで使い分ける重要な違い。
game → loopholes
元: Some players try to game the system by exploiting bugs.
後: Some players try to game the system by exploiting loopholes.
訳: 一部のプレイヤーは、抜け穴を悪用して制度を出し抜こうとする。
📝 bugs はソフトウェアの技術的な不具合を指すのに対し、loopholes はルール・制度そのものに存在する抜け道・欠陥を指すより一般的な語。対象がバグなのか制度上の欠陥なのかで使い分ける。

⚠️ 日本人の典型誤用
✕ Let's do a game after school.
「放課後にゲームをしよう」と言いたかったが、do a game は英語として不自然で、ネイティブには言い間違いとしてすぐ気づかれてしまう。
○ Let's play a game after school.
放課後にゲームをしよう。
Let's play a game after school.
日本語「ゲームをする」の『する』をそのまま do に置き換えてしまう誤り。英語では game（ゲーム・試合）は動詞 play と結びつくのが決まった言い方で、do a game とは言わない。
✕ I like game very much.
「ゲームが大好きだ」と一般論を言いたかったが、単数無冠詞の game は『特定の1試合』を指すように聞こえるか、文法的に不完全に響いてしまう。
○ I like games very much.
私はゲームが大好きだ。
I like games very much.
I like playing games very much.
『ゲーム』というジャンル全体を一般論として語る場合、英語では可算名詞 game を無冠詞の単数のままでは使えず、複数形 games（またはgamingという動名詞）にする必要がある。
✕ He gamed with his little brother yesterday.
「昨日、弟と一緒にゲームをして遊んだ」と言いたかったが、game という動詞をこの形で使うと不自然、あるいは『弟を相手に何か企んだ』のように誤解されかねない。
○ He played games with his little brother yesterday.
彼は昨日、弟と一緒にゲームをして遊んだ。
He played games with his little brother yesterday.
口語動詞 game は『（1人で）テレビゲームをして遊ぶ』という自動詞用法であり、game the system のような『不正操作する』の意味と混同されやすいうえ、with A（誰かと一緒に）を伴う一般的な『遊ぶ』の意味では使わない。誰かと一緒に遊ぶ場合は play games with を使う。
🗾 カタカナ語とのギャップ

カタカナ「ゲーム」は「ゲ・エ・ム」と3モーラで発音されがちだが、英語は1音節 /ɡeɪm/。母音は二重母音 /eɪ/ として「エ」から「イ」へ滑らせ1拍でつなげる。また語末の m は口を閉じたまま鳴らす鼻音で、日本語の「ム」のように母音「ウ」を足して伸ばさない。

🌱 Word family・語源
gamer（名詞）（特にテレビ）ゲームをする人、ゲーマー
gaming（名詞）ゲーム（をすること）、ゲーム業界；賭博
endgame（名詞）（チェスなどの）終盤戦；（物事の）最終局面、最終目標
gamesmanship（名詞）（ルール違反すれすれの）駆け引きの技術、狡猾な戦術
語源: 古英語 gamen（喜び、楽しみ、遊戯）に由来し、ゲルマン祖語の『共に楽しむこと』を表す語にさかのぼる。中英語期に語尾の -en が落ちて現在の game の形になった。
backgammon（バックギャモン）— back +（中英語で『ゲーム』を意味した）gammon の組み合わせで、game の古い語形が化石のように残っている
💡 覚え方: 古英語で『みんなで楽しむこと』を意味した gamen が短くなったのが game。盤双六 backgammon の後半 -gammon にその古い姿が今も残っている、とセットで覚える。
🔗 句動詞
game out — （計画・シナリオを）あらゆる可能性まで細かくシミュレーションする、机上で検討し尽くす
The team gamed out several scenarios before the launch.（チームはリリース前にいくつものシナリオを机上でシミュレーションした。）
⚖️ 類義語の使い分け
match — match は対戦相手同士の1回の勝負（特にスポーツ・格闘技）を指すことが多く、対戦色が強い。game はより広く、テレビゲームや子どもの遊びなど非対戦的なものまで含む包括的な語。
contest — contest はより形式的・公式な『競技会、コンテスト』を指し、審査や採点のニュアンスを伴うことが多い。game は日常的で幅広い活動全般を指す一般語。
exploit — 動詞の game（the systemをgameする）は『ルールの抜け穴をうまく利用する』という限定的な意味だが、exploit はより広く弱点・機会全般を利用する意味で使え、対象が制度やルールに限らない。
💬 熟語・慣用表現 (8)
game over 万事休す、（試みが）もう終わりだ、手遅れだ口語
Once the police found the evidence, it was game over for the gang.（警察が証拠を見つけた時点で、その一味はもう万事休すだった。）
🔍 元はビデオゲームの画面表示「ゲームオーバー」だが、比喩的に「これでおしまい、望みは絶たれた」という意味で日常会話にも定着している。
📝 感嘆的に単独で "Game over!" と言うことも多い。文字通りのゲーム終了の意味と、比喩的な「終わり」の意味の両方で使われる。
be game (for something) 乗り気である、（何か新しいこと・難しいことにも）やる気がある口語
I'm not sure if it'll work, but I'm game to try.（うまくいくかは分からないけど、やってみる気はあるよ。）
🔍 名詞game「勝負」から派生した形容詞的用法。「勝負に応じる用意がある」というニュアンスから「喜んで挑戦する」の意味に。
📝 be game for + 名詞（I'm game for anything.）や be game to do のどちらの形でも使う。誘いに対する快い返事としてよく使われる。
play games (with someone) (人を)もてあそぶ、だます、誠実に向き合わない口語
Just tell me how you really feel—stop playing games with me.（本当の気持ちをちゃんと教えてよ。駆け引きはやめて。）
🔍 「試合をする」の play a game とは違い、複数形 games を使い、対人関係で誠実さを欠く駆け引きをする、という否定的な意味の慣用表現。
📝 恋愛関係や交渉の場面で「はっきりしない態度で相手を振り回す」ことを非難するときによく使う。単数形 play a game（ゲームをする）とは意味も文法上の数も異なる点に注意。
ahead of the game 一歩リードしている、他より先んじている中立
By finishing the report early, she stayed ahead of the game.（報告書を早めに仕上げたおかげで、彼女は一歩リードした状態を保てた。）
🔍 「勝負の先を行く」という比喩から、競争や進捗において優位な立場にあることを表す。
📝 stay/keep ahead of the game の形でよく使われ、ビジネスや学業での「先手を打つ」文脈に頻出。
the only game in town 唯一の選択肢、他に代わりがないもの口語
This shop is the only game in town if you need spare parts on a Sunday.（日曜日に部品が必要なら、この店しか選択肢がないよ。）
🔍 「町で唯一開かれている賭博・勝負」という比喩から、他に選択肢がなく、それを選ぶしかない状況を表す。
📝 独占的な立場や、他に競合がない状況を表すときに使う。皮肉交じりに「仕方なくこれを選ぶしかない」というニュアンスを含むこともある。
beat someone at their own game 相手の得意技・やり方で相手を打ち負かす中立

🔍 「相手自身の勝負のやり方で相手を負かす」という直接的な比喩表現。
📝 相手の得意分野や手法をあえて逆手に取って勝つ、という状況で使われる。

In this business, customer service is the name of the game.（この業界では、顧客サービスこそが何より重要だ。）
🔍 「そのゲームの名前」という直訳から転じて、「その活動において本質的に重要なもの」を指す比喩表現。
📝 通例 "A is the name of the game" の形で、成功の鍵となる要素を強調するときに使う。
two can play at that game そっちがその気なら、こっちにも同じことができる口語
If he wants to play dirty, he'll find out that two can play at that game.（彼が汚い手を使いたいなら、こっちも同じ手でやり返せることを思い知るだろう。）
🔍 「その勝負には2人でも参加できる」という比喩から、「相手と同じ手段で対抗できる」という警告・宣言の決まり文句。
📝 相手の不誠実な・攻撃的な行動に対して「自分も同様の手段で対抗できる」と警告する決まり文句。単独でことわざ的に使われることが多い。
