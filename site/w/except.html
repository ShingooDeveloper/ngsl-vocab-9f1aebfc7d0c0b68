except

前置詞・接続詞・動詞米 /ɪkˈsept/ ／ 英 /ɪkˈsept/ex·cept（2音節）CEFR A2

🗣 強勢は第2音節 cept に落ちる（ɪkˈSEPT）。第1音節 ex- の母音は曖昧母音に近い短い /ɪ/ で軽く添える程度にし、「エク」とはっきり強く言わない

  1. 【前置詞】〜を除いて、〜以外は（全体を表す語の後に置き、そこから特定のものを除外する）（Everyone came to the party except Sam. サム以外は全員パーティーに来た。）
  2. 【接続詞】〜という点を除いて、ただし〜（前に述べた内容に対する例外・限定を加える。except that / except when / except if の形で節を導くことが多い）（I'd go with you, except I have to work tonight. 一緒に行きたいところだけど、今夜は仕事があるんだ。）
  3. 【動詞】〜を除く、除外する（フォーマルでまれな用法。特に過去分詞 excepted が「present company excepted（この場にいる人たちは別として）」のような決まった言い回しで使われる）（Present company excepted, few people in this town can be trusted. この場にいる人たちは別として、この町で信用できる人はほとんどいない。）

語形変化

形種類発音備考
excepts三人称単数現在（動詞用法）/ɪkˈsepts/動詞としての except 自体がまれな用法。語末は無声子音の連続 /pts/ で、母音を挟まず一気に発音する（「プツ」のように母音を足さない）
excepted過去形・過去分詞（動詞用法）/ɪkˈseptɪd/語幹が /t/ で終わるため -ed は /ɪd/ と発音する。動詞としてよりも過去分詞が形容詞化した決まり文句 present company excepted（この場にいる人たちは別として）でよく使われる
exceptinging形（動詞用法）/ɪkˈseptɪŋ/動詞のing形として使われることはまれで、実際にはこの形自体が except と同義の前置詞として使われることが多い（not excepting anyone = 誰も例外なく）

語法マーク

  • 前置詞・接続詞・(まれに)動詞の3品詞を持つが、実際の使用の大半は前置詞用法
  • except の後ろに節（S+V）を直接続けるのは口語的（except it's raining）で、書き言葉・フォーマルな場面では except that を使うのが標準
  • except は all / every / no / nothing / everyone など「全体・皆無」を表す語と共起し、そこから例外を1つ除く構文で使われることが多い。肯定文の一般的な文脈で単独で使うのはやや不自然で、except for が好まれる
  • 文頭に置いて「〜を除けば」と補足する場合は except for the noise, ... のように except for を使うのが普通で、except だけを文頭に置くのはまれ
  • except that / except when / except if は節を導く接続詞的な用法として頻出
  • 他品詞での用法: 動詞としての except は現代英語ではきわめてまれで、フォーマルな文体や決まり文句以外ではほとんど使われない。「（全体から）〜を除く」という他動詞的な意味を持ち、しばしば過去分詞 excepted の形で名詞の後ろに置かれ「〜は除いて」という後置修飾的な言い方になる（例: Sundays excepted, the store is open every day. 日曜を除いて、その店は毎日開いている）。現在分詞 excepting も動詞としてよりは、except や except for と同じ意味を持つ前置詞として使われることが多い（present company excepting）。名詞 exception（例外）はこの動詞 except から派生した語で、except の名詞形として非常によく使われる。

文型・頻出パターン

  • all/every/no/nothing など + except + 名詞全体・皆無を表す語の後に置き、そこから特定のものを除外する最も基本的なパターン
    Everyone except Tom passed the exam.トム以外は全員試験に合格した。
    There was nothing in the fridge except an old jar of jam.冷蔵庫には古いジャムの瓶以外何もなかった。
  • except that + S + V「〜という点を除いて」前述の内容に対する例外・限定を書き言葉的に加える
    The two rooms look the same, except that one has a window.その2部屋は同じに見えるが、片方には窓があるという違いがある。
  • except when / except if + S + V時間的・条件的な例外「〜のとき／〜の場合を除いて」
    The park is open every day except when it snows heavily.大雪のときを除いて、その公園は毎日開いている。
  • Except for + 名詞, S + V.文頭で「〜を除けば」と補足的な例外を加える。except 単独より for を伴うほうが自然
    Except for a small scratch, the car was in perfect condition.小さな傷を除けば、その車は完璧な状態だった。

他のNGSL語への置き換え

except → besides
元: All of the shops on this street are closed except the small bakery on the corner.
後: All of the shops on this street are closed besides the small bakery on the corner.
訳: この通りの店は、角の小さなパン屋以外は全部閉まっている。
📝 besides は本来「〜に加えて」という意味で、except の「〜を除いて」とは方向性が逆になりうる語。くだけた会話やアメリカ英語の一部の用法では except の意味で使われることもあるが、標準的な書き言葉では誤解を招くため避け、except や apart from を使うのが無難。
except → unless
元: I always walk to work, except when it's raining heavily.
後: I always walk to work, unless it's raining heavily.
訳: 大雨のときでない限り、いつも歩いて通勤している。
📝 unless は「〜でない限り」という条件を表す接続詞で、習慣的な行動の例外を述べるこの文では except when とほぼ同じ意味になる。ただし unless は必ず節（主語＋動詞）を伴うのに対し、except は except in winter のように名詞句も直接取れる点が異なる。
except → apart from
元: Nothing surprises me anymore, except for the fact that he still shows up late every single time.
後: Nothing surprises me anymore, apart from the fact that he still shows up late every single time.
訳: 彼が毎回遅刻してくること以外、もう何にも驚かなくなった。
📝 apart from は except for とほぼ同義で置き換え可能。apart from はイギリス英語でより好まれる傾向があり、except for はアメリカ・イギリス両方で広く使われる中立的な表現という違いがある。

頻出コロケーション

  • except for（〜を除いて — except の後に置かれ、名詞の前で使われる頻出パターン）
  • except that（〜という点を除いて — 節を導く接続詞のパターン）
  • except when（〜のときを除いて — 時間的な例外を表す頻出パターン）

⚠️ 日本人の典型誤用
✕ I like all fruits accept bananas.
「バナナ以外の果物は全部好き」と言いたかったが、accept を使うと「バナナを受け入れる」のような意味不明な文になり、ネイティブには単純な綴りミスとして映る。
○ I like all fruits except bananas.
バナナ以外の果物は全部好き。
I like all fruits except bananas.
accept（受け入れる、動詞）と except（〜を除いて、主に前置詞）は綴りが似ているため書き言葉で混同しやすい。意味も品詞もまったく別の語であり、日本人学習者に非常に多い綴りの取り違えの一つ。
✕ She likes all subjects except of math.
「数学以外の科目は全部好き」と言いたかったが、except of という形は存在せず、ネイティブには誤った前置詞の組み合わせに聞こえる。
○ She likes all subjects except math. / She likes all subjects except for math.
数学以外の科目は全部好き。
She likes all subjects except math.
She likes all subjects except for math.
except の後ろに名詞を直接続けるときは of を入れない。「〜を除いて」を強調したい、あるいは文頭に置きたい場合は except for を使う。except of という形は英語には存在しない。
✕ Everyone left except I.
「私以外はみんな帰った」と言いたかったが、except I は文法的に誤りで、ネイティブには不自然な言い間違いとして響く。
○ Everyone left except me.
私以外はみんな帰った。
Everyone left except me.
except は前置詞として働くことが多く、後ろに代名詞が来るときは目的格（me, him, her, them）を取るのが標準的。主格の I を続けるのは誤り。
🗾 カタカナ語とのギャップ

カタカナ「イクセプト」は「イ・ク・セ・プ・ト」5モーラで均等に聞こえがちだが、実際は2音節 /ɪkˈsept/ で強勢は第2音節 sept に集中する。第1音節の ɪ は曖昧で短く弱く発音し、はっきり「イク」と強く言わない。語末は破裂音の連続 /pt/ で、日本語のように「プト」と母音を足さず、しばしば t は不完全開放（t̚）で止めて音が消えるように終わる。

🌱 Word family・語源
exception（名詞）例外NGSL
exceptional（形容詞）並外れた、例外的な、非常に優れた
exceptionally（副詞）並外れて、例外的に
unexceptional（形容詞）取り立てて言うほどでもない、平凡な
語源: ラテン語 excipere（ex-「外へ」+ capere「取る、つかむ」）の過去分詞 exceptus に由来。古フランス語 excepter を経て中英語に入った。原義は「(全体から)外へ取り出す」で、そこから「除く、除外する」という意味が生まれた。
accept（受け入れる）— ac-（〜へ）+ cept（取る）で「自分の方へ取り込む」が原義。except の「外へ取り出す」と正反対の方向を表す仲間の語
concept（概念）— con-（共に）+ cept（取る）で「一緒に取りまとめたもの」が原義
capture（捕らえる、捕獲する）— except の cept と同じラテン語 capere（取る、つかむ）に直接由来する語
💡 覚え方: except = ex-（外へ）+ cept（取る、capere由来）→「全体から外へ取り出す」＝除く。同じ cept ファミリーの accept（ac-＝〜へ＋cept）は「自分の方へ取り込む」で except とは逆方向、と対比させて覚えると混同しにくい。
⚖️ 類義語の使い分け
apart from — except (for) とほぼ同義でイギリス英語でより好まれる傾向がある。apart from はやや会話的・柔らかい響きを持ち、書き言葉のフォーマルな except that ほど硬くはない。
but — nobody but him, nothing but trouble のように、否定語・indefinite代名詞の直後で「〜以外は」を表す前置詞として使われる。except より口語的で慣用句的に固定した組み合わせが多く、except ほど自由に名詞句を続けられるわけではない。
save — except とほぼ同義だが非常に文語的・古風で、日常会話ではまず使われない。法律文書や文学的・格式ばった文体で all, save one（1つを除いて全部）のように使われる。
besides — 本来は「〜に加えて」という意味で except の「〜を除いて」とは方向性が異なる語。くだけた口語やアメリカ英語の一部用法では except の意味でも使われるが、意味が曖昧になりやすく標準的な書き言葉では except や apart from を使うほうが誤解がない。
💬 熟語・慣用表現 (1)
present company excepted この場にいる人たちは別として、ここにいる人は除いて口語（やや古風・儀礼的）
People from this town tend to gossip a lot, present company excepted, of course.（この町の人はよく噂話をするけれど、もちろんこの場にいるあなたは別としてね。）
🔍 「目の前にいる人々（present company）は除かれて」。動詞 except の過去分詞 excepted を名詞の後ろに置く後置修飾の決まり文句で、これから述べる批判的な内容の対象から今その場にいる相手を外す、という前置きの挨拶的表現。
📝 単独では使えず、必ず何かを述べる文に添える挿入句として使う。相手を悪口の対象から外す社交辞令的な決まり文句で、しばしば冗談交じりに使われる。excepted を except に変えて present company except とは言えない点に注意。
