reference

名詞・動詞米 /ˈrefərəns/ ／ 英 /ˈrefrəns/ref·er·ence（3音節）CEFR B2

🗣 第1音節 REF に強勢が置かれる。第2音節の er は曖昧母音 /ə/ でごく弱く、第3音節 -ence も /əns/ と弱化する。referencing のように語尾が伸びても強勢位置は変わらない

  1. 【名詞】言及、参照すること（make reference to the accident その事故に言及する）
  2. 【名詞】参考文献、出典（a detailed reference list at the end of the report 報告書末尾の詳しい参考文献リスト）
  3. 【名詞】（就職などの）身元保証人、推薦者、推薦状（give someone's name as a reference 誰かの名前を照会先として挙げる）
  4. 【動詞】～に言及する、～を参照する（The article references several recent studies. その記事はいくつかの最近の研究を参照している。）

語法マーク

  • 名詞「言及」で make reference to ~（～に言及する）の形は無冠詞・不可算の定型。ただし個々の言及を数える文脈では a reference to ~ と可算にもなる（a clear reference to Shakespeare）
  • 名詞「参考文献」「照会先の人・推薦状」の意味は可算：a reference, two references のように使う
  • 動詞 reference は他動詞。目的語の前に to を挟まない（× reference to the report は誤り、○ reference the report）
  • 類義の refer to は自動詞＋前置詞（refer to the report）。reference と refer to は意味が近くても構文が違うので混同注意
  • in reference to ~ / with reference to ~ は「〜に関して」を表すフォーマルな前置詞句で、ビジネス文書・手紙の書き出しに頻出
  • 他品詞での用法: 動詞用法は名詞から転用された比較的新しい用法（いわゆる「名詞の動詞化」）で、学術・ビジネス文脈で「（文献・資料を）参照する、引用する」の意味で使われる。他動詞として直接目的語を取り（reference a source）、同義の refer to（自動詞＋前置詞）よりもインフォーマルで、特にアメリカ英語でよく使われる。名詞の「身元保証人」の意味では、a reference が推薦状そのものと、推薦してくれる人物本人の両方を指しうる点に注意（Can you be my reference?＝人／Please send two references.＝推薦状）。

文型・頻出パターン

  • reference + O（他動詞＋目的語）〜を参照する、〜に言及する（前置詞を挟まない直接目的語）
    reference the manualマニュアルを参照する
    reference a source出典を示す
  • in / with reference to + O〜に関して（ビジネス文書・手紙で使われるフォーマルな前置詞句）
    in reference to your emailあなたのメールに関しまして
    with reference to the contract契約書に関しまして
  • a reference to + O（名詞＋前置詞）〜への言及（reference が名詞として使われ、対象は to で示す）
    a reference to Shakespeareシェイクスピアへの言及
  • reference + 名詞（複合名詞）「参照用の〜」を表す複合名詞（形容詞的に reference を前置）
    reference number照会番号
    reference point基準点

他のNGSL語への置き換え

reference → checked
元: Several official documents were referenced during the interview to confirm his identity.
後: Several official documents were checked during the interview to confirm his identity.
訳: 面接の間、彼の身元を確認するためにいくつかの公式文書が確認された。
📝 reference は文書を情報源として「参照する・引用する」ニュアンスだが、checked は内容が正しいか「照合・確認する」という検証行為に焦点がある。何をしたいかで動詞の焦点が変わる。
reference → citation
元: If you quote someone else's words directly, you must always give a clear reference to avoid plagiarism.
後: If you quote someone else's words directly, you must always give a clear citation to avoid plagiarism.
訳: 他人の言葉をそのまま引用する場合は、盗用を避けるために必ず明確な出典表記を示さなければならない。
📝 reference は情報源を指し示す一般的な語だが、citation は学術論文などで使われる「正式な引用表記」を意味し、よりフォーマルで専門的な響きを持つ。日常会話では reference の方が自然。
reference → contact
元: Do you know anyone who could act as a reference when I apply for the new job?
後: Do you know anyone who could act as a contact when I apply for the new job?
訳: 私が新しい仕事に応募するとき、コネ（連絡先の人）になってくれそうな人を誰か知っていますか?
📝 reference は人柄や能力を保証してくれる「正式な推薦人・照会先」を意味するが、contact は単に「つながりのある知り合い」を指し、推薦や保証の役割を必ずしも伴わない。就職応募の文脈では reference の方が制度的な意味合いが強い。

🗾 カタカナ語とのギャップ

カタカナ「レファレンス」「リファレンス」は5モーラだが、英語は3音節 /ˈrefərəns/。強勢は最初の ref に強く1つだけ置き、er と ence の母音はどちらも曖昧母音 /ə/ に潰して弱く読む（全部の音を均等な強さで「レ・ファ・レ・ン・ス」と読まない）。また最初の音は /r/ なので「リ」ではなく「レ」に近い

🌱 Word family・語源
refer（動詞）言及する、参照する、委ねるNGSL
referral（名詞）紹介、（医療機関などへの）照会
referee（名詞・動詞）審判（員）；審判を務める
referendum（名詞）国民投票、住民投票
cross-reference（名詞・動詞）相互参照；相互参照をつける
語源: ラテン語 referre「運び戻す、報告する」（re-「戻す」＋ ferre「運ぶ」）に由来し、古フランス語 reference を経て英語に入った。「情報をこちらに運び戻す」というイメージが「参照する・言及する」の意味につながっている。
refer（言及する、参照する）— re-（戻す）+ fer（運ぶ）で reference の動詞の元になった語
prefer（〜を好む）— pre-（前に）+ fer（運ぶ）で「（他より）前に運び出す」→好んで選ぶ
transfer（移す、転送する）— trans-（向こうへ）+ fer（運ぶ）で「別の場所へ運ぶ」
💡 覚え方: ラテン語 ferre「運ぶ」が共通の核。re-（戻す）+ ferre で reference は「情報を運び戻して示す」＝参照。同じ ferre 系列の prefer は「前に運び出す」→好む、transfer は「向こうへ運ぶ」→転送、と芋づる式につながる。
⚖️ 類義語の使い分け
refer to — refer to は自動詞＋前置詞の構文（refer to a document）。reference は他動詞で前置詞なしに直接目的語を取る（reference a document）。意味はほぼ同じだが構文が異なる。
mention — mention は「話題として触れる」ことに重点があり、具体的な資料や出典を示すとは限らない。reference は情報源・文書を明示的に指し示す点でより形式的・具体的。
cite — cite は学術論文などで「（権威ある情報源を）引用する」という意味に特化しており、reference よりもフォーマルで書き言葉的。日常会話では reference の方が自然に使われる。
💬 熟語・慣用表現 (5)
frame of reference 判断の枠組み、物の見方、準拠枠中立／やや学術的
Growing up in a small village gave her a very different frame of reference from her city friends.（小さな村で育ったことで、彼女は都会育ちの友人たちとはまったく違う物の見方（判断基準）を持つようになった。）
🔍 「参照の枠組み」から転じて、人が物事を理解・判断する際の土台となる知識・経験・価値観の集合を指す比喩表現。
📝 心理学・社会学でよく使われる語。「参照の枠」という文字通りの意味ではなく、人が何をどう感じ・評価するかを決める価値観や経験の土台全体を指す比喩表現。
point of reference 比較・判断の基準となるもの、拠り所中立
Without a common point of reference, it's hard to say whether the new design is really better.（共通の判断基準がなければ、新しいデザインが本当に優れているかどうかを言うのは難しい。）
🔍 「参照の点」から、何かを評価・比較する際に基準として使うものを意味する。
📝 抽象的な価値観全体を指す frame of reference に対し、point of reference は比較のための一つの具体的な基準・目安を指すことが多い。
terms of reference （委員会・調査などの）権限の範囲、付託事項、任務規定ビジネス／フォーマル
The committee's terms of reference were expanded to include environmental issues.（その委員会の付託事項（権限の範囲）は、環境問題も含むよう拡大された。）
🔍 「参照の条件」から転じて、ある組織や調査が扱うべき範囲・目的・権限を公式に定めたものを指す。
📝 主にビジネス・行政・法律文書で使われる硬い表現。常に terms と複数形で用い、単数の a term of reference とは言わない。
in reference to 〜に関して、〜について（フォーマル）ビジネス／フォーマル
In reference to your email dated May 3rd, please find the attached report.（5月3日付のメールに関しまして、添付の報告書をご確認ください。）
🔍 「参照の中で」から「〜について言えば」という意味に固定化した前置詞句。
📝 ほぼ同義の with reference to も同じ意味で使われる（in reference to の方がややアメリカ英語寄り）。手紙・メールの書き出しに頻出する定型句で、日常会話ではほとんど使わない。
for reference 参考までに、参照用として中立／ビジネス
I've attached the original invoice for reference.（参考までに、元の請求書を添付しました。）
🔍 「参照のために」という意味で、冠詞を付けずに使う固定表現。
📝 無冠詞のまま定型化しており、× for the reference とは言わない（common_errors 参照）。相手に向けては for your reference の形もよく使われる。
