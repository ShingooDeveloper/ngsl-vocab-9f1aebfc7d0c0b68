program

名詞・動詞米 /ˈproʊɡræm/ ／ 英 /ˈprəʊɡræm/
  1. 【名詞】（テレビ・ラジオの）番組（I watched an interesting program on TV last night. 昨夜面白い番組をテレビで見た。）
  2. 【名詞】（コンピュータの）プログラム、ソフトウェア（She wrote a program to sort the data automatically. 彼女はデータを自動的に並べ替えるプログラムを書いた。）
  3. 【名詞】計画、講座、制度（教育・訓練など）（The university offers a new program in data science. 大学はデータサイエンスの新しい講座を提供している。）
  4. 【名詞】（式典・公演の）プログラム、次第（Check the program for the order of events. イベントの順番は式次第で確認してください。）
  5. 【動詞】（コンピュータに）プログラムを組む、プログラミングする（He programmed the robot to follow a specific path. 彼はロボットが特定の経路をたどるようにプログラムした。）
  6. 【動詞】（機器を）あらかじめ設定する、組み込む（You can program the thermostat to lower the temperature at night. サーモスタットは夜間に温度を下げるよう設定できる。）

他のNGSL語への置き換え

program → course
元: The training program was completely redesigned after the accident.
後: The training course was completely redesigned after the accident.
訳: その研修コースは事故のあと、全面的に見直された。
📝 program は組織的・体系的な「プログラム全体」の枠組みを指すことが多く、制度としての規模感がある。course は「一連の授業・講座」そのものを指し、学習内容に焦点がある。研修の枠組み全体なら program、具体的な講座内容なら course が自然。
program → set
元: Program the remote control before the guests arrive.
後: Set the remote control before the guests arrive.
訳: お客さんが来る前にリモコンを設定しておいて。
📝 program は「あらかじめ複雑な手順・条件を機器に組み込む」というニュアンスが強い。set はより一般的で単純な「（値・状態を）合わせる」動作を表し、日常会話ではこちらのほうが軽い響きになる。
program → tool
元: I used a Java program to generate a huge custom map for my Minecraft world.
後: I used a Java tool to generate a huge custom map for my Minecraft world.
訳: 巨大なカスタムマップを作るために、Javaのツールを使ってマインクラフトのワールドを生成した。
📝 program はコード全体・アプリケーションそのものを指すのに対し、tool はある作業を助ける実用的な道具というニュアンスが強く、より軽量・補助的なユーティリティを連想させる。大規模なソフトウェアなら program、簡易なユーティリティなら tool が自然。
