program

名詞・動詞米 /ˈproʊɡræm/ ／ 英 /ˈprəʊɡræm/pro·gram（2音節）CEFR A2

🗣 第1音節に強勢がある /ˈproʊ/。第2音節 -gram は弱く短く添えるだけで、カタカナ「グラム」のように重く伸ばさない。第1音節の母音は二重母音 /oʊ/ で、「オ」だけで止めない点に注意

  1. 【名詞】（テレビ・ラジオの）番組（I watched an interesting program on TV last night. 昨夜面白い番組をテレビで見た。）
  2. 【名詞】（コンピュータの）プログラム、ソフトウェア（She wrote a program to sort the data automatically. 彼女はデータを自動的に並べ替えるプログラムを書いた。）
  3. 【名詞】計画、講座、制度（教育・訓練など）（The university offers a new program in data science. 大学はデータサイエンスの新しい講座を提供している。）
  4. 【名詞】（式典・公演の）プログラム、次第（Check the program for the order of events. イベントの順番は式次第で確認してください。）
  5. 【動詞】（コンピュータに）プログラムを組む、プログラミングする（He programmed the robot to follow a specific path. 彼はロボットが特定の経路をたどるようにプログラムした。）
  6. 【動詞】（機器を）あらかじめ設定する、組み込む（You can program the thermostat to lower the temperature at night. サーモスタットは夜間に温度を下げるよう設定できる。）

語法マーク

  • 可算名詞。a program / two programs のように単数・複数を区別する
  • 動詞は主に他動詞で目的語をとる（program a robot / program the thermostat）。ただし program in Python（〜語でプログラムを組む）のように言語を示して自動詞的にも使え、learn to program のように目的語なしでも成立する
  • be programmed to do は「（機械が）〜するように設定・プログラムされている」の頻出受動態パターン。人を主語にすると比喩的に『〜するよう条件づけられている、育てられている』という意味にもなる（She was programmed to please others.）
  • 動詞用法はほぼ常にコンピュータ・機械の設定に限定される専門的な語で、一般的な『計画する』の意味では使わない（その意味では plan を使う）
  • 他品詞での用法: 名詞にはさらに抽象度の高い用法があり、単発の講座やイベントだけでなく「一連の行動計画・方針全体」を指すことがある: The government announced an ambitious reform program.（政府は野心的な改革プログラム［政策方針］を発表した。）。また表記面では、イギリス英語は『あらかじめ組まれた予定・内容』を指す名詞（TV番組、式次第、研修制度など）を programme とつづるが、コンピュータのソフトウェアの意味だけは英米ともに program とつづり分ける（Oxford辞書等で明記される慣用）。

文型・頻出パターン

  • program + O〜をプログラムする・設定する（他動詞の基本用法）
    program a computerコンピュータをプログラムする
    program a robotロボットをプログラムする
    program the thermostatサーモスタットを設定する
  • be programmed to do〜するようにプログラム・設定されている（受動態＋不定詞）。比喩的に人にも使い『〜するよう条件づけられている』の意味になる
    The machine is programmed to shut down automatically.その機械は自動的に停止するよう設定されている。
    He seems programmed to say yes to everything.彼は何でも『はい』と言うように条件づけられているようだ。
  • a/the ... program「〜プログラム・制度・講座」（前に名詞を置いて種類を限定する複合表現）
    a training program研修プログラム
    an exercise program運動プログラム
    a degree program学位課程
  • program in + 言語〜語でプログラムを組む（使用する言語を示す前置詞句）
    program in PythonPythonでプログラムを組む
  • a program of + 名詞〜という一連の計画・活動一式（抽象的な『方針・計画のまとまり』を指す用法）
    a program of economic reform経済改革の計画

他のNGSL語への置き換え

program → course
元: The training program was completely redesigned after the accident.
後: The training course was completely redesigned after the accident.
訳: その研修コースは事故のあと、全面的に見直された。
📝 program は組織的・体系的な「プログラム全体」の枠組みを指すことが多く、制度としての規模感がある。course は「一連の授業・講座」そのものを指し、学習内容に焦点がある。研修の枠組み全体なら program、具体的な講座内容なら course が自然。
program → set
元: Program the remote control before the guests arrive.
後: Set the remote control before the guests arrive.
訳: お客さんが来る前にリモコンを設定しておいて。
📝 program は「あらかじめ複雑な手順・条件を機器に組み込む」というニュアンスが強い。set はより一般的で単純な「（値・状態を）合わせる」動作を表し、日常会話ではこちらのほうが軽い響きになる。
program → tool
元: I used a Java program to generate a huge custom map for my Minecraft world.
後: I used a Java tool to generate a huge custom map for my Minecraft world.
訳: 巨大なカスタムマップを作るために、Javaのツールを使ってマインクラフトのワールドを生成した。
📝 program はコード全体・アプリケーションそのものを指すのに対し、tool はある作業を助ける実用的な道具というニュアンスが強く、より軽量・補助的なユーティリティを連想させる。大規模なソフトウェアなら program、簡易なユーティリティなら tool が自然。

🗾 カタカナ語とのギャップ

カタカナ「プログラム」は4モーラで各音がはっきり分かれて聞こえるが、英語は2音節 /ˈproʊɡræm/ で強勢は第1音節のみ。第2音節の母音は曖昧に短く処理され、カタカナの「グラム」のように重さを置いて発音しない。また「ラ」は舌先を弾く日本語のラ行の音とは異なる英語の /r/ で、混同しやすいので注意

🌱 Word family・語源
programmer（名詞）プログラマー、プログラムを組む人
programmable（形容詞）プログラム可能な、設定可能な
reprogram（動詞）（コンピュータ・機器を）再設定する、作り直す
programmatic（形容詞）計画的な、プログラムに基づいた（やや形式張った語）
語源: フランス語 programme を経て英語に入った語で、さらに遡るとギリシャ語 programma（pro-「前もって」+ gramma「書かれたもの」）に由来する。原義は「前もって書き示されたもの、公示・布告」。
grammar（文法）— 同じギリシャ語 gramma（書かれたもの）に由来。program の -gram 部分と grammar は語源的なきょうだい
diagram（図表）— dia-（通して）+ gram（書かれたもの）で「（線を通して）描かれたもの」→ 図表
telegram（電報）— tele-（遠く）+ gram（書かれたもの）で「遠くへ送られた文書」→ 電報
💡 覚え方: pro-（前もって）+ gram（書く）＝「前もって書かれた予定・手順」。テレビの番組表もコンピュータのコードも、どちらも『あらかじめ書かれた手順』という同じ原義でつながっている。
⚖️ 類義語の使い分け
schedule — schedule は個々の予定・時刻表そのものに焦点がある（会議や電車の時刻など具体的な日時の並び）。program はより組織立った一連の内容・講座・制度全体を指し、単なる時刻の並び以上の包括的な計画性を含む。
plan — plan は一般的な『計画』を指す最も広い語で、動詞としても日常的に『意図する』の意味で使う。program（動詞）はコンピュータ・機器の設定に用途が限定される専門的な語で、一般的な『計画する』の意味では使わない。
application — application（アプリ）は完成されたソフトウェア製品を指すことが多いのに対し、program はソースコードや処理手順そのものを含むより広い一般的な語。日常会話の『アプリ』には application/app、開発者視点でのコードには program がよく使われる。
