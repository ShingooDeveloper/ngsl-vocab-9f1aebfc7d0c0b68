table

名詞・動詞米 /ˈteɪbl̩/ ／ 英 /ˈteɪbl̩/ta·ble（2音節）CEFR A1

🗣 強勢は第1音節 ta にある。第1音節は二重母音 /eɪ/ を「エ」から「イ」へなめらかに滑らせて発音する。語末の -ble は母音を伴わず、成節子音 l̩ で終える（「ブル」と母音をはっきり発音しない）

  1. 【名詞】机、テーブル（家具）（Please put the plates on the table. お皿をテーブルに置いてください。）
  2. 【名詞】表、一覧表（データや数字を整理したもの）（Look at the table on page five. 5ページの表を見てください。）
  3. 【動詞】（主に米で）議案などを棚上げにする、先送りする／（主に英で）議案を提出する、審議にかける（The committee tabled the proposal until next month. 委員会はその提案を来月まで棚上げにした。）※米英で意味がほぼ逆になるので注意

語形変化

形種類発音備考

語法マーク

  • 名詞としては可算名詞。a table / two tables のように数える（不可算扱いにしない）
  • 動詞 table は他動詞で、議案・提案などの目的語を必ず伴う
  • 動詞の意味は米（棚上げ・先送り）と英（議題として提出）でほぼ正反対になるため、話者の出身・会議の主催国で判断する必要がある
  • set the table / clear the table のように定冠詞 the を伴う定型コロケーションが多い
  • 他品詞での用法: 名詞用法はさらに広がりがある。(1) 組織的に整理されたデータや情報の一覧という意味で、multiplication table（九九表）、periodic table（元素周期表）、table of contents（目次）のように複合語で頻出する。(2) 「食卓を囲む人々」を指して集合的に使うこともある（例: The whole table burst out laughing. テーブル全体＝そこにいた全員が笑いだした）。動詞 table の米英での意味の逆転（米＝棚上げして先送り、英＝正式に議題として提出）は、国際会議の議事録を読む際に特に注意が必要。

文型・頻出パターン

  • a table for + 数字〜人用のテーブル（レストランの予約などで使う表現）
    a table for two2人用のテーブル
    a table for four4人用のテーブル
  • at the table食卓について、テーブルの席に着いて
    Everyone was seated at the table by the time dinner started.夕食が始まる頃には、全員がテーブルの席に着いていた。
  • look at / see the table（データを整理した）表を見る、参照する
    See Table 2 for more details.詳細は表2を参照してください。
  • table + O（動詞：米＝棚上げにする、英＝議題として提出する）議案・提案などを棚上げにする、または議題として正式に提出する
    table the motion動議を棚上げにする（米）／動議を提出する（英）
  • table + 名詞（複合語）「表」「一覧」の意味を含む複合名詞
    table of contents目次
    periodic table元素周期表

他のNGSL語への置き換え

table → booked
元: I've already reserved a table for four at that new restaurant.
後: I've already booked a table for four at that new restaurant.
訳: あの新しいレストランに4人用のテーブルをもう予約してあるよ。
📝 book はレストラン・ホテル・チケットなど予約全般で reserve とほぼ同義に使われる口語的な語。reserve はやや改まった響きがあり、book の方が日常会話ではより自然なことが多い。
table → postponed
元: The proposal was tabled until the next board meeting.
後: The proposal was postponed until the next board meeting.
訳: その提案は次の役員会まで延期された。
📝 postpone は「延期する」を表す最も一般的で中立的な語で、あらゆる場面で使える。動詞 table（棚上げにする）は主に会議・議会の議事進行という専門的な文脈に限定される。
table → desk
元: This table is much bigger than the one we had in our old apartment.
後: This desk is much bigger than the one we had in our old apartment.
訳: この机は、前のアパートにあったものよりずっと大きい。
📝 table は食卓や作業台など汎用的な「テーブル」全般を指すのに対し、desk は勉強や仕事専用の「机」を指す。用途の違いによる使い分けがポイント。

✕ I always do my homework at my table in my room.
「自分の部屋の机で宿題をする」と言いたかったが、table を使うとネイティブには『（引き出しのない）共用のテーブルを部屋に置いている』ような、やや不自然な響きに聞こえてしまう。

I always do my homework at my desk in my room.
日本語の「机」を無条件に table と訳してしまう誤り。引き出しの付いた個人用の学習・仕事机は desk、脚と天板だけの共有の台は table と使い分ける。
✕ Please put the plates on table before guests arrive.

○ Please put the plates on the table before guests arrive.

日本語には冠詞が無いため、可算名詞 table の前の定冠詞 the を落としてしまう誤り。特定のテーブルを指す場合は on the table のように the を忘れずに付ける。
🗾 カタカナ語とのギャップ

カタカナ「テーブル」は4モーラだが、英語は teɪ-bl̩ のわずか2音節。特に語末の「ブル」は母音を伴わず、唇を閉じたbの直後に舌先を歯茎につけて成節的なl̩で軽く終える（はっきり「ブル」と発音しない）。また第1音節は伸ばす「テー」ではなく、二重母音/eɪ/を「エィ」とすばやく滑らせる点にも注意。

🌱 Word family・語源
tablet（名詞）錠剤；（iPadなどの）タブレット端末
tablecloth（名詞）テーブルクロス
tablespoon（名詞）大さじ（料理の計量単位）
turntable（名詞）（レコードプレーヤーの）ターンテーブル
語源: ラテン語 tabula「板、平らな面」に由来。古フランス語 table を経て中英語に入り、家具の「テーブル」から派生的に「板に書き記した一覧」＝データの「表」の意味も持つようになった。
tablet（錠剤／タブレット端末）— tabula の指小形（小さな板）。石板やろう板から、平らな錠剤・平らな板状の電子機器へ意味が広がった
tabulate（表にまとめる）— tabula + -ate（動詞化）で「板＝表の形に整理する」という原義が残る
💡 覚え方: ラテン語 tabula「平らな板」が全ての起源。tablet は「小さな板」→錠剤やタブレット端末、tabulate は「表の形に整理する」— どれも『平らな面に並べる』という tabula のイメージでつながっている。
⚖️ 類義語の使い分け
desk — desk は引き出しなどが付いた個人用の学習・仕事机を指すのに対し、table は脚と天板だけのシンプルな共有の台を指す。食卓や作業台としての table には desk のような収納機能はない。
chart — table は数値やデータを行と列に整理した表形式を指すのに対し、chart は棒グラフ・円グラフ・折れ線グラフなど視覚的な図形で傾向を示すもの。同じデータでも見せ方が違えば呼び方も変わる。
postpone — 動詞 table（米用法）は『議案などを棚上げにして先送りする』という意味で postpone とほぼ同義だが、会議・議会など議事進行の専門的な文脈に限られる。postpone はあらゆる予定の延期に使える一般語。
💬 熟語・慣用表現 (5)
turn the tables (on someone) 形勢を逆転させる、（人に）逆襲する中立
After losing the first two games, our team turned the tables and won the championship.（最初の2試合に負けた後、私たちのチームは形勢を逆転させて優勝した。）
🔍 「テーブルをひっくり返す」から、劣勢だった立場を一気に逆転させるという比喩。もとは盤を回転させて対戦相手と立場を入れ替えるゲームに由来するとされる。
📝 turn the tables on him のように on を伴い「〜に対して形勢を逆転する」と使うことが多い。The tables have turned.（形勢が変わった）のように tables を主語にした自動詞的な言い方もよく使われる。
under the table 裏で、こっそりと（不正に）口語／やや否定的
The manager was paying some workers under the table to avoid taxes.（その店長は税金を逃れるため、一部の従業員に裏で現金払いをしていた。）
🔍 「テーブルの下で」やり取りする＝人目につかない場所で行う不正な取引や現金の授受を表す比喩。
📝 主に賄賂・裏金など不正な金銭のやり取りに使う。drink someone under the table（〜より酒に強く相手を酔い潰す）という別の慣用句もあるので混同しないこと。
on the table （提案などが）検討・交渉の対象になっている、提示されているビジネス／中立
A new offer is on the table, but both sides still need to negotiate the details.（新しい提案が出されているが、両者はまだ詳細について交渉する必要がある。）
🔍 「テーブルの上に」置かれている状態＝正式に提案・検討の場に出されていることを表す比喩。
📝 put/lay something on the table で「〜を提案・提示する」の意味になる。英では動詞tableと同様、文脈によっては「一時棚上げにする」の意味で使われることもあるため注意。
bring something to the table （能力・経験・アイデアなどを）提供する、貢献するビジネス／中立
She brings a lot of experience and creativity to the table.（彼女は豊富な経験と創造力をもたらしてくれる。）
🔍 「〜をテーブルに持ってくる」から、交渉や協力の場に自分の強み・価値を差し出すという比喩。
📝 主にビジネスやチームワークの文脈で、人が貢献できる強み・価値を述べる際に使う。What do you bring to the table? は面接などの定番質問。
wait tables ウェイター（ウェイトレス）として働く、給仕をする口語
He waited tables at a diner to pay for college.（彼は大学の学費を払うため、食堂でウェイターとして働いていた。）
🔍 「テーブルを待つ」ではなく、この wait は「仕える、給仕する」という古い意味の名残で使われる特殊用法。
📝 wait on tables とも言う。tables は無冠詞・複数形が定型で、a table や the table にはしない。
