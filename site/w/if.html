if

接続詞・名詞米 /ɪf/ ／ 英 /ɪf/if（1音節）CEFR A1

🗣 1音節語で強勢の位置に迷いはない。母音は短母音 /ɪ/ で、日本語の「イ」よりも力を抜いて短く曖昧に発音する（heat の長母音 /iː/ と混同しない）

  1. 【接続詞】もし〜ならば、〜と仮定すると（起こるかどうか分からない条件を表す。If it rains tomorrow, we'll stay home. もし明日雨が降ったら、家にいよう）
  2. 【接続詞】〜すると必ず、〜するときはいつも（一般的な真理・習慣的な条件を表す。If you heat water to 100°C, it boils. 水を100度に熱すると沸騰する）
  3. 【接続詞】〜かどうか（＝whether。間接疑問で使う。Ask him if he's coming. 彼が来るかどうか聞いてみて）
  4. 【接続詞】〜だとしても、〜かもしれないが（譲歩的に「たとえそうであっても」の意味。形容詞句の前で使われることが多いややかたい用法。an interesting if unusual idea 面白いが風変わりなアイデア）
  5. 【名詞】仮定、不確定な条件（通常複数形 ifs で使う。too many ifs 不確定要素が多すぎる）

語形変化

形種類発音備考
ifs名詞の複数形/ɪfs/名詞用法（不確かな要素・仮定条件）でのみ使う複数形。接続詞の if には活用・複数形はない。決まり文句 no ifs, ands, or buts（一切の言い訳・条件なしに）でよく使われる

語法マーク

  • if 節は従属節であり、単独では文として成立しない。必ず主節とセットで使う（If it rains. だけでは文が完結しない）
  • 条件を表す if 節の中では、未来のことでも will を使わず現在形で表す（×If it will rain tomorrow, ... ○If it rains tomorrow, ...）
  • if 節を文頭に置くときは主節との間にカンマを打つのが原則。文末に置く場合はカンマ不要（We will stay home if it rains. にカンマは要らない）
  • whether と異なり、if は文の主語位置・前置詞の直後・to不定詞の直前では使えない（×If he will come is uncertain. ○Whether he will come is uncertain. ／×the question of if ／×if to go）
  • 名詞としての if は通例複数形 ifs で「不確定要素」を表す（too many ifs）。単数の an if も使われるが頻度は低い
  • 他品詞での用法: 名詞用法は主に複数形 ifs で使われ、「仮定・不確定な要素」を意味する（too many ifs 不確定要素が多すぎる）。口語の決まり文句 no ifs, ands, or buts（問答無用で、一切の言い訳や条件抜きに）でもこの名詞用法が使われる。単数形も There's always an if.（いつも「もし」という不確定要素がある）のように使われることがある。また接続詞 if には、形容詞句の直前に置いて「〜だとしても」という譲歩を1語で凝縮する用法がある（an interesting if unusual idea＝面白いが変わったアイデア。if unusual は even if it is unusual の省略的な言い方）。

文型・頻出パターン

  • If + 現在形, 現在形（ゼロ条件文）一般的な真理・自然の法則など「〜すると必ず…になる」を表す。この if は when に近い
    If you heat ice, it melts.氷を熱すると、溶ける。
    If water reaches 0°C, it freezes.水は0度に達すると凍る。
  • If + 現在形, will + 原形（第1条件文）実現しそうな未来の条件と結果を表す。if 節の中は未来のことでも現在形
    If you're free tonight, we can go out for dinner.今夜暇なら、夕食に出かけられるよ。
  • If + 過去形, would + 原形（仮定法過去）現在の事実に反する、または実現しにくい仮定とその帰結を表す
    If I won the lottery, I would travel around the world.宝くじが当たったら、世界中を旅するのに。
  • If + had + 過去分詞, would have + 過去分詞（仮定法過去完了）過去の事実に反する仮定と、それに対する後悔・推測を表す
    If you had told me earlier, I could have helped you.もっと早く言ってくれていたら、手伝えたのに。
  • 動詞 + if + 節（＝whether）ask, wonder, know, see, check などの動詞の目的語として「〜かどうか」を表す名詞節を導く
    I wonder if she remembers my name.彼女は私の名前を覚えているだろうか。
    Can you check if the door is locked?ドアに鍵がかかっているか確認してもらえる?
  • If only + S + 過去形／had + 過去分詞「〜さえすれば（いいのに）／〜していれば（よかったのに）」と強い願望・後悔を表す感嘆的な言い方
    If only I had studied harder for the exam.もっと一生懸命試験勉強していたらなあ。

他のNGSL語への置き換え

if → when
元: If it rains tomorrow, we will cancel the picnic.
後: When it rains tomorrow, we will cancel the picnic.
訳: 明日雨が降ったら（＝降ることが決まっている前提で）、ピクニックは中止にします。
📝 if は「雨が降るかどうか不確か」な条件を表すが、when に置き換えると「雨が降ること」を既定の事実として話していることになり不自然（天気予報が確実でない限り when は使わない）。if と when の混同は日本人学習者の典型的な誤りの一つ。
if → whether
元: I'm not sure if he has finished the report yet.
後: I'm not sure whether he has finished the report yet.
訳: 彼がもう報告書を終えたかどうか、よくわからない。
📝 間接疑問で「〜かどうか」を表す場合、if と whether はほぼ同じ意味で置き換え可能。ただし whether の方がフォーマルで書き言葉的、or not を直接続けられる（whether or not）。文頭・前置詞の後・to不定詞の前（例: the question of whether、whether to go）では if は使えず whether のみ可という違いがある。
if → whenever
元: If you mix blue and yellow paint, you get green.
後: Whenever you mix blue and yellow paint, you get green.
訳: 青と黄色の絵の具を混ぜるといつも、緑になるよ。
📝 ゼロ条件文の if は一般的真理・習慣的な条件を表す場合、意味的に whenever（〜するときはいつも）とほぼ置き換え可能。ただし if の方が中立的で自然、whenever は「毎回必ず」を強調するやや硬い言い方になる。

⚠️ 日本人の典型誤用
✕ If it will rain tomorrow, we will stay home.
「もし明日雨が降るだろうなら、家にいます」と未来の不確かさを丁寧に表そうとしたのだろうが、ネイティブにはif節にwillを重ねた文法的に不自然な文に聞こえる。
○ If it rains tomorrow, we will stay home.
もし明日雨が降ったら、家にいます。
If it rains tomorrow, we will stay home.
条件を表す if 節の中では、未来の内容でも will を使わず現在形で表すのが原則（主節側だけに will を使う）。日本語の「〜だろう」を意識してif節にもwillを入れてしまう典型的な誤り。
✕ If I would have more time, I would learn to play the guitar.
「もっと時間があれば、ギターを習うのに」と言いたかったのだろうが、if節にもwouldを重ねてしまうと、英語ネイティブには文法が崩れた不自然な文として響く。
○ If I had more time, I would learn to play the guitar.
もっと時間があれば、ギターを習うのに。
If I had more time, I would learn to play the guitar.
仮定法過去は If + 過去形, would + 原形 の形。if節の中にもwouldを入れてしまうのは非母語話者に多い誤りで、if節はあくまで過去形（had）に留め、wouldは主節側だけに置く。
✕ Unless you don't hurry, you'll miss the train.
「急がなければ電車に乗り遅れるよ」と言いたかったのだろうが、unlessはすでに「〜でなければ」の意味を持つため、don'tを足すと「急がなくなければ＝急げば」のような二重否定になり、ネイティブには意味が逆に取られかねない。
○ Unless you hurry, you'll miss the train. / If you don't hurry, you'll miss the train.
急がなければ、電車に乗り遅れるよ。
Unless you hurry, you'll miss the train.
If you don't hurry, you'll miss the train.
unless はそれ自体が「もし〜でなければ」というif...notの意味をすでに含んでいる。unless節の中にさらにdon'tを入れると否定を二重に重ねることになり、意味が逆転・混乱する。ifを使うならnotを残し、unlessを使うならnotを消す。
🗾 カタカナ語とのギャップ

カタカナ「イフ」は「イ・フ」と2拍で区切って読みがちだが、実際の /ɪf/ は1拍。母音 /ɪ/ は日本語の「イ」より力を抜いて短く曖昧に発音する。また if の直後に母音で始まる語（it, I, he など）が続くと語末の f と次の母音がなめらかに連結する（if it → イフィット）ため、「イフ・イット」と切って発音すると不自然に響く。

🌱 Word family・語源
iffy（形容詞）（口語）不確かな、あやしい、疑わしい
語源: 古英語 gif（「もし、〜かどうか」の意）に由来。ゲルマン祖語まで遡ると考えられているが、語頭の g- がどう生じたかなど詳細は不明な点が多い接続詞。中英語期にかけて語頭の g が落ち、現代の if の形に落ち着いた。
ob（ドイツ語）（〜かどうか）— ドイツ語の ob は if の「〜かどうか（whether）」の意味に対応する同系語
of（オランダ語）（または；〜かどうか）— オランダ語の of も同じゲルマン系統から発達し、「または」「〜かどうか」の両方の意味を併せ持つ
💡 覚え方: if はもともとの語頭の g が落ちて短くなった語。ドイツ語 ob・オランダ語 of と同じ「条件・不確かさ」を表すゲルマン語族の兄弟語だと覚えると、if に「〜かどうか（whether相当）」の意味があることにも納得がいく。
⚖️ 類義語の使い分け
whether — 「〜かどうか」の意味ではifとほぼ同義だが、whetherは文の主語位置・前置詞の直後・to不定詞の直前でも使え、if にはこれらの位置での使用ができない。whether or notの形も自然だが、if or notはやや不自然。
when — ifは起こるかどうか不確かな条件を表すのに対し、whenは「起こることが前提」の時間を表す。確実視される事柄にはwhen、不確かならifを使い分ける。
unless — unless = if...not（もし〜でなければ）に相当する否定条件を1語で表す。unless自体がすでに否定の意味を含むため、unless節の中でさらにnotを重ねると二重否定になる点に注意。
provided (that) / providing (that) — 「〜という条件で、〜さえすれば」とifよりも条件を強く限定するやや形式的な表現。契約書や公式な場面で好まれ、日常会話のifよりかたい響きを持つ。
💬 熟語・慣用表現 (6)
as if まるで〜であるかのように（事実に反するかもしれない仮定での比喩）中立
She talked to me as if she were my mother.（彼女はまるで私の母親であるかのように話しかけてきた。）
🔍 「〜のように」＋「もし」。実際にはそうではないかもしれないことを仮定して比較する、仮定法特有の言い回し。
📝 後ろは仮定法（was ではなく were を使うことが多い）。口語で単独で As if! と使うと「そんなわけない、まさか」という皮肉・強い否定の意味になる点も要注意。
if anything どちらかと言えば、むしろ（逆に）中立
The movie wasn't boring—if anything, it was too exciting.（その映画は退屈どころか、むしろ興奮しすぎるくらいだった。）
🔍 「もし何かあるとすれば」という原義から転じて、直前の内容を控えめに否定し、予想とは逆の方向を付け加えるときの決まり文句。
📝 文頭や文中に挿入句として置き、先行する内容を軽く修正しながら「むしろ〜」という方向を控えめに示す。ビジネス文書でも会話でも使われる。
if any たとえあるとしても（ごくわずか）、あるかどうか中立／ビジネス
He has few friends, if any, in this city.（彼はこの街に友人がほとんど、あるいは全くいない。）
🔍 if there are any の省略形。直前の名詞を受けて「もしあるとしても」とその存在自体をやんわり疑う言い方。
📝 few, little などの後ろに置いて「〜はごくわずかしかない、あるいは皆無に近い」と念押しするときに使う。
if worst comes to worst 最悪の場合には、いざとなったら口語
If worst comes to worst, we can always cancel the trip.（いざとなったら、旅行はいつでもキャンセルできる。）
🔍 「最悪が最悪になったら」という強調的な言い回しで、実際に最も悪い事態が起きてしまった場合を仮定する。
📝 if the worst comes to the worst や if worse comes to worst という異形も使われる。最悪の事態への備えを述べる場面での定型句。
no ifs, ands, or buts 問答無用で、一切の言い訳や条件なしに口語
You're cleaning your room today—no ifs, ands, or buts.（今日中に部屋を片付けなさい。言い訳は一切なしよ。）
🔍 反論・条件・言い訳を表す if, and, but という3つの機能語を並べ立てて「そうしたものは一切認めない」と強く否定する口語表現。
📝 親が子どもに、あるいは上司が部下に有無を言わさず何かを命じる場面でよく使われる。no ifs or buts のように短縮されることもある。
what if もし〜だったらどうする、〜としたら中立／口語
What if it rains during the festival?（もしお祭りの最中に雨が降ったらどうしよう。）
🔍 「何が起こる、もし〜なら」が凝縮され、動詞（happens など）を省いたまま仮定の状況への不安や疑問、提案を投げかける定型の問いかけ。
📝 What if S + V? の形で不安・提案を簡潔に切り出す決まった問いかけ方。答えの文を伴わずそれ単体で発話されることも多い。
