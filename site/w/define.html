define

動詞 (verb)米 /dɪˈfaɪn/ ／ 英 /dɪˈfaɪn/de·fine（2音節）CEFR B1

🗣 第2音節 fine に強勢（deFINE）。1音節目の de は弱く曖昧母音 /dɪ/、2音節目は二重母音 /aɪ/ を「ファ」から「イ」へなめらかに滑らせて強く長めに発音する

  1. 【動詞】（言葉・概念の）意味を定義する、明確に述べる
  2. 【動詞】（性質・境界を）決定づける、特徴づける（What defines a true friend? 真の友とは何によって決まるのか?）
  3. 【動詞】（輪郭・形を）くっきりと際立たせる（well-defined edges くっきりとした輪郭）

形種類発音備考

語法マーク

  • 他動詞。目的語を必ず伴う（× He defines. のように単独では使いにくい）
  • 受動態が非常に多い: be defined as ~「~と定義される」、well-defined「明確に定義された」
  • define + O + as + 名詞/形容詞句 の形で「OをAとして定義する」（define success as happiness）
  • wh節（what/how/who など）を目的語に取れる: define what 'fair' means
  • 進行形も可能な動作動詞: Defining the problem clearly is... のように動名詞・進行形どちらでも使われる
  • 他品詞での用法: 動詞のみの語（多品詞ではない）。ただし ing形が形容詞化して名詞を修飾する用法が頻出する: a defining moment（決定的な瞬間）、a defining feature/characteristic（際立った特徴）。これは分詞形容詞であり、名詞用法の take のような別品詞追加ではない。

文型・頻出パターン

  • define + O〜を定義する・明確にする（最も基本的な他動詞用法）
    define a term用語を定義する
    define the scope範囲を定義する
    define your goals自分の目標を明確にする
  • define + O + as + N/Adj〜を…として定義する
    The law defines an adult as someone over eighteen.その法律は成人を18歳を超える者と定義している。
  • be defined as ~ / well-defined受動態：〜と定義される／明確に定義された
    Success is defined differently by each person.成功が何を意味するかは、人によって定義が異なる。
    The border is well-defined.その境界は明確に定められている。
  • define + wh節〜が何か／どうかを定義する（間接疑問を目的語に取る）
    We need to define what counts as a success.何を成功とみなすかを定義する必要がある。
  • a defining + 名詞（分詞形容詞）決定的な〜、際立った〜
    a defining moment決定的な瞬間
    a defining feature際立った特徴

他のNGSL語への置き換え

define → explained
元: The rules of the game are clearly defined at the start of each match.
後: The rules of the game are clearly explained at the start of each match.
訳: そのゲームのルールは、各試合の開始時に分かりやすく説明される。
📝 define は基準や境界を確定させる客観的な行為を表すのに対し、explain は分かりやすく伝えて理解させることに重点がある。ルール自体の内容は変わらないが、聞き手の理解のしやすさに焦点が移る点が異なる。
define → assign
元: A good leader is someone who can clearly define each team member's role.
後: A good leader is someone who can clearly assign each team member's role.
訳: 優れたリーダーとは、各チームメンバーの役割をはっきりと割り振れる人のことだ。
📝 define は役割の内容・境界をはっきりさせること、assign は誰にその役割を割り当てるかという配分の行為を表す。define は「役割が何であるか」、assign は「誰にやらせるか」に焦点がある。
define → clarify
元: Before we sign the contract, we should define exactly what 'on time' means.
後: Before we sign the contract, we should clarify exactly what 'on time' means.
訳: 契約書にサインする前に、『時間通り』が正確に何を意味するのか明確にしておくべきだ。
📝 define は基準・意味をゼロから確定させるニュアンスが強く、clarify はすでにある発言や合意の曖昧な部分をはっきりさせるニュアンス。既存の契約条項の意味をはっきりさせる場面では clarify の方がより自然なことが多い。

⚠️ 日本人の典型誤用
✕ Can you define about this word for me?
「この単語の意味を定義してくれる?」と言いたいが、define about は英語として不自然で、余分な前置詞に聞こえてしまう。
○ Can you define this word for me?
この単語を定義してくれる?
Can you define this word for me?
define は他動詞で目的語を直接取り、前置詞 about は不要。explain about のような形と混同して about を付けてしまう誤り。
✕ We define success as make a lot of money.
「私たちは成功をお金をたくさん稼ぐことだと定義している」と言いたいが、as の後に動詞の原形を置くと文法的に破綻し、意味が通らなくなる。
○ We define success as making a lot of money.
私たちは成功を、お金をたくさん稼ぐことだと定義している。
We define success as making a lot of money.
define O as の後には名詞句・動名詞が来る必要があり、原形動詞 make をそのまま置くことはできない。前置詞 as の後は動名詞にする。
✕ Let me define why this happened.
「なぜこれが起きたか説明させて」と言いたいが、define why は不自然で、意味の定義をしようとしているかのように聞こえてしまう。
○ Let me explain why this happened.
なぜこれが起きたか説明させて。
Let me explain why this happened.
define は用語・概念・境界の「意味」を確定させる動詞であり、出来事の理由を述べる場合には使わない。理由を述べる文脈では explain が適切。
🗾 カタカナ語とのギャップ

「デファイン」という形でカタカナ語として定着してはいないため直接のズレは少ないが、発音する際は1音節目 de を弱く曖昧母音 /dɪ/ にし、強勢は2音節目 fine（/faɪn/）に置くこと。ローマ字読みで「デ・ファイン」と均等に区切って発音しないよう注意。

🌱 Word family・語源
definition（名詞）定義NGSL
definite（形容詞）明確な、確定したNGSL
definitely（副詞）間違いなく、明確にNGSL
undefined（形容詞）未定義の、はっきりしない
redefine（動詞）再定義する、見直す
definitive（形容詞）決定的な、最終的な
語源: 古フランス語 definer/definir を経て、ラテン語 definire（de-「完全に」+ finire「境界を定める」、finis「終わり・境界」由来）から。「境界線を完全に引く」が原義で、そこから「意味・範囲をはっきりさせる」に転じた。
finish（終える）— finis（終わり）から派生し、物事の「終わり」に到達する意
final（最終的な）— finis（終わり）に由来し、「これ以上先がない終着点」を表す
infinite（無限の）— in-（否定）+ finite（境界がある）で「境界が無い」
💡 覚え方: de-（完全に）+ fine（= finis「境界・終わり」）で「境界線をきっちり引く」→そこから「意味や範囲を確定する＝定義する」につながる。finish/final/infinite と同じ finis 一族。
⚖️ 類義語の使い分け
describe — describe は物事の特徴や様子を言葉で説明することに重点があり、境界や基準を確定させるわけではない。define は「何がその範囲に入るか」を厳密に決める点がより強い。
specify — specify は条件・詳細を一つひとつ具体的に挙げることに焦点があるのに対し、define は概念や語の意味・範囲そのものを確定させることに焦点がある。
characterize — characterize はある物事の典型的な特徴を述べて性格づけることを表し、必ずしも厳密な境界を示さない。define はより公式・厳密に意味や基準を定める場合に使う。
