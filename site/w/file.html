file

名詞・動詞米 /faɪl/ ／ 英 /faɪl/file（1音節）CEFR A2

🗣 1音節語なので強勢の位置に迷う余地はない。母音は二重母音 /aɪ/ で「ア」から「イ」へなめらかに滑らせて1拍で発音する。語末の l は暗いL（dark l）で、舌先を歯茎につけたまま母音のように「ゥ」寄りに響かせる

  1. 【名詞】書類、ファイル（整理してまとめられた一続きの文書）
  2. 【名詞】（コンピュータの）ファイル、データ
  3. 【名詞】列、隊列（in single file 一列縦隊で）
  4. 【動詞】（書類を）ファイルに綴じる、整理して保存する
  5. 【動詞】（正式に）提出する、申請する（file a complaint 苦情を申し立てる；file for divorce 離婚を申請する）
  6. 【動詞】一列になって歩く、進む（file in/out 一列になって入る・出る）

語形変化

形種類発音備考

語法マーク

  • 名詞用法（書類・データ・列・やすり）はすべて可算名詞: a file, two files
  • 動詞の基本用法は他動詞（file a complaint, file documents）。ただし file in / file out のような句動詞では自動詞的に「一列で進む」の意味になる
  • file for + 名詞（file for divorce, file for bankruptcy）のように、正式な申請には前置詞 for を伴うことが多い
  • file a complaint / file a report は against や with と結びつく（file a complaint against ~, file a complaint with the police）。to を使わない点に注意
  • 他品詞での用法: 「やすり」という工具の意味もある（entry.meaningsには未収録の追加義）。名詞: a nail file（爪やすり）、a metal file（金属やすり）。動詞: やすりで磨く・削る（file your nails 爪をやすりで整える、file down the rough edge ざらついた縁をやすりで削る）。この意味は、書類の file（ラテン語 filum「糸」由来）とは語源がまったく異なる別語が同じ綴りに合流したもの（詳細は etymology 参照）。

文型・頻出パターン

  • file + O（書類・訴訟など）正式な書類・申し立てなどを提出する（最も基本的な他動詞用法）
    file a complaint苦情を申し立てる
    file a lawsuit訴訟を起こす
    file a report報告書を提出する
  • file for + O〜を求めて正式に申請する
    file for divorce離婚を申請する
    file for bankruptcy破産を申請する
  • file + O + away〜を整理してしまい込む
    file the documents away書類を整理してしまっておく
  • file in / file out一列になって入る・出る（自動詞的な句動詞用法）
    The students filed into the classroom.生徒たちは一列になって教室に入った。
    The crowd filed out of the stadium.観客たちは一列になってスタジアムから出て行った。
  • in (single) file一列縦隊で（file が「列」の意味の名詞用法を使った決まり文句）
    walk in single file一列縦隊で歩く
    march in file一列になって行進する
  • a file of + 複数名詞〜の列・隊列
    a file of soldiers一列に並んだ兵士たち
    a long file of ants長い一列のアリの行列

他のNGSL語への置き換え

file → submitted
元: She has already filed her tax return for this year.
後: She has already submitted her tax return for this year.
訳: 彼女はすでに今年分の確定申告を提出済みだ。
📝 submit はより一般的でフォーマルな「提出する」。file は特に税務・法律関連の書類を、決まった手続きに則って正式に提出するというニュアンスが強い。
file → brought
元: The lawsuit was filed against the company last week.
後: The lawsuit was brought against the company last week.
訳: その訴訟は先週、その会社に対して起こされた。
📝 bring a lawsuit も file a lawsuit と同義で使われる定型表現。bring の方がやや口語的でニュース記事にも頻出し、file は書類提出という手続き面を強調する。
file → documents
元: This is the folder that contains all the files you need.
後: This is the folder that contains all the documents you need.
訳: これが、必要な書類が全部入っているフォルダーです。
📝 document は紙・電子を問わない「文書」全般を指す一般語。file はコンピュータ内の1つのデータ単位、または整理されたひとまとまりの書類を指すことが多く、より具体的なニュアンスを持つ。

⚠️ 日本人の典型誤用
✕ She filed a complaint to the manager about the noise.
言いたかったのは「彼女は騒音についてマネージャーに苦情を申し立てた」。to を使うと不自然な前置詞の組み合わせに聞こえ、ネイティブには違和感を与える。
○ She filed a complaint with the manager about the noise.
彼女は騒音についてマネージャーに苦情を申し立てた。
She filed a complaint with the manager about the noise.
file a complaint は「（誰）に対して」を表すのに前置詞 to ではなく with を使う。to をそのまま当てはめる誤りが多い。
✕ I have file the documents already.
言いたかったのは「もうその書類は提出済みです」。だが have の後に原形 file を置くと文法的に壊れており、ネイティブには単なる語の並べ間違いに聞こえる。
○ I have filed the documents already.
もうその書類は提出済みです。
I have filed the documents already.
現在完了は have + 過去分詞。file の過去分詞は filed だが、原形のまま使ってしまう典型的なミス。
✕ Please open this file to see all my photos.
言いたかったのは「このフォルダを開いて、私の写真を全部見てください」。しかし file を使うと、単一のファイル（1つのデータ）を開くよう指示しているように聞こえ、意図とずれて伝わる。
○ Please open this folder to see all my photos.
このフォルダを開いて、私の写真を全部見てください。
Please open this folder to see all my photos.
日本語のIT用語「ファイル」は複数のデータをまとめた入れ物（＝英語の folder）を指すことがあるが、英語の file は個々の書類・データ単位を指す。まとめて入れる箱を言いたいときは folder を使う。
🗾 カタカナ語とのギャップ

カタカナ「ファイル」は「ファ・イ・ル」の3〜4モーラだが、英語は1音節。母音は二重母音 /aɪ/ として1拍でつなげ、語末の l は母音を伴わない暗いL（「ル」と読まない）。またIT用語としてのカタカナ「ファイル」は日本語では英語の folder（複数のファイルをまとめる入れ物）の意味で使われることがあるが、英語の file はあくまで個々の書類・データ単位を指す点に注意。

🌱 Word family・語源
profile（名詞・動詞）横顔、輪郭；プロフィール、概要（を紹介する）NGSL
filament（名詞）フィラメント、繊維状のもの
fillet（名詞・動詞）（魚・肉の）ヒレ肉、切り身；ヒレに切り分ける
defile（名詞・動詞）隘路、峡谷（一列でしか通れない狭い道）；（軍隊などが）一列で行進する
filing cabinet（名詞）書類整理棚、ファイルキャビネット
語源: 実は語源の異なる2つの語が同じ綴りに合流した典型例。①書類・データの意味は、古フランス語 filer（紐に通す、束ねる）を経て、ラテン語 filum「糸」に由来する（もともと書類を紐に通して綴じて保管したことから）。②「やすり」という工具の意味は、まったく別系統の古英語 feol（ゲルマン祖語 *fīhalō「切る、磨く」に由来し、ドイツ語 Feile と同根）に由来し、綴りがたまたま一致しただけで語源上のつながりはない。
profile（輪郭、プロフィール）— pro-（前に）+ file（filum「糸」由来の細い線）＝「浮かび上がる輪郭線」が原義
filament（フィラメント）— ラテン語 filum「糸」をそのまま受け継いだ語。電球のフィラメントなど細い糸状のものを指す
fillet（ヒレ肉、細い帯）— filum の「細い糸・帯」の意味から、薄く細長く切り分けた肉や魚の部位を指すようになった
💡 覚え方: 「書類のfile」は元々ラテン語filum「糸」＝書類を紐で束ねたイメージ（profile・filamentも同じ「糸」の仲間）。一方「やすりのfile」は古英語feol「削る道具」で、たまたま綴りが同じになっただけの赤の他人。2つのfileは実は別人と覚える。

⚖️ 類義語の使い分け
submit — submit はどんな書類・作品でも「提出する」を表す一般語。file は裁判所・行政機関など決まった手続きに則って正式に提出するというニュアンスが強く、記録として残ることを含意する。
lodge — lodge a complaint のように、特に苦情や正式な申し立てを提出する意味で file とほぼ同義になるが、lodge はよりフォーマル・イギリス英語的な響きを持つ。
register — register は公式な名簿・データベースに「登録する」ことに焦点があるのに対し、file は実際の書類・データを「提出する」という行為そのものに焦点がある。
💬 熟語・慣用表現 (4)
rank and file 一般構成員、平社員、下っ端（幹部・指導部に対して）中立／ややフォーマル
The union leaders reached a deal, but the rank and file were furious about it.（組合の幹部は合意に達したが、一般組合員たちはそれに激怒した。）
🔍 元々は軍隊で兵士が整列する「横列（rank）」と「縦列（file）」から生まれた語。指揮官ではない一般の兵士たちを指した表現が転じて、組織の中の「トップではない一般会員・平社員」を表す。
📝 the rank and file で「一般構成員たち」という集合的な意味になる。形容詞的に rank-and-file members（一般会員）のようにハイフンでつないで使うこともある。
on file 記録として保管されて、ファイルに保存されてビジネス／中立
We already have your resume on file, so you don't need to send it again.（あなたの履歴書はすでに保管してありますので、再送していただく必要はありません。）
🔍 文字通りには「ファイルの上に」だが、実際は「（後で参照できるよう）記録・保管されている状態」を表す決まった言い方。
📝 keep ~ on file（〜を保管しておく）、have ~ on file（〜を保管している）の形でよく使われる。動詞 file が「提出する」という行為を表すのに対し、on file はすでに保管済みの「状態」を表す点が異なる。
circular file ごみ箱（書類などを捨てる場所を指すユーモラスな遠回し表現）口語（ユーモラス、主に米語）
He glanced at the junk mail and tossed it straight into the circular file.（彼はダイレクトメールをちらっと見ると、そのままゴミ箱に放り込んだ。）
🔍 丸い（circular）ファイル、つまり丸いゴミ箱を書類整理棚に見立てた、オフィスでのジョーク的な言い方。
📝 file a document in the circular file のように動詞 file と組み合わせて「捨てる」を遠回しに言う洒落表現。ビジネスの場でも軽いジョークとして使われる。
in single file 一列縦隊で、一列になって中立
The hikers walked in single file along the narrow mountain trail.（ハイカーたちは狭い山道を一列縦隊になって歩いた。）
🔍 file の「列・隊列」という名詞義を用いた決まり文句。単に in file とも言うが、in single file の形がより一般的。
📝 人や動物が縦一列に並んで進む様子を表す決まり文句。march in single file のように動詞と組み合わせて使われることも多い。
