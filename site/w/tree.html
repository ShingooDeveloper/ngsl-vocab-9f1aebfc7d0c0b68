tree

名詞 (noun)米 /triː/ ／ 英 /triː/tree（1音節）CEFR A1

🗣 1音節語。語頭の子音クラスタ /tr/ は日本語のように母音を挟まず一息で発音し、母音は長母音 /iː/ でしっかり伸ばす（「トゥ・リー」と2拍にならないよう注意）

  1. 【名詞】木、樹木（There's a big tree in front of my house. 家の前に大きな木がある。）
  2. 【名詞】樹形に見立てた図・構造（a family tree 家系図）

語形変化

形種類発音備考
trees複数形/triːz/規則変化。tree の語末は母音（有声）なので、複数語尾は /s/ ではなく濁った /z/ で発音する（「トゥリース」ではなく「トゥリーズ」）

語法マーク

  • 可算名詞。単数では必ず a/the などの限定詞を伴う（× I like tree. は誤り。○ I like trees. または I like the tree.）
  • 個々の木を指す tree（可算）と、素材としての「木材」を指す wood（原則不可算）は明確に区別される
  • family tree（家系図）、decision tree（決定木）のように、樹形に見立てた比喩的な図・構造を指す用法がある（entry の meanings 参照）
  • tree house（ツリーハウス）、tree trunk（幹）、treetop（梢）のように名詞+名詞の複合語を作りやすい
  • 他品詞での用法: まれに動詞としても使われる。「（犬などが獲物や動物を）木の上に追い上げる」の意味（The dogs treed a raccoon in the backyard. 犬たちがアライグマを裏庭の木の上に追い詰めた）。この用法は口語的でやや古風・地域的な響きがあり、日常会話ではほぼ名詞用法のみと考えてよい。

文型・頻出パターン

  • There + be + tree(s) + 場所句存在構文で木がどこにあるかを述べる
    There are trees along the street.通り沿いに木々が並んでいる。
    There is a tree behind the house.家の裏に木が1本ある。
  • the tree + 関係詞節関係代名詞で特定の木を修飾し、どの木かを限定する
    the tree that fell down in the storm嵐で倒れた木
    the tree my grandfather planted祖父が植えた木
  • 数詞・年数 + -year-old + tree樹齢を表す複合形容詞で木の年齢を述べる
    a 200-year-old tree樹齢200年の木
    a fully grown tree成木、育ちきった木
  • tree + house/trunk/top/root（複合名詞）tree を構成要素とする複合名詞で木の部分や関連物を表す
    a tree houseツリーハウス
    a tree trunk木の幹
  • family/decision + tree（比喩的な「木」構造）枝分かれした図・構造を樹木に見立てる比喩用法
    trace your family tree家系図をたどる
    build a decision tree決定木を作る

他のNGSL語への置き換え

tree → knock
元: The old tree was cut down after the storm damaged its roots.
後: The old tree was knocked down after the storm damaged its roots.
訳: その古い木は、嵐が根を傷めた後に倒れてしまった。
📝 cut down は人が道具を使って意図的に切り倒すことを含意するのに対し、knock down は嵐などの外力によって（人の手を介さず）倒れることを表す。自然災害の描写では knocked down / blown down の方が自然なことが多い。
tree → found
元: This is the tree that my grandfather planted when he was young.
後: This is the tree that my grandfather found when he was young.
訳: これは、祖父が若い頃に見つけた木なんです。
📝 plant は「自分の手で植えて育てた」という創造・関与のニュアンスを持つのに対し、find は「すでにそこにあったものを発見した」という意味になり、木との関わり方が大きく変わる。
tree → parent
元: Don't climb that tree without an adult watching you.
後: Don't climb that tree without a parent watching you.
訳: 親が見ていないところで、その木に登ってはいけません。
📝 adult は教師や近所の人なども含む広い範囲を指すのに対し、parent は「親」に限定される。子音で始まる parent の前では冠詞も an adult → a parent と変わる。

詳細
⚠️ 日本人の典型誤用
✕ This chair is made of tree.
「この椅子は木でできている」と言いたいが、tree は「生えている木」そのものを指すため、家具の材料として使うと意味が通らない。
○ This chair is made of wood.
この椅子は木（材）でできている。
This chair is made of wood.
日本語の「木」は生きている植物（tree）と素材としての木材（wood）の両方を指せるが、英語では明確に区別される。家具などの材料を言うときは wood を使う。
✕ There are a lot of tree in this park.
「この公園にはたくさんの木がある」と言いたいが、tree を単数のまま使うと文法的に誤りで不自然に響く。
○ There are a lot of trees in this park.
この公園にはたくさんの木がある。
There are a lot of trees in this park.
tree は可算名詞なので、複数を表すときは必ず -s をつける。日本語の「木」には単複の区別がないため、複数形を落としてしまう典型的な誤り。
✕ I climbed on the tree.
「その木に登った」と言いたいが、on を入れると「木の上に乗っかった」ような響きになり、登るという動作の意味がぼやけてしまう。
○ I climbed the tree.
私はその木に登った。
I climbed the tree.
climb は他動詞として目的語を直接取れるため、「木に登る」は climb the tree（または climb up the tree）が自然。climb on the tree は「木の上に乗った」ように聞こえ、登る動作というより一瞬乗った状態を指す不自然な表現になる。
🗾 カタカナ語とのギャップ

カタカナ「ツリー」（クリスマスツリー等）は「ツ・リー」の2モーラで、語頭が「ツ」に引っ張られがちだが、実際の英語は tr の子音クラスタを一息でつなげる /tr/ で「ツ」の音は含まれない。また母音 /iː/ は長母音として1拍長めに伸ばし、r は舌を反らせた英語のr（日本語のラ行とは異なる）で発音する。

🌱 Word family・語源
treehouse（名詞）ツリーハウス（木の上に建てた小屋）
treetop（名詞）梢、木のてっぺん
tree-lined（形容詞）並木の、木々が立ち並ぶ
語源: 古英語 trēow（木）に由来し、さらにゲルマン祖語 *trewą を経て印欧祖語の語根 *deru-/*dreu-「堅い、しっかりした（木・樫のように）」にさかのぼる。同じ語根から「堅く揺るがない」というイメージを共有する語が英語に複数残っている。
true（本当の、忠実な）— 「木のように堅く揺るがない＝信頼できる」という語源イメージを tree と共有する
trust（信頼）— true と同根で、「堅固さ・揺るがなさ」から「信頼」の意味に発展した
endure（耐える、持続する）— ラテン語 durus（堅い）を経由し、tree と同じ印欧祖語根「堅い」に由来する
💡 覚え方: 「木のようにどっしり堅い」というイメージが語源の核。true（忠実な）や trust（信頼）も『木のように揺るがない』という同じ発想でつながっている、と覚えると tree ファミリーが記憶に残りやすい。
⚖️ 類義語の使い分け
sapling — sapling は幹がまだ細い「若木」に限定される語で、大きさ・年齢を問わない一般語 tree とは範囲が異なる。
shrub — shrub（低木）は主幹が1本ではなく根元から複数に枝分かれし丈も低い植物を指し、通常1本の太い幹を持ち背が高い tree とは区別される。
wood — wood は伐採されて素材となった「木材」を指し、生きて根を張っている植物そのものを指す tree とは状態が異なる（common_errors 参照）。
💬 熟語・慣用表現 (5)
bark up the wrong tree 見当違いのことをする、的外れな相手・原因を追及する口語
If you think I broke the vase, you're barking up the wrong tree.（花瓶を割ったのが私だと思っているなら、それは見当違いだよ。）
🔍 獲物を見失った猟犬が間違った木に向かって吠え続ける様子から。「間違った木に向かって吠える」が転じて「見当違いの方向へ努力・非難を向ける」の意味になった。
📝 主に you're barking up the wrong tree の形で、疑う相手や追及する原因を間違えていると指摘するときに使う。
can't see the forest for the trees 木を見て森を見ず、細部にとらわれて全体像が見えない中立
He's so focused on tiny details that he can't see the forest for the trees.（彼は細かい点ばかり気にして、物事の全体像を見失っている。）
🔍 「木々ばかり見て森全体が見えない」という比喩から、細部にこだわりすぎて大局を見失う様子を表す。
📝 米式は forest、英式は can't see the wood for the trees と wood を使う。否定形（can't/couldn't）で使うのが基本。
money doesn't grow on trees お金は簡単には手に入らない、お金は木になるものではない口語
You can't buy a new phone every month—money doesn't grow on trees, you know.（毎月新しいスマホなんて買えないよ、お金は木になるわけじゃないんだから。）
🔍 「お金は木に生えているわけではない」という直接的なイメージから、お金は労せず手に入るものではないと諭す決まり文句になった。
📝 親が子どもの浪費をたしなめる際などによく使われる決まり文句。
up a tree 進退窮まって、行き詰まって口語（やや古風）
Without a backup plan, we're really up a tree now.（代わりの案もなく、今は本当に八方塞がりだ。）
🔍 犬に追われて木の上に逃げ込んだ動物が身動きできなくなるイメージから、「逃げ場がなく困り果てている」状態を表す。
📝 現代ではやや古めかしい響きがあり、in a fix や up a creek の方が今日ではよく使われる。
out of one's tree 頭がおかしい、正気じゃない口語（インフォーマル）
You must be out of your tree if you think that's a good idea.（それがいい考えだと思うなんて、どうかしてるよ。）
🔍 「自分の木から出てしまっている」という奇妙なイメージから、正気を失っている・まともでない状態を表すくだけた表現になった。
📝 out of one's mind とほぼ同義のくだけた表現で、ひどく酔っている状態を指すこともある。
