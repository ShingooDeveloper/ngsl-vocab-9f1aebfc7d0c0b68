try

動詞・名詞米 /traɪ/ ／ 英 /traɪ/try（1音節）CEFR A1

🗣 1音節語。子音連結 /tr/ を一息で発音し、続く母音は二重母音 /aɪ/。「ト・ラ・イ」と3拍に伸ばさず、「トゥライ」を1拍でなめらかに滑らせる

  1. 【動詞】試す、やってみる（Why don't you try it? やってみたら?）
  2. 【動詞】～しようとする、努力する（She tried to fix the printer herself. 彼女は自分でプリンターを直そうとした。）
  3. 【動詞】（忍耐・限界などを）試す、疲れさせる（The long meeting really tried everyone's patience. 長い会議はみんなの忍耐力を本当に試した。）
  4. 【名詞】試み、挑戦（It's worth a try. やってみる価値はある。）

語形変化

形種類発音備考
tries三単現/traɪz/子音+yで終わる語なのでy→iesに変化（trys は誤り）。語末は有声音なので/z/
tried過去形・過去分詞/traɪd/規則変化だがy→iedに変化（tryed は誤り）。過去形と過去分詞は同形
tryinging形/ˈtraɪɪŋ/-ingの前ではyは変化しない（そのままtry+ing）。形容詞化した trying（「つらい、しんどい」の意）と同形なので文脈で判断

語法マーク

  • try to + 動詞の原形（不定詞）と try + -ing（動名詞）で意味が異なる：try to do は「（苦労して）～しようとする」という努力・意図、try doing は「試しに～してみる」という実行・確認を表す
  • 他動詞用法（try + O）と、try to / try -ing を伴う準動詞的用法の両方を持つ
  • 名詞用法は可算名詞：a try, one try, three tries のように数えられる（give it a try, worth a try）
  • 進行形にできる動作動詞（She is trying her best now.）
  • 他品詞での用法: 名詞用法は「試み」（a try, have a try, give it a try）の他に、ラグビーなどのスポーツで得点行為を指す専門語としても使われる（score a try 得点する）。日本語のスポーツ用語「トライ」はこの意味から借用された。また口語表現 Nice try! は「（うまい言い訳・工作だけど）その手には乗らないよ」という皮肉の決まり文句として頻出する。

文型・頻出パターン

  • try + to不定詞～しようとする（努力・意図、成功したかは不明）
    try to fix the problem問題を解決しようとする
    try to be on time時間通りに来ようとする
  • try + -ing（動名詞）試しに～してみる（実際に行動して結果を確かめる）
    try restarting the computer試しにパソコンを再起動してみる
    try adding more salt試しに塩を足してみる
  • try + O（名詞目的語）～を試す・味わってみる
    try this cakeこのケーキを食べてみる
    try a new approach新しいやり方を試す
  • give/have a try（軽動詞構文）名詞 try を使った「ひとつ試してみる」という言い回し
    give it a try試しにやってみる
    It's worth a try.やってみる価値はある。

他のNGSL語への置き換え

try → attempt
元: I've tried this recipe twice, but it never turns out right.
後: I've attempted this recipe twice, but it never turns out right.
訳: このレシピを2回試みたけど、いつもうまくいかない。
📝 attempt は try よりも改まった・書き言葉的な語で、フォーマルな報告や説明文で好まれる。try は日常会話で圧倒的に多く使われる中立的な語。
try → study
元: If I tried harder, I could probably pass the exam.
後: If I studied harder, I could probably pass the exam.
訳: もっと熱心に勉強すれば、たぶん試験に合格できるんだけど。
📝 try harder は「もっと努力する」という一般的な表現だが、study harder は「勉強する」という具体的な行動を指す。文脈に応じて具体性の度合いを調整できる。
try → suggest
元: The solution that we tried didn't work at all.
後: The solution that we suggested didn't work at all.
訳: 私たちが提案した解決策は全く効果がなかった。
📝 tried は実際にその解決策を試してみたことを含意するが、suggested は口頭・文書で提案しただけで、実行されたかどうかは含意しない。行動の有無で意味が変わる点に注意。

⚠️ 日本人の典型誤用
✕ He tries to fixing the leak every weekend.
「彼は毎週末、その水漏れを直そうとしている」と言いたかったが、to の後に動名詞を続けると文法的に誤りで、ネイティブには不自然な文に聞こえてしまう。
○ He tries to fix the leak every weekend.
彼は毎週末、その水漏れを直そうとしている。
He tries to fix the leak every weekend.
try to の後ろは動詞の原形（to不定詞）。動名詞 fixing を続けるのは誤り。tries -ing という形と混同しないよう注意。
✕ The room was stuffy, so I tried to open the window a little.
「部屋が蒸し暑かったので、試しに窓を少し開けてみた」という軽い行動のつもりが、tried to open では「窓を開けようと苦労した（開けられたかは不明）」という努力・困難のニュアンスに聞こえてしまう。
○ The room was stuffy, so I tried opening the window a little.
部屋が蒸し暑かったので、試しに窓を少し開けてみた。
The room was stuffy, so I tried opening the window a little.
単に「試しに～してみた」という何気ない行動を表すには try + -ing が自然。try to do は「（苦労して・意図して）～しようとした」という努力のニュアンスを持ち、成功したかどうかが不明な響きになる。
✕ It looks hard, but let's give a try it.
「難しそうだけど、試しにやってみよう」と言いたかったが、語順を間違えると英語として不自然になり、意味が正しく伝わらない。
○ It looks hard, but let's give it a try.
難しそうだけど、試しにやってみよう。
It looks hard, but let's give it a try.
give + 目的語(it) + a try という語順（SVOO）が正しい。試す対象の it を a try より先に置く必要があり、順番を逆にすると英語として成立しない。
🗾 カタカナ語とのギャップ

カタカナ「トライ」は「ト・ラ・イ」と3モーラで発音されがちだが、英語 try は1音節 /traɪ/。子音連結 /tr/ を一息で発音し、母音は二重母音 /aɪ/ のみ（3拍に伸ばさない）。またラグビー等スポーツ用語の「トライ」は英語 try の得点行為の意味から借用されたものだが、日常語としての try は「試み全般」を指す点で意味の幅が広いことに注意。

🌱 Word family・語源
trial（名詞）裁判、試験、試練NGSL
tryout（名詞）（米口語）入団・出演のための選考テスト、オーディション
retry（動詞）もう一度試す、再試行する
trying（形容詞）つらい、しんどい、我慢を強いる（a trying day）
語源: 古フランス語 trier「選り分ける、選別する」に由来（原義は農作物などを「より分ける」）。中英語でこの「選別する」意味が「吟味する、試す」に転じ、さらに「（努力して）～しようとする」へと意味が広がった。
trial（試み・裁判）— trier「選り分ける」の名詞化。「証拠を吟味して選び分ける」が裁判の原義
retry（再試行する）— re-（再び）+ try で「もう一度選り分け直す・試す」
💡 覚え方: 元は「（良い物を）選り分ける」という意味だったのが、「あれこれ選別しながら試す」→「試みる」に変化した語。ふるいにかけて選び出すイメージで try を覚えるとよい。
🔗 句動詞
try on — （服・靴などを）試着する
She tried on three dresses before choosing one.（彼女は1着選ぶ前に3着のドレスを試着した。）
try out — 試しに使ってみる、試用する；（チームなどの）選考テストを受ける
He tried out for the school basketball team.（彼は学校のバスケットボール部の選考テストを受けた。）
⚖️ 類義語の使い分け
attempt — attempt はより形式張った書き言葉的な語で、失敗の可能性がある難しい行為に使われることが多い（attempt a solution）。try は日常会話で使う中立的で幅広い語。
endeavor — endeavor はさらに格式張った語で、長期的・組織的な努力（プロジェクトなど）を指すことが多く、日常会話ではほとんど使わない。try ほど汎用的ではない。
experiment — experiment は結果がどうなるか分からないものを検証する「実験的に試す」というニュアンスが強い。try は単に「～してみる」という広い意味で使え、実験的な含みは必須ではない。
💬 熟語・慣用表現 (5)
try one's hand at （何かに）初めて挑戦してみる、腕試しをする中立
After retiring, she decided to try her hand at painting.（退職後、彼女は絵を描くことに挑戦してみることにした。）
🔍 「自分の手を試す」という直訳から、「不慣れな分野に挑んでみる」という意味に転じた定型表現。
📝 対象はたいてい初めて・不慣れな活動。at の後ろには名詞または動名詞が続く（try one's hand at cooking）。
try one's luck 運試しをする、一か八かやってみる中立
He moved to the city to try his luck as an actor.（彼は俳優として一旗揚げようと都会に出た。）
🔍 「自分の運を試す」という直訳から、「結果が運や偶然に左右される物事に挑戦する」の意味になった。
📝 後ろに at/as/in などを伴い対象を示すことが多い（try one's luck at the casino, try one's luck as a singer）。
Nice try. （皮肉で）おしい、その手には乗らないよ口語
"I swear the dog ate my homework." "Nice try, but I know you just forgot."（「本当に犬が宿題を食べちゃったんだ」「おしいね、でも忘れただけでしょ」）
🔍 直訳は「良い試み」だが、実際には相手の言い訳や小細工がばれた・失敗した時に使う皮肉な決まり文句。
📝 単独の間投詞的な決まり文句として使う。字義通り「よくやった」の意味で使われることは稀で、ほぼ常に皮肉・冗談として使われる。
try as one might どんなに頑張っても（〜できなかった）中立／やや文語
Try as she might, she couldn't get the stubborn jar open.（どんなに頑張っても、彼女はその頑固な瓶のふたを開けられなかった。）
🔍 「試みるにせよ」という譲歩構文が倒置された形。「いくら〜しようとしても」という意味を作る。
📝 mightの代わりにcould/wouldも使われる（try as she could）。通常は「〜できなかった」という否定的な結果の文が後に続く。
try something on for size （考え・案などを）試しに採用してみる、様子を見る口語
Let's try this new schedule on for size before we commit to it.（本格的に決める前に、この新しいスケジュールを試しに使ってみよう。）
🔍 元は服が「サイズに合うか試着してみる」の意味だが、比喩的にアイデアや計画を「ひとまず試してみる」の意味で使われる。
📝 文字通り服の試着にも使うが（この場合は phrasal verb の try on と重なる）、比喩的にアイデア・計画・役割を試す文脈で on for size まで続けて使う点が特徴。
