contain

動詞 (verb)米 /kənˈteɪn/ ／ 英 /kənˈteɪn/con·tain（2音節）CEFR B1

🗣 第2音節に強勢（conTAIN）。第1音節 con- は弱形の schwa /kən/ で「コン」と強く読まない。第2音節の母音は二重母音 /eɪ/ で「テイン」と1拍でなめらかに滑らせる

  1. 【動詞】（容器などの中に）～を入れている、～が入っている（This box contains old photographs. この箱には古い写真が入っている。）
  2. 【動詞】（成分・要素として）～を含む（This drink contains caffeine. この飲み物にはカフェインが含まれている。）
  3. 【動詞】（感情などを）抑える、こらえる（She could barely contain her excitement. 彼女は興奮を抑えきれなかった。）
  4. 【動詞】（火災・被害・病気の拡大などを）封じ込める、食い止める（Firefighters managed to contain the blaze. 消防士たちは火災を食い止めた。）

語法マーク

  • 他動詞。目的語を必ず伴う（× The box contains. のように単独では使えない）
  • 「（容器の中に）〜が入っている／〜を含む」の意味では状態動詞として働き、原則として進行形にしない（× This box is containing photos.）
  • 「（感情を）抑える」の意味では contain oneself のように再帰代名詞を目的語に取ることが多い
  • 「（火災・被害・病気の拡大を）封じ込める」の意味では be contained（受動態）で使われることが非常に多い
  • 他品詞での用法: contain は動詞のみで、名詞用法は持たない（「容器」を表す名詞は別語の container）。状態動詞（stative verb）としての用法（「〜が入っている／含まれている」）では通常進行形にしない点に注意。また目的語に再帰代名詞を取る contain oneself（自分を抑える）という定型的な言い方がある。

文型・頻出パターン

  • contain + O（物・情報など）〜が入っている、〜を含む（最も基本的な状態動詞用法）
    The report contains several errors.その報告書にはいくつかの誤りがある。
    This box contains old photographs.この箱には古い写真が入っている。
  • contain + one's + 感情名詞 ／ contain oneself（感情を）抑える、こらえる
    He couldn't contain his joy.彼は喜びを抑えられなかった。
    She could barely contain herself.彼女は自分を抑えるのがやっとだった。
  • be contained in + 場所〜の中に含まれている（受動態）
    The answer is contained in chapter three.答えは第3章に含まれている。
  • contain + the spread／fire／outbreak 等（火災・病気などの拡大を）封じ込める、食い止める
    Doctors are working to contain the outbreak.医師たちは感染拡大の抑え込みに取り組んでいる。

他のNGSL語への置き換え

contain → hold
元: This bottle contains about two liters of water.
後: This bottle holds about two liters of water.
訳: このボトルは約2リットルの水が入る容量がある。
📝 contain は「実際に今、中に入っている」ことを述べるのに対し、hold は「収容できる・入れられる」という容量・キャパシティのニュアンスが強い。空のボトルの容量を言う場合は hold の方が自然なことも多い。
contain → control
元: She tried to contain her anger, but her hands were shaking.
後: She tried to control her anger, but her hands were shaking.
訳: 彼女は怒りを制御しようとしたが、手が震えていた。
📝 contain は「あふれそうなものを内側に堰き止める」という一時的・物理的な抑制のイメージが強く、control は「制御する」というより一般的・持続的な意味合い。とっさに込み上げる感情を抑える場面では contain がより生き生きした表現になる。
contain → stop
元: If the fire had not been contained quickly, it could have spread to the whole building.
後: If the fire had not been stopped quickly, it could have spread to the whole building.
訳: もしその火事がすぐに止められていなかったら、建物全体に燃え広がっていたかもしれない。
📝 contain は「範囲を限定して封じ込める」という専門的・的確なニュアンスを持ち、消防や報道でよく使われる。stop はより一般的な「止める」全般を指し、範囲を限定するニュアンスは薄い。

  • contain oneself（自分を抑える、平静を保つ）

⚠️ 日本人の典型誤用
✕ This drink contains of caffeine.
「この飲み物にはカフェインが含まれている」と言いたかったが、of を付けると不自然な文になり、consist of と混同していることがネイティブに伝わってしまう。
○ This drink contains caffeine.
この飲み物にはカフェインが含まれている。
This drink contains caffeine.
consist of との混同で contain の後に不要な of を付けてしまう誤り。contain は他動詞で目的語を前置詞なしで直接取る。
✕ This box is containing old photos.
「この箱には古い写真が入っている」と言いたかったが、進行形にすると不自然で、まるで箱が今この瞬間だけ一時的に写真を『含んでいる最中』であるかのような奇妙な響きになる。
○ This box contains old photos.
この箱には古い写真が入っている。
This box contains old photos.
contain は「今まさに入っている」という一時的な動作ではなく、恒常的な状態を表す状態動詞なので、原則として進行形にしない。
✕ He tried to contain from his anger.
「彼は怒りを抑えようとした」と言いたかったが、from を挟むと文法的に破綻し、refrain from と混同しているように聞こえてしまう。
○ He tried to contain his anger.
彼は怒りを抑えようとした。
He tried to contain his anger.
contain は他動詞で「〜を抑える」の目的語を直接取り、from のような前置詞は不要（refrain from との混同）。
🗾 カタカナ語とのギャップ

contain 自体のカタカナ表記は一般的ではないが、派生語「コンテナ」(container) はよく使われるため、con- を「コン」と強く読んでしまいがち。実際は第1音節は弱形の schwa /kən/ で軽く「クン」程度に流し、強勢は第2音節の /ˈteɪn/（テイン）に置く。「コンテイン」と均等に読まない。

🌱 Word family・語源
container（名詞）容器、コンテナ
content（名詞）中身、内容（物）；（含有）量NGSL
containment（名詞）封じ込め、抑制
語源: ラテン語 continere（con-「共に」＋ tenere「保持する、つかむ」）に由来。古フランス語 contenir を経て、中英語 conteinen として英語に入った。
content（中身、内容）— con- + tenere で「中に保持されているもの」が原義→容器の中身
continent（大陸）— con- + tenere で「（陸地を）ひとまとめに保持するもの」が原義
maintain（維持する）— manu-（手で）+ tenere（保持する）で「手で保ち続ける」
💡 覚え方: ラテン語 tenere「つかむ・保持する」を芯に持つ語。con-（共に）+ tenere で『ひとまとめに保持する』→『中に入れておく／含む』。content（中身）は「保持されているもの」、continent（大陸）は「陸地をまとめ保持するもの」、maintain（維持する）は「手で保ち続ける」— どれも同じ tenere の血統。
⚖️ 類義語の使い分け
include — include は「全体の中の一部として持つ・数え入れる」という意味合いが強く、リストや集合の要素を挙げる文脈でよく使われる。contain は容器や物理的・抽象的な『入れ物』の中に実際に何が入っているかを中立的に述べる語。
hold — hold は「収容できる容量・キャパシティ」に焦点が当たることが多い（This hall holds 500 people.）。contain は容量ではなく『実際に今、中に入っている』内容物そのものに重点がある。
restrain — 感情を抑える意味での contain と近いが、restrain は『（人や力を）外部から抑えつける』ニュアンスが強く、他者の行動を制止する文脈でも使われる。contain は自分の内側にあるものを堰き止めるイメージが中心。
💬 熟語・慣用表現 (1)
self-contained 自己完結型の、独立した（住居・設備などが他に頼らず必要なものを内部に備えている様子。人については感情を表に出さない、他人に頼らないの意）中立／ビジネス
The cottage is completely self-contained, with its own kitchen, bathroom, and entrance.（そのコテージは専用のキッチン、バスルーム、出入り口を備え、完全に独立している。）
🔍 「自分自身（self）を内に含んでいる（contained）」が文字通りの意味だが、そこから転じて「必要なものがすべて内部に揃っていて外部に依存しない」という比喩的な意味で使われる。self- と contained を単純に足しただけでは学習者には意味を予測しにくい定型表現。
📝 住居・ユニット・システムなど「他と切り離されて単独で機能するもの」を形容する際によく使う。人の性格を形容すると a self-contained person（感情を表に出さず、他人に頼らない人）という意味になる。
