project

名詞・動詞米 /ˈprɑːdʒekt/（名詞）；/prəˈdʒekt/（動詞） ／ 英 /ˈprɒdʒekt/（名詞）；/prəˈdʒekt/（動詞）pro·ject（2音節）CEFR A2

🗣 名詞用法は第1音節に強勢 /ˈprɑːdʒekt/、動詞用法は第2音節に強勢 /prəˈdʒekt/。この名詞→動詞の強勢移動は record, present, contract などと同じパターンで、品詞を聞き分ける最大の手がかりになる。動詞のときは第1音節の母音が弱まって曖昧母音 /ə/ になる（「プロ」ではなく軽く「プロ」寄りの弱い音）。

  1. 【名詞】計画、事業、企画（She's working on a new project at school. 彼女は学校で新しいプロジェクトに取り組んでいる。）
  2. 【名詞】(米用法で)公営住宅地区、団地（He grew up in a housing project in the city. 彼は市内の公営住宅地区で育った。）
  3. 【動詞】(データ・傾向から)予測する、見積もる（The report projects strong sales growth. その報告書は売上の大幅な伸びを予測している。）
  4. 【動詞】(光・映像を)映し出す、投影する；(声を)張る、届かせる（project an image onto a screen スクリーンに映像を映し出す）
  5. 【動詞】(建物・岩などが)突き出る、張り出す（The rocks project out over the sea. その岩は海の上に突き出ている。）

語形変化

形種類発音備考
projects名詞:複数形/ˈprɑːdʒekts/名詞の複数形。語末は無声子音クラスターなので /s/（有声化しない）。強勢は第1音節のまま変わらない。
projects動詞:三人称単数現在/prəˈdʒekts/動詞の三単現。綴りは名詞複数形と同じだが、強勢が第2音節に移る点が異なる（文脈でしか区別できない）。

projectinging形/prəˈdʒektɪŋ/規則変化。動詞用法のみでとる形（名詞に -ing は付かない）。

語法マーク

  • 名詞は可算名詞。単数は a/the project、複数形は projects。
  • 動詞は名詞と綴りが同じだが強勢移動で区別する（名詞は第1音節、動詞は第2音節）。record, present, contract, produce など英語の名詞・動詞同形語に共通するパターン。
  • 「予測する」の意味では他動詞。目的語に that節・名詞句のどちらも取れる: project (that) sales will rise / project a rise in sales。受動態 be projected to do もよく使われる。
  • 「映し出す」「(声を)張る」の意味でも他動詞: project an image onto the screen / project your voice。
  • 「突き出る」の意味では自動詞: The balcony projects over the street.
  • 他品詞での用法: 心理学由来の用法として、自分の感情・欠点を無意識に他人へ『投影する』意味の project (onto) がある: He's projecting his own insecurities onto his students.（彼は自分自身の不安を生徒に投影している。）日常会話でも「それってあなたの思い込みを相手に投影してるだけだよ」を You're projecting. と言う口語表現がある。また project + 抽象名詞で「(イメージ・印象を)醸し出す、与える」の意味もある: project confidence（自信があるように見せる）、project an image of success（成功者というイメージを演出する）。

文型・頻出パターン

  • project + O（名詞用法）計画・事業に取り組む（軽動詞的なコロケーション）
    work on a projectプロジェクトに取り組む
    launch a projectプロジェクトを立ち上げる
    a research project研究プロジェクト
  • project (that) + S + V〜と予測する、見積もる（動詞:予測）
    Analysts project that demand will double by 2030.アナリストたちは需要が2030年までに倍増すると予測している。
  • be projected to + 動詞の原形〜すると予測されている（受動態＋to不定詞）
    Sales are projected to grow by 15% next year.売上は来年15%伸びると予測されている。
  • project + O + onto + O〜を…に映し出す／(感情などを)〜に投影する
    project an image onto a screenスクリーンに映像を映し出す
    project your feelings onto someone自分の感情を誰かに投影する
  • project + 抽象名詞(イメージ・印象を)醸し出す、与える
    project confidence自信があるように見せる
    project an image of success成功者のイメージを演出する
  • project + 副詞(out/over など)（自動詞）(建造物・岩などが)突き出る、張り出す
    The rocks project out over the sea.その岩は海の上に突き出ている。

他のNGSL語への置き換え

project → displayed
元: The map was projected onto the wall so that everyone could see the route clearly.
後: The map was displayed on the wall so that everyone could see the route clearly.
訳: 地図は壁に表示され、みんながルートをはっきり見られるようになっていた。
📝 project は光や画像を機器（プロジェクター）で「投影する」ことに限定されるニュアンスがあるのに対し、display はもっと一般的に「表示する、展示する」を意味し、投影技術を伴わないポスターや看板の掲示にも使える。
project → predict
元: If the current trend continues, experts project a sharp rise in electricity demand.
後: If the current trend continues, experts predict a sharp rise in electricity demand.
訳: 今の傾向が続けば、専門家たちは電力需要が急増すると予測している。
📝 project は過去のデータや傾向に基づいた数値的な推計（projection）を指すことが多く、統計・経済・気象などでよく使われるのに対し、predict はより一般的な「予測する」で、根拠が直感や経験の場合にも使える。project の方がやや客観的・定量的なニュアンスを持つ。
project → task
元: Don't start a new project until you've finished the one you're currently working on.
後: Don't start a new task until you've finished the one you're currently working on.
訳: 今取り組んでいるものを終えるまでは、新しい作業を始めないで。
📝 project は複数のステップや一定の期間を要する比較的大きな取り組み・事業を指すのに対し、task はその中の個々の作業・仕事単位を指すことが多い。project が「企画全体」であるのに対し、task は「一つの作業」という規模感の違いがある。

⚠️ 日本人の典型誤用
✕ We project to increase our sales next year.
「来年は売上を伸ばす予定です」と言いたかったが、project に to不定詞を続けると英語として不自然に響き、plan/intend との混同に聞こえてしまう。
○ We project that we will increase our sales next year.
私たちは来年、売上が伸びると予測している。
We project that we will increase our sales next year.
We're projecting an increase in sales next year.
日本語「〜する予定だ」の感覚で plan to のパターンを project に流用してしまう誤り。project(予測する)は plan to のように to不定詞を直接目的語に取らず、that節や名詞句を続けるのが基本。
✕ Please protect the presentation slides onto the screen.
「スクリーンにスライドを映してください」と言いたかったが、protect(守る)を使うと「スライドを守ってください」という全く違う意味になってしまう。
○ Please project the presentation slides onto the screen.
スクリーンにスライドを映してください。
Please project the presentation slides onto the screen.
project(投影する)と protect(守る)は綴りが似ているため混同しやすい。特に project の名詞義(プロジェクト)からの連想で言い誤ったり、タイプミスでそのまま書いてしまうケースが多い。
✕ I'm projecting to finish the report by Friday.
「金曜までにレポートを終わらせるつもりです」と言いたかったが、個人の予定に project を使うと、統計や見積もりのような他人事めいた不自然な響きになる。
○ I'm planning to finish the report by Friday.
金曜までにレポートを終わらせるつもりです。
I'm planning to finish the report by Friday.
個人の予定・意志を表すときに project を使ってしまう誤り。project は主にデータや傾向に基づく客観的な予測（ビジネス・統計の文脈）に使う語で、自分個人のスケジュールの「〜するつもり」には plan to や intend to の方が自然。
🗾 カタカナ語とのギャップ

カタカナ「プロジェクト」は名詞用法の強勢パターンしか反映しておらず、動詞用法(第2音節に強勢 /prəˈdʒekt/)との違いを示せない。また最初の母音は実際には /ɑː/（「プラ」に近い音）であり、カタカナの「プロ」/oʊ/寄りの音とはズレがある。動詞では第1音節が弱まって曖昧母音 /ə/ になるため、はっきり「プロ」と発音すると不自然になる。

🌱 Word family・語源
projection（名詞）予測、見積もり；投影、映写
projector（名詞）映写機、プロジェクター
projectile（名詞・形容詞）飛翔体、発射体；推進性の
語源: ラテン語 proicere（前へ投げる）の過去分詞 proiectus に由来。pro-（前へ）+ iacere（投げる）という要素から成り、「前方に投げ出す」が原義。同じ iacere/ject語根を持つ英単語には object（対して投げる→反対する・対象）、subject（下に投げる→服従させる・主題）、reject（後ろへ投げる→拒否する）などがあり、いずれも『投げる』のイメージでつながる。
projection（予測、投影）— project + -ion（動作・結果を表す名詞化接尾辞）で「投げ出す(こと)」→予測・投影という結果を表す
projector（映写機）— project + -or（〜する人・物）で「投げ出すもの」→光や映像を投げ出す機械
💡 覚え方: ラテン語 jacere「投げる」のイメージで覚える。project は pro-（前へ）+ ject（投げる）なので『前方へ投げ出す』―計画を未来へ投げ出す(予定・予測)、光や声を前へ投げ出す(投影・張る)、岩が前へ突き出る、というふうに全部『前に投げる』でつながる。仲間の object(反対に投げる→反論)、reject(後ろに投げ返す→拒否)も同じ根っこ。
⚖️ 類義語の使い分け
forecast — forecast は天気・経済など数値データに基づく将来予測でよく使われ、project よりもさらに専門的・統計的な響きを持つ。project は予測に加え、計画や投影の意味も持つ点で意味範囲が広い。
predict — predict はより一般的な『予測する』で、直感や経験に基づく予測にも使える。project はデータ・傾向からの推計に限定されやすく、ビジネスや統計の文脈でよく使われる。
plan — plan は個人の意図・計画一般を指す基本語。project(名詞)は複数人・複数段階からなる規模の大きい継続的な取り組みを指すことが多く、plan よりも組織的・長期的なニュアンスを持つ。
💬 熟語・慣用表現 (2)
pet project 個人的にお気に入りの企画、特別に肩入れしているプロジェクト中立
The community garden was Mr. Davis's pet project, and he tended it every weekend.（その地域の庭園はデイビスさんが特に力を入れているプロジェクトで、彼は毎週末そこの手入れをしていた。）
🔍 pet はここでは「ペット、愛玩動物」ではなく「特別にかわいがっている、個人的に思い入れの強い」という形容詞的用法。pet theory（お気に入りの持論）、pet peeve（個人的にいらだつこと）と同じ“pet + 名詞”パターンで、project 単体の意味からは pet が付く理由は予測できない。
📝 本来の業務とは別に、個人的な思い入れが強い企画を指すことが多く、他人からは大して重要視されていない場合にも使われる。同パターンの pet theory、pet peeve と合わせて覚えるとよい。
vanity project 自己満足のための企画、見栄・自己顕示のために始めたプロジェクト中立／やや批判的
Critics dismissed the billionaire's museum as a vanity project rather than a genuine contribution to the arts.（批評家たちは、その億万長者の美術館を芸術への真の貢献ではなく、自己満足のためのプロジェクトだとこき下ろした。）
🔍 vanity（虚栄心）+ project。実用性や需要よりも、本人のエゴやイメージ作りのために行われる企画を指し、批判的なニュアンスを伴う点が project 単体からは読み取れない。
📝 主に映画・建築・出版などの分野で、実力や必要性よりも本人の名声・自己顕示欲のために行われた企画を揶揄する際に使われる。
