set

動詞・名詞・形容詞米 /set/ ／ 英 /set/set（1音節）CEFR A2

🗣 1音節語。母音は短母音 /e/ で、日本語の「エ」よりやや口を大きく開けて短く鋭く発音する。sat /sæt/（座った）と母音の質が違うので混同しないこと。語末の /t/ は母音を伴わない

  1. 【動詞】(物を)ある場所・状態に置く、据える（Set the box down carefully. その箱をそっと下に置いて）
  2. 【動詞】(目標・時刻・ルールなどを)設定する、決める（They set a new deadline for the project. 彼らはプロジェクトの新しい締め切りを設定した）
  3. 【動詞】(太陽・月が)沈む（The sun sets early in winter. 冬は日が早く沈む）
  4. 【動詞】(ゼリー・接着剤・骨などが)固まる（Let the glue set overnight. 接着剤を一晩置いて固めて）
  5. 【名詞】ひとそろい、一式（a set of dishes 食器ひとそろい）
  6. 【名詞】(映画・演劇の)舞台装置、セット；(テニスなどの)セット（the final set of the match 試合の最終セット）
  7. 【形容詞】(あらかじめ)決まった、固定の（at a set price 決まった価格で）
  8. 【形容詞】(準備が)整った、用意ができた（叙述用法：be all set）（Are you all set to go? もう出発の準備はできた?）

語法マーク

  • 文字通り「物を置く」の意味では目的語の後に場所を示す語句を伴うことが多い（set O + on/in/at ...）。put とほぼ同義だが、set は「意図的にきちんと据え置く」ニュアンスがやや強い
  • 「設定する・決める」の意味では抽象名詞を目的語にとる軽動詞構文が生産的（set a goal / a record / a price / a date）
  • 「(太陽・月が)沈む」の意味では自動詞。The sun sets / set / has set のように使う
  • 「(ゼリー・接着剤などが)固まる」の意味でも自動詞（Let the jelly set overnight.）
  • 形容詞用法は限定用法（a set price）にも叙述用法（be all set）にも使われる。be set on 〜「〜と心に決めている」という用法もある
  • 他品詞での用法: 名詞用法もある。(1) 数学・論理学の専門用語としての「集合」: a set of numbers（数の集合）。中学・高校数学で習う「集合」はこの set の訳語で、カタカナ「セット理論」もここから。(2) 主に古風・くだけた言い方で「(共通の趣味・階層でつながる)仲間、連中」を指す用法: the fast set（遊び好きの連中）。現代の日常会話ではやや古めかしく、clique や crowd の方が普通に使われる。

文型・頻出パターン

  • set + O + 場所を表す句〜を（場所）に置く・据える
    Set the vase on the shelf.花瓶を棚に置いて。
  • set + O + to + 値/状態〜を（値・状態）に設定する
    Set the oven to 200 degrees.オーブンを200度に設定して。
    Set your phone to silent mode.スマホをマナーモードに設定して。
  • set + 抽象名詞（軽動詞構文）目標・記録・日付・価格などを設定する・決める
    set a goal目標を設定する
    set a record記録を樹立する
    set a date日取りを決める
  • set + O + doing〜が…し始めるようにする（使役的）
    The news set everyone talking.そのニュースはみんなを話題騒然にさせた。
  • be set in/at + 場所（受身）（物語・映画などの）舞台が〜に設定されている
    The novel is set in 19th-century London.その小説は19世紀のロンドンを舞台にしている。
  • (the sun/moon) set(太陽・月が)沈む（自動詞）
    The sun set behind the mountains.太陽が山の向こうに沈んだ。

他のNGSL語への置き換え

set → broken
元: A new world record was set by the runner during yesterday's race.
後: A new world record was broken by the runner during yesterday's race.
訳: 昨日のレースで、そのランナーによって新記録が破られた。
📝 break a record は既存の記録を更新して上回るニュアンスを持つ。set a record はゼロから新たに記録を打ち立てることを表し、前人未到の記録が初めて生まれた場合にも使われる。broken はすでにある基準を超えたことを強調する対比的な語。
set → established
元: The rules that the committee set last year are still in effect today.
後: The rules that the committee established last year are still in effect today.
訳: 委員会が去年制定した規則は、今日でも有効だ。
📝 establish はより公式的・制度的なニュアンスを持ち、法律や制度を正式に定める場面で好まれる硬い語。set はより中立的・口語的な「決める」という意味合いが強く、日常的なルール決めにも使える。
set → switch
元: Set your phone to silent mode before the movie starts.
後: Switch your phone to silent mode before the movie starts.
訳: 映画が始まる前に、スマホをマナーモードに切り替えて。
📝 switch は「ある状態から別の状態へ切り替える」という変化そのものに焦点を当てる動詞。set はより中立的に「設定する」ことを表し、初期設定にも変更にも使える。switch を使うと「今の状態から変える」という動作性がより明確になる。

⚠️ 日本人の典型誤用
✕ I setted my alarm for six o'clock.
「アラームを6時にセットした」と言いたかったが、setted という語形は英語には存在せず、単なる自然さの欠如ではなく明確な誤りとして響く。
○ I set my alarm for six o'clock.
アラームを6時にセットした。
I set my alarm for six o'clock.
set は原形・過去形・過去分詞がすべて同形の不規則動詞。規則動詞と誤って類推し -ed を付けて setted としてしまう誤り。
✕ Please set down on this chair.
「この椅子に座ってください」と言いたかったが、set は他動詞で「(物)を据える」という意味なので、目的語が抜けた不完全な文に聞こえ、何を据えるのか分からず戸惑わせてしまう。
○ Please sit down on this chair.
この椅子に座ってください。
Please sit down on this chair.
set（据える、他動詞で目的語が必要）と sit（座る、自動詞）は形も意味も似ているため混同しやすい。「(人が)座る」は sit を使う。
✕ I bought set of dishes for my new apartment.
「新しいアパートのために食器ひとそろいを買った」と言いたかったが、冠詞 a が抜けているため、数えられる名詞の前に何もない据わりの悪い不自然な文に聞こえる。
○ I bought a set of dishes for my new apartment.
新しいアパートのために食器ひとそろいを買った。
I bought a set of dishes for my new apartment.
「set of ~」（ひとそろいの~）の set は可算名詞で、単数なら冠詞 a が必要。日本語には冠詞がないため抜け落ちやすい。
🗾 カタカナ語とのギャップ

カタカナ「セット」は「セ・ッ・ト」で促音を含む3拍だが、英語の set は1音節。語末の /t/ は母音を伴わず（「ト」と読まない）、破裂を伴わないまま次の音へ移ることも多い。また「セット」は名詞（ひとそろい・舞台装置・髪のセット）のイメージが強く、日常的な動詞用法（置く・設定する・沈む・固まる）を連想しにくい点にも注意。

upset（動詞・形容詞・名詞）動揺させる；動揺した；番狂わせNGSL

語源: 古英語 settan「座らせる、据える」に由来。これは sittan（sit の古い形）の使役形（〜させる動詞）で、文字通り「座らせる」が原義。ゲルマン語派に共通し、ドイツ語 setzen、オランダ語 zetten と同語源。
upset（動揺させる、ひっくり返す）— up（上下逆に）+ set（据える）→「ひっくり返して据える」から「動揺させる」へ転じた
onset（(病気などの)始まり）— on（〜に向けて）+ set →もともと「攻撃を仕掛ける」の意から「(病気などが)始まる」へ転じた
subset（部分集合）— sub（下位の）+ set（集合）→ 全体集合の中の一部分の集合
💡 覚え方: sit（座る）の使役形が set ―「座らせる」から「物を据え置く」「状態を固定する」へ意味が広がった動詞。upset は「ひっくり返して据える」→動揺させる、subset は「集合(set)の下位」→部分集合、と派生語も『据える・集合』のイメージでつながる。

⚖️ 類義語の使い分け
put — put はより一般的・中立的な「置く」動詞。set はより意図的に「きちんと据え置く」ニュアンスや、「設定する・決める」という抽象的な用法を持つ点で put より意味の幅が広い。
place — place は丁寧・慎重に「配置する」というフォーマルな響きを持つ。set はより日常的で頻度が高く、置く・固定する・設定する・沈むなど幅広い場面で使える汎用性の高い語。
establish — establish は制度・組織・事実などを公式に「確立する」硬い語。set は「(ルール・目標・記録などを)決める・設定する」というより軽く日常的な語で、establish の方がフォーマルで永続性を含意する。
💬 熟語・慣用表現 (8)
set foot in/on 〜に足を踏み入れる、〜に行く中立
I had never set foot in Japan before this trip.（この旅行の前は、日本に一度も行ったことがなかった。）
🔍 「足を据える」という直訳からは「入る・訪れる」の意味は予測しにくい。多くは否定文や完了形で「一度も行ったことがない」を強調する際に使われる。
📝 否定文（never set foot in）で使われることが非常に多い。単に「訪れる」だけでなく、初めて・久しぶりにという含みを持つことが多い。
set in stone 確定している、変更できない中立
These plans aren't set in stone, so feel free to suggest changes.（この計画はまだ確定ではないので、遠慮なく変更を提案してください。）
🔍 「石に刻まれて据えられている」という比喩から「絶対に変わらない」の意味へ。文字通りの「据える」からは決定事項というニュアンスは読み取れない。
📝 多くは否定形（not set in stone）で「まだ変更の余地がある」と伝える際に使われる。
set the stage for 〜のお膳立てをする、〜の下地を作る中立
The peace agreement set the stage for years of economic growth.（その和平合意が、その後何年もの経済成長のお膳立てとなった。）
🔍 舞台（stage）を設営するという文字通りの意味から転じ、比喩的に「次に起こることの条件を整える」を表す。物理的な舞台とは無関係な文脈でも使われる。
📝 ニュース記事や歴史的な出来事の説明でよく使われるやや書き言葉寄りの表現。
set your sights on 〜を目標に定める、〜を狙う中立
After the win, she set her sights on the world championship.（その勝利の後、彼女は世界選手権を目標に定めた。）
🔍 「照準（sights）を据える」という射撃由来の比喩表現。文字通りの意味からは「強い意欲を持って目指す」というニュアンスは推測しにくい。
📝 set a goal（目標を設定する）よりも意志の強さ・野心を強調する言い方。
set the record straight 誤解を正す、事実をはっきりさせる中立
I want to set the record straight: I never said that.（はっきりさせておきたいのですが、私はそんなことは一言も言っていません。）
🔍 「記録をまっすぐに据える」という直訳からは「誤った情報を訂正する」という意味は読み取れない。record は公式な記録・世間の認識を指す比喩。
📝 誤解や噂を公に訂正する場面で使われる。for the record（念のため言っておくと）と混同しないよう注意。
be dead set against/on 断固反対している／〜を固く決意している口語
My parents are dead set against me moving abroad alone.（両親は私が一人で海外に移住することに断固反対している。）
🔍 dead は「完全に、徹底的に」を強める副詞的用法。against なら断固反対、on なら固い決意を表し、文字通りの「据える」からは意味が推測できない。
📝 against（反対）と on（決意）で意味が逆になる点に注意。感情の強さを伝えるくだけた表現。
all set 準備万端で、用意ができて口語
Grab your coat, and we'll be all set to leave.（コートを取ったら、もう出発の準備は万端だよ。）
🔍 「すべて設定された」状態からは「準備が完全に整った」という意味は直接読み取れない、慣用化した形容詞的表現。
📝 be all set (to do / for ~) で「準備ができている」の意。別れ際に単独で Are you all set? （もう大丈夫？）とも使われる。
set eyes on 初めて〜を見る、〜に目を留める文語的／やや口語
The moment I set eyes on the painting, I knew I had to buy it.（その絵を初めて目にした瞬間、私はそれを買わなければと確信した。）
🔍 「目を据える」という直訳からは「(初めて)目にする」という意味は推測しにくい。しばしば first と共に使われる。
📝 the first time I set eyes on ~ のように「初めて見た時」を強調する表現としてよく使われる。過去形でも set のまま変化しない。
