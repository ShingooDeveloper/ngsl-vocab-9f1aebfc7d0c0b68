add

動詞 (verb)米 /æd/ ／ 英 /æd/add（1音節）CEFR A1

🗣 1音節語。母音は短母音 /æ/ で、日本語の「ア」よりも口を大きく横に開いて出す。語末の /d/ は弱く破裂させる程度でよい

  1. 【動詞】（材料・数量などを）加える、足す（Add a pinch of salt to the soup. スープに塩をひとつまみ加えて。）
  2. 【動詞】（言葉を）付け加える、言い添える（She added that she would be late. 彼女は遅れると付け加えた。）
  3. 【動詞】（数を）合計する、足し算する（Add these two numbers together. この2つの数字を足して。）
  4. 【動詞】（価値・魅力などを）増す、加える（New paint adds value to an old house. 新しい塗装は古い家の価値を高める。）

語形変化

形種類発音備考
adds三単現/ædz/語末は有声音なので /z/（「アッツ」ではなく「アッズ」に近い）
added過去形・過去分詞/ˈæɾɪd/規則変化。語末が /d/ で終わるため -ed は独立音節 /ɪd/ になり2音節に。米音では語中の d が母音に挟まれフラップ化する
addinging形/ˈædɪŋ/規則変化。語尾の d を重ねずそのまま -ing を付ける

語法マーク

  • 他動詞が基本。目的語を必ず伴う（add A、または add A to B の形が中心）
  • give/tell のような SVOO（add + 人 + 物）にはならない。人を関わらせる場合は add A for 人、または add A to 人's ... のように前置詞句で表す
  • add that + 節 で「〜と言い添える」という発言内容の追加を表せる
  • 受動態にしやすい動詞（A is added to B）。レシピ・手順・会計などの説明文で頻出
  • 進行形にできる動作動詞（is adding, was adding など）
  • 他品詞での用法: 基本的に動詞専用語で、独立した名詞用法はほとんどない。編集・出版業界などのごく限られたインフォーマルな文脈で「追記した部分」を指して an add（例: We need a quick add to this article.）と使われることがあるが、一般的な用法ではない。

文型・頻出パターン

  • add + O〜を加える・足す（最も基本的な他動詞用法）
    add some pepperコショウを少し加える
    add a new feature新しい機能を追加する
  • add A to BAをBに加える
    Add the eggs to the flour.卵を小麦粉に加えて。
    A tax is added to the price.価格に税金が加算される。
  • add that + 節〜と付け加えて言う（発言内容の追加）
    She added that she was tired.彼女は疲れているとも言い添えた。
  • add A and B (together)AとBを足す（数の合計を求める）
    Add 5 and 3 together to get 8.5と3を足すと8になる。
  • add to + N（目的語なしの自動詞的用法）〜を増す、〜の一因となる
    The delay only added to our frustration.その遅れが私たちのいら立ちをさらに増しただけだった。

他のNGSL語への置き換え

add → introduced
元: The feature that they added last month made the app much faster.
後: The feature that they introduced last month made the app much faster.
訳: 彼らが先月導入した機能のおかげで、アプリはずっと速くなった。
📝 introduce は「新しく導入する・取り入れる」というニュアンスで、addよりもフォーマルで大掛かりな変更・発表を暗示することが多い。addは単に何かを付け加える中立的な語。
add → included
元: A service charge is added to every bill automatically.
後: A service charge is included in every bill automatically.
訳: サービス料はすべての請求書に自動的に含まれる。
📝 include は「（最初から）含む」というニュアンスで、addのように後から加える動作ではなく、元々組み込まれていることを示す。前置詞もto→inに変わる点に注意。
add → pour
元: If you add too much water, the mixture will become too thin.
後: If you pour too much water, the mixture will become too thin.
訳: 水を注ぎすぎると、その混ぜたものは薄くなりすぎてしまう。
📝 pour は「（液体を）注ぐ」という具体的な動作を表し、addよりも液体を加える様子を視覚的に描写する。addは液体に限らず一般的に何かを加える際に使う、より広い語。

⚠️ 日本人の典型誤用
✕ Please add me more time.
「もっと時間を加えてください」のつもりだったが、addをgiveのように人を直接目的語にとる形で使っており、英語としては不自然で意味が通りにくい。
○ Please give me more time.
もっと時間をください。
Please give me more time.
give や tell と同じ感覚で add を「add + 人 + 物」のSVOOで使ってしまう誤り。add にはこの型はなく、人に何かを与える意味なら give を使うか、add A for 人 のように前置詞を補う必要がある。
✕ Add, I like coffee too.
「それに加えて、コーヒーも好きです」と言いたかったが、動詞addを接続表現のように文頭でそのまま使っており、英語として不自然。
○ In addition, I like coffee too.
それに加えて、コーヒーも好きです。
In addition, I like coffee too.
日本語の接続詞的な「加えて」を動詞 add でそのまま文頭に置いてしまう誤り。add はあくまで他動詞で、文をつなぐ副詞句の働きはしない。「加えて／それに加え」は in addition や additionally で表す。
✕ I have add two more eggs to the bowl.
「ボウルに卵をあと2つ加えた」と言いたかったが、have の後の動詞を原形のままにしてしまい、文法的に誤り。
○ I have added two more eggs to the bowl.
ボウルに卵をあと2つ加えた。
I have added two more eggs to the bowl.
現在完了 have/has の後で過去分詞に変化させ忘れる誤り。addは規則変化なので -ed を必ず付けて added にする必要がある。
🗾 カタカナ語とのギャップ

add単体のカタカナ語は一般的ではないが、「アドオン（add-on）」「アディショナルタイム（additional time）」のように関連語がカタカナ化している。英語の/æ/は「ア」と「エ」の中間のような口を大きく開けた音で、日本語の「アド」の「ア」より明るく強い音になる。また語末の/d/には母音を付けず、「アドゥ」のように伸ばさない。

🌱 Word family・語源
addition（名詞）追加、足し算、加算NGSL
additional（形容詞）追加の、付加的なNGSL
additionally（副詞）加えて、その上
add-on（名詞）追加部品、追加機能
added（形容詞）（形容詞的に）追加された、増した（an added bonus 思わぬ余得）
語源: ラテン語 addere（ad-「〜へ向けて」+ dare「与える、置く」）に由来。「〜に向けて与える・置く」が原義で、そこから「加える」の意味が生まれ、14世紀後半に中英語 adden として英語に取り入れられた。
addition（追加、足し算）— add + -ition（名詞化語尾）で「加えること」
additional（追加の）— addition + -al（形容詞化語尾）で「追加に関する」
add-on（追加部品）— add + on（副詞辞）が名詞化し「加えて乗せるもの」
💡 覚え方: ラテン語 ad-（〜へ）+ dare（与える）＝「そこへ何かを与え置く」イメージ。塩を鍋に「置き与える」→「加える」、会話に一言「置き与える」→「言い添える」と考えるとつながる。
🔗 句動詞
add up — 合計が合う、つじつまが合う
His story just doesn't add up.（彼の話はどうもつじつまが合わない。）
add in — （追加で）加える、含める
Don't forget to add in the shipping cost.（送料を加えるのを忘れないで。）
add on — 上乗せする、追加で付け加える
They added on an extra fee at the last minute.（彼らは土壇場で追加料金を上乗せした。）
⚖️ 類義語の使い分け
include — include は「（すでにある集合・範囲の中に）含む」という状態を表し、後から加える動作を意味しない点でaddと異なる。addは外から新しく足す動作そのものに焦点がある。
increase — increaseは量・程度が「増える／増やす」こと全般を表す抽象度の高い語で、具体的に何を足すかを問わない。addは足す対象（物や言葉）を目的語にとる、より具体的で動作寄りの語。
append — appendは文書やリストの「末尾に付け加える」ことを指すフォーマルな語で、主に書き言葉・事務的な文脈で使う。addは口語でも書き言葉でも使える中立的で汎用性の高い語。
💬 熟語・慣用表現 (2)
add insult to injury 傷口に塩を塗る、追い打ちをかける、踏んだり蹴ったりにする中立（ややインフォーマル寄り）
My flight was delayed for six hours, and to add insult to injury, the airline lost my luggage.（飛行機が6時間も遅れた上に、追い打ちをかけるように航空会社は私の荷物を紛失した。）
🔍 「怪我（injury）に侮辱（insult）を加える」が直訳。すでに傷ついている人にさらに一撃を加えるイメージから転じて、悪い状況にさらに悪いことが重なる場面全般を指す比喩表現として定着している。
📝 多くの場合 to add insult to injury という形で文中に挿入句として使い、「さらに悪いことに」という導入表現になる。悪いことが立て続けに重なる状況で非常によく使われる。
add fuel to the fire 火に油を注ぐ、事態をさらに悪化させる中立
Raising his voice only added fuel to the fire, and the argument got even worse.（彼が声を荒げたことが火に油を注ぐ結果になり、口論はさらにひどくなった。）
🔍 「火に燃料を加える」が直訳。すでに悪化している状況（怒り・対立・噂など）をさらに激化させる比喩で、日本語の「火に油を注ぐ」とほぼ同じ発想の表現。
📝 fire の代わりに flames を使う add fuel to the flames も同義でよく使われる。怒り・論争・噂など感情的・対立的な状況の悪化を表し、実際の火事の描写にはあまり使わない。
