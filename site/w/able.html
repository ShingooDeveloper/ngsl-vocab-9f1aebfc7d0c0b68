able

形容詞 (adjective)米 /ˈeɪbl̩/ ／ 英 /ˈeɪbl̩/a·ble（2音節）CEFR A1

🗣 第1音節に強勢。第2音節は母音を持たず、l が音節主音になる成節子音 l̩（「ブル」と母音を足さず、舌先を歯茎につけたまま息を止めて終える）

  1. 【形容詞】（be able to ~ の形で）〜することができる、可能である
  2. 【形容詞】有能な、能力の高い（an able leader 有能な指導者）

語形変化

形種類発音備考
abler比較級/ˈeɪblɚ/1音節語なので -er 型比較級が可能。ただし書き言葉では more able も同じくらいよく使われ、どちらも正用法。able の l̩ が音節性を失い、そのまま ɚ につながる発音になる点に注意（『エイブラー』ではなく1拍で滑らかに）
ablest最上級/ˈeɪbləst/-est 型最上級。more able / most able も同様によく使われる。one of the ablest ~ のようなやや硬い書き言葉でよく見る形

語法マーク

  • 形容詞。単独の動詞のようには使えず、原則 be動詞と共に be able to + 原形 の形をとる（can/could の代用としてあらゆる時制・形で使える点が can にない利点）
  • able to の後ろは動詞の原形のみ（able to swimming のように動名詞にはできない）
  • 叙述用法（be動詞の補語）が基本。限定用法（名詞の前に置く）で使うのは『有能な』という意味のときのみ（an able assistant）
  • 比較級・最上級は abler/ablest（-er/-est型）と more able/most able（迂言的）の両方が可能。フォーマルな文章では more able/most able がやや好まれる傾向
  • able to の主語は基本的に人など『能力を持つ動作主』。無生物や出来事を主語にして『〜されうる』と言いたい場合は able ではなく can be done や it is possible to を使う
  • 他品詞での用法: able は基本的に形容詞1品詞のみ（多品詞語ではない）。ただし用法は大きく2つに分かれる：(1) be able to ~ の形で述語的に使われ「〜する能力・可能性がある」を表す最頻用法。(2) 名詞を直接修飾する限定用法で「有能な」を表す（an able leader）。この2つ目の意味では、very ではなく highly able / less able のように程度を表す語を選ぶ点に注意（collocations参照）。

文型・頻出パターン

  • be + able to + 原形〜することができる（能力・可能を表す最も基本の形）
    He is able to drive a car now.彼はもう車を運転できる。
    We were able to finish the project on time.私たちは予定通りプロジェクトを終えることができた。
  • not able to（= cannot）〜することができない
    I'm not able to attend the meeting tomorrow.明日の会議には出席できません。
  • able + 名詞（限定用法）有能な〜（名詞を直接修飾する形容詞用法）
    an able assistant有能な助手
    the ablest students in the classクラスで最も優秀な生徒たち
  • as able as ~〜と同じくらい有能だ（原級比較）
    She is as able as any manager I've worked with.彼女はこれまで一緒に働いたどのマネージャーにも劣らず有能だ。

他のNGSL語への置き換え

able → capable
元: She is able to speak three languages fluently.
後: She is capable of speaking three languages fluently.
訳: 彼女は3か国語を流暢に話すことができる（能力がある）。
📝 capable も能力を表すが、able to + 原形とは異なり capable of + 動名詞という形を取る。capable の方がやや硬く、書き言葉的な響きを持つ。
able → ready
元: After a few more lessons, you will be able to swim across the pool.
後: After a few more lessons, you will be ready to swim across the pool.
訳: あと数回レッスンを受ければ、プールを横断する準備ができるよ。
📝 able は「（技能として）できる」という能力そのものを表すのに対し、ready は「準備・心構えができている」という状態を表す。技能の習得を言うなら able、心身の準備を言うなら ready が自然。
able → free
元: If I were able to take a week off, I would travel around Kyushu.
後: If I were free to take a week off, I would travel around Kyushu.
訳: もし1週間休みを取ってよい状況なら、九州を旅して回るんだけどな。
📝 able は「（能力・条件的に）可能だ」という意味なのに対し、free は「（許可・制約の無さから）自由にできる」という意味。休みが取れない理由が制度・立場上の制約なら free、そもそも都合がつかないなら able がより自然。

⚠️ 日本人の典型誤用
✕ I able to play the guitar.
「私はギターが弾ける」と言いたかったが、be動詞が抜けているため英語として文法的に成立しておらず、ネイティブには言いかけの途中の文のように聞こえる。
○ I am able to play the guitar.
私はギターを弾くことができる。
I am able to play the guitar.
can の感覚で able を助動詞のように扱い、be動詞を落としてしまう誤り。able は形容詞なので、必ず am/is/are/was/were などの be動詞とセットで使う。
✕ This problem is able to be solved easily.
「この問題は簡単に解決できる」と言いたかったが、able を無生物主語に使うと、問題自体に意志や能力があるかのような不自然な響きになる。
○ This problem can be solved easily.
この問題は簡単に解決できる。
This problem can be solved easily.
able to は基本的に人など『能力を持つ動作主』を主語にとる。problem のような無生物・出来事を主語にして『〜されうる』と言いたい場合は、able ではなく can be done や it is possible to do を使うのが自然。
✕ She is able speak three languages.
「彼女は3か国語を話せる」と言いたかったが、able の後の to が抜けているため、can と able to の構文を混同した不完全な文に聞こえる。
○ She is able to speak three languages.
彼女は3か国語を話すことができる。
She is able to speak three languages.
able の後ろには必ず to 不定詞を置く必要がある。can のように助動詞の後に動詞の原形を直接続ける形と混同してしまう誤り。
🗾 カタカナ語とのギャップ

カタカナ「エイブル」は「エ・イ・ブ・ル」の4モーラだが、英語は2音節。1音節目は二重母音 /eɪ/ で「エ」から「イ」へ滑らかに移り、2音節目の l は母音を伴わない成節子音 l̩ なので、『ブル』のように余分な母音（ウ）を足して発音しないこと。

🌱 Word family・語源
ability（名詞）能力、才能NGSL
unable（形容詞）〜できない、無能なNGSL
enable（動詞）〜を可能にする、〜できるようにするNGSL
disability（名詞）障がい、（心身の）不自由
disabled（形容詞）障がいのある
disable（動詞）（機能・能力を）不能にする、無効にする
語源: 古フランス語 able を経て、ラテン語 habilis「扱いやすい、有能な」に遡る。habilis はさらにラテン語動詞 habēre「持つ、扱う」に由来する。英語に入った当初は h のない able という綴りだったが、16世紀にラテン語源に近づけようとする綴り字改革の影響で habile という綴りも一時使われた（現代英語では h なしの able に落ち着いている）。
ability（能力）— able + -ity で『できる状態』→能力
habit（習慣）— 同じラテン語 habēre（持つ）に遡る遠い親戚。『身に持っている』というイメージでつながる
exhibit（展示する、示す）— ex-（外に）+ habēre（持つ）→『外に持ち出して見せる』
💡 覚え方: able の語源はラテン語で『持つ・扱う』を意味する habēre。何かを『手に持って自在に扱える』→『〜できる』というイメージ。habit（身に持っている習慣）や exhibit（外に持ち出して見せる）も、たどれば同じ語根 habēre の遠い親戚。
⚖️ 類義語の使い分け
capable — capable は of + 動名詞という形をとる（capable of doing）点が able to + 原形と異なる。意味はほぼ同じだが capable の方がやや硬い書き言葉的な響きを持つ。
competent — competent は『必要な基準・水準を満たしている』という有能さに焦点があり、仕事・専門分野の文脈でよく使われる（a competent engineer）。able より評価のニュアンスが強い。
qualified — qualified は資格・学歴・訓練などの『条件を満たしている』ことを強調する語で、能力そのものというより公式な資格の有無に焦点がある（a qualified doctor）。
