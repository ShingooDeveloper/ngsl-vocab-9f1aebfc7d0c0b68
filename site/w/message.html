message

名詞・動詞米 /ˈmesɪdʒ/ ／ 英 /ˈmesɪdʒ/mes·sage（2音節）CEFR A2

🗣 第1音節に強勢 /ˈmesɪdʒ/。第2音節の母音は曖昧母音 /ɪ/ で弱く短く添えるだけ。語末の-ageはフランス語由来の-age(garage等)のように/ɑːʒ/にはならず、英語化した/ɪdʒ/（「イジ」に近い）で読む点に注意

  1. 【名詞】伝言、(電話・メール・SNSなどで伝える)メッセージ（Did you get my message? 私のメッセージ届いた?）
  2. 【名詞】(映画・本・出来事などが伝える)教訓、メッセージ性（The film has an important message about friendship. その映画には友情についての大切なメッセージが込められている。）
  3. 【動詞】(人に)メッセージを送る、連絡する（I'll message you when I arrive. 着いたらメッセージするね。）

語形変化

形種類発音備考
messages複数形・三単現/ˈmesɪdʒɪz/語末が破擦音/dʒ/なので-esを付けて/ɪz/を追加（名詞の複数形と動詞の三人称単数現在形が同形）

語法マーク

  • 名詞は可算名詞。単数では必ず冠詞（a/the）を伴う（× I got message. は誤り、○ I got a message.）
  • 動詞用法は他動詞。目的語は人が来ることが多く、toを挟まない（message him であって message to him ではない）
  • 「（作品などが伝える）教訓・メッセージ性」の意味では通例単数・the messageの形で使われる（the message of the film）
  • 「伝言」の意味ではleave/take/get/sendなどの軽動詞と組み合わせて使われることが多く、message単独より軽動詞構文での使用頻度が高い
  • 他品詞での用法: 動詞用法（message + 人＝人にメッセージを送る）は、名詞messageから転用された比較的新しい用法（denominal verb）。携帯電話・SNSでの短文のやり取りが一般化した2000年代以降、text（テキストする）と同じパターンで口語に定着した。改まった文章では send a message のように名詞＋軽動詞の組み合わせの方が好まれる。

文型・頻出パターン

  • leave + a message伝言を残す（電話・留守番電話などに）
    leave a message after the beep発信音の後に伝言を残す
    Can I leave a message for him?彼に伝言をお願いできますか?
  • take + a message（電話などで）伝言を預かる
    Can I take a message?伝言を承りましょうか?
    He took a message for his boss.彼は上司宛ての伝言を預かった。
  • message + 人（他動詞の動詞用法）（人に）メッセージを送る（前置詞toを伴わない）
    message me when you get home家に着いたらメッセージして
    I'll message the group later.あとでグループにメッセージするね。
  • the message + (that節／of ~)（作品・出来事が伝える）教訓・メッセージ性
    the message of the storyその物語が伝えるメッセージ性
    I didn't quite understand the message that the film was trying to send.その映画が伝えようとしているメッセージがいまいち理解できなかった。
  • get / receive + a messageメッセージを受け取る
    I got a message from an unknown number.知らない番号からメッセージが来た。
    Did you receive my message?私のメッセージ届いた?

他のNGSL語への置き換え

message → note
元: I got your message about the meeting, but I couldn't reply until this morning.
後: I got your note about the meeting, but I couldn't reply until this morning.
訳: 会議についてのメモは受け取ったけど、今朝まで返事できなかったんだ。
📝 note は紙やメモ書きなど「短い書き置き」のニュアンスが強く、机に置いておく・手渡しするような物理的なメモを連想させる。message は電話・メール・SNS・音声など伝達手段を問わない広い語で、この文脈（会議についての連絡）では message の方が自然。
message → email
元: A text message was sent to all the students before the exam.
後: An email was sent to all the students before the exam.
訳: 試験の前に、全生徒にメールが送られた。
📝 text message はスマートフォンのSMSやアプリで送る短いメッセージを指すのに対し、email はメールアドレスを通じて送る、より公式・詳細な連絡手段。学校からの一斉連絡なら実際には email の方がよく使われる。
message → texted
元: I've already messaged her twice, but she hasn't replied yet.
後: I've already texted her twice, but she hasn't replied yet.
訳: もう2回もあの人にテキストを送ったのに、まだ返事が来ていない。
📝 text は電話番号を使ったSMSに限定される動詞で、message よりも意味範囲が狭い。message はLINEやチャットアプリなど媒体を問わず使える点で汎用性が高い。

⚠️ 日本人の典型誤用
✕ I left message on his desk.
「彼の机にメッセージを残した」と言いたかったが、可算名詞messageが単数形なのに冠詞a/theが抜けており、ネイティブには文法的に不完全でぎこちなく響く。
○ I left a message on his desk.
彼の机にメッセージを残した。
I left a message on his desk.
messageは単数では必ず冠詞を伴う可算名詞。日本語には冠詞という概念がないため、学習者はaやtheを省いてしまいがち。
✕ I received a messages from him.
「彼からメッセージを受け取った」と言いたかったが、単数の冠詞aと複数形messagesが同時に使われていて矛盾した文になっている。
○ I received a message from him.
彼からメッセージを受け取った。
I received a message from him.
冠詞aは単数名詞にしか付けられない。複数形にしたい場合はaを外してmessagesだけにする必要がある。
✕ I'll message to you tomorrow.
「明日あなたにメッセージするね」と言いたかったが、動詞messageはtoを取らない他動詞なので、ネイティブにはtoが余分で不自然に響く。
○ I'll message you tomorrow.
明日あなたにメッセージするね。
I'll message you tomorrow.
動詞のmessageは他動詞で、目的語（人）を直接取る。日本語「〜にメッセージを送る」の「に」に引きずられてtoを入れてしまう誤り。
🗾 カタカナ語とのギャップ

カタカナ「メッセージ」は5モーラだが英語は2音節 /ˈmesɪdʒ/。第1音節「メ」を強く短く発音し、第2音節はほぼ添える程度。語末は伸ばさず有声の/dʒ/（「ジ」）で止める。また綴り・音が似ている「マッサージ」massage /məˈsɑːʒ/ と混同しないよう注意（massageは第2音節に強勢があり、語末母音も異なる）。

🌱 Word family・語源
messenger（名詞）使者、伝令、（アプリの）メッセンジャー
語源: 古フランス語 message「伝言」を経て、俗ラテン語 *missaticum に遡り、さらにラテン語 mittere「送る」の過去分詞形 missus に由来する。mittere は英語に非常に多くの派生語を残した動詞で、message はその「送られたもの」を意味する語。
messenger（使者）— message + -er（送られたものを運ぶ人）＝「使者、伝令」
mission（使命）— mittereの異形語幹miss-からmissio「送ること」→「送られてする仕事＝使命」
promise（約束）— pro-（前もって）+ mittere「送る」＝「先に言葉を送っておく」→「約束」
💡 覚え方: ラテン語 mittere「送る」から生まれた仲間たち。message は「送られたもの＝伝言」、messenger は「それを運ぶ人＝使者」、mission は「送られてする仕事＝使命」、promise は「先に送っておく言葉＝約束」。すべて『何かを人から人へ送り届ける』イメージでつながる。
⚖️ 類義語の使い分け
note — noteは紙などに書く短い書き置きを指し、物理的・手書きのニュアンスが強い。messageは電話・メール・SNSなど伝達手段を問わない広い語。
text — textは携帯電話番号を使ったSMSでのやり取りに限定される動詞・名詞。messageはLINEやチャットアプリなど媒体を問わず使える、より汎用的な語。
notification — notificationはアプリやシステムが自動的に発する通知を指すことが多い。messageは基本的に人から人への意図的な伝達を表す。
💬 熟語・慣用表現 (3)
get the message (それとなく伝えられた意図・態度を)察する、理解する口語
After I stopped returning his calls for a month, he finally got the message and left me alone.（1か月電話を折り返すのをやめたら、彼はようやく察して私にかまわなくなった。）
🔍 「メッセージを受け取る」という文字通りの意味ではなく、言葉にされていない意図やほのめかしを態度から読み取って理解すること。
📝 命令形 Get the message! は「いい加減わかれよ」という苛立ちを込めた強い言い方になる。言葉で直接伝えず、態度や沈黙で意図を伝える文脈でよく使われる。
message received 了解した、わかったよ(時に皮肉を込めて)口語
"Please don't leave your shoes in the hallway again." "Message received."（「もう玄関に靴を置きっぱなしにしないでね」「わかったよ」）
🔍 「メッセージは受信済みだ」という無線・軍隊用語由来の応答表現が、日常会話で「言いたいことは伝わったよ」という意味に転用された定型句。
📝 文の一部としてではなく、それ単独の返答として使う決まり文句。素直な了承にも、少しうんざりした皮肉な同意にも聞こえる。
message in a bottle (誰に届くか分からないまま発信する)当てのないメッセージ中立
Posting the question in an old forum felt like sending a message in a bottle—he never expected a reply.（その古いフォーラムに質問を投稿するのは瓶に手紙を入れて海に流すようなもので、彼は返信なんて期待していなかった。）
🔍 文字通りは「瓶に入れて海に流す手紙」だが、比喩的に「受け取り手が誰か分からないまま発信する伝達」全般を指す。
