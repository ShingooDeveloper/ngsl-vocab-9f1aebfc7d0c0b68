size

名詞 (noun)米 /saɪz/ ／ 英 /saɪz/size（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。母音は二重母音 /aɪ/ で、「ア」から「イ」へなめらかに滑らせて1拍で発音する。語末の /z/ は有声のまま息を止めずに終える（「ザイズ」と2拍に伸ばさない）

  1. 【名詞】大きさ、寸法（サイズ）（What's the size of your suitcase? あなたのスーツケースの大きさはどれくらい?）
  2. 【名詞】(衣類・靴などの)サイズ、号数（Do you have this in a bigger size? これのもっと大きいサイズはありますか?）
  3. 【名詞】規模（組織・グループ・都市などの大きさ）（a city of this size このくらいの規模の都市）
  4. 【名詞】(問題・被害・数量などの)程度、規模（the size of the problem 問題の深刻さ・規模）

語法マーク

  • 「大きさ・規模」という一般的な意味では不可算（in size, increase in size, the same size のように無冠詞・単数扱いが基本）
  • 「(衣類・靴の)サイズ番号」を指すときは可算名詞になる（a size 10, this shoe comes in five sizes）
  • the size of + 名詞、で「〜の大きさ」という定型パターンを作る（the size of a room）
  • double/increase/decrease + in size、で「大きさ・規模が倍になる/増える/減る」という変化を表す頻出表現
  • 他品詞での用法: entryでは名詞用法のみを扱っているが、size は「(衣類・機械などを)特定の寸法に合わせて作る・調整する」という他動詞としても使われる（Please size this ring to fit my finger. この指輪を私の指に合うサイズに調整してください）。ただし日常会話で圧倒的に多いのは句動詞 size up（下記phrasal_verbs参照）で、単独の動詞用法はやや専門的・技術的な文脈に限られる。

文型・頻出パターン

  • the size of + N〜の大きさ・規模（比較や説明の基準として使う）
    the size of a football fieldサッカー場ほどの大きさ
    What's the size of the audience?観客の規模はどれくらいですか?
  • N + comes in + (several/different) sizes(商品などが)複数のサイズ展開がある
    This jacket comes in four sizes.このジャケットは4種類のサイズがある。
  • double/increase/decrease + in size大きさ・規模が倍になる/増える/減る
    The population has increased in size over the past decade.この10年で人口の規模は増加した。
  • a size + 比較級(smaller/bigger/up/down)(衣類・靴などが)ワンサイズ小さい/大きい
    Could I try a size up?ワンサイズ大きいものを試着できますか?
  • the same size as ~〜と同じ大きさ・サイズ
    My feet are the same size as my brother's.私の足のサイズは兄と同じだ。

⚠️ 日本人の典型誤用
✕ The size of people in this town is very large.
「この町は人口がとても多い」と言いたかったが、the size of people という言い方は英語として不自然で、「人々の(体の)大きさ」のように誤解されかねない。
○ The population of this town is very large.
この町は人口がとても多い。
The population of this town is very large.
日本語の「人口の大きさ」を直訳して size を使ってしまう誤り。人や物の「数」を表すときは size ではなく population（人口）や number（数）を使う。size はあくまで物理的な大きさ・規模に限る。
✕ This bag is same size with mine.
「このバッグは私のと同じサイズだ」と言いたかったが、冠詞の欠落と前置詞の誤りで語法として不自然に響く。
○ This bag is the same size as mine.
このバッグは私のと同じサイズだ。
This bag is the same size as mine.
「〜と同じ」を表す the same ~ as … の型で、冠詞 the が抜け、比較の前置詞に with を使ってしまう誤り。the same size as が正しい定型表現で、with は使わない。
✕ Can you tell me your size of shoes?
「あなたの靴のサイズを教えてください」と言いたかったが、size of shoes は回りくどく不自然に聞こえる。
○ Can you tell me your shoe size?
あなたの靴のサイズを教えてください。
Can you tell me your shoe size?
「靴のサイズ」を size of shoes と後ろから説明しようとする誤り。英語では shoe size のように名詞+名詞の複合語で表すのが自然（同様に clothing size, ring size など）。
🗾 カタカナ語とのギャップ

カタカナ「サイズ」は「サ・イ・ズ」の3モーラだが、英語は1音節。母音は二重母音 /aɪ/ として1拍でつなげ、語末の /z/ に母音を足して「ズ」と伸ばさない（「サイズゥ」にしない）。「ズ」を強く発音しすぎると2音節に聞こえてしまう点に注意。

🌱 Word family・語源
sizable（形容詞）かなり大きな、相当な規模の
oversized（形容詞）大きすぎる、特大の
undersized（形容詞）規格より小さい、小さすぎる
downsize（動詞）(人員・規模を)縮小する、小型化する
語源: 中英語 sise を経て、古フランス語 sise（assise の短縮形）に由来。assise はさらにラテン語 assidere「そばに座る」(ad- + sedere「座る」)から来ており、もともと「(規則・税額・寸法などを定めるために)人々が一緒に座って開いた会議・法廷」を意味した。そこで定められた「規定の基準・寸法」という意味が残り、現在の「大きさ・サイズ」に発展した。
assize（巡回裁判、(古い)法定基準）— ラテン語 assidere「一緒に座る」から。sizeと同じ語源で、もともと基準を定める会議を指した
session（会期、セッション）— sedere「座る」から派生し、「座って行う会合」を意味する点でsizeの語源と繋がる
sedentary（座りがちな、座位の）— sedere「座る」を直接受け継ぐ語で、size の語源にある「座る」イメージを共有する
💡 覚え方: 「人々が一緒に座って(sit=sedere)基準を定めた会議」が size の語源イメージ。服のサイズ表のような「みんなで決めた規格」というニュアンスが、assize（法廷）やsession（会期）と共通の「座る」語根で繋がっていると覚える。
🔗 句動詞
size up — (人・状況などを)見極める、判断する
She quickly sized up the situation and knew what to do.（彼女はすぐに状況を見極め、何をすべきか分かった。）
⚖️ 類義語の使い分け
dimensions — size が漠然とした「大きさ」全般を指すのに対し、dimensions は縦・横・高さなど具体的な数値で測った寸法を指す、より技術的・複数形の語。
scale — size が個々の物の大きさを指すのに対し、scale は「規模・範囲」というより抽象的な広がりを表し、計画やプロジェクトの大小を語る際によく使われる。
measurement — size が一般的な大きさの概念であるのに対し、measurement は実際に道具で測って得た具体的な数値そのものを指す。
💬 熟語・慣用表現 (4)
cut someone down to size (思い上がった人の)鼻をへし折る、身の程を思い知らせる口語
The coach cut the rookie down to size after his cocky remarks to the press.（コーチは、生意気な発言をした新人選手の鼻をへし折った。）
🔍 「(誰かの)サイズを切り詰める」から転じて、自惚れて大きく見せている人の慢心をくじき、身の丈相応の扱いに戻すこと。
📝 someone の部分に対象の人を入れる。傲慢さや自信過剰な態度を指摘して謙虚にさせる場面で使い、悪意ある嫌がらせではなく「妥当な指摘」というニュアンスが強い。
one size fits all 全てに当てはまる、画一的な(解決策・やり方)中立
There is no one-size-fits-all solution to reducing traffic in a big city.（大都市の交通渋滞を減らすのに、万能な解決策というものは存在しない。）
🔍 元は衣類の「フリーサイズ」を指す表現だが、比喩的に「一つのやり方が全員・全状況に通用する」という意味で使われる。
📝 形容詞的にハイフンでつなぎ one-size-fits-all solution/approach のように使うことが多い。否定文で「画一的なやり方は通用しない」と述べる場面が典型的。
that's about the size of it だいたいそういうことだ、まさにその通り口語
So we lost the contract because we missed the deadline? — Yeah, that's about the size of it.（「つまり締め切りに遅れて契約を失ったってこと?」「まあ、だいたいそんなところだね。」）
🔍 「それがそのサイズ・程度についてだ」という言い方から転じて、相手が述べた状況の要約に同意する決まり文句。
📝 相手の状況説明に「概ねその通りだ」と相槌を打つ表現。フォーマルな文章ではほぼ使わない、会話専用の定型句。
try something on for size (考え・提案などを)試しに使ってみる、お試しで採用してみる口語
Let's try this new schedule on for size before we make it permanent.（本採用にする前に、まずこの新しいスケジュールを試しに使ってみよう。）
🔍 「服を試着してみる」という文字通りの意味が、比喩的に「アイデアや案が自分(達)に合うか試す」へ拡張された表現。
📝 文字通り服の試着にも使うが、ビジネスや日常会話では「アイデア・計画を試験的に導入してみる」の比喩として使われることが多い。
