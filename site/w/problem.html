problem

名詞 (noun)米 /ˈprɑːbləm/ ／ 英 /ˈprɒbləm/prob·lem（2音節）CEFR A1

🗣 第1音節 prob に強勢（/ˈprɑːbləm/）。母音は /ɑː/ で、日本語の「オ」より口を大きく開けて喉の奥で出す音。第2音節 -lem は曖昧母音 /ə/ を伴う軽い音節で、日本語のようにはっきり「レム」と踏み込まず、力を抜いて滑らせる

  1. 【名詞】問題、困りごと、やっかいな事態
  2. 【名詞】（数学などの）問題、設問（solve a math problem 数学の問題を解く）
  3. 【名詞】（健康上の）不調、持病（a heart problem 心臓の持病）

語形変化

形種類発音備考
problems複数形/ˈprɑːbləmz/語末の m は有声音なので -s は /z/ になる（「プラブレムズ」であって「プラブレムス」ではない）。規則変化

語法マーク

  • 可算名詞が基本（a problem / problems）。数えられない漠然とした「困難・厄介」を表したいときは不可算の trouble を使う
  • 前置詞は with が基本（a problem with my computer）。貧困・失業など社会的な事柄を抽象的に指すときは of も使われる（the problem of poverty）
  • have a problem, solve a problem, cause a problem など動詞との組み合わせが非常に固定的なコロケーションを作る
  • 名詞のまま別の名詞の前に置いて形容詞的に修飾できる（a problem child, a problem area）
  • No problem. は依頼を快く引き受けるとき、または感謝への返答として使う口語の決まり文句
  • 他品詞での用法: NGSLでは名詞のみのレンマ（動詞用法は無い）。意味は大きく3つ: (1) 一般的な「問題・困りごと」（a serious problem）。(2) 数学・試験などの「問題・設問」（solve a math problem）。(3) 健康上の「不調・持病」（a heart problem）。また名詞のまま別の名詞を修飾する属性的用法があり、形容詞のように働く: a problem child（問題児）、a problem area（問題のある領域）。

文型・頻出パターン

  • have a problem with + 名詞/-ing〜に問題がある・〜するのに苦労する
    have a problem with something〜に問題がある
    She has a problem with her neighbors.彼女は近所の人たちとトラブルを抱えている。
  • solve / fix / deal with a problem問題を解決する・対処する
    solve a problem問題を解決する
    We need to deal with this problem immediately.この問題にすぐ対処する必要がある。
  • cause / create a problem問題を引き起こす
    cause a problem問題を引き起こす
    Heavy rain caused a problem for the event.大雨がそのイベントに問題を引き起こした。
  • No problem.問題ないよ／どういたしまして（依頼への快諾・感謝への返答として使う決まり文句）
    "Can you help me?" "No problem."「手伝ってくれる?」「もちろん、大丈夫だよ。」
  • the problem is (that) ...問題は〜ということだ（話の核心・原因を導入する）
    The problem is that we don't have enough time.問題は、時間が十分にないということだ。
  • problem + 名詞（属性的用法）「問題のある〜」（名詞のまま別の名詞を形容詞的に修飾する）
    a problem child問題児
    a problem area問題のある領域・エリア

他のNGSL語への置き換え

problem → issue
元: The problem was finally solved after hours of discussion.
後: The issue was finally solved after hours of discussion.
訳: 何時間も話し合った末に、その懸案はようやく解決した。
📝 issue は「取り上げて議論すべき事柄」という中立的なニュアンスが強く、必ずしも悪いこと・故障を含意しない。problem は「何かがうまくいっていない」という否定的な含みが issue より強い。
problem → trouble
元: If you have a problem with the schedule, please let me know.
後: If you have trouble with the schedule, please let me know.
訳: スケジュールで困ったことがあれば、教えてください。
📝 trouble は不可算名詞で「困難・厄介」という状態そのものに焦点があり、problem（可算・具体的な事柄）よりも漠然とした苦労・支障のニュアンスになる。
problem → glitch
元: There must be a problem with the internet connection because nothing is loading.
後: There must be a glitch with the internet connection because nothing is loading.
訳: 何も読み込まれないから、インターネット接続に不具合があるに違いない。
📝 glitch は口語的で「一時的な小さな技術的不具合」を指し、深刻さの度合いが problem より軽い。IT・機械関連の文脈で特によく使われる。

⚠️ 日本人の典型誤用
✕ I have a trouble with my smartphone.
スマホの調子が悪い、と言いたかったが、不可算名詞 trouble に a を付けてしまっており、ネイティブには文法的に不自然な響きに聞こえてしまう。
○ I have a problem with my smartphone.
スマホの調子が悪い。
I have a problem with my smartphone.
trouble は不可算名詞なので原則として a を付けない。日本語では「問題」も「トラブル」も同じ感覚で数えてしまいがちだが、具体的で数えられる1件の不具合を言うときは可算の problem を使うのが自然。
✕ There is a problem of the printer.
プリンターに問題がある、と言いたかったが、of を使うと「プリンターの（抽象的な）問題」のようにやや不自然で堅い響きになり、機械の故障を指す文脈では違和感を与える。
○ There is a problem with the printer.
プリンターに問題がある。
There is a problem with the printer.
「〜の問題」を機械的に of で直訳してしまう誤り。problem of は poverty（貧困）や inequality（不平等）のような抽象的・社会的な事柄に使い、目の前の具体的な物・人の不具合には problem with を使う。
✕ Let's discuss this problem at the meeting.
会議でこの案件について話し合おう、と中立的な話題のつもりで言ったのに、problem を使うとすでに何か悪いことが起きているかのように聞こえてしまい、意図と違う否定的な印象を与える。
○ Let's discuss this issue/topic at the meeting.
会議でこの件について話し合おう。
Let's discuss this issue at the meeting.
Let's discuss this topic at the meeting.
日本語の「議題」「案件」を「問題」と考えて何にでも problem を使ってしまう誤り。problem は「何かがうまくいっていない」という否定的な含みを強く持つため、まだ良し悪しが決まっていない中立的な議題には issue や topic を使う方が自然。
🗾 カタカナ語とのギャップ

カタカナ「プロブレム」は「プ・ロ・ブ・レ・ム」と5拍で均等に読みがちだが、英語は2音節 /ˈprɑːbləm/。まず pr- は「プロ」と母音を挟まず1つの子音連続として一気に発音し、母音は /ɑː/ で「オ」より喉の奥で大きく開ける。第2音節 -blem は曖昧母音を伴う弱い音節で、「レム」とはっきり踏み込まない。語末の m も「ム」と母音を足さず、唇を閉じたまま終える。

🌱 Word family・語源
problematic（形容詞）問題のある、やっかいな
problematically（副詞）問題があるやり方で
problem-solving（名詞・形容詞）問題解決（の）
語源: ギリシャ語 próblēma（πρόβλημα）「前に投げ出されたもの、障害物」に由来。pro-（前に）+ ballein（投げる）という構成で、「目の前に立ちはだかって解決を迫る課題」が原義。ラテン語 problema を経てフランス語 problème から英語に入った。
symbol（象徴、シンボル）— 同じ ballein（投げる）系。syn-（共に）+ ballein で「一緒に投げ合わせて示す印」が原義
emblem（紋章、象徴）— em-（中に）+ ballein で「中に投げ込んで表す図案」が原義
metabolism（代謝）— meta-（変化して）+ ballein で「投げて変化させること」→ 体内で物質を変化させる働き
💡 覚え方: ballein（投げる）の仲間として、problem は「目の前に投げ出された障害物」、symbol は「共に投げ合わされた印」、emblem は「中に投げ込まれた図案」、metabolism は「変化するよう投げられた働き」— 全部『投げる』イメージでつながる。
⚖️ 類義語の使い分け
issue — issue は良し悪しが決まっていない中立的な「議題・懸案」を指すことが多く、必ずしも悪いことを含意しない。problem は「何かがうまくいっていない」という否定的な含みが issue より強い。
trouble — trouble は不可算名詞で「困難・厄介」という漠然とした状態そのものに焦点がある。problem は可算名詞で、具体的に特定できる1件の事柄を指す。
difficulty — difficulty は何かを成し遂げる過程で生じる「難しさ・苦労」に焦点があり（have difficulty doing）、problem は解決すべき対象・事柄そのものに焦点がある。
💬 熟語・慣用表現 (2)
No problem. どういたしまして、大丈夫だよ、問題ないよ（依頼への快諾・感謝への返答の決まり文句）口語
"Thanks for helping me move all this furniture." "No problem!"（「家具の移動、手伝ってくれてありがとう。」「どういたしまして！」）
🔍 文字通りには「問題は存在しない」。そこから転じて、依頼を快く引き受ける返事や、感謝された時の返答として社交的に使われる決まり文句になっている点が非構成的（文字通りの意味だけでは会話でのこの使われ方は予測しにくい）。
📝 Not a problem. もほぼ同義でよく使われる。依頼される前に先回りして言うと「お安い御用ですよ」の意味にもなる。
What's your problem? 一体何なんだよ？、何をそんなに怒ってるんだ？（相手の失礼・攻撃的な態度を非難する言い方）口語
Hey, what's your problem? There's no need to yell at me like that.（ちょっと、一体何なんだよ？ そんな怒鳴らなくてもいいだろ。）
🔍 文字通りには「あなたの問題は何?」という素朴な質問だが、実際には相手の態度に苛立って詰め寄る口語表現として使われる。相手の悩みを心配して尋ねる文ではなく、非難・喧嘩腰のニュアンスになる点が非構成的。
📝 かなり攻撃的・挑発的に響くため使う場面には注意が必要。純粋に相手の悩みを心配して尋ねたい場合は What's wrong? や Is something bothering you? の方が穏当。
