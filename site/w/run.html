run

動詞・名詞米 /rʌn/ ／ 英 /rʌn/run（1音節）CEFR A1

🗣 1音節語。母音は短母音 /ʌ/（sun, fun と同じ）。カタカナ「ラン」につられて母音を長く伸ばしたり、語末のnを「ン」とはっきり切って発音したりしない。舌先を上の歯茎に軽く当てて鼻に息を抜くだけでよい

  1. 【動詞】走る（She runs five kilometers along the river every morning before breakfast. 彼女は毎朝、朝食前に川沿いを5キロ走る。）
  2. 【動詞】（機械・プログラムなどが）動く、作動する（The software update is still running, so please don't unplug the computer. ソフトウェアのアップデートがまだ進行中なので、コンピューターの電源を抜かないでください。）
  3. 【動詞】（バス・電車などが）運行する（If the trains ran more frequently, fewer people would need to drive to work. 電車がもっと頻繁に運行していれば、車で通勤する人はもっと少なくなるだろう。）
  4. 【動詞】（会社・店などを）経営する、運営する（My grandfather has run this small bakery since 1985. 祖父は1985年からこの小さなパン屋を経営している。）
  5. 【動詞】（検査・手続きなどを）実施する、行う（A background check is run on every new employee before they start. 新しく入る社員には皆、勤務開始前に身元調査が行われる。）
  6. 【動詞】（液体が）流れる（Tears ran down her cheeks as she watched the sad movie. 悲しい映画を見ながら、彼女の頬に涙が流れた。）
  7. 【動詞】（選挙に）立候補する（She's planning to run for mayor next year. 彼女は来年、市長選に立候補する予定だ。）
  8. 【名詞】ひとっ走り、ランニング（I like to go for a run before work. 私は仕事前にひとっ走りするのが好きだ。）
  9. 【名詞】（一定期間の）連続、続き（The team is on a run of five straight wins. そのチームは5連勝中だ。）

形種類発音備考

語法マーク

  • 自動詞用法が基本（She runs every day.）だが、run a business / run a test / run a red light のように他動詞としても非常によく使われる
  • 不規則変化 run-ran-run。過去形と過去分詞が同じ形（run）に戻る点に注意（have ran は誤り、正しくは have run）
  • 進行形にしやすい動作動詞。ただし「（バスなどが）定期的に運行する」という習慣的な意味では現在形（The bus runs every hour.）が基本で、進行形は「今まさに動いている」ことを強調したいときに使う（The engine is running now.）
  • run for + 役職名で「〜に立候補する」（run for president / mayor）。forは省略できない
  • 他品詞での用法: 名詞用法も幅広い。(1) 野球などの「得点」: score a run（1点入れる）。(2) ストッキングの「伝線」: There's a run in my tights.（ストッキングが伝線している）。(3) 印刷・生産の「ロット、刷り」: a production run of 10,000 units（1万個の生産ロット）。(4) 銀行の「取り付け騒ぎ」: a run on the bank（銀行への取り付け騒ぎ）。カタカナ「ラン」は「走ること」の意味に限定されがちだが、英語のrunの名詞用法はそれよりずっと幅が広い。

  • run + 距離／コース（ある距離・コースを）走る

  • run + a／an + 名詞（経営・実施の軽動詞構文）〜を経営する・実施する（動詞1語の代わりの言い回し）

  • run + O + 副詞辞／前置詞句（拡張的な句動詞構文）目的語を挟んで「（意見・数字などを）人に確かめてもらう」を表す形

他のNGSL語への置き換え

run → manage
元: My grandfather has run this small bakery since 1985.
後: My grandfather has managed this small bakery since 1985.
訳: 祖父は1985年からこの小さなパン屋を管理・運営している。
📝 run は日常会話で最も自然な「経営する」の言い方。manage はよりフォーマルで、日々の運営やスタッフの管理といった実務的な側面を強調するニュアンスを持つ。
run → sprint
元: He ran three miles to catch the last train, but he arrived a minute too late.
後: He sprinted three miles to catch the last train, but he arrived a minute too late.
訳: 彼は最終電車に間に合うように3マイル全力疾走したが、1分遅れで着いてしまった。
📝 run は「走る」を表す最も中立的な語であるのに対し、sprint は「全速力で走る」という一時的な全力疾走のニュアンスを強調する、より限定的な語。
run → operate
元: If the trains ran more frequently, fewer people would need to drive to work.
後: If the trains operated more frequently, fewer people would need to drive to work.
訳: 電車がもっと頻繁に運行していれば、車で通勤する人はもっと少なくなるだろう。
📝 run は交通機関の運行を表す日常的でくだけた言い方。operate はよりフォーマル・技術的な響きを持ち、公式なアナウンスや報告書などで好まれる。

⚠️ 日本人の典型誤用
✕ I have ran this bakery for ten years.
「10年間このパン屋を経営してきた」と言いたかったが、have ran は活用を間違えた非標準の言い方で、ネイティブには稚拙な誤りに聞こえる。
○ I have run this bakery for ten years.
10年間このパン屋を経営してきた。
I have run this bakery for ten years.
不規則動詞 run-ran-run は過去形と過去分詞が同じ run という形に戻るが、多くの学習者は過去形 ran を have の後にも使ってしまう。現在完了・受動態など have/be + 過去分詞の位置には常に run を置く。
✕ She's planning to run mayor next year.
「彼女は来年、市長選に立候補する予定だ」と言いたかったが、for が抜けると英語として成り立たず、ネイティブには言い間違いに聞こえる。
○ She's planning to run for mayor next year.
彼女は来年、市長選に立候補する予定だ。
She's planning to run for mayor next year.
「〜に立候補する」は run for + 役職という形が基本で、for は省略できない。日本語「市長に立候補する」の「に」に引きずられて for を落としてしまう誤り。
✕ My nose is flowing because I have a cold.
「風邪で鼻水が出ている」と言いたかったが、flow を使うと大げさで不自然な響きになり、ネイティブには違和感を与える。
○ My nose is running because I have a cold.
風邪で鼻水が出ている。
My nose is running because I have a cold.
日本語「鼻水が流れる」を直訳して flow を選んでしまう誤り。鼻水のような体液が出ることを表すときは run が決まった言い方で、flow は川など大量の液体の流れに使うことが多い。
🗾 カタカナ語とのギャップ

カタカナ「ラン」は「ラ・ン」の2モーラで「走ること」の意味しか連想させない。しかし英語のrunは非常に多義的で、経営する（run a business）、（機械が）動く（the engine is running）、（液体が）流れる（tears ran down her face）、（選挙に）立候補する（run for mayor）など「ラン」からは連想しにくい意味で使われることが多い。発音面でも、母音は短母音/ʌ/で日本語の「ラ」より喉の奥寄りの曖昧な音であり、語末のnもはっきり「ン」と区切らず軽く鼻に抜ける程度に留める。

🌱 Word family・語源
runner（名詞）走者、ランナー；（引き出しなどの）レール金具
runway（名詞）（空港の）滑走路
runaway（名詞・形容詞）家出人、逃亡者；制御不能の、暴走した
rerun（名詞・動詞）（テレビ番組の）再放送；再び実行する
overrun（動詞）（害虫・雑草などが）はびこる；（予算・時間を）超過する
語源: 古英語の動詞rinnan（流れる、走る）と、その使役形ærnan（走らせる）が中英語期に融合してできた語。古ノルド語の同根語rinnaの影響も受けたとされ、ゲルマン語族に共通する「絶え間なく動く・流れる」という基本義を今も残している。
runner（走者、ランナー）— run + -er（〜する人）で「走る人」
runaway（家出人、逃亡者）— run + away（離れて）で「走り去った（人・状態）」
overrun（はびこる、超過する）— over（覆って）+ runで「覆うように広がって走る」→溢れる、超過する
💡 覚え方: ゲルマン祖語由来の「流れ続ける・動き続ける」動詞。runner（走る人）、runaway（走り去った人）、overrun（覆うように溢れる）は、どれもrunの「絶え間なく動く」イメージから枝分かれしている。

⚖️ 類義語の使い分け
jog — jogは健康法として一定の楽なペースでゆっくり走ることに限定される。runはペースを問わず「走る」全般を表す包括的な語。
sprint — sprintは短距離を全力疾走する一時的な動作に限定される。runは短距離から長距離まで、持続的な走行にも使える一般的な語。
manage — 「経営する」の意味ではrunと近いが、manageはよりフォーマルで日々の管理・組織運営に焦点がある。runは口語的で「切り盛りする」という日常的なニュアンスが強い。
💬 熟語・慣用表現 (12)
in the long run 長い目で見れば、結局は中立
Buying a more expensive but durable pair of boots will save you money in the long run.（値は張っても長持ちするブーツを買っておいた方が、長い目で見れば得になる。）
🔍 「長い距離を走った先で」という比喩から「最終的には、長期的に見れば」という意味になった副詞句。目先の結果ではなく将来的な結果に注目するときに使う。
📝 文頭・文末どちらにも置ける。対義表現は in the short run（短期的には）。
on the run 逃走中で、追われて中立
The escaped prisoner has been on the run for three days.（その脱獄囚は3日間逃走を続けている。）
🔍 「走っている最中」の状態から「（警察などから）逃げ回っている」の意味に転じた表現。
📝 on the run from the police のように from を伴って「〜から逃げている」を表せる。比喩的に「大忙しで動き回っている」の意味でも使われる（I've been on the run all day.）。
run in the family （体質・才能・癖などが）一族に代々受け継がれている中立
Red hair runs in the family on my mother's side.（赤毛は母方の家系に代々受け継がれている。）
🔍 「家系の中を走っている」という比喩から「世代を超えて受け継がれる」の意味になった表現。
📝 主語には遺伝的な特徴・体質・才能・病気などが入ることが多い（Heart disease runs in the family.）。
run-of-the-mill ありふれた、平凡な中立
The restaurant serves run-of-the-mill pasta dishes, nothing you'd remember.（そのレストランのパスタはごく平凡な出来で、特に印象に残るものではない。）
🔍 もとは製粉所（mill）で選別せずそのまま出荷される「並みの」小麦粉を指した工業用語から、「特に優れても劣ってもいない、ごく普通の」という意味に転じた形容詞。
📝 常にハイフンでつないだ形容詞として名詞の前に置く。「たいしたことない」という否定的なニュアンスを含むことが多い。
run its course （病気・状況などが）自然の成り行きで一通り終わる中立
The doctor said the flu just needs to run its course.（医師は、そのインフルエンザは自然に治まるのを待つしかないと言った。）
🔍 「自らのコース（進路）を走り切る」という比喩から「人為的に止めずに自然に終わりまで進む」という意味になった表現。
📝 病気・恋愛関係・ブームなど「時間とともに自然消滅・収束するもの」に使う。let it run its course「なすがままにさせる」の形も頻出。
hit the ground running 出だしから全力で取り組む、幸先よく勢いをつけてスタートする中立
The new manager hit the ground running and reorganized the team within a week.（新しいマネージャーは着任早々から精力的に動き、1週間でチームを立て直した。）
🔍 「（飛行機やパラシュートから）着地するなり走り出す」という軍事由来の比喩表現から、「準備を整えて即座に全力で始める」という意味に転じた。
📝 新しい仕事・プロジェクトを「助走なしで即座に軌道に乗せる」というビジネス文脈で特によく使われる。
give someone a run for their money 〜に対して互角の勝負をする、〜に負けない実力を見せつける中立
The rookie gave the defending champion a run for his money in the final match.（その新人は決勝戦で前チャンピオンに互角の勝負を挑んだ。）
🔍 「賭けた金に見合うだけの走り（レース）を見せる」という競馬由来の比喩から、「強い相手に引けを取らない競争・接戦をする」という意味に転じた。
📝 主に「予想外に善戦する・強敵になる」という肯定的な文脈で使う。money は「賭け金」由来で今では意味が薄れている。
run the show （組織・状況を）仕切る、実権を握る口語
On paper he's just an assistant, but everyone knows who really runs the show.（肩書きはただのアシスタントだが、実際に仕切っているのが誰かは誰もが知っている。）
🔍 「ショー（興行）を仕切る」という比喩から「実質的な責任者として物事を取り仕切る」という意味になった口語表現。
📝 「本当の権力者は別にいる」という含みで使われることが多い（who really runs the show）。
cut and run （責任や困難を放り出して）一目散に逃げる、さっさと手を引く口語
Instead of fixing the problem, the company decided to cut and run.（その会社は問題を解決するのではなく、さっさと手を引くことにした。）
🔍 もとは船の錨綱を切って（cut）すぐに出航する（run）という航海用語から、「状況を収拾せずに突然逃げ出す・撤退する」という意味に転じた。
📝 「困難な状況から無責任に逃げる」という否定的なニュアンスを伴うことが多い。
run dry （井戸・資源・資金などが）涸れる、尽きる中立
After the drought, the well finally ran dry.（干ばつの末、その井戸はついに涸れてしまった。）
🔍 「（川や井戸が）乾いた状態まで流れ果てる」という文字通りの意味から、資金・アイデア・忍耐など無形のものにも比喩的に使われるようになった。
📝 資金（funds run dry）やインスピレーション（ideas run dry）など、比喩的な「枯渇」にもよく使われる。
run a fever 熱がある、発熱している中立
My son has been running a fever since last night, so I kept him home from school.（息子が昨夜から熱を出しているので、学校を休ませた。）
🔍 直訳の「熱を走らせる」では意味が通らず、run が have に近い意味で使われる特殊な決まり文句。
📝 have a fever よりも「熱がある状態がしばらく続いている」という進行的なニュアンスを持つ。イギリス寄りの言い方では run a temperature も使われる。
make a run for it （危険から逃れるために）一気に走って逃げる、逃げ切ろうとする口語
When the alarm went off, the thieves made a run for it and disappeared into the crowd.（警報が鳴ると、泥棒たちは一気に走って逃げ、人混みに紛れて姿を消した。）
🔍 「それ（状況）のために一気に走る」から、危険や捕まりそうな状況から素早く逃げ出す決断を表す口語表現。
📝 it は特定の対象を指さない慣用の it。危険から逃げ出す一瞬の決断・行動を表す。
