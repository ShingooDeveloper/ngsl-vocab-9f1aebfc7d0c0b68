true

形容詞・動詞米 /truː/ ／ 英 /truː/true（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。子音連結 tr- のあとに長母音 /uː/ が続く。日本語話者は「ト・ゥ・ル・ー」と4拍に分けがちだが、英語では tr をひとまとまりの子音として素早く発音し、そのあとの母音部分だけをしっかり伸ばす

  1. 【形容詞】本当の、真実の（Is it true that the museum is closed on Mondays? その美術館は月曜休みというのは本当ですか?）
  2. 【形容詞】本物の、真の（She showed true courage when she spoke up for her classmate. 彼女はクラスメートのために声を上げたとき、本物の勇気を見せた。）
  3. 【形容詞】（人・約束などに）忠実な、誠実な（He has stayed true to his promise even when it was difficult. 大変な状況でも彼は約束に忠実であり続けた。）
  4. 【形容詞】（機械・道具などが）狂いのない、正確に水平・垂直な（Check that the wheel is true before you ride the bike. 自転車に乗る前に車輪に狂いがないか確認して。）
  5. 【動詞】（車輪・面などの）狂いを直す、正確に調整する（true up: The mechanic trued the wheel so it would spin smoothly. 整備士は車輪の狂いを直し、スムーズに回るようにした。）

語法マーク

  • 形容詞としては限定用法（a true story）・叙述用法（The story is true.）のどちらでも使える
  • 比較級・最上級は -er/-est を直接つける規則変化（truer/truest）。more true / most true は標準的ではない
  • true to ~ で「〜に忠実な」という形容詞句になり、stay/remain/hold true to の形で頻出
  • come true は「（夢・予言などが）実現する」の意味。come の後に形容詞 true が補語として置かれる構文で、進行形にはしない
  • 動詞としては及物動詞。目的語（車輪・面・列など）を伴い、専門的・技術的な文脈が中心で日常会話ではあまり使わない
  • 他品詞での用法: 動詞用法は主に工作・機械の文脈で使われるやや専門的な語（自転車の車輪の芯出し、木工でまっすぐに削る、建築で水平・垂直を正確に出す等）。true up の形（The carpenter trued up the beam. 大工は梁の狂いを直した。）でもほぼ同じ意味で使われる。なお ring true（本当らしく聞こえる）、run true（型通りにいく）のような一部の固定表現では副詞的に響くこともあるが、これらはあくまで慣用表現内の用法であり、true 自体が独立した副詞として確立しているわけではない。

文型・頻出パターン

  • It is/It's true that + S + V一般的な事実や伝聞を述べる形式主語構文
    It is true that written English differs from spoken English.書き言葉の英語が話し言葉と異なるというのは本当だ。
  • stay/remain/hold true (to + O)（人・約束・原則などに）忠実であり続ける／（事実・規則などが）当てはまり続ける
    This advice still holds true today.このアドバイスは今日でも当てはまる。
    She stayed true to her word.彼女は約束を守り通した。
  • S + come true（夢・予言などが）現実になる、実現する
    Her dream of becoming a doctor finally came true.医者になるという彼女の夢がついに実現した。
  • true + 名詞（限定用法）「本物の〜、真の〜」と名詞を直接修飾する用法
    a true friend本当の友達
    the true meaning of the storyその物語の真の意味
  • 比較級・最上級: truer / truest「より真実に近い／もっとも真実に近い」など程度を比べる用法
    No truer words have ever been spoken.これほど真実を言い当てた言葉はかつてなかった。
  • true（動詞）+ O ／ true up + O（車輪・面などの）狂いを正確に直す
    The technician trued the wheel.技術者は車輪の狂いを直した。

他のNGSL語への置き換え

true → accurate
元: The rumor has turned out to be true after all.
後: The rumor has turned out to be accurate after all.
訳: その噂は結局のところ正確だったとわかった。
📝 true は「事実に反しない、本当である」という一般的な語で日常会話向き。accurate は「正確さ・精度」に焦点を当てるやや客観的でフォーマルな語で、データ・情報・測定などの文脈でよく使われる。
true → loyal
元: She is the only friend who has stayed true to me since we were kids.
後: She is the only friend who has stayed loyal to me since we were kids.
訳: 彼女は子供の頃からずっと私に対して誠実であり続けてくれた唯一の友達だ。
📝 true to は「約束・信念・人を裏切らない」という一般的な忠実さを表すのに対し、loyal はより強く「忠誠心・支持を持ち続ける」ことを強調し、組織・国家・友人関係などでも使われる。stay true to / stay loyal to はどちらも定型表現。
true → genuine
元: If his story were true, the police would already believe him.
後: If his story were genuine, the police would already believe him.
訳: もし彼の話が本物（偽りのないもの）であれば、警察はもう彼を信じているだろう。
📝 true は「事実と一致している」という意味の最も一般的な語。genuine は「本物の、偽りのない」ことを強調し、話・書類・感情などの真偽が疑われている文脈でよく使われる。

✕ This story is more true than that one.
「この話はあの話より本当らしい」と言いたかったが、more true は回りくどく不自然に響く。
○ This story is truer than that one.
この話はあの話よりも本当のことに近い。
This story is truer than that one.
true は1音節の形容詞なので、比較級は more を使わず -er を直接つけて truer にするのが標準。長い形容詞の比較級を作る感覚で more true としてしまう誤り。
✕ I true believe that she is innocent.
「本当に彼女は無実だと思う」と言いたかったが、true を副詞として動詞の前に置くのは文法的に誤りで、不自然に聞こえる。
○ I truly believe that she is innocent.
本当に彼女は無実だと思う。
I truly believe that she is innocent.
「本当に」と動詞を修飾したいときは副詞 truly を使う。true は基本的に形容詞であり、ring true や run true のようなごく限られた慣用表現以外では、一般の動詞を修飾する副詞として使えない。
✕ His prediction trued last year.
「彼の予言は去年当たった」と言いたかったが、動詞 true にその意味はなく、機械調整の文脈以外では意味が通じない。
○ His prediction came true last year.
彼の予言は去年当たった。
His prediction came true last year.
His prediction proved true last year.
動詞 true は「（車輪・面などの）狂いを機械的に直す」という専門的な意味に限られ、「（予言などが）当たる・実現する」という意味では使えない。その意味では come true や prove true を使う。
🗾 カタカナ語とのギャップ

カタカナ「トゥルー」は「ト・ゥ・ル・ー」の4モーラに分解されがちだが、英語 /truː/ は1音節。tr は子音連結としてひと息に発音し、r は舌を巻く音ではなく舌先を口の中で丸めて構える接近音。母音は長母音 /uː/ で、「ル」に相当する独立した子音+母音は存在しない。

🌱 Word family・語源
truth（名詞）真実、真理NGSL
truly（副詞）本当に、心からNGSL
untrue（形容詞）真実でない、不誠実な
truthful（形容詞）正直な、真実を語る
語源: 古英語 trīewe（西サクソン方言）・trēowe（マーシア方言）「忠実な、信頼できる」に由来し、さらに遡るとゲルマン祖語 *trewwjaz「信義に厚い」に行き着く。この語根はさらに印欧祖語の *deru-「堅固な、しっかりした」（原義は「木」）に連なり、truth（真実）とは同じゲルマン語根を共有し、trust（信頼）とはこの印欧祖語の語根を介した遠い親戚にあたる。木のようにどっしりと動かず揺るがないというイメージを核に持つ。
truth（真実）— true と同じ語根から派生した名詞形。「本当であること」そのものを表す
trust（信頼）— true とは異なり古ノルド語 traust を経由した語だが、さらに遡ると true と同じ印欧祖語の語根 *deru-「堅固な」に行き着く遠い同根語。「揺るがず信頼できる」という核イメージを共有する
truce（休戦）— 元は「誓約、信義」を意味した中英語 trewes（true の古い複数形）に由来し、同根の語
💡 覚え方: true・truth・trust は同じ「揺るがない、信頼できる」というゲルマン語根から生まれた三兄弟語。「本当のこと(truth)を信じられる(trust)なら、それは true」と結びつけて覚える。
🔗 句動詞

⚖️ 類義語の使い分け
genuine — true は「事実と一致している、本当である」という最も一般的な語。genuine は「偽物ではない、本物である」ことを強調し、書類・感情・ブランド品などの真正性が疑われる文脈でよく使われる。
accurate — true は日常的に「本当かどうか」を広く指すのに対し、accurate は数値・測定・説明などの「正確さ・精度」に焦点を当てたややフォーマルな語。
loyal — 「忠実な」の意味に限ると、true to は約束・信念など抽象的な対象にも人にも使える一般的な表現。loyal はより強く「支持・忠誠を持ち続ける」ことを強調し、組織・国・友人などに対してよく使われる。
💬 熟語・慣用表現 (7)
come true （夢・予言などが）実現する、本当になる中立
Her lifelong dream of opening her own bakery finally came true this spring.（自分のパン屋を開くという彼女の長年の夢が、この春ついに実現した。）
🔍 come（〜になる）+ true（本当の）で「本当のものになる」という比喩。主語には dream, wish, prediction など「望んでいたこと」が来ることがほとんど。
📝 進行形にはせず、過去形や完了形でよく使う。反対に「実現しない」は never come true や fail to come true と言う。
too good to be true 話がうますぎる、信じられないほど良すぎる口語
A brand-new laptop for only ten dollars sounds too good to be true.（新品のノートパソコンがたった10ドルなんて、話がうますぎる。）
🔍 「本当であるには良すぎる」から転じて、条件が良すぎてかえって疑わしいという含意を持つ。
📝 詐欺や誇大広告への警戒を表すことが多い。"If it sounds too good to be true, it probably is." という決まり文句もよく使われる。
show one's true colors （隠していた）本性を現す、正体を現す中立
He seemed kind at first, but he showed his true colors when he refused to help us.（彼は最初は親切そうだったが、私たちを助けるのを拒んだときに本性を現した。）
🔍 船が自国の旗（colors）を掲げて真の所属を示したことに由来し、「本当の姿・性格を現す」という意味に転じた。
📝 裏切りや失望を伴う否定的な文脈で使われやすい。イギリス綴りは colours。
ring true 本当らしく聞こえる、真実味がある中立／やや文語
Her excuse for missing the deadline didn't quite ring true.（締め切りに遅れた彼女の言い訳は、どうも本当らしく聞こえなかった。）
🔍 硬貨をはじいて澄んだ音が鳴れば本物、という言い伝えから転じて、話や説明が「本物らしく響く」という意味になった。
📝 主語には人ではなく発言・説明・話などの「内容」が来る。否定形（doesn't ring true）で「怪しい、嘘くさい」の意味で使われることが多い。
true to form いつも通りに、案の定中立／ややくだけた
True to form, he showed up twenty minutes late for the meeting.（案の定、彼は会議に20分遅れてやって来た。）
🔍 「（いつもの）型に忠実に」から転じて「相変わらず、予想通りに」という意味。
📝 文頭に副詞句として置いて使うことが多い。良い意味にも皮肉にも使える。
true to life 現実そのままの、実物そっくりの中立／ややフォーマル
The characters in the novel felt true to life.（その小説の登場人物たちは、まるで実在するかのようにリアルに感じられた。）
🔍 「人生・現実に忠実な」から転じて、描写や表現が「リアルで真に迫っている」という意味。
📝 小説・映画・絵画など、現実を描いた作品の写実性を褒めるときによく使う。ハイフンで true-to-life とつなぎ名詞の前に置く形容詞としても頻出。
true blue 生粋の、忠実で信頼できる口語／ややイギリス英語的
He's a true-blue supporter of the local team; he never misses a game.（彼はその地元チームの生粋のファンで、試合を欠かしたことがない。）
🔍 元は色落ちしにくい濃く純粋な青の染料を指したことから、「本物の、揺るがず忠実な」という意味に転じた。
📝 名詞の前ではハイフンで true-blue とつなぐことが多い。イギリスでは保守党支持者を指す政治的な意味で使われることもある。
