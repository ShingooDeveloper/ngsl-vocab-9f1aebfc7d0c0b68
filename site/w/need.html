need

動詞・名詞米 /niːd/ ／ 英 /niːd/need（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。母音は feed, seed と同じ長母音 /iː/。kid, bid の短い /ɪ/ と混同して短く発音しないこと

  1. 【動詞】～を必要とする、～が要る（I need your help. あなたの助けが必要だ）
  2. 【動詞】need to do：～する必要がある、しなければならない（You need to see a doctor. 医者に診てもらう必要がある）
  3. 【動詞】（否定文で）don't need to：～する必要はない（You don't need to worry. 心配する必要はない）
  4. 【名詞】必要、必要性（不可算）（There's no need to rush. 急ぐ必要はない）
  5. 【名詞】（needsの形で）必要なもの、要求、ニーズ（meet the needs of customers 顧客のニーズを満たす）

語形変化

形種類発音備考
needs三単現/niːdz/規則変化。語末は有声の子音連続 /dz/。日本語話者は母音を挟んで「ニーズ」と3拍で言いがちだが、実際は母音を入れず一息で /niːdz/
needed過去形・過去分詞/ˈniːdəd/規則変化。語幹が d で終わるため -ed は独立音節 /əd/ になる（needed は2音節）。受動態 be needed でも同じ形
needinging形/ˈniːdɪŋ/規則的に -ing を付加。進行形（be needing）や動名詞として用いる。なお needs fixing 型の受動的構文で -ing になるのは need ではなく後続の動詞（fixing）の方である点に注意

語法マーク

  • 通常の一般動詞としては do-support を使う: Do you need help? / I don't need it.（三単現は needs、否定・疑問は don't/doesn't）
  • 他動詞として名詞・to不定詞・O+to不定詞を取る catenative動詞: need help / need to go / need you to help me
  • イギリス英語ではまれに準助動詞（modal）としても使われる: 否定・疑問文で -s を付けず to なしの原形を続ける（You needn't wait. / Need I say more?）。アメリカ英語では You don't need to wait. の形が普通で、needn't はやや形式ばった書き言葉的響き
  • need + -ing（動名詞）は『〜される必要がある』という受動的意味を表す（= need to be + 過去分詞）: This bike needs fixing. = This bike needs to be fixed.
  • 不可算名詞としての need（There's no need to worry.）と、複数形 needs で使う可算的な用法（meet the needs of ~）を区別する
  • needn't（不要）と mustn't（禁止）は意味が逆であり混同しやすい: needn't は『しなくてよい』、mustn't は『してはいけない』
  • 他品詞での用法: 名詞用法もある。(1) 不可算名詞として「必要性」: There's no need for that.（そんな必要はない）、if need be（もし必要なら）のような定型表現で頻出。(2) 複数形 needs で「必要とするもの、ニーズ」: special needs（特別な支援を要すること）、meet someone's needs（〜のニーズを満たす）。(3) in need of ~（〜を必要としている）、in need（困窮して、困っている）のように、困難な状況にあることを示す言い方にもなる（a friend in need is a friend indeed ということわざにも残る）。

文型・頻出パターン

  • need + O（名詞）〜が必要だ（最も基本的な他動詞用法）
    need help助けが必要だ
    need more timeもっと時間が必要だ
    need a break休憩が必要だ
  • need to do〜する必要がある
    I need to finish this report today.今日中にこの報告書を終わらせる必要がある。
  • don't need to do〜する必要はない（不必要）
    You don't need to knock; just come in.ノックする必要はないよ、そのまま入って。
  • S + need + -ing主語が『〜される必要がある』（受動的意味、= need to be + 過去分詞）
    The garden needs watering.庭に水をやる必要がある。
  • there's no need for/to ~〜の必要はない（定型表現）
    There's no need for an umbrella today.今日は傘の必要はない。
  • in need of + 名詞〜を必要としている（状態を表す）
    The old house is in need of repair.その古い家は修理が必要な状態だ。

他のNGSL語への置き換え

need → require
元: Everyone needs a good night's sleep to stay healthy and focused.
後: Everyone requires a good night's sleep to stay healthy and focused.
訳: 誰でも健康を保って集中力を維持するには、ぐっすり眠ることを要する。
📝 require は need よりも硬く、書き言葉やフォーマルな説明・規則で好まれる。need は日常会話で最も中立的で頻度の高い語。
need → have
元: You don't need to bring your own tent; we already have one for you.
後: You don't have to bring your own tent; we already have one for you.
訳: テントは自分で持ってこなくていいよ、こっちで用意してあるから。
📝 否定形の don't need to と don't have to はどちらも「する必要がない」を表し、ほぼ言い換え可能。肯定文では have to が外的な規則・事情による義務、need to が話し手自身の判断による必要性を表す傾向があるが、否定文ではその差がほとんど消える。
need → want
元: More volunteers are needed to help clean up the park this weekend.
後: More volunteers are wanted to help clean up the park this weekend.
訳: この週末、公園の掃除を手伝ってくれるボランティアがもっと求められている。
📝 be wanted は「求人・募集」の掲示（Help wanted など）で使われる呼びかけのニュアンスが強く、be needed は実際に不足している・必要だという客観的事実を強調する。

詳細
⚠️ 日本人の典型誤用
✕ I need go now.
「もう行かなくちゃ」と言いたかったが、to を落とすと英語として文法的に壊れており、canやmustと同じ助動詞のつもりで使っているのだと伝わってしまう。
○ I need to go now.
もう行かなくちゃ。
I need to go now.
need を can や must のような助動詞と混同し、後ろに続く動詞から to を落としてしまう誤り。need は一般動詞（catenative動詞）なので、動詞を続けるときは必ず to不定詞にする。
✕ You needn't smoke in here.
「ここでタバコを吸ってはいけません」と禁止を伝えたかったが、needn't では『吸う必要はない（吸ってもよい）』という正反対の意味に聞こえてしまう。
○ You mustn't smoke in here.
ここでタバコを吸ってはいけません。
You mustn't smoke in here.
needn't（〜する必要はない＝してもしなくてもよい）と mustn't（〜してはいけない＝禁止）を混同する誤り。禁煙を伝えたいなら禁止を表す mustn't（または can't）を使う必要がある。
✕ I need a coffee, please.
「コーヒーをお願いします」と丁寧に頼みたかったが、need を使うと店員には妙に切羽詰まった、あるいはやや高圧的な注文のように響くことがある。
○ I'd like a coffee, please.
コーヒーをお願いします。
I'd like a coffee, please.
カフェなどでの注文に need を使うと、単なる希望ではなく『どうしても必要だ』という強い切迫感を与え、ぶしつけ・大げさに響くことがある。丁寧な依頼には would like を使うのが無難。
🗾 カタカナ語とのギャップ

カタカナ「ニード」は「ニ・ー・ド」3拍だが、英語は1音節 /niːd/。長母音 /iː/ を保ちながら語末の /d/ には母音を付けずに止める（「ド」と発音しない）。複数形・三単現の needs も「ニーズ」と3拍にせず、[niːdz] と一息の子音連続で発音する。

🌱 Word family・語源
needy（形容詞）困窮した、生活に困っている
needless（形容詞）不必要な（needless to say「言うまでもなく」の形でよく使う）
needlessly（副詞）不必要に、無駄に
neediness（名詞）困窮、（人に頼りたがる）欠乏感
語源: 古英語 nēod（必要、緊急、苦難）に遡り、さらにゲルマン祖語 *naudiz「困窮・強いられた状態」に由来する、ラテン語系ではない英語本来の基本語彙。もとは『困っている状況』そのものを指す語で、そこから『（困っているからこそ）必要とする』という現在の意味へ発展した。
Not（ドイツ語）（緊急事態、苦境）— 同じゲルマン祖語 *naudiz に由来し、『困窮・逼迫』という核の意味を共有する
nood（オランダ語）（必要、緊急）— 同系のゲルマン語で『困窮・必要』の意味を今も保つ
💡 覚え方: need はもともと『困っている状態』そのものを指した語（ドイツ語 Not「緊急事態」と同語源）。そこから『困っているからこそ必要とする』へ意味が広がったと考えると、in need（困窮して）や a friend in need（困った時の友）に原義の『苦境』が残っていることが腑に落ちる。
⚖️ 類義語の使い分け
require — require は need よりも硬く形式的で、規則・条件・手続き上の必要性を述べる書き言葉やビジネス文書で好まれる。need は日常会話で最も中立的で頻度の高い語。
want — want は『欲しい・したい』という主観的な欲求を表すのに対し、need は『必要性』というより客観的・切実な度合いが強い。need を want の代わりに軽い希望に使うと大げさに響くことがある。
lack — lack は『〜が欠けている・不足している』という現状を述べる語で、必要性そのものを表す need とは焦点が異なる（lack water = 水が不足している／need water = 水を必要としている）。
💬 熟語・慣用表現 (3)
if need be もし必要なら、いざとなれば中立／やや書き言葉
We can always take a taxi if need be.（いざとなれば、いつでもタクシーに乗ればいいよ。）
🔍 「必要が存在するならば」という意味の古い仮定法的表現。need はここでは名詞、be は仮定法現在の名残で is/was のように活用しない特殊な形。
📝 if it is necessary とほぼ同義の完全に固定化された言い回し。be を活用させて if need is/was のようにはしない。
in need 困窮して、生活に困って中立
The charity provides food and shelter to families in need.（その慈善団体は、困窮している家庭に食料と住居を提供している。）
🔍 本来「(何かを)必要としている」状態を表す語だが、of 以下の目的語を伴わず単独で使うと「困っている、助けを要する状態にある」という一般的な意味に定着する。
📝 in need of + 名詞（〜を必要としている）と違い、of以下を省いた in need は「貧しい、困っている」という状態そのものを表す定型句として使われる。
a friend in need is a friend indeed 困った時の友こそ真の友（ことわざ）ことわざ／やや古風
She helped me move out at midnight—a friend in need is a friend indeed.（彼女は真夜中の引っ越しを手伝ってくれた。困った時の友こそ真の友だね。）
🔍 「困窮している時の（そばにいてくれる）友は、本当に(indeed)友と呼べる」という意味の古いことわざ。need と indeed の押韻が記憶の助けになっている。
📝 全体で一つの定型句として引用される。前半だけを A friend in need... と言って余韻を残す使い方もある。
