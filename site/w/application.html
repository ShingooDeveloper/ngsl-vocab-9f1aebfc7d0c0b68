application

名詞 (noun)米 /ˌæplɪˈkeɪʃən/ ／ 英 /ˌæplɪˈkeɪʃn̩/ap·pli·ca·tion（4音節）CEFR B1

🗣 第3音節 -ca- に主強勢（/ˈkeɪ/、「ケイ」）が置かれ、第1音節 ap- に第2強勢（/ˌæp/）が来る。第2音節 -pli- と第4音節 -tion は無強勢であいまい母音化し、それぞれ /plɪ/、/ʃən/ と弱く短く発音される（『シヨン』のようにはっきり発音しない）

  1. 【名詞】申し込み、応募（書類）（She filled out an application for the summer internship. 彼女はその夏季インターンシップの申込書に記入した）
  2. 【名詞】（スマホ・PCの）アプリ、アプリケーションソフト（This application lets you edit photos on your phone. このアプリを使うとスマホで写真を編集できる）
  3. 【名詞】適用、応用（技術・理論などを実際に用いること）（the application of new technology in medicine 医療への新技術の応用）
  4. 【名詞】（クリームなどの）塗布、一回分の塗る量（Apply a thin application of cream to the affected area. 患部に薄く一塗りのクリームを塗ってください）

語形変化

形種類発音備考

語法マーク

  • 語義(1)(2)(4)は可算名詞（a/an application, applications）。語義(3)「適用・応用」は原則として不可算で、単数形のまま冠詞なしで使われることが多い
  • 『申し込み』の意味では、何を求めるかを表す対象には for（application for a job/visa）、出願先の組織・機関には to（application to a university）を使い分ける
  • 『適用』の意味では application of X to Y（XをYに適用すること）という of...to...の構文が典型
  • 動詞 apply と対応関係にあるが、名詞化しても前置詞はそのまま引き継がれる（apply for → application for、apply to → application to）
  • 他品詞での用法: 単一品詞（名詞）だが4つの主要な語義を持つ多義語で、可算・不可算が語義によって切り替わる点に注意。(1)申し込み・出願：可算（an application, applications）。例: a job application（就職の申込書）。(2) スマホ・PCの「アプリ」：可算で、口語では app と略されることが多い。(3) 適用・応用：原則として不可算で、the application of A to B の形を取る。(4) 塗布・一回分の塗る量：可算（a thin application of cream）。

文型・頻出パターン

  • application for + N〜への申し込み・応募（求める対象を示す）
    an application for a student visa学生ビザの申請
    She submitted an application for the position.彼女はその職への応募書類を提出した。
  • application to + 組織〜（大学・機関）への出願・申し込み
    her application to Oxford Universityオックスフォード大学への彼女の出願
  • the application of A to BAをBに適用すること・応用すること
    the application of AI to medical diagnosis医療診断へのAIの応用
  • fill out / submit / process an application申込書に記入する／提出する／処理する（頻出する動詞との組み合わせ）
    You need to fill out an application before Friday.金曜日までに申込書に記入する必要がある。
    It can take weeks to process an application.申込書の処理には数週間かかることがある。

他のNGSL語への置き換え

application → request
元: Her application was accepted by the university last month.
後: Her request was accepted by the university last month.
訳: 彼女の要望は先月、その大学に受理された。
📝 request は「要望・依頼」全般を指す中立的な語で、必ずしも書式のある正式書類を意味しない。application は入学・就職などの正式な出願・応募の書類や手続きそのものを指す点が異なる。
application → form
元: The application that I submitted last week still hasn't been processed.
後: The form that I submitted last week still hasn't been processed.
訳: 先週提出した用紙はまだ処理されていない。
📝 form は「用紙・書式」という書類そのものを指す中立語。application は「申し込み・出願という行為とその書類一式」を指し、目的（何かに応募する）が含意される点が違う。
application → resume
元: Please attach a recent photo to your application before you send it.
後: Please attach a recent photo to your resume before you send it.
訳: 送る前に、あなたの履歴書に最近撮った写真を添付してください。
📝 resume（履歴書）は経歴・職歴をまとめた書類そのものを指す。application は応募という手続き全体、または履歴書を含む申込書一式を指すことが多く、resumeより範囲が広い。

⚠️ 日本人の典型誤用
✕ I applicated for the summer internship last week.
「先週、夏季インターンシップに申し込んだ」と言いたかったが、applicate という動詞は英語に存在せず、聞き手には作り話の単語のように響いてしまう。
○ I applied for the summer internship last week.
先週、夏季インターンシップに申し込んだ。
I applied for the summer internship last week.
名詞 application から存在しない動詞 "applicate" を作ってしまう誤り。「申し込む」の動詞は apply であり、名詞形 application とは不規則な対応関係にある。
✕ Please send application by email.
「申込書をメールで送ってください」と言いたいが、application が冠詞なしの裸の単数形で使われており、英語としては非文法的で舌足らずに響く。
○ Please send an application by email.
申込書をメールで送ってください。
Please send an application by email.
Please send your application by email.
application は可算名詞なので、単数形で使う際は必ず冠詞（a/an/the）や所有格を伴う必要がある。日本語には冠詞に相当する概念がないため脱落しやすい。
✕ I submitted my application to the job.
「その仕事に応募書類を提出した」と言いたいが、job の前に to を使うと、出願先の機関名の前で使う to と混同しているように聞こえ不自然。
○ I submitted my application for the job.
その仕事に応募書類を提出した。
I submitted my application for the job.
「〜への応募」の意味では、応募先の組織・機関には to（application to Harvard）、応募が求める対象の職・地位・許可には for（application for the job, for a visa）を使う。job のような「求める対象」には for が正しい。
🗾 カタカナ語とのギャップ

カタカナ「アプリケーション」は6モーラで各音が同じ長さに聞こえるが、英語は4音節で主強勢が第3音節 -ca- (/keɪ/) に集中し、他の母音（特に第1音節と第4音節の -tion）はあいまい母音化して弱く短くなる。また日常語の『アプリ』はスマホアプリの意味に限定されるが、英語の application は『申込書』『適用』『塗布』など幅広い意味を持ち、カタカナ「アプリ」のイメージだけで捉えると意味を誤解しやすい。

🌱 Word family・語源
apply（動詞）申し込む、適用する、当てはまるNGSL
applicant（名詞）応募者、志願者
applicable（形容詞）適用できる、該当する
appliance（名詞）（家庭用の）電化製品、器具
reapply（動詞）再度申し込む、再度塗る
語源: ラテン語 applicare（ad-「〜へ」+ plicare「折りたたむ、重ねる」）に由来し、原義は『〜に折り重ねて取り付ける、押し当てる』。古フランス語 aplicacion を経て中英語に入った。『申し込む』という意味は、比喩的に『自分をある地位・組織にぴったり当てはめる、寄り添わせる』という発想から発展したと考えられる。
apply（申し込む、適用する）— 同じ動詞語根 applicare の直接の派生語で、名詞形が application
reply（返事する）— re-（後ろへ）+ plicare（折る）＝『折り返す』が原義で、返答するという意味に発展
imply（暗に示す、ほのめかす）— im-（中に）+ plicare（折る）＝『中に折り込む』→言外に含める、ほのめかす
complicate（複雑にする）— com-（共に）+ plicare（折る）の派生語で『幾重にも折り重ねる』→事を複雑にする
💡 覚え方: 語根 -pli-/-plic- は『折る』という意味。ラテン語 applicare は『〜へ(ad-)折り重ねて(plicare)押し当てる』が原義。紙を折って書類を提出先に『ぴったり当てはめる』イメージで application（申込書）を覚えると、apply・reply・imply などの-ply/-plic-仲間ともつながる。
⚖️ 類義語の使い分け
request — request は「依頼・要望」全般を指す中立語で、正式な書式や手続きを必ずしも伴わない。application は入学・就職などの正式な出願手続きとその書類一式を指し、より形式張った文脈で使われる。
app — app は application（ソフトウェアの意味）のくだけた省略形。日常会話やスマホの文脈ではほぼ app が使われ、application は取扱説明書やビジネス文書などフォーマルな場面で好まれる。
use — 『適用・応用』の意味での application は学術的・技術的な文脈で使われる硬い語（the application of a theory）。use はより口語的で汎用的な『使用』を表し、意味範囲が広い。
