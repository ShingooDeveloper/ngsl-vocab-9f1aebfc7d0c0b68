pull

動詞・名詞米 /pʊl/ ／ 英 /pʊl/pull（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。母音は短母音 /ʊ/（book, put と同じ）で、long /uː/（pool, fool）と混同しないよう注意。語末の l は暗い l（dark l）で、舌先を上の歯茎につけたまま奥にこもった響きで終える

  1. 【動詞】（ロープ・取っ手などを）引く、引っ張る
  2. 【動詞】（筋肉を）痛める、（歯を）抜く
  3. 【動詞】（人の注意・関心を）引きつける、引き込む
  4. 【動詞】（pull over/up/out などの形で）車が方向を変えて動く・停止する・出る
  5. 【名詞】引くこと、一引き；（比喩的に）引力、影響力・コネ（Give the rope a strong pull. ロープを強く引いて。／He has a lot of pull in this town. 彼はこの町でかなり顔が利く。）

語形変化

形種類発音備考
pulls三単現/pʊlz/語末 l は有声音なので -s は /z/（「プルズ」であって「プルス」ではない）
pulled過去形・過去分詞/pʊld/規則変化。語末 l は有声音なので -ed は /d/（「プルド」であって「プルット」ではない）
pullinging形/ˈpʊlɪŋ/規則変化。語末がすでに子音 l 一つなので、ing を付ける際に子音を重ねる必要はない（× pulling ではなく綴りはそのまま pull + ing）

語法マーク

  • 基本は他動詞で目的語を必ず伴う（pull the rope, pull the door）。目的語を示さず自動詞的に使うのは pull over / pull up / pull out のような句動詞の形が中心
  • 結果構文 pull + O + 形容詞/副詞辞が可能: pull the door shut（ドアを引いて閉める）, pull the drawer open（引き出しを引いて開ける）
  • 進行形にできる動作動詞（She is pulling the rope now.）
  • give/take のような二重目的語（SVOO）の型は取りにくい（× pull him the rope は不自然。pull the rope for him や pull the rope toward him のように前置詞句で表す）
  • 他品詞での用法: 名詞用法もある。(1) 基本の「引くこと・一引き」: Give the door a hard pull. （ドアを強く引いて。）(2) 比喩的な「引力・魅力」: the pull of the ocean（海の引力・海に引かれる感覚）のように、物理的な力だけでなく心理的に引きつけられる感覚にも使う。(3) スポーツ用語として、野球で打者が球を自分の利き手側（引っ張る方向）へ打つことを pull と言い、そうした打者を a pull hitter（プルヒッター）と呼ぶ。ゴルフでは（右打ちの場合）まっすぐ左へそれる打球を a pull と言う。

文型・頻出パターン

  • pull + O〜を引く・引っ張る（最も基本的な他動詞用法）
    pull the ropeロープを引く
    pull the doorドアを引く
    pull a weed雑草を抜く
  • pull + O + 形容詞/前置詞句（結果構文）引いて〜の状態にする
    pull the door shutドアを引いて閉める
    pull the drawer open引き出しを引いて開ける
  • 句動詞 pull + 副詞辞（自動詞的）車の動きや状況の変化を表す（止まる・出る・抜け出す 等）
    pull over車を路肩に寄せて止める
    pull out（車が）発進して出る、（計画などから）手を引く
  • pull a + 身体部位/動作名詞体の一部を痛める、または特定の動作をする（決まった言い方）
    pull a muscle筋肉を痛める
    pull a face変な顔をする
  • S + pull(s) + O + toward/to ~（比喩的用法）〜を…へ引きつける（物理的にも心理的にも）
    The current pulled the swimmer toward the rocks.潮の流れが泳いでいた人を岩の方へ引き寄せた。

他のNGSL語への置き換え

pull → strained
元: I've pulled a muscle in my back, so I can't lift anything heavy.
後: I've strained a muscle in my back, so I can't lift anything heavy.
訳: 背中の筋肉を痛めてしまったので、重い物が持てないんです。
📝 strain はやや医学的・フォーマルな響きで、医者との会話やスポーツ医学の文脈でも使われる。pull a muscle は日常会話で最も一般的な言い方。
pull → torn
元: The old shed in the backyard was pulled down last month to make room for a new garage.
後: The old shed in the backyard was torn down last month to make room for a new garage.
訳: 裏庭の古い物置は、新しいガレージのスペースを作るために先月取り壊された。
📝 tear down は解体全般に使われる一般的な表現でニュース記事などで頻出。pull down は人手や機械で「引き倒す」イメージがやや強い。
pull → yanked
元: If you pulled that plug out now, the whole computer would shut down immediately.
後: If you yanked that plug out now, the whole computer would shut down immediately.
訳: もし今そのプラグをぐいっと引き抜いたら、コンピューター全体がすぐに落ちてしまうだろう。
📝 yank は「勢いよく・乱暴に引っ張る」という急激な動作を強調する。pull は中立的な動作で、yank ほどの荒っぽさは含まない。

⚠️ 日本人の典型誤用
✕ I pulled a cold last week.
「先週風邪をひいた」と言いたかったが、英語では意味が通らず、「先週風邪を引っ張った」のような奇妙な文に聞こえてしまう。
○ I caught a cold last week.
先週、風邪をひいてしまった。
I caught a cold last week.
日本語「風邪をひく」の「ひく」を pull と直訳してしまう誤り。英語で病気に「かかる」は catch（またはget）を使うのが決まった言い方で、pull は使わない。
✕ She pulled a line under the important word.
「大事な単語の下に線を引いた」と言いたかったが、pull を使うと線をロープのように物理的に引っ張るような不自然な意味に聞こえてしまう。
○ She drew a line under the important word.
彼女は大事な単語の下に線を引いた。
She drew a line under the important word.
日本語「線を引く」の「引く」を pull と直訳してしまう誤り。線や絵を「引く・描く」は英語では draw を使い、pull は物理的にロープや取っ手などを手前に引く動作に限られる。
✕ He pushed the trigger by accident.
「誤って引き金を引いてしまった」と言いたかったが、push を使うと「引き金を押した」という不自然で状況の伝わらない文になってしまう。
○ He pulled the trigger by accident.
彼は誤って引き金を引いてしまった。
He pulled the trigger by accident.
ボタンのように押すイメージから push を選んでしまう誤り。銃の引き金は指を手前に引いて動かす仕組みなので、英語でも日本語同様 pull the trigger が決まった言い方。
🗾 カタカナ語とのギャップ

カタカナ「プル」は「プ・ル」の2モーラだが、英語は1音節。母音は短母音 /ʊ/（book, put と同じ）で、カタカナの「ウ」よりも口をあまり丸めず力を抜いて短く発音する。また語末の l は舌先を歯茎につけたまま奥にこもらせる暗い l で、「ル」のように母音をはっきり出さない。

🌱 Word family・語源
pullover（名詞）プルオーバー、頭からかぶって着るセーター
pull-up（名詞）懸垂（運動）
pullback（名詞）（軍の）撤退；（相場の）一時的な反落・押し目
語源: 古英語 pullian「（羊毛や羽根などを）むしり取る、引っ張る」に由来。起源ははっきりしないが西ゲルマン語系の固有語とされ、もとは布地や羽根を「むしり取る」という限定的な意味だったのが、次第に一般的な「引く」動作全般を表すようになった。
pullover（プルオーバー）— pull（引く）+ over（上に）で「頭の上から引っ張ってかぶる」衣服 → その名の由来
pull-up（懸垂）— pull（引く）+ up（上へ）で「体を上へ引き上げる」運動動作そのものが名詞化した語
💡 覚え方: 羊毛をむしり取るイメージが「グイッと引く」動作全般に広がった語。pullover は「頭の上から引っ張ってかぶる」セーター、pull-up は「体を上へ引き上げる」懸垂。なお repel/compel/propel など -pel 系の語はラテン語 pellere（押す・駆り立てる）由来で、綴りが似ていても pull とは語源的に無関係な空似語なので注意。

⚖️ 類義語の使い分け
drag — drag は重い物を地面や床に沿って引きずるイメージが強く、動きが遅く重々しい。pull はより中立的で、方向を問わず「手前に力を加えて動かす」動作全般を指す。
tug — tug は短く強い力で急に引っ張る動作を表し、しばしば繰り返しの小刻みな動きを含意する（tug at someone's sleeve）。pull は一回の持続的な引く動作にも使える点でより一般的。
draw — draw はよりフォーマル・文語的で、カーテンを引く（draw the curtains）や剣を抜く（draw a sword）など決まった言い回しに使われることが多い。pull は日常会話での基本動詞。
💬 熟語・慣用表現 (9)
pull someone's leg 〜をからかう、冗談を言う口語
Don't worry, I was just pulling your leg about forgetting your birthday.（心配しないで、誕生日を忘れたなんて言ったのはただの冗談だから。）
🔍 「人の脚を引っ張る」だが実際には脚とは無関係で、「冗談を言ってからかう」という意味に完全に慣用化した表現。
📝 進行形（I'm pulling your leg.）で使うことが多い。相手が本気にしてしまった時に "I'm just pulling your leg!" と種明かしする決まり文句。
pull strings （裏で）コネを使う、影響力を行使する中立
My uncle pulled some strings to get me an interview at the company.（おじが裏でコネを使って、その会社の面接を取り付けてくれた。）
🔍 「（操り人形の）糸を引く」イメージから、裏で人脈や権力を使って物事を有利に進めるという比喩表現。
📝 良い意味にも、コネの乱用というやや否定的な意味にも使われる。pull a few strings のように a few/some を伴うことが多い。
pull yourself together 気持ちを立て直す、平静を取り戻す口語
Pull yourself together and tell me exactly what happened.（落ち着いて、何があったのか正確に話して。）
🔍 「自分自身をまとめて引き寄せる」から、動揺・混乱した状態から立ち直り、冷静さを取り戻すという意味。
📝 命令形で使われることが多く、パニックや涙で取り乱している相手を励ます・叱咤する場面で使われる。
pull the wool over someone's eyes 人をだます、欺く口語
The salesman tried to pull the wool over our eyes about the car's mileage.（そのセールスマンは車の走行距離について私たちをだまそうとした。）
🔍 「（かつらの）羊毛を人の目にかぶせる」から視界を塞ぐイメージで、「だまして真実を見えなくする」という意味の慣用句。
📝 しばしば try to と共に使われ、「だまそうとしたが見抜かれた」という文脈で使われることが多い。
pull your weight 自分の分担分の仕事をこなす、応分の努力をする中立
Everyone on the team needs to pull their weight if we want to finish this on time.（期限内に終わらせたいなら、チームの全員が自分の分担をきちんとこなす必要がある。）
🔍 「自分の体重分を引く」というボート漕ぎ（オール）由来の比喩で、チームの中で自分の役割・責任を果たすという意味。
📝 否定形 not pull your weight「サボる、自分の分の仕事をしない」という形で不満を表す時によく使われる。
pull the plug on （計画・支援などを）打ち切る、中止するビジネス／中立
The investors pulled the plug on the startup after months of losses.（数ヶ月にわたる赤字の末、投資家たちはそのスタートアップへの支援を打ち切った。）
🔍 「（コンセントの）プラグを抜く」という物理的動作から転じて、「資金・支援を止めて計画そのものを終わらせる」という比喩表現。
📝 資金提供・プロジェクト・関係を「打ち切る」際に使う比喩表現。文字通りコンセントを抜く場合は pull the plug out のように out を伴い、区別される。
pull rank 地位・権限を笠に着る、職権を振りかざす中立
The manager pulled rank to force the team to work on the weekend.（部長は職権を振りかざして、チームに週末出勤を強いた。）
🔍 「階級（rank）を引き出す」から、上司や上位者が自分の地位を利用して部下に指図するという意味の慣用句。
📝 たいてい否定的なニュアンスを伴い、「本来の議論ではなく地位で押し切る」ことへの批判として使われる。pull rank on someone の形も可。
pull off （難しいことを）うまくやり遂げる、成功させる口語
Nobody thought they could win, but the team pulled it off in the final minute.（誰も勝てるとは思っていなかったが、そのチームは終了間際に見事にやり遂げた。）
🔍 字面通りには「引き剥がす」だが、比喩的に「困難なことを見事に成し遂げる」という意味に転じた表現。
📝 しばしば pull it off の形で使われ、驚き・称賛のニュアンスを伴う。詐欺やいたずらを「まんまと成功させる」の意味でも使われる（pull off a heist）。
pull a fast one (on someone) 人をだしぬく、ずるい手を使ってだます口語
He tried to pull a fast one on me by changing the price at the last minute.（彼は土壇場で値段を変えて、私をだまそうとした。）
🔍 「素早い一手を繰り出す」イメージから、相手に気づかれないようにずるい手段でだます・出し抜くという口語表現。
📝 主に try to pull a fast one の形でよく使われ、ずるさ・不誠実さを非難するニュアンスが強い。
