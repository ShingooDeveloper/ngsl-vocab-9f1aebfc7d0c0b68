start

動詞・名詞米 /stɑːrt/ ／ 英 /stɑːt/start（1音節）CEFR A1

🗣 1音節語。母音は長母音 /ɑː/ のみで、語頭は子音連結 st-、語末は -rt で終わる。「ス・タ・ー・ト」という4拍のカタカナのリズムに引きずられず、st を1つの塊として1拍でまとめて発音する

  1. 【動詞】(～を)始める、開始する（She started her own bakery last year. 彼女は去年、自分のパン屋を始めた。）
  2. 【動詞】(物事が)始まる（The movie starts at seven. 映画は7時に始まる。）
  3. 【動詞】(エンジン・車などが)始動する、かかる（The car wouldn't start this morning. 今朝、車がかからなかった。）
  4. 【動詞】(～し)始める、着手する（start doing/to do ~）（He started working here two months ago. 彼は2か月前にここで働き始めた。）
  5. 【名詞】(時点としての)始まり、開始（the start of the meeting 会議の始まり）
  6. 【名詞】(競争・物事の)出だし、スタート（get off to a good start いいスタートを切る）

語法マーク

  • 他動詞・自動詞の両方で使われる：他動詞 start + O「〜を始める」（He started the engine.）、自動詞 start「（物事が）始まる」（The movie starts at seven.）
  • start doing / start to do はどちらも「〜し始める」を表し、意味の差はほとんどない（begin と同様）。ただし直後に否定を伴う場合など一部の文脈では to不定詞がやや好まれる
  • 名詞としては可算名詞。a slow start, from the start, get off to a good start のように冠詞や形容詞を伴って使うのが基本で、無冠詞の裸の start は使わない
  • start は単純な『(場所から)出発する』という移動そのものは表しにくい。通勤・外出などで場所を離れる意味では leave や set out の方が自然な場合が多い
  • 他品詞での用法: 名詞用法は可算名詞で、(1) 時点としての「始まり」（the start of the race レースのスタート地点・開始時刻）と、(2) 状態としての「出だし・滑り出し」（a good start, a slow start）の2つの側面を持つ。定冠詞 the start は具体的な開始時点そのものを指すのに対し、不定冠詞 a start は出だしの「様子・質」を評価する言い方になる（get off to a good start は entry の collocations を参照）。

文型・頻出パターン

  • start + O（他動詞）〜を始める・始動させる（目的語を直接取る基本の他動詞用法）
    start a war戦争を始める
    start a family家庭を持ち始める、子供をもうけ始める
    start a rumorうわさを流す
  • start（自動詞）（物事が）始まる、（エンジン・機械などが）動き出す
    The race starts in five minutes.レースは5分後に始まる。
    The old engine won't start.その古いエンジンはどうしてもかからない。
  • start + doing〜し始める（動作の開始。動名詞を伴う形がより一般的）
    She suddenly started crying.彼女は突然泣き始めた。
  • start + to do〜し始める（不定詞。doing とほぼ同義）
    It started to rain just as we left.私たちが出たちょうどそのとき、雨が降り始めた。
  • start with + Oまず〜から始める、〜を手始めにする
    Let's start with the basics.まずは基本から始めよう。

他のNGSL語への置き換え

start → begins
元: The concert starts at eight o'clock sharp, so we should leave now.
後: The concert begins at eight o'clock sharp, so we should leave now.
訳: コンサートは8時ちょうどに始まるから、もう出た方がいい。
📝 begin は start とほぼ同じ意味だが、より改まった・書き言葉的な響きを持つ。案内文やプログラムなど公式な場面では begin が好まれ、日常会話ではより中立的な start の方が圧倒的によく使われる。
start → created
元: This tradition was started by our grandparents over fifty years ago.
後: This tradition was created by our grandparents over fifty years ago.
訳: この伝統は50年以上前に祖父母たちが作り上げたものだ。
📝 create は「新しく作り出す・生み出す」という創造性・独自性を強調するのに対し、start は単に「始めた」という行為そのものに焦点があり、独創性の含みは弱い。
start → rough
元: We got off to a slow start, but things improved by halftime.
後: We got off to a rough start, but things improved by halftime.
訳: 出だしはひどかったけど、ハーフタイムまでには持ち直した。
📝 slow start は単に立ち上がりが遅いという中立的な描写だが、rough start は混乱やトラブルの多い、より否定的でつまずきの大きい出だしを示す。

⚠️ 日本人の典型誤用
✕ I start from my house at seven every morning.
「毎朝7時に家を出る」と言いたかったが、"start from my house" だと「家を拠点に何かを始める」ように聞こえ、単なる出発の意味では不自然、または意図が伝わりにくい。
○ I leave my house at seven every morning.
毎朝7時に家を出る。
I leave my house at seven every morning.
日本語の「(家を)出発する」を安易に start と訳してしまう誤り。start は「開始する」が中心の意味で、単純な移動としての「出る・出発する」には leave や set out の方が自然。
✕ I start learning Japanese when I was ten years old.
「10歳のときに日本語を学び始めた」と過去のことを言いたいが、start を現在形のままにすると時制が合わず不自然に響く。
○ I started learning Japanese when I was ten years old.
10歳のときに日本語を学び始めた。
I started learning Japanese when I was ten years old.
過去の一時点(when I was ten)を示す語があるのに動詞を現在形のまま使ってしまう誤り。過去の出来事を述べる文では start も過去形 started にする必要がある。
✕ I start work from nine to five.
「9時から5時まで働く」と一続きの勤務時間を言いたかったが、start に from nine to five をつなげると意味がねじれて不自然、または誤解を招く表現になる。
○ I work from nine to five.
9時から5時まで働く。
I work from nine to five.
start は「始まる・開始する」という一点しか表せない動詞であり、「〜から〜まで」という期間全体を述べるのには使えない。勤務時間の幅を言うときは work のような動詞に from A to B を添えるのが正しい。
🗾 カタカナ語とのギャップ

カタカナ「スタート」は「ス・タ・ー・ト」と4拍で発音するが、英語の /stɑːrt/ は s と t の間に母音を挟まない子音連結で、1拍でまとめて発音する（「スゥタート」のようにならない）。また語末の t は破裂させずに止めることが多く、カタカナの「ト」のように母音を伴わせない。r も日本語のラ行のように舌先を弾かず、どこにも触れさせない米音の r になる。

🌱 Word family・語源
starter（名詞）（競技の）出走者、スターター；(エンジンの)始動装置；(英)前菜
restart（動詞・名詞）再開する、再起動する；再開
startup（名詞）新興企業（スタートアップ）；起動、立ち上げ
upstart（名詞）成り上がり者、生意気な新参者
startle（動詞）びっくりさせる、ぎょっとさせる
語源: 中英語 sterten「跳ねる、突然飛び出す」に由来し、古英語の記録には残らない *steartian に遡ると考えられる。ゲルマン系の語で、オランダ語 storten やドイツ語 stürzen「急に落ちる、突進する」と同根。もともと「はっと飛び上がる、急に動く」という意味だったものが、「物事が急に動き出す→開始する」という現在の意味へと発展した。
startle（びっくりさせる）— start と同じ語源から生まれた派生語で、「はっと飛び上がらせる」という原義を色濃く残している
German stürzen（（ドイツ語）急に落ちる、突進する）— 英語 start と同じゲルマン祖語の語根に遡る同根語で、「急な動き」という中心イメージを共有する
💡 覚え方: もとは「はっと飛び跳ねる」という突然の動きを表す語。驚いて飛び上がる startle と兄弟語で、そこから『物事が急に動き出す→始まる』という今の意味に育った。

⚖️ 類義語の使い分け
begin — begin は start とほぼ同義だが、より書き言葉的・フォーマルな響きを持つ。日常会話では中立的な start の方が圧倒的によく使われる。
launch — launch は新しい事業・製品・計画などを大々的に、勢いよく『打ち出す』ニュアンスが強く、ビジネスや公式な文脈で好まれる。start は中立的な『開始する』全般に使える。
commence — commence はかなり形式張った語で、式典・法的文書・正式な報告書などで使われる。日常会話ではほとんど使われず、start が自然。
💬 熟語・慣用表現 (7)
head start 有利なスタート、(他より)先んじたスタート中立
Learning to code in high school gave her a head start in her career.（高校でプログラミングを学んだことが、彼女のキャリアに有利なスタートを与えた。）
🔍 「頭の分だけ先を行くスタート」から転じて、他者より早く・有利に始めることで得られるアドバンテージを指す。head は「先頭・先んじた分」の比喩。
📝 have/get a head start (on/over 人) の形でよく使う。競争や人生における「先行者利益」を表す定番表現。
from start to finish 始めから終わりまで、一部始終中立
The plan went smoothly from start to finish.（その計画は最初から最後までスムーズに進んだ。）
🔍 文字通り「開始から終了まで」だが、過程の最初から最後まですべてを強調する決まり文句として使われる。
📝 前置詞なしの副詞句として使う。from beginning to end とほぼ同義。
false start (失敗して仕切り直しとなる)出だしのつまずき、空振りのスタート中立
The project had a false start when the funding fell through.（資金調達が頓挫し、そのプロジェクトは出だしでつまずいた。）
🔍 陸上競技で「合図前に飛び出してしまう不正スタート」が原義。そこから比喩的に「うまくいかず最初からやり直すことになった試み」全般を指す名詞に発展した。
📝 陸上競技のフライングという文字通りの意味でも、ビジネス・人生の比喩的な意味でも使われる。
for starters まず第一に、手始めに口語
I don't want to go—for starters, I have no money.（行きたくないな。まず第一に、お金がないし。）
🔍 料理のコースの前菜（starter）になぞらえ、話や理由の「最初の一皿」として挙げる口語表現。
📝 文頭に置いて複数ある理由・項目のうち最初のものを挙げる際に使う。イギリス英語では for a start ともいう。
start from scratch ゼロから始める、一から出直す中立
The old file was corrupted, so we had to start from scratch.（古いファイルが壊れていたので、私たちはゼロからやり直さなければならなかった。）
🔍 scratch はもともと徒競走の「基準線・スタートライン」を指し、そこから「何もない状態・振出し」を意味するようになった。既存の材料や成果を一切使わずに最初から始めることを表す。
📝 build/cook (something) from scratch のように start 以外の動詞とも組み合わさる汎用表現。
jump-start (物事に)勢いをつけて始動させる、てこ入れするビジネス／中立
The government hopes new spending will jump-start the economy.（政府は新たな歳出が景気にてこ入れすることを期待している。）
🔍 「バッテリーが上がった車をブースターケーブルで始動させる」が原義。そこから「停滞している物事に一気に弾みをつけて動き出させる」の比喩に発展した。
📝 名詞としても jump-start（起爆剤・てこ入れ）の形で使われる。ハイフンを省いて jumpstart と綴られることも多い。
start something (けんか・トラブルを)引き起こす、事を起こす口語
Stop pushing him—do you want to start something?（彼を押すのはやめなよ。もめ事でも起こしたいの?）
🔍 something が具体的に何かを指さない慣用の代名詞で、文脈上「もめ事・けんか」を暗示する。
📝 疑問文・否定命令文でよく使われ、けんかや対立を仄めかす警告表現になる。
