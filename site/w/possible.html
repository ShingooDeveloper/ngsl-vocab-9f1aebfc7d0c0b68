possible

形容詞 (adjective)米 /ˈpɑːsəbl̩/ ／ 英 /ˈpɒsəbl̩/pos·si·ble（3音節）CEFR A2

🗣 第1音節 pos に第一強勢が置かれる（/ˈpɑːsəbl̩/）。第2音節は弱いあいまい母音 /ə/ で軽く添えるだけ。語尾の -ble は母音を持たない成節子音 l̩ で、『ブル』と母音（ウ）を足さずに舌先を歯茎につけたまま止める。

  1. 【形容詞】起こり得る、あり得る（It's possible that the exam will be moved to next week. その試験は来週に延期される可能性がある。）
  2. 【形容詞】（条件が整えば）実現可能な、～できる（Is it possible for you to send me the report before noon? 正午までに報告書を送っていただくことは可能でしょうか。）
  3. 【形容詞】考えられる、想定される（選択肢として）（Can you think of any possible reason that she didn't come to the party? 彼女がパーティーに来なかった理由として、何か考えられるものはありますか。）
  4. 【形容詞】（as ~ as possible や the+最上級+possible で）できる限りの（Please reply to this email as soon as possible. できるだけ早くこのメールにご返信ください。）

語法マーク

  • 叙述用法（It's possible.）・限定用法（a possible solution）のどちらでも使える形容詞。
  • 非人称の it を主語にした形式主語構文 It is possible that S+V / It is possible for 人 to do が基本形。人をそのまま主語にして『（人）が可能だ』の意味では使えない（× I am possible to swim.）。能力を言うときは be able to や can を使う。
  • possible は本来『可能性があるかないか』を表す絶対的な形容詞とされ、比較級 more possible / most possible は理論上作れても、フォーマルな文章では『可能性に程度はない』として避けられる傾向がある（unique と同様の扱い）。
  • as + 形容詞/副詞 + as possible（できるだけ〜）、the + 最上級 + possible + 名詞（できる限り最高の〜）という定型の比較構文で非常によく使われる。
  • It is possible to do（for句なし）は『一般に・誰にとっても可能』という意味になり、for 人 を加えると『（その人）にとって可能』と意味が狭まる。

文型・頻出パターン

  • It is possible that + S + V〜という可能性がある（事実・出来事についての可能性を述べる）
    It is possible that the flight will be delayed.そのフライトは遅れる可能性がある。
  • It is possible for + 人 + to do（人）が〜することは可能だ
    Is it possible for you to come earlier?もっと早く来ていただくことは可能ですか。
  • as + 形容詞/副詞 + as possibleできるだけ〜（程度の上限を示す定型表現）
    Come as quickly as possible.できるだけ早く来て。
  • the + 最上級 + possible + 名詞できる限り最高の〜（理論上の上限を強調する）
    We chose the best possible option.私たちはできる限り最良の選択肢を選んだ。
  • possible + 名詞（限定用法）あり得る〜、想定される〜
    List all the possible causes of the problem.その問題として考えられる原因をすべて挙げてください。
  • if possibleもし可能なら（依頼・提案を和らげる慣用表現）
    Could you send it today, if possible?もし可能なら、今日それを送っていただけますか。

他のNGSL語への置き換え

possible → likely
元: It's possible that the exam will be moved to next week.
後: It's likely that the exam will be moved to next week.
訳: その試験は来週に延期される可能性が高い。
📝 possible は「実現の余地がある（低確率でも成立）」ことを示すのに対し、likely は「実現する可能性が高い」という確信の強さを表す。possible は0でない可能性、likely はその中でも高い確率を指す点が違う。
possible → available
元: We want to offer our customers the best possible service.
後: We want to offer our customers the best available service.
訳: 私たちはお客様に、現時点で提供できる最高のサービスを提供したいと考えています。
📝 best possible は理論上あり得る最高の水準（仮想的な上限）を表すのに対し、best available は今実際に用意できる選択肢の中での最高を表す。possible は可能性の上限、available は現実にアクセスできる範囲を指す。
possible → easy
元: If it were possible, I would finish this project by myself.
後: If it were easy, I would finish this project by myself.
訳: もしそれが簡単なら、一人でこのプロジェクトを終わらせるのだが。
📝 possible は「実現できるかどうか」という可能性の有無を、easy は「どれくらい難しくないか」という難易度を表す。意味の焦点が全く異なるので置き換えると文の主張が変わる点に注意。

⚠️ 日本人の典型誤用
✕ Possible, he will come to the party.
「たぶん彼はパーティーに来るだろう」と言いたかったが、possible は形容詞なので文頭に単独では置けず、ネイティブには文法的に破綻した文に聞こえる。
○ He will possibly come to the party.
彼はたぶんパーティーに来るだろう。
He will possibly come to the party.
possible は形容詞であり、文全体を修飾する副詞としては使えない。『たぶん』のように文頭や動詞の前に置いて確率を表したいときは副詞形 possibly（または maybe）を使う。
✕ I am possible to attend the meeting tomorrow.
「私は明日の会議に出席することが可能だ」と言いたかったが、possible の主語に人を置く形は英語として不自然で、ネイティブには違和感のある文に聞こえる。
○ It is possible for me to attend the meeting tomorrow.
明日の会議に出席することは私にとって可能だ。
It is possible for me to attend the meeting tomorrow.
I am able to attend the meeting tomorrow.
possible は『（人）が可能だ』という形で人を主語にはとれない。日本語の『私は〜可能だ』を直訳するとこの誤りが起きる。人の能力・都合を言いたいときは be able to（またはcan）を使うか、possible を使うなら形式主語 it を立てる。
✕ Please contact me possible soon.
「できるだけ早く連絡してください」と言いたかったが、as が抜けているため定型表現が崩れており、ネイティブには意味が通じにくい不完全な文に聞こえる。
○ Please contact me as soon as possible.
できるだけ早く連絡してください。
Please contact me as soon as possible.
as soon as possible は as ~ as possible という『as + 形容詞/副詞 + as possible』の型が丸ごと定型化した表現で、最初の as を省略できない。possible だけを形容詞・副詞の直前に置いても『できるだけ〜』の意味にはならない。
🗾 カタカナ語とのギャップ

カタカナ「ポッシブル」は「ポ・ッ・シ・ブ・ル」の5拍で読まれがちだが、英語は3音節 /ˈpɑːsəbl̩/。まず母音は「オ」ではなく口を大きく開けた /ɑː/（「パー」に近い）で、カタカナの促音「ッ」に当たる詰まりも実際の英語には無い。語尾の -ble も母音を足さず、舌先を歯茎につけたまま息を止める成節子音 l̩ で終える。

🌱 Word family・語源
impossible（形容詞）不可能な、あり得ないNGSL
possibility（名詞）可能性、あり得ることNGSL
possibly（副詞）たぶん、ひょっとすると；（can/couldとともに）どうにかしてNGSL
語源: ラテン語 possibilis（できる、可能な）に由来。possibilis は posse「〜する力がある、できる」（potis「できる」+ esse「である」の合成）の形容詞形。古フランス語を経て14世紀頃に英語へ入った。
potential（潜在的な、可能性のある）— 同じラテン語 posse/potis（できる）から。『まだ表に出ていない力』を表す
power（力、権力）— 古フランス語 poeir（< 俗ラテン語 potere、posseの変形）から。『〜できること』そのものを指す
potent（強力な、効き目の強い）— posse の現在分詞形 potens（力を持っている）に由来
💡 覚え方: 西部劇でおなじみの『ポッシー（posse＝保安官が召集する追っ手の一団）』は、ラテン語 posse comitatus『郡の実力（集団）』が語源。possible の posse も同じ『〜できる、力がある』の意味で、power・potent・potential も全部この posse の子孫だとつなげて覚える。
⚖️ 類義語の使い分け
likely — likely は『起こる可能性が高い』という蓋然性の高さを表すのに対し、possible は蓋然性の高低を問わず『理論上あり得る（可能性が0ではない）』ことを示す。
feasible — feasible は計画・方法などが実務的に実行可能かどうかに焦点があり、ビジネスや技術の文脈でよく使われる。possible はより一般的・日常的な『あり得る』を表す。
potential — potential は名詞的に『潜在能力・将来性』を表し、まだ実現していないが芽があるというニュアンスが強い。possible は単に『理論上ありうる』という中立的な判断を表す。
