language

名詞 (noun)米 /ˈlæŋɡwɪdʒ/ ／ 英 /ˈlæŋɡwɪdʒ/lan·guage（2音節）CEFR A1

🗣 第1音節 lan- に強勢がある。つづり -guage は「ゲージ」ではなく /wɪdʒ/（ウィッジ）と読む点に注意（gu は /ɡw/、age は /ɪdʒ/）。母音は1音節目が /æ/、2音節目が弱い /ɪ/

  1. 【名詞】言語、ことば（人間が意思疎通に用いる体系全般を指す。language skills 語学力）
  2. 【名詞】（英語・日本語など）個々の言語（a foreign language 外国語；a native language 母語）
  3. 【名詞】言葉遣い、表現のしかた（bad language 下品な言葉づかい；formal language 改まった言い方）
  4. 【名詞】（コンピュータの）プログラミング言語（a programming language）
  5. 【名詞】（身振り・芸術などによる）伝達手段（body language 身振り言語；the language of music 音楽という言葉）

語形変化

形種類発音備考
languages複数形/ˈlæŋɡwɪdʒɪz/語末が破擦音 /dʒ/ のため -es をつけ、発音上 /ɪz/ が1拍増える（「ラングウィッジ」→「ラングウィッジィズ」）。個々の言語を数える可算用法でのみ使い、『言語という能力・体系全般』を指す不可算用法には複数形を使わない

語法マーク

  • 可算・不可算の両方で使われる。『言語という能力・システム全般』を指す時は不可算（無冠詞・単数）：Language is unique to humans. 個々の言語を指す時は可算：a language / two languages
  • speak / learn / study + 言語名 は前置詞なしで直接目的語を取る（× speak in Japanese ではなく speak Japanese が基本。ただし『英語を使って話す』という手段を強調する in English は文脈により可）
  • bad language（下品な言葉づかい）は常に不可算・単数扱いの決まった言い方
  • the language of + 名詞 で『〜という言葉・表現手段』という比喩的な意味になる（the language of music など）
  • 他品詞での用法: language は基本的に名詞としてのみ使われ、現代英語に動詞用法はない（"to language something" とは言わない）。ただし名詞のまま別の名詞を直接修飾する『名詞＋名詞』の複合表現を非常に多く作る: language school（語学学校）、language skills（語学力）、language barrier（言葉の壁）、language learner（言語学習者）。

文型・頻出パターン

  • speak + 言語（無冠詞）（言語）を話す。言語名を直接目的語に取る
    She speaks three languages fluently.彼女は3つの言語を流暢に話す。
    Do you speak Japanese?日本語は話せますか?
  • learn / study + a language言語を学ぶ（可算・冠詞つき）
    It's never too late to learn a new language.新しい言語を学ぶのに遅すぎるということはない。
  • the language of + NNの言葉、Nを象徴的に表す伝達手段（比喩表現）
    Music is often called the language of emotion.音楽はしばしば感情の言葉と呼ばれる。
  • in + 形容詞 + language〜な言葉づかいで、〜な文体で
    Try to explain the rule in plain language.そのルールをわかりやすい言葉で説明してみて。
  • a / the language barrier言葉の壁（意思疎通を妨げる言語の違い）
    They managed to communicate despite the language barrier.彼らは言葉の壁があっても何とか意思疎通した。

他のNGSL語への置き換え

language → book
元: This novel has been translated into more than thirty languages.
後: This book has been translated into more than thirty languages.
訳: この本は30以上の言語に翻訳されている。
📝 novel は虚構の「小説」に限定されるが、book はノンフィクションや専門書も含むより広い「本」全般を指す中立語。翻訳される対象が小説とは限らない場合は book の方が自然。
language → master
元: If you learned the local language, it would be much easier to make friends here.
後: If you mastered the local language, it would be much easier to make friends here.
訳: 現地の言葉を使いこなせるようになれば、ここで友達を作るのがずっと楽になるだろう。
📝 learn は学び始め・習得の過程を表す中立的な語だが、master は流暢に使いこなせるレベルまで極めることを強調する。基礎段階の話なら learn、高い到達度を言いたいなら master が適切。
language → words
元: Please choose your language carefully when you're writing a formal email.
後: Please choose your words carefully when you're writing a formal email.
訳: 正式なメールを書くときは、言葉選びを慎重にしてください。
📝 language は言葉遣い・文体全体を指す抽象的な語だが、words は個々の単語の選び方に焦点を当てる。choose your words carefully は日常会話で非常によく使われる、より口語的で具体的な定型表現。

詳細
⚠️ 日本人の典型誤用
✕ I want to learn language.
『言語を学びたい』と言いたかったが、language を無冠詞・単数のまま使うと『言語というもの（人間の能力としての言語）そのものを学びたい』という不自然で漠然とした意味に聞こえてしまう。
○ I want to learn a language.
私は言語を学びたい。
I want to learn a language.
I want to learn languages.
特定の一つの言語を学びたい場合は可算名詞として冠詞 a が必要。無冠詞・単数のままだと不可算の『言語という能力全般』の意味と紛れてしまう。複数の言語について言うなら languages と複数形にする。
✕ I can speak many language.
『たくさんの言語を話せる』と言いたかったが、many の後に単数形 language を置くと文法的に破綻しており、不自然に聞こえる。
○ I can speak many languages.
私はたくさんの言語を話せる。
I can speak many languages.
many の後には可算名詞の複数形が来る。個々の言語を数えているので language を languages と複数形にする必要がある。
✕ I don't know this language. What does it mean?
『この単語がわからない』と言いたかったが、language を使うと『この言語（体系）自体を知らない』という意味に聞こえてしまい、単語の意味を尋ねる文脈とかみ合わない。
○ I don't know this word. What does it mean?
この単語がわからないんですが、どういう意味ですか?
I don't know this word. What does it mean?
language は個々の単語ではなく、言語という体系全体を指す。一つの単語の意味がわからないと言いたい時は word を使う。
🗾 カタカナ語とのギャップ

カタカナ「ランゲージ」は「ラ・ン・ゲ・ー・ジ」の5モーラだが、英語は2音節。最初の音節 lan- の母音 /æ/ は日本語の「ア」より口を横に引いた音で「ラ」とは異なる。また -guage の部分は「ゲージ」ではなく /ɡwɪdʒ/（グ+ウィッジ）で、「ゲ」の音は入らない点に注意。

🌱 Word family・語源
linguistics（名詞）言語学
linguist（名詞）言語学者
linguistic（形容詞）言語の、言語学上の
bilingual（形容詞・名詞）2言語を話せる（人）
languageless（形容詞）言葉にならない、言語を持たない（まれ・文語的）
語源: 古フランス語 langage「言葉、話し方」に由来し、これはさらに俗ラテン語 linguaticum、古典ラテン語 lingua「舌、言語」に遡る。舌を使って話すことから『言語』を意味するようになった。
linguistics（言語学）— ラテン語 lingua「舌・言語」から。言語を科学的に研究する学問を指す
bilingual（2言語を話せる）— bi-（2つの）+ lingua（言語）で『2つの言語を操れる』
lingua franca（共通語）— ラテン語 lingua がそのまま使われた表現。異なる母語を持つ人々の間の共通のやり取り言語を指す
💡 覚え方: language の語源は『舌』を意味するラテン語 lingua。舌を動かして話すこと=言語。bilingual（2言語を話せる）や linguistics（言語学）も同じ『舌』にルーツを持つ仲間。
⚖️ 類義語の使い分け
tongue — tongue は『舌』の意味に加え、文学的・やや古風な言い方で『言語』を指す（mother tongue=母語）。日常一般の言語を指す文脈では language の方が中立的で頻度が高い。
speech — speech は話し言葉・発話行為そのもの、または『演説』を指すことが多く、language が持つ体系全体（書き言葉も含む）という広い意味とは異なる。
dialect — dialect はある言語の中の地域的・社会的な変種（方言）を指し、独立した言語体系全体を指す language とは階層が異なる（言語の下位区分）。
💬 熟語・慣用表現 (3)
speak the same language 気が合う、考え方が同じで理解し合える中立
As soon as they met, they realized they spoke the same language when it came to raising children.（出会ってすぐ、二人は子育てに関する考え方がぴったり合うと気づいた。）
🔍 文字通りには「同じ言語を話す」だが、実際に同じ母語かどうかではなく、価値観や考え方が一致していて意思疎通がしやすいことを比喩的に表す定型表現。
📝 逆に意見や価値観がまったく合わない相手には we don't speak the same language / we're speaking different languages のように使う。文字通り「同じ言語を話す」の意味と区別するのは文脈次第。
watch your language 言葉遣いに気をつけろ、下品な言葉を使うな口語
Watch your language! There are children in the room.（言葉遣いに気をつけて。部屋に子どもがいるんだから。）
🔍 「自分の言葉遣いを見張れ」という命令表現で、乱暴・下品な言葉（特に罵り言葉）を使わないよう相手に注意する時の決まり文句。
📝 イギリス英語では mind your language も同義でよく使われる。親や教師が子どもをたしなめる時の典型的な表現。
strong language 下品な言葉遣い、罵り言葉ビジネス／中立
The movie was rated R for violence and strong language.（その映画は暴力描写と下品な言葉遣いのためR指定を受けた。）
🔍 「強い言語」という意味ではなく、罵り言葉や卑猥な表現など「きつい・下品な言葉遣い」を指す固定表現。映画やドラマの視聴者向け注意書き（contains strong language）でよく使われる。
📝 bad language とほぼ同義だが、strong language の方が特に激しく攻撃的な罵倒表現を強調するニュアンスを持つ。人に直接 "Please, no strong language here." のように注意する時にも使われる。
