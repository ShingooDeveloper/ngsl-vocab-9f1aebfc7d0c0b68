sure

形容詞 (adjective)米 /ʃʊr/ ／ 英 /ʃɔːr/sure（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷いはない。母音はrに彩られた/ʊr/で、口をすぼめた曖昧な「ウ」からそのままrへ途切れなく移行させる（「シュ・ア」と2拍に割らない）。なお現代米語ではshoreと同じ/ʃɔːr/という発音も非常によく聞かれ、どちらも標準的とされる。

  1. 【形容詞】（人が）確信している、確かだと思う（I'm not sure if he'll come. 彼が来るかどうか確信が持てない。）
  2. 【形容詞】（〜することが）確実な、必ず〜する（She's sure to be surprised by the news. 彼女はきっとその知らせに驚くだろう。）
  3. 【形容詞】確かな、信頼できる、まず間違いのない（There's no sure way to predict the weather. 天気を確実に予測する方法はない。）
  4. 【形容詞】（返答で単独に用いて）もちろん、いいよ（"Could you help me?" "Sure!" 手伝ってくれる? もちろん!）

語形変化

形種類発音備考
surer比較級/ˈʃʊrər/1音節でrで終わるため-erを直接付加（more sureも可だが surer の方が一般的）。語末に無強勢の-erが加わり2音節になる

語法マーク

  • 基本は「be動詞 + sure」の叙述用法。名詞を直接修飾する限定用法は a sure sign（確かな兆候）、a sure win（確実な勝ち）など一部の慣用的な組み合わせに限られ、自由には使えない（× a sure book のようには使わない）
  • be sure that + 節／be sure of・about + 名詞／be sure to do の3つの構文をとる。ofとaboutはどちらも使えるがofの方がやや一般的
  • 段階的形容詞として比較変化する（sure - surer - surest、または more sure - most sure）。very, quite, really などの程度副詞で修飾できる
  • 返答として単独で使う Sure!（もちろん、いいよ）は快諾のあいづちで、I'm sure（確信している）とは意味が異なる別の用法。混同しないこと
  • Are you sure? / I'm not sure. のように疑問文・否定文で頻出し、日常会話での確認・不確実性の表明に多用される
  • 他品詞での用法: 口語的なアメリカ英語では、副詞的に「本当に、確かに」（certainly/reallyに近い意味）で使われることがある: That sure was a great movie!（あれは本当にいい映画だったね!）/ It sure is hot today.（今日は本当に暑いね。）。この用法はかなりカジュアルで、書き言葉やフォーマルな場面では certainly/really/definitely の方が適切。また sure enough という副詞句で「案の定、思った通り」という意味を表す（Sure enough, it started raining the moment we left the house. 案の定、家を出た途端に雨が降り出した。）。

文型・頻出パターン

  • be sure (that) + 節〜だと確信している
    I'm sure (that) she'll understand.彼女はきっとわかってくれると思う。
  • be sure of/about + 名詞〜について確信している
    Are you sure about your decision?自分の決断に自信はある?
  • be sure to do必ず〜する、きっと〜する（高い確実性を表す）
    He's sure to be late again.彼はまたきっと遅刻するだろう。
  • make sure (that) + 節 / make sure to do〜を確実にする、必ず〜するようにする
    Make sure you save your work before closing the file.ファイルを閉じる前に必ず保存してね。
  • for sure（口語の副詞句）確かに、間違いなく
    That's one thing I know for sure.それだけは確かに分かっている。
  • Sure!（単独の返答）もちろん、いいよ（依頼・提案への快諾）
    "Can I borrow your pen?" "Sure!"「ペン借りてもいい?」「もちろん!」

他のNGSL語への置き換え

sure → remember
元: Make sure you turn off the stove before you leave the kitchen.
後: Remember to turn off the stove before you leave the kitchen.
訳: 台所を出る前に、コンロの火を消すのを忘れないでね。
📝 make sureは「確認して確実に実行させる」という念押しのニュアンスが強いのに対し、remember toは「忘れずに」と記憶に焦点を当てた表現で、念押し感はやや弱い。
sure → likely
元: If you don't stretch before running, you're sure to hurt your knees eventually.
後: If you don't stretch before running, you're likely to hurt your knees eventually.
訳: ランニングの前にストレッチをしないと、いずれ膝を痛める可能性が高い。
📝 sure toはほぼ確実だという高い確信を表すのに対し、likely toは「〜しそうだ」という蓋然性を表し、確信の度合いがsureより低い。
sure → best
元: Getting a good night's sleep is often the surest way to feel better.
後: Getting a good night's sleep is often the best way to feel better.
訳: ぐっすり眠ることが、気分を良くする一番の方法であることが多い。
📝 surestは「確実性」に焦点を当てるのに対し、bestは「効果の高さ」に焦点を当てる。sureからbestへの置き換えで「確実に効く」から「一番効果的」へとニュアンスが変わる。

⚠️ 日本人の典型誤用
✕ I'm sure for his honesty.
「彼の誠実さを確信している」と言いたかったが、sure forという組み合わせは英語として不自然で、ネイティブには意味が伝わりにくい。
○ I'm sure of his honesty.
彼の誠実さを確信している。
I'm sure of his honesty.
sureの後で「〜について確信している」を表す前置詞はof/aboutであり、forは使わない。日本語の「〜について」という感覚からforを選んでしまう典型的な誤り。
✕ Please for sure to turn off the lights when you leave.
「出るときは必ず電気を消してください」と言いたかったが、for sure toという形は文法的に誤りで、ネイティブには不自然に響く。
○ Please make sure to turn off the lights when you leave.
出るときは必ず電気を消してください。
Please make sure to turn off the lights when you leave.
for sureは「確かに、間違いなく」を表す口語的な副詞句で、to不定詞を伴って動詞のように使うことはできない。誰かに「必ず〜するように」と指示・依頼する場合は make sure to do / make sure that を使う。
✕ Sure he will pass the exam this time.
「きっと彼は今回試験に受かるだろう」と言いたかったが、文頭に裸でSureを置くと依頼への「もちろん」という返事のように聞こえてしまい、意図した確信のニュアンスが伝わらない。
○ I'm sure he will pass the exam this time.
きっと彼は今回試験に受かるだろう。
I'm sure he will pass the exam this time.
He will surely pass the exam this time.
文頭の裸のSureは依頼・提案に対する「もちろん」という返答の間投詞であり、「きっと〜だろう」という話者自身の確信を表す文の頭に置くことはできない。その意味を表すには I'm sure (that) 〜、または副詞 surely/certainly を使う。
🗾 カタカナ語とのギャップ

カタカナ「シュア」は「シュ・ア」の2モーラだが、英語は1音節。/ʃʊr/は口をすぼめた曖昧な母音からそのまま途切れなくrへ移行させ、「ア」のようにはっきり分離した母音を作らない。現代米語ではshoreと同じ/ʃɔːr/で発音されることも多いが、いずれにせよ2拍に割らないことが重要。

🌱 Word family・語源
assure（動詞）〜を安心させる、〜に保証するNGSL
ensure（動詞）〜を確実にする、保証するNGSL
insure（動詞）〜に保険をかけるNGSL
insurance（名詞）保険NGSL
surely（副詞）きっと、確かにNGSL

語源: 中英語期に古フランス語 seür（現代フランス語 sûr）を経由して英語に入った語で、さらに遡るとラテン語 securus（se-「〜なしに」+ cura「心配」=「心配のない」）に行き着く。secure と同じラテン語源から分かれた二重語（doublet）で、フランス語経由で縮約した形が sure、ラテン語形により忠実な学術的借用語として後から入ったのが secure である。
secure（安全な）— sureと同じラテン語securusに由来する双子の語。フランス語を経て縮約したsureに対し、secureはラテン語形に近い姿のまま英語に借用された
assure（安心させる、保証する）— as-（ad-「〜へ」）+ sureで「（人を）sureな状態にする」が原義の動詞
ensure（確実にする）— en-（「〜にする」）+ sureで「（物事を）確実な状態にする」が原義の動詞
💡 覚え方: secureとsureは実は「同じ言葉」のフランス語版と学術語版という双子。どちらも「心配（cure）がない」状態が原義で、assure/ensure/insureはみな「相手や物事をsureな状態にする」という同じイメージから枝分かれした動詞。
⚖️ 類義語の使い分け
certain — certainはやや硬めでフォーマルな響きを持ち、客観的な確実性（It is certain that ~）にも使いやすい。sureはより口語的で、話者個人の主観的な確信（I'm sure ~）を表す場面で好まれる。
confident — confidentは自分の能力・実力に対する自信（I'm confident I can do this.）に重点があるのに対し、sureは事実や状況が正しい・そうなるという確信（I'm sure it's true.）に重点がある。
certainly — certainlyは文修飾の副詞で「確かに、間違いなく」と文全体の確実性を強めるのに対し、sureは形容詞として人の状態を述べる（口語ではsureも副詞的に使われるが、certainlyの方がフォーマルで書き言葉向き）。
💬 熟語・慣用表現 (3)
for sure 確かに、間違いなく口語
"Is she coming to the party?" "For sure, she wouldn't miss it."（「彼女はパーティーに来る?」「間違いなく、見逃すはずないよ。」）
🔍 前置詞forの後にsureを置く形が口語表現として定着したもので、文字通りに分解しても意味は出てこない。強い断定を表す決まり文句。
📝 文末に置いて直前の内容を強調することが多い（She's the best candidate, for sure.）。単独で「もちろん」の意の返事にも使えるが、Sure!より断定の響きが強い。
sure enough 案の定、思った通り中立
I told him the bridge was closed, and sure enough, we had to turn back.（橋が閉鎖されていると彼に言っておいたが、案の定、私たちは引き返さなければならなかった。）
🔍 「十分に確かな」という直訳ではなく、「予想していた通りに実際そうなった」ことを確認する副詞句として定型化している。
📝 直前に述べた予測・警告・懸念が現実になったことを示す際に使う。文頭やコンマの後に置くのが基本で、as expected とほぼ同義。
sure thing （返答で）もちろん、いいとも／（名詞として）確実なこと、絶対に上手くいくもの口語
"Can you send me the file today?" "Sure thing, I'll do it right now."（「今日中にファイルを送ってもらえる?」「もちろん、今すぐやるよ。」）
🔍 「確かなもの」という字義から転じて、依頼への快諾の返事、または「絶対に成功・実現する物事」を指す口語表現に固定化した。
📝 返答としてはSure!よりくだけた響き。名詞用法では a sure thing（賭けなどで必ず当たる・絶対確実な物事）としても使われる。
