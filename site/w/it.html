it

代名詞 (pronoun)米 /ɪt/ ／ 英 /ɪt/it（1音節）CEFR A1

🗣 1音節語。母音は緩い（弛緩した）短母音 /ɪ/ で、日本語の「イ」よりも口をゆるめ、「エ」に近い曖昧な音。機能語なので文中では基本的に無強勢で、はっきり「イット」と切らずに前後と溶け込む

  1. 【代名詞】それ、それを（前に出た物・事・動物などを指す）
  2. 【代名詞】（天候・時間・距離・寒暖などを表す）it（It's raining. / It's ten o'clock. / It's cold.）
  3. 【代名詞】（形式主語）真主語の to不定詞・that節を後回しにする it（It's hard to say. / It's true that ...）
  4. 【代名詞】（形式目的語）make / find / think + it + 形容詞 + to do の it
  5. 【代名詞】（漠然とした状況・事情を指す）How's it going? / That's it. など

語形変化

形種類発音備考
its所有格（所有限定詞）/ɪts/「それの」。名詞の前に置く所有限定詞（its color）。アポストロフィ付きの it's は it is / it has の短縮形で全くの別物（混同が最頻出の綴りミス）
itself再帰代名詞／強調/ɪtˈself/「それ自身」。再帰用法（The cat hurt itself.）と強調用法（The plan itself is good.）。強勢は第2音節 self 側に置く

語法マーク

  • 3人称・単数・中性の代名詞。主格と目的格は同形（both it）で語形変化しない
  • 原則として人以外（物・事・動物・状況・天候）を指す。ただし生後間もない赤ん坊や性別不明の動物には it を使うこともある
  • 機能語で通例は無強勢・弱く発音される。強調構文の焦点部分など、特別に力を込める場合のみ強く読む
  • 「虚辞の it」= 特定の何かを指さず文の形を整えるためだけに置く it（形式主語・形式目的語・天候の it）がある。この it は日本語に訳出されない
  • 同じ「〜がある」でも存在を新情報として導入するのは There is 〜、it is 〜 ではない点に注意
  • 他品詞での用法: 品詞は代名詞のみで、他の品詞転用はない。ただし単に前出の名詞を受ける通常の代名詞用法のほかに、特定の物を指さない「虚辞（きょじ）の it」＝形式主語・形式目的語・天候や時間の非人称の it という特殊用法が非常に発達している点が最大の特徴。学習上はこの「中身のない it」をどう使うかが要。

文型・頻出パターン

  • It + be + 形容詞 + to do / that節（形式主語）真主語の to不定詞・that節を後ろに回し、頭に軽い it を立てる定番構文
    It is hard to explain.それを説明するのは難しい。
    It is true that he left.彼が去ったというのは本当だ。
  • make / find / think + it + 形容詞 + to do（形式目的語）真の目的語である to不定詞を後ろに回し、目的語の位置に仮の it を置く
    The noise made it hard to sleep.その騒音のせいで眠りにくかった。
    I found it easy to use.私はそれを使いやすいと感じた。
  • It + 動詞（天候・時間・距離・寒暖の非人称 it）特定の主語を持たない自然現象・時間・距離などを表す形式的な主語の it
    It is raining.雨が降っている。
    It is ten o'clock.10時だ。
    It is five miles to the station.駅まで5マイルある。
  • It is / was + 焦点 + that / who …（強調構文・分裂文）文の一部を It is … that で挟んで「〜こそが」と強調する cleft 構文
    It was you that I wanted to see.私が会いたかったのはあなただ。
    It is here that we first met.私たちが初めて会ったのはここだ。
  • 漠然とした状況を指す it（慣用表現）その場の状況・事情・成り行きを漠然と指す口語の it
    How's it going?調子はどう？
    That's it.それだ／それで終わり。
    Take it easy.気楽にね。

強調構文（It is/was ～ that/who …）

強調構文（It is/was ～ that/who …）。主語 John を It was ... who で挟んで強調する分裂文（cleft）。人が焦点なので who を使う。

/ɪt wəz ˈdʒɑːn ˈhuː ˈleft ðə ˈdɔːr ˈoʊpən ˈlæst ˈnaɪt/

  • door open → アメリカ英語の語末rが後続の母音と連結 [ˈdɔːroʊpən] ドーロウプン

形式目的語の it（make + it + 形容詞 + for ... to do）。真の目的語である to play outside を後ろに置き、it が仮に目的語の位置を埋める。make は「～の状態にする」の使役。

他のNGSL語への置き換え

it → when
元: If it rains this weekend, we'll just stay home and watch a movie.
後: When it rains this weekend, we'll just stay home and watch a movie.
訳: 今週末に雨が降ったときは、家でのんびり映画でも見よう。
📝 if は「雨が降るかどうか分からない」不確実な条件、when は「（今週末のいずれか）降ったそのときには」と、降ることをほぼ前提にした言い方。天候の it を使う条件文で if / when の対比はよく問われるポイント。
it → that
元: It was John who left the door open last night.
後: It was John that left the door open last night.
訳: 昨夜ドアを開けっぱなしにしたのは、ジョンだったんだ。
📝 強調構文で焦点が「人」のとき、who も that もどちらも使える。who はやや丁寧で人物を明確に指す響き、that はより口語的で中立。意味はほぼ同じで、cleft の関係詞の選び方を示す例。
it → hard
元: The heavy rain made it impossible for us to play outside.
後: The heavy rain made it hard for us to play outside.
訳: 激しい雨のせいで、私たちは外で遊びにくくなった。
📝 impossible は「まったく不可能」、hard は「難しいが不可能ではない」。どちらも make + it（形式目的語）+ 形容詞 + to do の型を保ったまま、程度だけが変わる。形式目的語の it の型を練習できる置き換え。

⚠️ 日本人の典型誤用
✕ Rained a lot yesterday.
「昨日たくさん降った」と言いたかったが、英語では主語のない壊れた文になり、「何が降ったのか」が示されない不完全な発話に聞こえる。
○ It rained a lot yesterday.
昨日はたくさん雨が降った。
It rained a lot yesterday.
日本語「昨日はたくさん雨が降った」には主語がないため、そのまま主語を省いてしまう誤り。英語は天候・時間の文でも必ず主語が要り、中身のない形式的な it を立てる。
✕ Is difficult to speak English well.
「英語を上手に話すのは難しい」と言いたかったが、主語のない文になり、英語として成立していない。
○ It is difficult to speak English well.
英語を上手に話すのは難しい。
It is difficult to speak English well.
「英語を上手に話すのは難しい」の主語（形式主語の it）を落としてしまう誤り。to不定詞句を真主語として後ろに置く場合、文頭には形式主語の it が必須。
✕ It is a cat under the table.
「テーブルの下に猫がいる」と言いたかったが、It is だと「それ（＝すでに話に出ている何か）はテーブルの下の猫だ」という妙な意味に聞こえ、存在を伝えられない。
○ There is a cat under the table.
テーブルの下に猫がいる。
There is a cat under the table.
「テーブルの下に猫がいる」のように、初めて話題に出す物の存在を示すときは There is 〜 を使う。It is 〜 は既出のものを「それは〜だ」と説明する形なので、新情報の導入には使えない。
🗾 カタカナ語とのギャップ

カタカナ「イット」は「イ・ッ・ト」の3モーラで語末に母音「ト（to）」まで付けて強く読みがちだが、英語の it は機能語で通常は弱く1音節。母音は日本語「イ」より緩んだ /ɪ/、語末の /t/ には母音を付けず、後ろに子音が続くと未開放（飲み込む）、母音が続くと連結（It is → イリズ）やフラップ化を起こす。単独で強く「イット」と切ることはむしろ少ない。

🌱 Word family・語源
its（限定詞）それの（所有格）NGSL
itself（代名詞）それ自身（再帰代名詞）NGSL
語源: 古英語 hit（3人称単数中性の代名詞）に由来。もとは語頭に h があったが、無強勢で使われるうちに h が脱落して it になった。所有格は長らく his（男性形と同形）が使われ、現在の its が定着したのは16〜17世紀と比較的新しい。
its（それの）— it の所有格として後から作られた形。it + 所有の -s
that（あれ・それ）— 同じゲルマン語系の指示・中性代名詞の系統に連なる語
💡 覚え方: もとは h 付きの hit。無強勢でいつも弱く発音されるうちに頭の h が取れて it になった、「弱いから縮んだ」典型例。だから今も文中では弱く軽く発音するのが自然。
⚖️ 類義語の使い分け
this — this は話し手に近いもの・今まさに話題化するものを指し示す指示詞で、注意を向ける力が強い。it は既に話題に出たものを軽く受け直すだけで、指し示す力はない（Look at this! は言えるが Look at it! は「すでに話題のそれ」を見る意味になる）。
that — that は話し手から離れたもの・前の発話全体を「それ・あのこと」と明示的に指すのに使い、強勢を伴い指示力がある。it は無強勢で前出の特定名詞を淡々と受けるだけで、新たに指し示すニュアンスはない。
💬 熟語・慣用表現 (10)
call it a day （その日の仕事などを）切り上げる、おしまいにする口語
We've done enough for now—let's call it a day.（今日はもう十分やったね。この辺で切り上げよう。）
🔍 「それを1日と呼ぶ」から「もう1日分やったことにして終わりにする」。it は特定の何かを指さない慣用の it。
📝 仕事・作業・活動を「今日はここまで」と終える時の定番。長年の仕事や習慣をやめる意味で call it a career / call it quits と言うこともある。
make it 成功する；（時間に）間に合う；（会などに）来られる口語
Sorry, I can't make it to the party tonight.（ごめん、今夜のパーティーには行けないんだ。）
🔍 「それを作る」ではなく「うまくやり遂げる・たどり着く」。目的地・目標・締切に「達する」イメージ。it は漠然とした状況を指す。
📝 文脈で「間に合う（列車に）」「出席できる」「（病人が）持ちこたえる」「一流になる」など幅広く使い分ける。make it big で「大成功する」。
get it （意味・冗談を）理解する、わかる口語
I explained the joke twice, but he still didn't get it.（その冗談を二度説明したのに、彼はまだわからなかった。）
🔍 「それを手に入れる」から「（言われたことを）飲み込む・ピンとくる」。it は話の内容や要点を漠然と指す。
📝 Got it. だけで「わかった／了解」の返事になる。You're going to get it! のように「叱られる・罰を受ける」という別用法もあるが、単独の get it は「理解する」が中心。
that's it それだ（その通り）；これで終わり；もう限界だ口語
That's it—I'm not waiting any longer.（もう限界だ、これ以上は待たないぞ。）
🔍 「それがそれだ」から、正解を指す「まさにそれ」、区切りを示す「以上・おしまい」、我慢の限界を示す「もうたくさん」まで文脈で変わる。
📝 肯定的には「正解！」、作業の締めには「これで完了」、怒りの表明では「いい加減にしろ」。声の調子と文脈で意味が決まる。
take it or leave it いやなら結構、この条件で受けるか断るか（妥協なし）口語
The price is fixed at fifty dollars—take it or leave it.（値段は50ドルで固定です。いやなら結構、というわけです。）
🔍 「それを取るか置いていくか」から「これ以上の交渉はしない、二者択一だ」。提示条件の最終通告。
📝 交渉や取引で「これ以上まけない」と突き放す言い方。can take it or leave it だと「あってもなくてもどちらでもいい」の意になる。
lose it 冷静さを失う、キレる；正気を失う口語
When he heard the news, he completely lost it and started shouting.（その知らせを聞いて、彼は完全にキレて怒鳴り始めた。）
🔍 「それを失う」の it は自制心・平静を漠然と指し、「感情が抑えられなくなる」こと。怒り・笑い・涙のいずれにも使う。
📝 怒りだけでなく、おかしくて「笑いが止まらなくなる」時にも使う。年配者について「衰えてぼけてきた」の意で使うこともある。
as it is （今の）そのままで；実際のところ、ただでさえ中立
Don't add anything—I like the design as it is.（何も足さないで。今のデザインのままが気に入ってるの。）
🔍 「それがある通りに」から「現状のままで」。また文頭で「ところが実際は」と現実を示す。
📝 「現状のまま」の意と、文頭で「ただでさえ〜なのに」と事情を述べる意がある。過去なら as it was。仮定に対し「（しかし）現実には」と対比する働き。
as it were いわば、言ってみればフォーマル／文語
He became, as it were, a second father to me.（彼は私にとって、いわば第二の父のような存在になった。）
🔍 仮定法過去 were を用いた古風な定型で「まるでそうであるかのように」。断定を避けて比喩的な言い回しに添える。
📝 たとえ・比喩を挿入するときのクッション表現で、so to speak とほぼ同義。やや改まった書き言葉・スピーチで使う。
go for it 思い切ってやってみる、頑張れ口語
If you really want that job, go for it!（その仕事が本当に欲しいなら、思い切って挑戦しなよ！）
🔍 「それに向かって行け」から「ためらわず挑戦しろ」。it は目標・チャンスを漠然と指す励ましの定型。
📝 背中を押す掛け声として「やっちゃえ！頑張れ！」。自分について I decided to go for it なら「腹を決めてやることにした」。
have had it もううんざりだ；（物が）もうダメだ、寿命だ口語
I've had it with all these delays.（こう遅れてばかりで、もう本当にうんざりだ。）
🔍 完了形 have had と it の組み合わせで「もう十分それを味わった」から「我慢の限界／使い果たした」。it は状況・物を漠然と指す。
📝 人が主語なら「もう耐えられない」、物が主語なら「壊れて使い物にならない（This car has had it）」。had it だけでも同義で使う。
