object

名詞・動詞米 /ˈɑːbdʒɪkt/（名詞）；/əbˈdʒekt/（動詞） ／ 英 /ˈɒbdʒɪkt/（名詞）；/əbˈdʒekt/（動詞）noun: ob·ject／verb: ob·ject（2音節）CEFR

🗣 名詞と動詞で強勢の位置が変わる典型的な名詞/動詞アクセント移動語（OBject⇔obJECT）。名詞は第1音節に強勢が置かれ /ˈɑːbdʒɪkt/、第1音節の母音は口を大きく開けた /ɑː/。動詞は強勢が第2音節に移り /əbˈdʒekt/ となり、第1音節は弱母音 /ə/ に縮小する。同じ綴りでもこの強勢差だけで品詞を聞き分ける必要がある。

  1. 【名詞】もの、物体（There's a strange object floating near the ceiling. 天井の近くに奇妙な物体が浮かんでいる。）
  2. 【名詞】(努力・欲望などの)対象、目的（Winning the championship became the main object of her training. 優勝することが彼女のトレーニングの主な目的になった。）
  3. 【名詞】(文法)目的語（In "I saw him," "him" is the object of the verb. "I saw him"では"him"が動詞の目的語だ。）
  4. 【動詞】反対する、異議を唱える（Several parents objected to the change in the school schedule. 何人かの保護者が学校のスケジュール変更に反対した。）

語形変化

形種類発音備考
objects名詞複数形／動詞三人称単数現在（同形・アクセント違い）/ˈɑːbdʒɪkts/（名詞複数）；/əbˈdʒekts/（動詞三単現）綴りは同じだが強勢位置で品詞が変わる。名詞は第1音節強勢のまま複数の -s が付き語末は無声音の連続で /s/。動詞は第2音節強勢のまま三単現の -s が付き、こちらも語末は /s/（objectsで「ズ」にはならない）。
objected過去形・過去分詞/əbˈdʒektɪd/規則変化（動詞用法のみ、名詞に過去形はない）。語幹が破裂音 /t/ で終わるため -ed は独立音節 /ɪd/ として加わり、objected は3音節になる。強勢は object と同じく第2音節のまま変わらない。
objectinging形/əbˈdʒektɪŋ/規則変化（動詞用法のみ）。語幹の t の直後に -ing が続くため、フラップ化はせず /t/ の音価がそのまま保たれる（objectingは「オブジェリング」のようにはならない）。

語法マーク

  • 動詞 object は自動詞。反対する対象は前置詞 to を伴う前置詞句で示す（object to a plan）。× object a plan は誤り
  • 動詞は object that + 節（フォーマル）でも使える: They objected that the process was unfair.
  • 名詞 object（物・対象の意味）は可算名詞。常に a/an や複数形 -s を伴って使える
  • 強勢移動によって名詞・動詞が入れ替わる語のペア（同種のパターンに record, permit, present などがある）
  • 他品詞での用法: 動詞用法は自動詞で、反対する対象を示す際は前置詞 to を伴う（object to a proposal）。strongly / firmly / formally など様態の副詞と結びつくことが多い。フォーマルな文体では that節を伴うこともある（object that ~）。名詞用法にはさらに、コンピュータ分野の専門用語として「オブジェクト」（データと処理をひとまとめにした単位）を指す意味もある（In object-oriented programming, everything is treated as an object.）。また文法用語としては、直接目的語（direct object）と間接目的語（indirect object）を区別して使う（'Give her a book.' では her が間接目的語、a book が直接目的語）。

文型・頻出パターン

  • object to + 名詞／動名詞〜に反対する、異議を唱える
    object to the planその計画に反対する
    object to being treated unfairly不当に扱われることに異議を唱える
  • object that + 節〜だと異議を唱える（フォーマルな言い方）
    He objected that the process was unfair.彼はその手続きは不公平だと異議を唱えた。
  • If nobody/no one objects, ...誰も反対しなければ〜（合意・許可を確認する定型表現）
    If no one objects, we'll go with this plan.誰も反対しなければ、この計画で進めます。
  • the object of + 名詞〜の対象、〜の的
    the object of criticism批判の対象
    the object of her affection彼女の愛情の対象
  • 形容詞 + object（名詞）物体・対象を表す名詞用法の定型的な組み合わせ
    a foreign object異物
    a solid object固体・実体のある物体

他のNGSL語への置き換え

object → protest
元: Local residents are objecting to the construction of the new factory.
後: Local residents are protesting the construction of the new factory.
訳: 地元の住民たちは、新しい工場の建設に抗議している。
📝 protest は前置詞を伴わず他動詞として使え、object to よりも公然と声を上げる抗議行動のニュアンスが強い。object to は意見・立場としての反対を静かに述べる場合にも使える、より中立的な表現。
object → item
元: Handle this object carefully because it could break easily.
後: Handle this item carefully because it could break easily.
訳: この品物は壊れやすいから、慎重に扱ってね。
📝 item は一覧・在庫・買い物などの文脈で「（複数ある中の）一点」を指すのに対し、object はもっと一般的でどんな物理的な物にも使える中立的な語。
object → complains
元: If nobody objects to the plan, we will move forward with the new schedule this week.
後: If nobody complains about the plan, we will move forward with the new schedule this week.
訳: 誰も計画に文句を言わなければ、今週中に新しいスケジュールで進める予定だ。
📝 complain about は不満を表明することに重点があり、口語的でやや感情的な響きを持つ。object to はより形式的で、提案や行動そのものに対する明確な反対・異議を示す。

⚠️ 日本人の典型誤用
✕ I object the plan.
「私はその計画に反対だ」と言いたくて対象をそのまま目的語にしてしまったが、object は自動詞なので to がないと文法的に破綻して聞こえる。
○ I object to the plan.
私はその計画に反対だ。
I object to the plan.
動詞 object は自動詞なので、反対する対象を直接目的語として続けることはできない。前置詞 to を挟む必要がある。他動詞 oppose（oppose the plan）と混同しやすい。
✕ He objects on the new policy.
「彼は新しい方針について反対している」と言いたかったが、on を使うとネイティブには前置詞の選択が不自然に響き、意味が取りづらくなる。
○ He objects to the new policy.
彼は新しい方針に反対している。
He objects to the new policy.
反対する対象を示す前置詞は on ではなく to。日本語の「〜について反対する」の「について」につられて on や about を選んでしまう誤り。
✕ The object of today's meeting is discuss the budget.
「今日の会議の目的は予算について話し合うことだ」と言いたかったが、is の直後に動詞の原形を置いてしまい、英語としては不完全な文になる。
○ The object of today's meeting is to discuss the budget.
今日の会議の目的は予算について話し合うことだ。
The object of today's meeting is to discuss the budget.
the object of ~ is to do の型では is の後に to不定詞が必要。学習者は to を落として動詞の原形をそのまま続けてしまいがち。
🗾 カタカナ語とのギャップ

カタカナ「オブジェクト」はほぼ均等な4拍で発音されがちだが、英語の名詞は第1音節に強勢がある2音節語 /ˈɑːbdʒɪkt/で、最初の母音はカタカナの「オ」よりも口を大きく開けた /ɑː/ に近い。さらに動詞用法では強勢が第2音節に移り /əbˈdʒekt/ となり、カタカナ「オブジェクト」の均等なリズムからは想像しにくい別の音形になる点に注意。

🌱 Word family・語源
objection（名詞）異議、反対（理由）NGSL
objective（名詞・形容詞）目的、目標；客観的なNGSL
objectionable（形容詞）不快な、いただけない
objector（名詞）反対者（例: conscientious objector 良心的兵役拒否者）
語源: ラテン語 objectum（ob-「〜に向かって」+ jacere「投げる」の過去分詞）に由来。「目の前に投げ出されたもの」が原義で、そこから「（視線・意識が向けられる）対象・物体」と「相手に向かって異議を投げつける＝反対する」という一見別々の名詞・動詞の意味の両方が発展した。
project（計画、企画）— pro-（前へ）+ ject（投げる）で「前に投げ出す」→ 計画・案
reject（拒否する）— re-（後ろへ）+ ject（投げる）で「投げ返す」→ 拒絶する
subject（主題、対象；服従させる）— sub-（下に）+ ject（投げる）で「下に投げ置かれたもの」→ 主題・支配下に置かれた者
💡 覚え方: ob-（〜に向かって）+ ject（投げる）＝「目の前に投げ出されたもの」。名詞の object は目の前に“投げ出されて置かれている”モノのイメージ、動詞の object は相手に向かって“異議を投げつける”イメージでつながる。project（前に投げる＝計画）、reject（投げ返す＝拒否）、subject（下に投げられたもの＝主題）と、ject「投げる」ファミリーでまとめて覚えるとよい。
⚖️ 類義語の使い分け
oppose — oppose は他動詞で反対の対象を直接目的語として取る（oppose the plan）。object は自動詞で to を伴う（object to the plan）。oppose の方が態度・行動としての反対がより強く持続的なニュアンスを持つ。
protest — protest は公然と声を上げる抗議行動に重点があり、しばしば集団的・公的な行動を伴う。object はより静かに意見や立場として異議を述べる場合にも使え、会議などフォーマルな場でもよく使われる。
mind — mind は口語的で『気にする』というニュアンス（Do you mind if I open the window?）。object はより形式的で、正式な異議・反対を表す（I object to your using my desk. は堅い響きになる）。
💬 熟語・慣用表現 (2)
money is no object お金に糸目はつけない、費用は問題ではない中立
For this wedding, money is no object—they want everything to be perfect.（この結婚式に関しては費用は問題ではない。彼らはすべてを完璧にしたいのだ。）
🔍 「お金は問題にならない」。ここでの object は「妨げ・障害となるもの」という古い意味で使われており、現代でよく知られた「物」「対象」という意味からは意味が予測しにくい。
📝 money 以外にも time is no object（時間は問題ではない）、distance is no object（距離は問題ではない）のように主語を入れ替えて使う定型表現。動詞の object（反対する）とは無関係なので混同しないこと。
object lesson （〜に関する）教訓となる実例、生きた見本中立／ややフォーマル
The company's bankruptcy was an object lesson in the dangers of overexpansion.（その会社の倒産は、過度な事業拡大の危険性を示す教訓的な実例だった。）
🔍 「実物教育」が原義で、そこから「（望ましい/望ましくない結果を鮮やかに示す）生きた教訓」を意味するようになった比喩表現。object の「物」という基本的な意味からは、この教訓的な意味は予測しにくい。
📝 an object lesson in ~ の形で「〜に関する教訓的な実例」として使われることが多い。良い手本にも悪い手本にも使え、ニュース記事や評論でよく見かける表現。
