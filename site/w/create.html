create

動詞 (verb)米 /kriˈeɪt/ ／ 英 /kriˈeɪt/cre·ate（2音節）CEFR A2

🗣 強勢は第2音節 /kriˈeɪt/ に置かれる。第1音節 cre- は弱く /kri/（「クリ」を短く軽く）、第2音節 -ate が二重母音 /eɪ/ を含み強く長めに「エイト」と発音される。第1音節を強く読む誤りが多いので注意

  1. 【動詞】（何もないところから）創造する、作り出す（create a new world 新しい世界を創り出す）
  2. 【動詞】（問題・機会・印象などを）引き起こす、生じさせる（create a problem 問題を引き起こす）
  3. 【動詞】（役職・組織などを）新設する、設ける（create a new department 新しい部署を新設する）

語形変化

形種類発音備考
creates三単現/kriˈeɪts/語末は無声音 t の後なので /s/（「クリエイツ」であって「クリエイズ」ではない）
created過去形・過去分詞（規則変化）/kriˈeɪɾɪd/動詞が /t/ で終わるため -ed は独立音節 /ɪd/ として発音される。米音では母音に挟まれた t がフラップ化し [ɾ] となり「クリエイリド」に近く聞こえる
creatinging形/kriˈeɪɾɪŋ/語末の e を落として -ing。米音では created と同様、母音間の t がフラップ化し「クリエイリング」に近く響く（強く t を発音しすぎない）

語法マーク

  • 常に他動詞。目的語を必ず伴う（× He created. のように単独では使えない）
  • 進行形も可能な動作動詞: They are creating a new logo for the company.
  • 受動態が非常に多用される: be created by ~（~によって作られる）
  • 抽象名詞（a problem, an opportunity, a good impression 等）を目的語に取り「引き起こす・生じさせる」の意味になる比喩的用法が多い
  • make ほど日常的ではなく、独創性・新規性を含意する語（料理や単純作業には通常使わない）
  • 他品詞での用法: この語は他動詞としての用法のみで、名詞・形容詞としての用法は確立していない（名詞は別語 creation、形容詞は別語 creative が担う）。抽象名詞（problem, opportunity, impression 等）を目的語に取る比喩的用法が非常に多く、具体物を「一から作り出す」用法と、状況・雰囲気を「引き起こす」用法の2系統があると意識するとよい。

文型・頻出パターン

  • create + O（具体物）〜を（無から）作り出す・創造する
    create a new world新しい世界を創り出す
    create a piece of art芸術作品を作り出す
  • create + O（抽象名詞）（問題・機会・印象などを）引き起こす、生じさせる
    create a problem問題を引き起こす
    create an opportunity機会を生み出す
    create a good impression好印象を与える
  • be created by + 行為者〜によって作られる（受動態で行為者を明示）
    This app was created by a small team of students.このアプリは少人数の学生チームによって作られた。
  • create + O + for + 人（人）のために〜を作る
    We created a special plan for new members.私たちは新会員のために特別なプランを作った。
  • create + O + out of / from + 材料〜（材料）から〜を作り出す
    She created a sculpture out of recycled metal.彼女はリサイクルされた金属から彫刻を作り出した。

他のNGSL語への置き換え

create → started
元: The two friends created a small business together after college.
後: The two friends started a small business together after college.
訳: その二人の友人は、大学卒業後に一緒に小さなビジネスを始めた。
📝 start は単に「始める・スタートさせる」という中立的な語で、create ほど「無から新しく生み出す」という創造性は強調しない。既存の枠組みに沿ってビジネスを開始した、というニュアンスに近づく。
create → made
元: This sculpture was created by a famous artist over three hundred years ago.
後: This sculpture was made by a famous artist over three hundred years ago.
訳: この彫刻は300年以上前に、ある有名な芸術家によって作られたものだ。
📝 make は最も一般的で日常的な「作る」。create は「独創的に、新しく生み出す」という創造性・オリジナリティのニュアンスを強く持つため、芸術作品の価値を強調したい場合は create の方が好まれる。
create → produce
元: If we had a bigger budget, we could create a much better advertisement.
後: If we had a bigger budget, we could produce a much better advertisement.
訳: もっと予算があれば、もっと良い広告を制作できるのだけれど。
📝 produce は組織的・工業的なプロセスを経て「制作する・生産する」というニュアンスで、広告や映像の文脈では create とほぼ同義に使えるが、より事務的・実務的な響きになる。

⚠️ 日本人の典型誤用
✕ My mother created dinner for us every night.
「母は毎晩私たちに夕食を作ってくれた」と言いたかったが、create を使うと、まるで母が毎晩何か前例のない斬新な料理を発明しているかのような大げさな響きになってしまう。
○ My mother made dinner for us every night.
母は毎晩私たちに夕食を作ってくれた。
My mother made dinner for us every night.
日本語「（料理を）作る」を無条件に create に当てはめてしまう誤り。create は独創性・新規性を含意する語で、日常の料理や繰り返し行う作業には通常 make を使う。
✕ I have create a new plan yesterday.
「私は昨日新しい計画を作った」と言いたかったが、現在完了と yesterday を組み合わせると文法的に矛盾しており、ネイティブには時制が崩れた不自然な英語に聞こえる。
○ I created a new plan yesterday.
私は昨日新しい計画を作った。
I created a new plan yesterday.
yesterday のような明確な過去の時点を示す語句は現在完了形と共に使えない。過去の一回の出来事は単純過去形 created で表す。
✕ She wants to create for her students a fun activity.
「彼女は生徒たちのために楽しいアクティビティを作りたい」と言いたかったが、for句を目的語の前に挟むと不自然な語順になり、ネイティブには読みにくく回りくどい英文に聞こえる。
○ She wants to create a fun activity for her students.
彼女は生徒たちのために楽しいアクティビティを作りたいと思っている。
She wants to create a fun activity for her students.
日本語の語順（「生徒のために〜を作る」＝目的語より先に相手を言う）に引きずられ、for句を目的語より前に置いてしまう誤り。英語では create + O + for + 人 の語順が基本。
🗾 カタカナ語とのギャップ

カタカナ「クリエイト」「クリエイター」はビジネス・創作分野で定着しているが、日本語では各モーラを均等に発音するため強勢の位置がぼやける。英語では第2音節に強勢があり /kriˈeɪt/（クリ・エイト、後ろを強く長めに）。第1音節「ク」を強く読む、または全体を平板に読むと通じにくくなる。

🌱 Word family・語源
creation（名詞）創造、創作物、（the Creation で）天地創造NGSL
creative（形容詞）創造的な、独創的なNGSL
creativity（名詞）創造性、創造力
creator（名詞）創造者、作成者（大文字 Creator で「神」）
recreate（動詞）再現する、作り直す
recreation（名詞）娯楽、気晴らし、レクリエーション（意味が create から大きく離れている点に注意）
語源: ラテン語 creare「（無から）生み出す、生む」に由来し、古フランス語 creer を経て中英語 createn として英語に入った。もともと神が世界を創造する行為（Creation, Creator）を表す宗教的な語であったことから、「無から新しいものを生み出す」という創造性の含意が今も色濃く残る。
creature（生き物）— 同じラテン語 creatura「創造されたもの」に由来。神に創造された存在という原義から一般の「生き物」の意味へ広がった
recreation（娯楽、気晴らし）— re-（再び）+ create（元気にする、作り直す）から「心身を作り直す＝気晴らし」の意味に発展し、現代では create の創造性とは離れた意味を持つ
creative（創造的な）— 動詞 create + 形容詞語尾 -ive（性質を表す）
💡 覚え方: ラテン語 creare は「無からガシッと生み出す」神の創造のイメージ。Creator（創造主）から creative（創造的な）、creativity（創造性）まで、すべて「無から何かを生む」という核イメージでつながっている。ただし recreation（娯楽）だけは意味がだいぶ離れているので別枠で覚える。
⚖️ 類義語の使い分け
make — make は最も一般的で日常的な「作る」。create は無から新しく生み出す独創性・新規性を強調する語で、料理や単純作業のような日常的な行為には通常 make を使う。
produce — produce は組織的・工業的なプロセスを経て「生産する・制作する」というニュアンス。create より実務的・量産的な響きになる（例: produce a film／a product）。
generate — generate は電力・データ・利益など、仕組みやシステムが自動的に「生み出す」場合に使うことが多く、人の創造性よりもプロセス・機構が生み出すニュアンスが強い（generate electricity／income）。
invent — invent は「今までになかったものを初めて考え出す・発明する」に焦点があり、create よりも新規性・独創性が強く、具体的な発明品（機械・技術・仕組み）に使われることが多い。
💬 熟語・慣用表現 (2)
create a monster （自分の行動が原因で）手に負えない存在・事態を生み出してしまう口語
By giving the kids so much candy at every visit, we've created a monster—now they refuse to eat anything else.（来るたびに子どもたちにお菓子をあげすぎたせいで、とんでもないことになった。もう他のものを食べようとしないんだ。）
🔍 フランケンシュタイン博士が作った怪物が制御不能になった物語に由来する比喩。良かれと思ってしたことが、後になって自分の手に負えない厄介な結果を招いてしまうことを表す。
📝 通常 have created a monster のように現在完了形で使い、「もう手遅れだ」と嘆くニュアンスを伴う。人だけでなく、暴走するようになった制度やシステムにも使われる。
create a scene 人前で騒ぎを起こす、大声で感情的になって周囲を困らせる口語
He created a scene at the restaurant when the waiter brought him the wrong order.（ウェイターが注文と違う料理を持ってきたとき、彼はレストランで騒ぎを起こした。）
🔍 scene は本来「（舞台の）一場面」の意味だが、そこから「公共の場での大げさな感情表現・修羅場」という比喩的な意味に転じた。
📝 同義表現の make a scene の方が使用頻度は高い。恥ずかしい・迷惑な公共の場での感情の爆発を指し、単に「場面を作る」という文字通りの意味にはならない。
