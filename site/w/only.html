only

副詞・形容詞・接続詞米 /ˈoʊnli/ ／ 英 /ˈəʊnli/on·ly（2音節）CEFR A1

🗣 第1音節に強勢（ˈONly）。強勢のある母音は二重母音 /oʊ/ で、日本語の単母音「オ」より唇を丸めて「オ」から「ウ」へなめらかに滑らせる。第2音節 -ly は弱勢だが母音はあいまい化せずはっきり /li/ と発音し、伸ばさず短く切る

  1. 【副詞】ただ〜だけ、〜しかない（数量・程度を限定する。I only have five dollars with me. 手持ちは5ドルしかない）
  2. 【副詞】〜だけが、〜のみが（他を除外する。Only members are allowed to enter. 会員だけが入場できる）
  3. 【副詞】ほんの、たった今（時間的な近さ・短さを強調する。I only just got here. ほんのついさっき着いたばかりだ）
  4. 【副詞】〜して初めて、〜してようやく（only で始まる副詞句を文頭に置くと主語と助動詞が倒置する。Only later did I understand what he meant. あとになってようやく彼の言いたかったことがわかった）
  5. 【形容詞】唯一の、ただ一つの（the only + 名詞の形で使う。She was the only one who noticed the mistake. 間違いに気づいたのは彼女だけだった）
  6. 【接続詞】ただし〜だが、もっとも〜なんだけど（口語で軽い留保・言い訳を加える。I'd love to help, only I don't have the time. 手伝いたいのはやまやまなんだけど、時間がないんだ）

語法マーク

  • 焦点副詞（focus adverb）: only がどの語句にかかるかは位置と文強勢の両方で決まる。書き言葉では曖昧さを避けるため、限定したい語のすぐ前に置くのが原則
  • Only + 副詞句 が文頭に来ると、主語と助動詞が倒置する（Only later did I realize the truth. / Not only was she late, but she also forgot the documents.）
  • if only は仮定法を伴い「〜でさえあれば」という強い願望・後悔を表す（If only I had studied harder!）。only if（〜という場合に限り、条件の限定）とは語順も意味も別物なので混同しないこと
  • the only + 名詞（+ 関係詞節）で「唯一の〜」という形容詞用法。原則として the が必須で、最上級に近い排他的な限定を表す
  • 接続詞 only は口語で but に近い軽い譲歩・言い訳を導く。フォーマルな文章では使わない
  • 他品詞での用法: 副詞としての only は「文中のどの位置に置くか」で限定する対象（焦点）が変わる焦点副詞（focus adverb）である。I only lent him the book.（貸しただけで、あげたのではない＝動詞を限定）と I lent only him the book.（彼にだけ貸した＝目的語を限定）では意味が異なるが、話し言葉では文法的にどこに置いても許容され、実際には文の強勢（イントネーション）で焦点を示すことが多い。書き言葉では誤解を避けるため、限定したい語の直前に置くのが原則。接続詞用法（口語で but とほぼ同義。I'd go, only it's raining. 行きたいんだけど、あいにく雨でね）はくだけた話し言葉に限られ、フォーマルな文章では but や however に置き換える。

文型・頻出パターン

  • S + only + V（中間位置）動詞句を弱く限定し「ただ〜するだけ」を表す最も基本の語順
    She only smiled and said nothing.彼女はただ微笑んだだけで、何も言わなかった。
    We only need one more signature.あとサイン1つもらうだけでいい。
  • Only + 副詞句, + 倒置only で始まる副詞句を文頭に置くと主語と助動詞が入れ替わる強調構文
    Only in an emergency should you call this number.緊急の場合に限り、この番号にかけてください。
  • not only ~ but (also) ~「〜だけでなく…も」相関接続詞。not only が文頭に来ると倒置が起こる
    Not only is the hotel expensive, but it's also far from the station.そのホテルは高いだけでなく、駅からも遠い。
  • the only + 名詞（+ who/that ...）「唯一の〜」形容詞用法。関係詞節で修飾されることが多い
    This is the only key that opens the back door.これが裏口を開ける唯一の鍵だ。
  • if only + 仮定法「〜でさえあれば」という強い願望・後悔を表す感嘆的構文
    If only I had brought an umbrella!傘を持ってきてさえいれば!

他のNGSL語への置き換え

only → just
元: Only two tickets were left when we finally got to the counter.
後: Just two tickets were left when we finally got to the counter.
訳: ようやくカウンターにたどり着いたときには、ちょうどチケットが2枚残っていた。
📝 only は「それだけしか残っていない」という不足・限定のニュアンスを強く出すのに対し、just は「ちょうど・たまたま」という中立的な事実描写に近く、残り少ないという含みが弱まる。どちらも数詞の直前に置ける。
only → one
元: He's the only person in the office who remembers her birthday every year.
後: He's one person in the office who remembers her birthday every year.
訳: 彼はオフィスの中で彼女の誕生日を覚えている人のひとりだ。
📝 the only は「彼以外にはいない」という排他性を表すのに対し、one にすると「その中のひとり」という意味になり、他にも同様の人がいる可能性を示す。排他性の有無が大きく変わる好例で、冠詞 the も外れる。
only → simply
元: If she only spoke a little more slowly, everyone in the room would understand her perfectly.
後: If she simply spoke a little more slowly, everyone in the room would understand her perfectly.
訳: 彼女がもう少しゆっくり話しさえすれば、部屋の全員が完璧に理解できるだろう。
📝 only と simply はどちらも「ただ〜するだけで」という条件の単純さを強調する用法で意味はほぼ同じ。simply はやや書き言葉寄りでフォーマルな響きを持ち、口語的な only/just よりも硬い印象を与える。

⚠️ 日本人の典型誤用
✕ I only can speak a little English.
「英語は少ししか話せません」と言いたかったが、only can のような語順はネイティブにはやや据わりが悪く、書き言葉としては誤りとされやすい。
○ I can only speak a little English.
英語は少ししか話せません。
I can only speak a little English.
only の定位置は助動詞 can の後ろ・本動詞の前。日本語「私はただ〜だけ」の語順につられて助動詞の前に置いてしまう誤り。標準的な書き言葉の語順は助動詞の直後。
✕ Only if I had more free time, I would travel more.
「もっと自由な時間さえあれば、もっと旅行するのに」と言いたかったが、only if だと「もっと自由な時間があるときに限り」という条件限定の意味に聞こえ、後悔・願望という話の趣旨が伝わらない。
○ If only I had more free time, I would travel more.
もっと自由な時間さえあれば、もっと旅行するのに。
If only I had more free time, I would travel more.
「〜さえあれば」という強い願望を表したいのに、if only と only if の語順を逆にしてしまう誤り。if only は仮定法とともに後悔・願望を表すのに対し、only if は「その条件のときに限り」という全く別の条件限定構文。
✕ Not only she is smart, but she is also kind.
「彼女は頭がいいだけでなく優しくもある」と言いたかったが、not only を文頭に置きながら倒置しないと英語としては不自然で、文法的な誤りとしてすぐに気づかれてしまう。
○ Not only is she smart, but she is also kind.
彼女は頭がいいだけでなく、優しくもある。
Not only is she smart, but she is also kind.
否定の副詞句 not only が文頭に来る場合、主語と（助）動詞を倒置させる必要がある（is she の語順）。日本語話者は倒置を忘れて通常の平叙文の語順のまま続けてしまいがち。
🗾 カタカナ語とのギャップ

カタカナ「オンリー」は「オ・ン・リー」の3拍で語末を伸ばしがちだが、英語 /ˈoʊnli/ は2音節で強勢は最初の1つだけ、語末の -ly は伸ばさず短く /li/ で切る。またカタカナの「オ」は単母音だが、英語は二重母音 /oʊ/ で「オ」から「ウ」へ滑らせる点にも注意。

one（1つの）— only の直接の語根。one + -ly（〜のような）＝「1つきりの」が原義
alone（1人で）— all + one「まったくの1つ」が縮まった語で、only と同じく one に由来する兄弟語
once（一度、かつて）— one に副詞をつくる古い語尾が付いた形で、only とは one を共有する派生語同士
💡 覚え方: only は one（1）＋ -ly（〜のような）＝「1つのような」から「唯一の、ただ〜だけ」へと育った語。onceやaloneも同じ one から枝分かれした親戚だと知ると、oneファミリー全体を芋づる式に覚えられる。
⚖️ 類義語の使い分け
just — just は「ちょうど、たった今」という中立的な事実描写にも、口語で only とほぼ同じ「ただ〜だけ」の意味にも使われる幅広い語。only は書き言葉でも自然で、排他性（それ以外にはない）をより明確に示す。
merely — merely はフォーマルな書き言葉で「単に、それだけの取るに足らないもの」という軽視・矮小化のニュアンスを伴うことが多い（merely a suggestion）。only はニュートラルで日常会話でも普通に使う。
solely — solely はビジネス文書や法律文などかたい書き言葉で「もっぱら、〜のみ」を強調する語で、責任や理由が唯一であることを明確にする場面で好まれる（solely responsible for）。only はカジュアルな会話から書き言葉まで幅広く使える汎用語。

if only 〜でさえあれば（なあ）、〜だったらよかったのに中立
If only I had listened to my parents' advice.（両親の忠告を聞いていさえすれば良かったのに。）
🔍 「もし〜だけであれば」という条件節から転じて、実現しなかった強い願望や後悔を表す感嘆的な決まり文句。単なる条件の if とは異なり、単独で感嘆文としても使われる。
📝 仮定法過去（現在の後悔）または仮定法過去完了（過去の後悔）を伴う。文を最後まで言い切らず If only I had more time... のように余韻を残す使い方も多い。only if（〜という条件に限り）と混同しないこと。
not only A but (also) B Aだけでなく（Bも）中立
Not only did she finish the marathon, but she also set a personal record.（彼女はマラソンを完走しただけでなく、自己ベストまで更新した。）
🔍 not（否定）と only（〜だけ）を組み合わせて一度「〜だけではない」と否定し、but (also) で補足を加える相関構文。文頭に置くと倒置を伴う点が定型として覚えるべきポイント。
📝 文頭に置くと主語と助動詞が倒置する（did she のように）。文中（S + not only + V）に置く場合は倒置しない。also は省略されることもある。
in name only 名ばかりの、名目上だけの中立
He's the CEO in name only; his brother makes all the real decisions.（彼はCEOとは名ばかりで、実際の決定はすべて弟が下している。）
🔍 「名前の点においてのみ」という文字通りの意味から、実質が伴わない「肩書きだけ」の状態を表す慣用句。
📝 肩書き・地位・関係などが形式だけで実態が伴わないことを皮肉交じりに表す。in title only という言い方もほぼ同義。
only too この上なく〜、非常に〜（喜んで／十分すぎるほど）中立／やや丁寧
I'd be only too happy to help you move next weekend.（来週末の引っ越し、喜んでお手伝いしますよ。）
🔍 「〜だけに過ぎない」はずの only が too（あまりに）と結びつくと、逆に「この上なく／十分すぎるほど」という強調表現になる、直感に反する慣用表現。
📝 only too happy to do / only too glad to do の形で「快く〜する」という申し出の決まり文句として頻出。only too well（痛いほどよく）のように理解の深さを強調する用法もある（I know only too well how hard it is.）。
the one and only 唯一無二の、正真正銘の口語／やや誇張的
Ladies and gentlemen, please welcome the one and only Freddie Mercury!（皆様、正真正銘のフレディ・マーキュリーの登場です！）
🔍 「その一人であり、かつ唯一である」という重複による強調表現で、有名人や特別な存在を紹介するときの決まり文句。
📝 コンサートの紹介やユーモラスな強調でよく使われる定型句。the one and only + 名前 の形で名詞の直前に置く。
only time will tell それは時が経てばわかる、今はまだ何とも言えない中立
Will the new policy actually reduce traffic? Only time will tell.（新しい政策で本当に渋滞は減るのか。それは時が経ってみないとわからない。）
🔍 「時間だけが（答えを）教えてくれるだろう」という比喩で、現時点では結果や真偽が判断できないことを表す決まり文句。
📝 会話や記事の締めくくりとしてよく使われる。only を省いた Time will tell も同義で使われる。
only so much/many 〜にも限度がある、限られた〜しかできない口語／中立
There's only so much I can do without more information.（これ以上の情報がなければ、私にできることにも限度がある。）
🔍 「これだけの量しか」という文字通りの意味から、「できることには限界がある」という不満・諦めのニュアンスを込めた慣用的な言い回し。
📝 There's only so much/many ... の形で「限界がある」という不満・諦めを表す。後ろに can do, one person can handle など可能を表す節が続きやすい。
