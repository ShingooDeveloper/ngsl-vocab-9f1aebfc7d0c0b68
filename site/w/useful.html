useful

形容詞 (adjective)米 /ˈjuːsfəl/ ／ 英 /ˈjuːsfəl/use·ful（2音節）CEFR A2

🗣 第1音節 use に強勢。母音は二重母音 /juː/ でそのまま伸ばし、-ful は弱化した曖昧母音 /fəl/ で軽く添える（フルを強く言わない）

  1. 【形容詞】役に立つ、有用な、便利な（実際に使ってみて助けになる）
  2. 【形容詞】都合がよい、ありがたい（状況・行為について。It would be useful if you could help me. 手伝ってもらえると助かります。）

語形変化

形種類発音備考
more useful比較級/mɔːr ˈjuːsfəl/2音節以上の形容詞なので -er は付けず more を前に置く（uselier や usefuler は誤り）
most useful最上級/moʊst ˈjuːsfəl/the most useful の形が基本。usefulest のような -est 変化はしない

語法マーク

  • 限定用法（a useful tool）・叙述用法（This tool is useful.）のどちらでも使える
  • 比較変化は more useful / most useful（2音節以上の規則形容詞なので -er / -est にしない）
  • useful for + 名詞／動名詞、useful to + 動詞原形の両方を伴える
  • 人を主語にすると「道具のように機能的に役立つ」というニュアンスになりやすい。人柄・親切さを言うときは helpful の方が自然
  • 他品詞での用法: 形容詞用法のみで、名詞・動詞に品詞転換することはない。関連する派生語として名詞 usefulness（有用性）、副詞 usefully（役立つように）があるが、useful自体がそのまま名詞・動詞として使われることはない。

文型・頻出パターン

  • useful for + -ing〜するのに役立つ（用途を表す最も基本的な形）
    useful for cleaning掃除に役立つ
    useful for beginners初心者に役立つ
  • useful to + 原形〜するのに役立つ、〜すると良い（不定詞で用途・助言を表す）
    useful to know知っておくと役立つ
    useful to have持っておくと便利だ
  • find + O + useful（SVOC）〜が役立つと感じる、〜を役立つと思う
    I found this useful.これが役に立つと思った。
  • more useful / most useful（比較級・最上級）2音節以上の形容詞なので -er/-est ではなく more/most を用いる比較変化
    more useful than a map地図より役立つ
    the most useful tool最も役立つ道具
  • It is/would be useful to do形式主語 it を立てて「〜すると役立つだろう」と助言・提案する定番構文
    It would be useful to check twice.二回確認すると良いだろう。

他のNGSL語への置き換え

useful → helpful
元: A flashlight is more useful than a candle when the power goes out.
後: A flashlight is more helpful than a candle when the power goes out.
訳: 停電のときは、ろうそくより懐中電灯の方が助けになる。
📝 useful は物の実用性・機能面での有用さを表すのに対し、helpful は人を助ける・親切に役立つというニュアンスが強い。道具の性能を言うなら useful がより自然。
useful → wise
元: It would be useful to bring a flashlight in case the lights go out.
後: It would be wise to bring a flashlight in case the lights go out.
訳: 停電に備えて懐中電灯を持って行くのが賢明だろう。
📝 useful は行動そのものの実用性を表すが、wise はその判断の賢さ・思慮深さを強調する。実利を言うなら useful、判断の良さを言うなら wise。
useful → valuable
元: I found her feedback really useful when I was revising my essay.
後: I found her feedback really valuable when I was revising my essay.
訳: エッセイを書き直しているとき、彼女のフィードバックが本当に貴重だった。
📝 useful は実用面で役立つことを表すが、valuable は価値の高さ・重要性を評価する語で、ややフォーマルな響きを持つ。

⚠️ 日本人の典型誤用
✕ My father is very useful because he always helps me with my homework.
「父はいつも宿題を手伝ってくれてとても役に立つ」と言いたかったが、useful を人に使うと父を道具のように扱っているような不自然な響きになってしまう。
○ My father is very helpful because he always helps me with my homework.
父はいつも宿題を手伝ってくれてとても頼りになる。
My father is very helpful because he always helps me with my homework.
useful は道具・情報など「物」の実用性を表す語で、人を主語にすると「機能として役立つ」という物扱いのニュアンスになる。人の親切さ・力添えを言うときは helpful を使う。
✕ This dictionary is more usefuler than that one.
「この辞書はあれより役に立つ」と言いたかったが、more と usefuler（-er語尾）を両方使ってしまい、英語として誤った二重比較級になっている。
○ This dictionary is more useful than that one.
この辞書はあの辞書より役に立つ。
This dictionary is more useful than that one.
2音節以上の形容詞は more を前に置いて比較級を作り、-er を語尾に付けない。more と -er を重ねて使うのは典型的な誤り。
✕ Is this seat useful now?
「この席、今使えますか」と聞きたかったが、useful は「役に立つ」という意味なので、席の空き状況を尋ねる文としては意味がずれて伝わる。
○ Is this seat free now?
この席、今空いていますか。
Is this seat free now?
日本語の「使える」を直訳して useful を当ててしまう誤り。座席や設備が「今使用可能かどうか」は useful（役立つ）ではなく free / available（空いている・利用可能）で表す。
🗾 カタカナ語とのギャップ

カタカナで書くと「ユー・ス・フ・ル」と4拍に分けたくなるが、英語は2音節 /ˈjuːs.fəl/。後半の sfəl は弱く曖昧母音でひと息に発音し、s と f の子音連続を1つずつ丁寧に区切ると不自然に聞こえる。

🌱 Word family・語源
use（動詞・名詞）使う；使用NGSL
useless（形容詞）役に立たない、無駄なNGSL
usefully（副詞）役立つように、有効に
usefulness（名詞）有用性、役立つこと
usable（形容詞）使用できる、使える状態の
misuse（動詞・名詞）誤用する、悪用する；誤用
語源: ラテン語 usus（動詞 uti「使う」の過去分詞）が古フランス語を経て英語 use に入り、そこに古英語由来の接尾辞 -ful（「〜に満ちた」）が付いて「使うことに満ちた」→「役立つ」の意になった。
usage（使用、語法）— 同じ語根 use から派生した名詞形
utility（実用性、効用）— ラテン語 utilis（use と同じ uti 由来）から英語に入った同系語
utilize（利用する）— utility と同じラテン語根から派生した動詞
💡 覚え方: use（使うこと）+ -ful（〜で満ちた）＝「使うことで満たされている」→ そこから「役に立つ」。utility や utilize も同じ『使う』のラテン語根でつながっている。
⚖️ 類義語の使い分け
helpful — helpful は人の親切な手助けに重点を置く語。useful は物・情報・方法などの実用性に重点があり、人に使うと物扱いのニュアンスになりやすい。
handy — handy はより口語的で、持ち運びやすさ・手近にあって便利という具体的・身近なニュアンスが強い。useful はより中立的・一般的な語。
practical — practical は理論より実践面で機能的・現実的に役立つことを強調する。useful は単に役立つかどうかという結果に焦点がある。
valuable — valuable は価値の高さ・重要性を評価する語でよりフォーマル。useful は実用面での役立ち方を表し、価値の大小までは含意しない。
💬 熟語・慣用表現 (2)
come in useful （後で）役に立つ、いざという時に役立つ口語
Keep that old rope—it might come in useful one day.（その古いロープは取っておきなよ。いつか役に立つかもしれないから。）
🔍 「役立つ状態でやって来る」という文字通りの意味からはすぐに意味が読み取れない定型表現。come in は「入ってくる」ではなく「（後になって）〜だと分かる・機能する」という特殊な使われ方で、単独の come in や useful の意味を足しただけでは出てこないニュアンス。
📝 イギリス英語で特によく使われる。ほぼ同義で come in handy もあり、こちらの方がアメリカ英語ではより一般的。未来のいつか分からない場面での有用性を示唆する点が特徴。
make yourself useful （暇にしていないで）何か役に立つことをする、手伝う口語
Stop watching TV and make yourself useful—help me carry these boxes.（テレビばかり見てないで、何か手伝ってよ。この箱を運ぶの手伝って。）
🔍 文字通りは「自分を役立つ状態にする」だが、実際にはたいてい命令形で「ぼーっとしてないで手伝いなさい」という軽い叱責・催促のニュアンスを帯びる決まり文句。字面だけでは相手を促す口調までは読み取れない。
📝 多くは命令形（Make yourself useful!）で使われ、やや冗談交じり・軽くたしなめる響きを持つ。丁寧な依頼というより、家族や親しい間柄でのくだけた一言として使われることが多い。
