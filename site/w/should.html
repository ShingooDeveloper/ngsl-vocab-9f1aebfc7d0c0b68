should

助動詞 (modal verb)米 /ʃʊd/ ／ 英 /ʃʊd/should（1音節）CEFR A2

🗣 1音節語。母音は短い /ʊ/（book, good と同じ）。綴りの l は would・could と同様まったく発音されない黙字（silent l）。ただし would のような同音異義の一般語（wood）は should には存在しない点が異なる

  1. 【助動詞】〜すべきだ、〜した方がよい（助言・提案）（You should see a doctor if the pain continues. 痛みが続くなら医者に診てもらった方がいい）
  2. 【助動詞】（should have + 過去分詞で）〜すべきだったのに（過去への後悔・非難）（I should have studied harder for the test. もっとテスト勉強をしておくべきだった）
  3. 【助動詞】当然〜のはずだ、〜するはずだ（推量・期待）（She should be home by now. 彼女はもう家に着いているはずだ）
  4. 【助動詞】（If節の代わりに文頭に置く倒置で；フォーマル）万一〜の場合は（Should you have any questions, please contact us. 万一ご質問がございましたら、ご連絡ください）
  5. 【助動詞】（提案・必要を表す語に続くthat節で；that S (should) + 原形）〜するように（It is important that you should arrive on time. 時間通りに到着することが重要だ）
  6. 【助動詞】（Why should ~?で不満・反発を表す）なぜ〜しなければならないのか（Why should I apologize for something I didn't do? やってもいないことのために、なぜ謝らなければならないのか）

語法マーク

  • 法助動詞（modal auxiliary）。三単現の-sを付けず（× She shoulds は誤り）、後ろは常に動詞の原形（bare infinitive）。to不定詞にはしない（× should to go は誤り）
  • 疑問文・否定文はdo-supportを使わず、should自身を主語の前に出す／直後にnotを置く（Should I ...? / should not、短縮形 shouldn't）
  • shouldそれ自体に進行形・完了形・過去形はなく活用しない（defective）。「〜すべきだったのに」はshould have + 過去分詞、「〜しているはずだ」はshould be -ingという迂言的な形で表す
  • shallの過去形として生まれた語だが、現代英語（特に米語）ではshallとほぼ切り離され、助言・義務・推量・仮定を表す独立した助動詞として使われる。アメリカ英語ではshall自体がほとんど使われないため、shouldをshallの過去形と意識することは少ない
  • mustが外部から課された絶対的な義務・強い確信を表すのに対し、shouldは話者の主観的な助言・一般的に望ましいという穏やかな判断を表す（強さの違いに注意）
  • 口語ではshould haveが短縮されshould've /ʃədəv/となり、これを聞き覚えでshould ofと書いてしまう誤りが非常に多い
  • 提案・要求・必要を表す語に続くthat節では、アメリカ英語は原形（仮定法現在）を好み、イギリス英語はthat + 主語 + should + 原形を好む（It is important that you arrive on time.［米式］／that you should arrive on time.［英式］はどちらも可）
  • 他品詞での用法: 法助動詞である should には、entry.meanings に挙げた6用法以外にも会話でよく使う型がある。(1) 控えめな意見・推測を述べる決まり文句（主にイギリス英語でやや形式張った響き）: I should think it'll be fine.（大丈夫だと思いますが）、I should imagine so.（そうだと思います）。(2) 相手の発言を受けて「そうすべきだ」と短く同意する応答: 'I'm going to apologize.' 'Yes, you should.'（うん、そうすべきだね）。助動詞としての用法のみで、should に独立した名詞・動詞用法はない。

文型・頻出パターン

  • should + 原形〜すべきだ、〜した方がよい（助言・提案の基本形）
    You should drink more water.もっと水を飲んだ方がいい。
    You should apologize to her.彼女に謝った方がいいよ。
  • should have + 過去分詞〜すべきだったのに（しなかった）：過去の行動への後悔・非難
    I should have called before I came.来る前に電話しておくべきだった。
  • should be + 過去分詞（受動態）〜されるべきだ：義務・必要性を受動態で表す
    This mistake should be corrected.この間違いは訂正されるべきだ。
  • 主語 + should + 原形（推量・期待）（当然）〜のはずだ、〜するはずだ
    The package should arrive by Friday.荷物は金曜日までに届くはずだ。
    She should be home by now.彼女はもう家に着いているはずだ。
  • Should + 主語 + 原形, ...（倒置による仮定法・フォーマル）万一〜の場合は（if節の代用。書き言葉・公的な場でよく使う）
    Should you need any help, please ask.万一お困りのことがあれば、お尋ねください。
  • It is 形容詞（important/necessary等）+ that + 主語 + (should) + 原形〜するように、〜すべきだという（提案・必要を表すthat節）
    It is important that you (should) arrive on time.時間通りに到着することが重要だ。
  • Why should + 主語 + 原形?なぜ〜しなければならないのか（不満・反発のニュアンス）
    Why should I be the one who apologizes?なぜ私が謝らなければならないのか。

他のNGSL語への置き換え

should → must
元: You should drink more water every day.
後: You must drink more water every day.
訳: 毎日もっと水を飲まなければならない。
📝 should は「〜した方がいい」という穏やかな助言なのに対し、must は絶対的な必要性・強い義務を示し、選択の余地がないニュアンスになる。健康上の強い警告や規則を述べる場面では must の方が自然。
should → must
元: This mistake should be corrected before the report is submitted.
後: This mistake must be corrected before the report is submitted.
訳: この間違いは、報告書が提出される前に必ず訂正されなければならない。
📝 should be corrected は一般的な義務・提案として「訂正されるべきだ」と述べるのに対し、must be corrected は「必ず訂正しなければならない」という強制力・緊急性を伴う。上司からの強い指示や規則違反の場面では must が選ばれやすい。
should → must
元: Why should I be the one who apologizes?
後: Why must I be the one who apologizes?
訳: なぜ私が謝らなければならないのか。
📝 should を使うと「（道理として）なぜ私が〜しなければならないのか」という納得のいかなさを表すのに対し、must に替えると「（外部の力によって）強制されている」という被害者意識・反発がより強く前面に出る、やや対決的な響きになる。

🗾 カタカナ語とのギャップ

カタカナ「シュッド」は「シュ・ッ・ド」の3モーラだが、英語は1音節 /ʃʊd/。母音は短い/ʊ/で伸ばさず、語末のdもはっきり破裂させきらないことが多い。さらに文中の助動詞shouldはほとんど強勢を持たず弱形/ʃəd/になり、should haveはさらに融合してshould've [ʃədəv] シュダヴのように聞こえる。「シュッド ハブ」と一語一語区切って発音すると不自然に響く。

🌱 Word family・語源
shall（助動詞）〜するつもりだ、〜しましょう（意志・提案；現代米語では稀）NGSL
語源: 古英語 sceolde「shall（sceal：〜する義務がある、〜しなければならない）」の過去形に由来する。will/wouldの関係とまったく同じ構造で、shall/shouldはゲルマン祖語の語根*skulaną（「義務を負う、借りがある」）に遡る。もともとは単なる過去形だったが、時間的な過去が心理的な距離（婉曲・控えめさ）に転じ、助言・後悔・推量・仮定法条件節などを表す独立した法助動詞に発達した。
shall（〜するつもりだ、〜しましょう）— shouldはもともとshallの過去形。will/wouldと並行する対の関係で、現代アメリカ英語ではshall自体があまり使われないため、shouldの方が独立した助動詞として広く定着している
sollen（ドイツ語）（〜すべきだ、〜することになっている）— 同じゲルマン祖語の語根*skulaną由来。ドイツ語の話法助動詞sollenは英語のshall/shouldと同じ「義務を負う」という原義を今も色濃く残している
💡 覚え方: shouldは「shallの過去の姿」。古英語sceal（shall）の過去形sceoldeが由来で、「（返すべき）借りがある、義務を負う」という原義が、時間的な過去→心理的な距離を経て、現代の「〜すべきだ」という助言や「〜すべきだったのに」という後悔（should have）に発展した。ドイツ語のsollen（〜すべきだ）にも同じ語根の「義務」のイメージがそのまま残っている。
⚖️ 類義語の使い分け
must — mustは客観的な強い義務・絶対的な必要性、または高い確信度の推量を表すのに対し、shouldは話者の主観的な助言・一般的に望ましいという穏やかな判断を表す。You must see a doctor. は「見なければならない」という強い勧告、You should see a doctor. は「見た方がいい」という助言にとどまる。
had better — had betterはshouldよりも切迫感・警告のニュアンスが強く、「そうしないと困ったことになる」という含みを持つ。目上の人に使うと高圧的に響くことがあるが、shouldは一般的な助言として幅広い場面で使いやすい。
ought to — ought toはshouldとほぼ同じ意味（義務・助言）で使えるが、後ろにtoを伴う点が形式上の違い。現代英語ではshouldの方がはるかに口語で普通に使われ、ought toはやや形式張った響きを持つ。
💬 熟語・慣用表現 (4)
you shouldn't have （贈り物や親切な行為に対して）そんなことしなくてもよかったのに、ありがとう口語
Oh, you shouldn't have! This necklace is beautiful.（まあ、そんな気を遣わなくてもよかったのに！このネックレス、素敵ね。）
🔍 文字通りは「あなたはすべきではなかった」という非難の形だが、実際には非難ではなく、贈り物や親切な申し出への恐縮と感謝を表す社交儀礼の決まり文句。
📝 本気で相手を叱っているのではなく、贈り物をもらった瞬間の礼儀正しい驚き・感謝を表す定型表現。実際にはもらって嬉しい気持ちを込めて言うのが普通。
you should have seen ~ （実際に見せたかった、見ればよかったのに）～ったらすごかったんだから口語
You should have seen his face when he opened the box — he screamed like a kid!（彼が箱を開けたときの顔ったら見せたかったよ、子どもみたいに叫んでたんだから！）
🔍 文字通りは「～を見ておくべきだったのに」という後悔の表現だが、実際は聞き手がその場に居合わせなかったことを惜しみながら、出来事の驚き・面白さ・すごさを誇張して伝える決まり文句。
📝 should have + 過去分詞の「後悔」の意味というより、実際に目撃した出来事の凄さ・面白さを強調する誇張表現。後ろに his face / the look on her face のような具体的な対象を続けるのが定番。
how should I know? そんなの知るわけないでしょ、知るはずないじゃん口語
'Where did she put the keys?' 'How should I know? I just got here.'（「彼女、鍵をどこに置いたの？」「そんなの知るわけないでしょ、来たばかりなんだから。」）
🔍 文字通りは「私がどうやって知り得ようか」という反語疑問文だが、実際には単に「知らない」ことを強調し、時に軽い苛立ちや突き放しのニュアンスを込めて伝える決まり文句。
📝 情報を求められて答えられない時、単なる I don't know. よりも防御的・時に不機嫌な響きを伴う。相手の質問自体を「聞くだけ無駄」と突き放すニュアンスがある。
I should hope so 当然そうであってほしい、そうでなきゃ困る口語
'I already apologized to her.' 'I should hope so, after what you said.'（「もう彼女に謝ったよ。」「当然そうすべきでしょうね、あんなことを言ったんだから。」）
🔍 文字通りは「私はそう望むべきだ」という控えめな表現だが、実際には相手の発言や期待される結果を強く肯定し、「当然そうであるべきだ」と念を押すニュアンスを添える決まり文句。
📝 単なる同意（I hope so.）より語気が強く、「そうでなければおかしい」という当然視・軽い念押しのニュアンスを持つ。皮肉っぽく使われることも多い。
