use

動詞・名詞米 /juːz/（動詞）、/juːs/（名詞） ／ 英 /juːz/（動詞）、/juːs/（名詞）use（1音節）CEFR A1

🗣 1音節語。母音は /juː/ で、半母音 /j/（「ユ」の子音部分）と長母音 /uː/ が一体化した音。日本語の「ユー」よりも唇をすぼめて長めに伸ばす。動詞は語末が有声の /z/（juːz）、名詞は無声の /s/（juːs）で、この有声/無声の違いが唯一にして最大のポイント

  1. 【動詞】（道具・手段などを）使う、利用する（use a computer コンピューターを使う）
  2. 【動詞】（機会・時間などを）活用する、生かす（use this chance このチャンスを活かす）
  3. 【動詞】かつて〜したものだ、以前は〜だった〔過去の習慣・状態で、今はしていない〕（used to ＋ 動詞の原形。He used to live in Tokyo. 彼は昔東京に住んでいた。）
  4. 【動詞】〜に慣れている、〜に慣れる（be/get used to ＋ 名詞・動名詞。I'm used to the cold. 私は寒さに慣れている。）
  5. 【名詞】使用、利用〔不可算〕（the use of smartphones スマートフォンの使用）
  6. 【名詞】用途、使いみち〔可算〕（This tool has several uses. この道具にはいくつか使いみちがある。）
  7. 【名詞】役に立つこと、効用（It's no use worrying about it. それを心配しても仕方がない。）

語形変化

形種類発音備考
uses三単現（動詞）/ˈjuːzɪz/study型と同じ-es語尾。語幹が有声のzで終わるため、-esは[ɪz]がそのまま加わり「ユーズィズ」となる
uses複数形（名詞）/ˈjuːsɪz/名詞のuseは語末が無声の/juːs/なので、複数形も無声のまま[s]+[ɪz]で「ユースィズ」。綴りは動詞の三単現usesと同じだが発音が異なる点に要注意
used過去形・過去分詞/juːzd/規則変化。語幹が有声子音zで終わるため-edは有声の[d]。ただし「used to＋動詞原形」(かつて〜した)のusedはこの発音とは別に無声化して[ˈjuːstə]となる独特の音変化がある（下記grammar_marks参照）

語法マーク

  • 動詞は他動詞が基本。use + O の形で目的語を必ず伴う（× I use every day. のように単独では意味が成立しにくい。何をuseするのか目的語が必要）
  • 動詞/juːz/（有声z）と名詞/juːs/（無声s）で発音が変わる、英語に多い「動詞は有声・名詞は無声」パターンの代表例（house, close なども同じ現象）
  • be used to + 名詞／動名詞「〜に慣れている」と be used to + 動詞原形「〜するために使われる（受動態）」は形が似ているため混同しやすい。前者のtoは前置詞、後者のtoはto不定詞
  • used to + 動詞原形「かつて〜したものだ」は過去の習慣・状態のみを表し、現在形は存在しない（× He uses to live in Tokyo. は誤り）。否定はdidn't use to〜またはused not to〜
  • 名詞のuseは不可算（使用という行為）にも可算（用途、複数形usesで使うことが多い）にもなる。文脈で判断する
  • 他品詞での用法: 名詞のuseには不可算用法と可算用法がある: 不可算「使用・利用という行為そのもの」(the use of computers コンピューターの使用)、可算「個々の用途・使いみち」(This device has three uses. この装置には3つの使いみちがある)。さらに過去分詞usedは形容詞化して「中古の、使用済みの」の意味でも使われる(a used car 中古車、used clothes 古着)。この形容詞用法のusedも動詞の過去分詞と同じく有声の/juːzd/で発音し、無声の名詞useとは区別される。

文型・頻出パターン

  • use + O〜を使う（最も基本的な他動詞用法）
    use a computerコンピューターを使う
    use your imagination想像力を働かせる
  • use + O + to do / for -ing〜を使って…する（目的・手段を示す）
    use a key to open the door鍵を使ってドアを開ける
    use flour for baking breadパンを焼くのに小麦粉を使う
  • be used to do（受動態）〜するために使われる
    This room is used to store old furniture.この部屋は古い家具を保管するために使われている。
  • be/get used to + 名詞・動名詞〜に慣れている／慣れる
    I'm used to working late.私は遅くまで働くことに慣れている。
    You'll get used to the weather here.ここの天気にはそのうち慣れるよ。
  • used to + 動詞原形かつて〜したものだ（過去の習慣・状態、今はしていない）
    used to play the piano昔ピアノを弾いていたものだ
  • make use of / put ~ to use〜を活用する（軽動詞構文）
    make use of every chanceあらゆる機会を活用する

他のNGSL語への置き換え

use → operate
元: Doctors often use this device to check a patient's heart rate.
後: Doctors often operate this device to check a patient's heart rate.
訳: 医者はこの機器をよく操作して、患者の心拍を確認する。
📝 operate は機械・装置を「操作する」ことに特化し、専門的な扱いを強調する。use はどんな道具・状況にも使える最も中立的で幅広い語。
use → download
元: This app is used by millions of people around the world every day.
後: This app is downloaded by millions of people around the world every day.
訳: このアプリは世界中の何百万もの人々によって毎日ダウンロードされている。
📝 download は「ダウンロードする」という一回的な取得行為を表す。is used by の「使われている」が継続的な利用状況を示すのに対し、download はその手前の瞬間的な動作に焦点が移る。
use → would
元: He used to smoke a lot, but he quit five years ago.
後: He would smoke a lot, but he quit five years ago.
訳: 彼はよくたばこを吸っていたものだったが、5年前にやめた。
📝 would も過去の習慣的な行動の反復を表せるが、used to と違い状態動詞（like, live, have など）には使えない（例: He used to like coffee. を would には置き換えられない）。この文の smoke（動作動詞）では would も可能だが、used to の方が「今はもうしていない」という現在との対比がより明確に伝わる。

⚠️ 日本人の典型誤用
✕ I am used to get up early.
言いたかったのは「私は早起きに慣れている」。しかしto以下を動詞原形にすると、ネイティブには「私は早起きするために使われている（道具のように）」という不自然な受動態の意味に聞こえてしまう。
○ I am used to getting up early.
私は早起きに慣れている。
I am used to getting up early.
「〜に慣れている」の be used to の to は前置詞なので、後ろには動詞の原形ではなく動名詞（-ing）または名詞が続く。「〜するために使われる」の be used to do（to不定詞）と形が似ているため混同しやすい。
✕ I use to play soccer every weekend when I was a child.
言いたかったのは「子どもの頃、毎週末サッカーをしたものだった」。しかし use to（現在形）はこの意味では文法的に誤りで、ネイティブには単純な書き間違い・言い間違いとして違和感を持たれる。
○ I used to play soccer every weekend when I was a child.
子どもの頃、毎週末サッカーをしたものだった。
I used to play soccer every weekend when I was a child.
過去の習慣を表す used to は現在形 use to では成立しない。過去の話をしているので必ず -ed の付いた used to にする。
✕ It's no use to cry over spilt milk.
言いたかったのは「こぼれた牛乳を嘆いても仕方ない（覆水盆に返らず）」。to不定詞にすると英語として不自然で、ネイティブには型崩れの表現として引っかかって聞こえる。
○ It's no use crying over spilt milk.
こぼれた牛乳を嘆いても仕方ない。
It's no use crying over spilt milk.
「〜しても無駄だ」の it's no use の後ろは to不定詞ではなく動名詞（-ing）を取る決まった構文。there's no point in -ing と同様のパターン。
🗾 カタカナ語とのギャップ

カタカナ「ユーズ」「ユース」は英語の動詞/名詞の区別と対応しており便利だが、日本語では有声・無声の違いを意識せず曖昧に発音しがち。動詞useは語末を必ず有声の[z]（喉を震わせるズ）で、名詞useは無声の[s]（喉を震わせないス）で区別する。「ユーザー」(user)もzの音を保ったまま「ユーザー」でよいが、母音を足して「ユーズァー」のように間延びさせないこと。

🌱 Word family・語源
useful（形容詞）役に立つ、有用なNGSL
user（名詞）使用者、ユーザーNGSL
useless（形容詞）役に立たない、無駄な
usage（名詞）（言葉・言語などの）使い方、慣用；使用（量）
misuse（動詞・名詞）誤用する、悪用する；誤用、悪用
reuse（動詞・名詞）再利用する；再利用
語源: ラテン語 ūsus「使うこと」（動詞 ūtī「使う」の過去分詞から）に由来し、古フランス語 user を経て中英語に入った借入語。名詞・動詞どちらの用法もこの同じラテン語の語幹を受け継いでいる。
abuse（乱用する、虐待する）— ab-（外れて、悪い方向へ）+ use で「使い方を誤る」が原義
usual（いつもの、普通の）— 「いつも使われている」→「ふつうの」へ意味が発展
utility（実用性、公共事業）— 同じ語根 ūt-（使う）から派生したラテン語 utilis「役立つ」に由来
💡 覚え方: コアはラテン語「使う」。abuse=ab（悪い方向）+use=「使い方を誤る」→乱用、usual=「いつも使われている」→ふつうの、utility=「使うこと」に基づく実用性——どれも use の「使う」イメージから枝分かれしている。
🔗 句動詞
use up — （在庫・資源などを）使い切る、消費し尽くす
We used up all the firewood before winter was over.（冬が終わる前に薪を全部使い切ってしまった。）
⚖️ 類義語の使い分け
utilize — use よりかたく、書き言葉やビジネス・学術的な文脈で好まれる。日常会話ではuseの方がはるかに自然。
employ — 「（手段・方法）を用いる」というフォーマルな言い方。人を「雇う」という別義もあるため文脈に注意。
apply — 「（技術・力・規則など）を適用する」という意味に限定され、useより当てはめる対象が専門的・限定的。
💬 熟語・慣用表現 (7)
used to do かつて〜したものだ（過去の習慣・状態）中立
I used to live in Osaka when I was a child.（子どもの頃、私は大阪に住んでいたものだった。）
🔍 現在形の use to は存在せず、used to はひとまとまりで過去の習慣を表す助動詞的表現。動詞 use の「使う」という意味からは予測できない、完全に独立した慣用構文。
📝 否定は didn't use to／used not to。「今はもうしていない」ことを暗示する点で現在完了などと意味が異なる。
be/get used to 〜に慣れている／慣れる中立
It took me a while to get used to the cold winters here.（ここの寒い冬に慣れるまでしばらくかかった。）
🔍 この to は前置詞で、後ろに名詞・動名詞を取る。動詞 use の「使う」ではなく「慣れ」という状態・変化を表す別の慣用構文。
📝 be used to do（〜するために使われる、受動態）と形が似ているため混同しやすい。
it's no use (doing) 〜しても無駄だ、仕方がない中立／やや口語
It's no use arguing with him—he never changes his mind.（彼と言い争っても無駄だよ。彼は絶対に考えを変えないから。）
🔍 「するための use（効用）が無い」から「効果がない、無駄だ」。後ろは動名詞を取る決まった構文で、to不定詞は使えない。
📝 It's useless to do とほぼ同義だが、no use の後は必ず -ing。there's no point in -ing も同じパターン。
what's the use (of doing)? 〜して何になるのか（無駄だ、という反語表現）口語
What's the use of complaining? Nothing will change.（文句を言って何になるの？何も変わらないよ。）
🔍 疑問形だが実際には答えを求めておらず「意味がない」と述べる修辞疑問。字面通りではない点が非構成的。
📝 What's the point (of doing)? とほぼ同義。単独で What's the use?（もういいよ、無駄だ）とも使う。
have no use for 〜を必要としない、〜が嫌いだ中立／やや口語
My grandfather has no use for modern technology.（祖父は最新技術を毛嫌いしている。）
🔍 文字通りは「〜のための使い道が無い」だが、実際は物だけでなく人にも使い「毛嫌いする、認めない」という否定的評価を表す比喩表現。
📝 対象が人の場合は「相手を評価しない、嫌う」という強い否定的ニュアンスになる。
come into use 使われるようになる、普及するビジネス／中立
This word came into use in the early twentieth century.（この単語は20世紀初頭に使われるようになった。）
🔍 「使用の中に入ってくる」から「新たに採用され広まる」。come into being などと同型の慣用パターン。
📝 対義表現は go out of use（使われなくなる、廃れる）。制度・道具・言葉の普及について使う。
go out of use 使われなくなる、廃れる中立
Many traditional tools went out of use after machines were introduced.（機械が導入された後、多くの伝統的な道具は使われなくなった。）
🔍 「使用の外へ出て行く」から「使われなくなる」。stop being used と言い換えられるが、慣用的な決まり文句として定着している。
📝 come into use の対義表現。道具・習慣・言葉などが廃れていく場合に使う。
