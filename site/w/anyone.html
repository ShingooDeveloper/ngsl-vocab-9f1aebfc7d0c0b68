anyone

代名詞 (pronoun)米 /ˈeniwʌn/ ／ 英 /ˈeniwʌn/an·y·one（3音節）CEFR A1

🗣 第1音節 AN に強勢が置かれる（/ˈeniwʌn/）。母音は /e/ で日本語の「エ」より短く鋭い。第2音節 -y- はあいまい母音 /i/ で軽く添えるだけ、第3音節 -one は数字の「1」と同じ /wʌn/ だが無強勢で軽く発音し、one 単独で言うときほど強く伸ばさない

  1. 【代名詞】（疑問文・否定文・if節で）誰か、誰も（存在するかどうかまだ分からない人を指す）（Is there anyone who can help me? 誰か私を手伝ってくれる人はいますか?）
  2. 【代名詞】（肯定文で、any系の自由選択用法）誰でも（相手を問わない自由な選択を表す）（Anyone can join this club. この会は誰でも参加できます）
  3. 【代名詞】（not anyone の形で、nobodyとほぼ同義の強い否定）誰も〜ない（I didn't tell anyone about it. そのことは誰にも話さなかった）
  4. 【代名詞】（anyone + 形容詞の語順で、形容詞が後ろに置かれる点に注意）〜な人が誰か（Is there anyone available to help right now? 誰か今手伝える人はいる?）
  5. 【代名詞】（or anyone / anyone else の形で、口語）ほかに誰か（Did anyone else see what happened? ほかに誰か何が起きたか見た人はいる?）

語形変化

形種類発音備考
anyone's所有格/ˈeniwʌnz/語末が有声の鼻音 /n/ の後ろなので 's は /z/（「エニワンズ」であって「エニワンス」ではない）。anything's が be/have の短縮形にしかならないのと違い、anyone は人を指すため anyone's は純粋な所有格として働く（It's anyone's guess. ＝誰にも予想がつかない、という決まり文句でも頻出）

語法マーク

  • 三人称単数扱いの代名詞。動詞は必ず単数形で受ける（Anyone is welcome. であって Anyone are welcome. ではない）
  • anyone を修飾する形容詞は後ろに置く（anyone available, anyone else。×available anyone のように前置しない）
  • 疑問文・否定文・if節では『（存在するか分からない）誰か』、肯定の平叙文では『（相手を問わない）誰でも』という two-faced な意味を持つ点は any/anything と共通。肯定の平叙文で単に『誰かがいる』という事実を述べたいときは someone を使う
  • 後方照応の代名詞は伝統的に he/his が使われてきたが、現代英語では性中立の単数 they/their/them が標準的（Anyone can bring their own laptop.）
  • not anyone ~ は no one とほぼ同義の強い全否定（I didn't see anyone. = I saw no one.）。ただし not と no one を同じ文で重ねる二重否定は誤り
  • 1語 anyone と2語 any one の違いに注意：any one は『（集合の中の）どれか一つ／どの一人か』を強調し、any one of ~ の形をとることが多い
  • 他品詞での用法: 品詞は代名詞1用法のみ。名詞・動詞への転用はない。1語の anyone（誰でも／誰か）と2語の any one（〜のうちのどれか一人、しばしば of句を伴う）を混同しないこと。any one は集合の中の個々を強調する表現で、人にも物にも使えるのに対し、1語の anyone は人だけを指す（物を指す自由選択は anything が担当する）。

文型・頻出パターン

  • anyone who + V〜する（ような）人は誰でも／誰か（関係代名詞 who で anyone を後ろから修飾）
    Anyone who breaks the rules will be disqualified.規則を破った人は誰でも失格になる。
    I don't know anyone who has read that book.その本を読んだことのある人を私は誰も知らない。
  • Is/If there anyone + (who) ~誰かいるか／もし誰かいれば（存在するかどうか不明な人物を問う・条件づける）
    Is there anyone who disagrees with this plan?この計画に反対する人は誰かいますか?
    If there's anyone waiting outside, let them in.もし外に誰か待っている人がいたら、中に入れてあげて。
  • not ... anyone（= no one）誰も〜ない（強い全否定）
    She hasn't spoken to anyone since the accident.彼女はその事故以来、誰とも話していない。
  • anyone + 形容詞（後置修飾）〜な人が誰か
    We need anyone qualified to run the machine.その機械を操作できる資格のある人なら誰でも必要だ。
    Is there anyone new joining the team this month?今月チームに加わる新しい人は誰かいますか?
  • anyone else / or anyone（口語）ほかに誰か／〜か誰かほかの人
    Ask anyone else in the office; they'll tell you the same thing.オフィスのほかの誰に聞いても、同じことを言うはずだ。

他のNGSL語への置き換え

anyone → allow
元: Don't let anyone into the server room without a key card.
後: Don't allow anyone into the server room without a key card.
訳: キーカードなしでは誰もサーバールームに入ることを許可しないで。
📝 let + O + into（前置詞句）は直接的でぶっきらぼうな禁止の響きを持つのに対し、allow + O + into も同じ構造で使えるが、規則・方針としてのややフォーマルな響きが強まる。掲示や規則の説明文でよく使われる交換。
anyone → somebody
元: Is there anyone here who can translate this letter for me?
後: Is there somebody here who can translate this letter for me?
訳: この手紙を私のために翻訳できる人がここにいますか?
📝 anyone は「誰でもいいので該当者がいるか」という中立的・網羅的な問いかけであるのに対し、somebody に替えると話し手が漠然と特定の誰かの存在をより強く想定している響きになる。日常会話ではほぼ同義に使われるが、anyone の方が掲示文や正式な依頼にも使いやすい。
anyone → know
元: I looked around the room, but I didn't recognize anyone there.
後: I looked around the room, but I didn't know anyone there.
訳: 部屋を見回したけれど、そこにいる人は誰も知らなかった。
📝 recognize は「（見て）それと分かる、見覚えがある」で既に会ったことのある人を視覚的に思い出すニュアンスを持つのに対し、know はそもそも面識・知り合いであること自体を表すより一般的な動詞。recognize は「顔は覚えているが誰か分からない」という状況でも使えるが、know は面識の有無そのものを問う点が異なる。

⚠️ 日本人の典型誤用
✕ Anyone are welcome to attend the meeting.
「誰でも会議に参加できます」と言いたかったが、areを使うと主語と動詞の単数・複数が食い違っており、文法的に崩れた文としてすぐに気づかれてしまう。
○ Anyone is welcome to attend the meeting.
誰でも会議に参加できます。
Anyone is welcome to attend the meeting.
anyone は意味的には『誰でも／みんな』を連想させても、文法上は常に三人称単数として扱われる。動詞は必ず単数形（is/has/knows など）で受ける。
✕ Any one can join this club if they want.
「誰でもこのクラブに参加できる」と言いたかったが、2語の any one を使うと『（何かの中の）どの一人か』を強調する不自然な響きになってしまう。
○ Anyone can join this club if they want.
誰でも希望すればこのクラブに参加できる。
Anyone can join this club if they want.
『誰でも参加できる』という一般的な意味では1語の anyone を使う。2語の any one は『（特定の集団の中の）どれか一人』を強調する語で、通常 any one of ~ の形で使う。
✕ Anyone left a message for you this morning.
「今朝、誰かがあなたに伝言を残していった」と事実を伝えたかったが、anyoneを使うと『誰でもいいから残していった』というぼやけた奇妙な意味に聞こえてしまう。
○ Someone left a message for you this morning.
今朝、誰かがあなたに伝言を残していきました。
Someone left a message for you this morning.
肯定の平叙文で『（実際に）誰かがいた』という事実を報告したいときは someone を使う。anyone を肯定文で使うと『（種類を問わず）誰でも』という自由選択の意味になり、実際に起きた出来事を述べる文脈とかみ合わない。
🗾 カタカナ語とのギャップ

カタカナ「エニワン」は「エ・ニ・ワ・ン」と4モーラ均等に聞こえがちだが、実際の英語 /ˈeniwʌn/ は最初の音節 an- に強勢が集中し、後ろの -y- や -one は弱く短く発音される。one の母音は /wʌn/ で、日本語の「ワン」ほどはっきり口を開かないあいまいな音になる点にも注意。

🌱 Word family・語源
any（限定詞・代名詞・副詞）いくつかの、何も（否定文で）、どれでもNGSL
anybody（代名詞）誰でも、（疑問・否定文で）誰か（anyoneとほぼ同義。ややくだけた響き）NGSL
anything（代名詞）何でも、（疑問・否定文で）何かNGSL
anywhere（副詞）どこでも、（疑問・否定文で）どこかNGSL
someone（代名詞）誰か（存在を前提とした肯定文での不特定の人）NGSL
everyone（代名詞）皆、全員NGSL
nobody（代名詞）誰も〜ないNGSL
語源: 古英語 ǣnig（「1つでも、どれか1つの」＝ any の祖先）と ān（「1つ」＝ one の祖先）が中英語期に結びついてできた複合代名詞。any の持つ「疑問文・否定文でのあるかどうか不明な誰か」「肯定文での自由選択（誰でも）」という二面性を受け継いだまま、one と結合して人を指す1語の代名詞になった。
anybody（誰でも）— any + body（人）。one の代わりに body が人を表す口語的な双子語
anything（何でも）— any + thing（もの・こと）。one（人）の代わりに thing（物・事）が結びついた対の語
anywhere（どこでも）— any + where（場所）。one（人）の代わりに where（場所）が結びついた語
💡 覚え方: any（1つでもあれば）+ one（1人）＝『誰であれ1人』というのが core イメージ。anybody・anything・anywhere とは one の部分だけが違う兄弟語で、対になる someone・everyone・no one とセットで覚えると、any-/some-/no-/every- × -one/-body/-thing/-where の複合代名詞システムが一気に整理できる。
⚖️ 類義語の使い分け
anybody — anyoneとほぼ同義だが、anybodyの方が口語的・くだけた響きを持つ。書き言葉やフォーマルな文脈ではanyoneが好まれる。
someone — anyoneは疑問文・否定文・条件節で『（存在するかどうか分からない）誰か』、肯定文で『（相手を問わない）誰でも』を表すのに対し、someoneは肯定の平叙文で『（存在を前提とした）特定されない一人』を表す。肯定文と疑問・否定・if節とで使い分ける。
whoever — whoeverも『誰であれ』という自由選択を表す点でanyoneの肯定文用法と重なるが、whoeverの方がより強調的・譲歩的（whoever comes first＝誰が最初に来ようと）で、名詞節を作る接続詞的な働きも持つ点が異なる。
💬 熟語・慣用表現 (3)
anyone's guess 誰にも予想がつかないこと、皆目見当がつかないこと中立／口語
Whether the concert will be canceled because of the storm is anyone's guess.（その嵐のせいでコンサートが中止になるかどうかは、誰にも分からない。）
🔍 「誰かの推測」という意味ではなく、「結果がどうなるか誰にも確信を持って言えない」ことを表す決まり文句。whether/how/what節などを主語にして It's anyone's guess (that/wh-節) の形で使う。
📝 How/whether/what などの疑問詞節・whether節を主語にとることが多い。単に someone's guess（誰かの推測）という文字通りの意味とは切り離して覚える。
(it's) anyone's game （勝敗や結果が）まだどちらに転ぶか分からない状況、五分五分の勝負口語（スポーツ実況・選挙報道などでよく使われる）
With both teams tied in the final minute, it's anyone's game.（最終盤で両チームが同点のため、勝敗はまだどちらに転ぶか分からない。）
🔍 「誰のゲームでもありうる」という比喩から、「まだ勝者が決まっていない、結果が読めない」競争や勝負を表す。
📝 game の部分は race, election, ballgame など状況に応じて入れ替えられる（it's anyone's race など）。文字通り「誰のゲーム」という意味ではなく、結果の不確実性を表す比喩表現。
anyone home? （ぼんやりしている相手に）ちゃんと聞いてる? 意識ある?口語
Hello? Anyone home? I've been talking to you for five minutes.（ねえ、聞いてる? もう5分も話しかけてるんだけど。）
🔍 文字通りは「誰か家にいますか?」だが、目の前の相手が上の空だったりぼんやりしている時に、頭をノックする仕草とともに冗談めかして「頭の中に誰かいる?」と問いかける口語表現。
📝 からかい半分・親しい間柄で使うくだけた表現で、フォーマルな場では使わない。文字通り家に人がいるか尋ねる質問（Is anyone home?）とは文脈で区別する。
