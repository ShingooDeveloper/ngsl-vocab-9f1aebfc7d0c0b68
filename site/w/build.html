build

動詞・名詞米 /bɪld/ ／ 英 /bɪld/build（1音節）CEFR A1

🗣 1音節語。綴りの ui は発音上は短母音 /ɪ/ のみを表し、u の音は現れない（「ビュイルド」のように読まない）。語末の子音クラスタ -ld ははっきり2音とも出す

  1. 【動詞】（建物・道路などを）建てる、建設する（build a house 家を建てる）
  2. 【動詞】（関係・信頼・力などを）築く、育てる（build trust with someone 誰かとの信頼を築く）
  3. 【動詞】（勢い・圧力などが）徐々に高まる、増大する（The pressure began to build. 圧力が高まり始めた）
  4. 【名詞】体格、体つき（He has a muscular build. 彼はがっしりした体格をしている）

語形変化

形種類発音備考
builds三単現/bɪldz/語末は有声子音の後なので /z/（有声の「ビルヅ」寄り。無声化して「ビルツ」にならない）
built過去形/bɪlt/不規則変化。規則変化のように builded とするのは誤り
built過去分詞/bɪlt/過去形と同形。受動態（was built）・現在完了（have built）に加え、形容詞的に a well-built house のようにも使われる
buildinging形/ˈbɪldɪŋ/進行形・動名詞で使用。綴りが同じ名詞 building（建物）と文脈で混同しないよう注意

語法マーク

  • 不規則動詞: build - built - built（過去形と過去分詞が同形）
  • 基本は他動詞で目的語を必須にとる（「（勢いなどが）高まる」の自動詞用法を除き、単独の I build. は不自然）
  • 名詞（体格）の意味では基本的に単数・可算扱い: a good build, an athletic build
  • 過去分詞 built は形容詞としても頻出: a well-built house, a custom-built car, built-in shelves
  • 他品詞での用法: 名詞用法もある。「体格・体つき」を表し、通例単数・可算で使う: He has a muscular build.（彼はがっしりした体格をしている）/ What's your build like?（体格はどんな感じ？）。この意味では builds と複数形になることはまれ。

文型・頻出パターン

  • build + O〜を建てる・築く（最も基本的な他動詞用法）
    build a house家を建てる
    build a bridge橋を架ける
    build a teamチームを作り上げる
  • build + O + out of / from + 材料〜（材料）で…を作る
    They built the cabin out of local timber.彼らはその小屋を地元の木材で建てた。
  • build + O2 + for + O1（build + O1 + O2 も可）（人）のために〜を建ててあげる
    He built a treehouse for his kids.彼は子どもたちのためにツリーハウスを建てた。
    He built his kids a treehouse.彼は子どもたちにツリーハウスを建ててあげた。
  • be built to + 原形〜するように作られている（耐久性・目的を表す受動態表現）
    This truck is built to last.このトラックは長持ちするように作られている。
  • build + O + into + 状態・組織〜を…へと築き上げる（段階的な発展・成長を表す）
    She built the small shop into a nationwide chain.彼女はその小さな店を全国チェーンにまで築き上げた。

他のNGSL語への置き換え

build → construct
元: They are building a new shopping mall near the station.
後: They are constructing a new shopping mall near the station.
訳: 駅の近くに新しいショッピングモールを建設中です。
📝 construct は build よりフォーマルで専門的な響きを持ち、建築・土木のニュース記事や公式な文脈でよく使われる。build は日常会話で使う最も一般的な語。
build → develop
元: She has built a successful career in the fashion industry.
後: She has developed a successful career in the fashion industry.
訳: 彼女はファッション業界で成功したキャリアを発展させてきた。
📝 develop は既にあるものが徐々に成長・向上していく過程に焦点を当てる。build は土台から一つずつ積み上げて作り上げるニュアンスが強い。
build → buy
元: If we had more money, we would build a bigger house.
後: If we had more money, we would buy a bigger house.
訳: もっとお金があれば、もっと大きな家を買うのに。
📝 buy は既存の家を「購入する」ことを意味し、build の「一から建てる」とは行為も費用構造も大きく異なる。

⚠️ 日本人の典型誤用
✕ She builded her career in the fashion industry.
「彼女はファッション業界でキャリアを築いた」と言いたかったが、builded という形は英語として存在せず、動詞の活用を知らない学習者の発話に聞こえてしまう。
○ She built her career in the fashion industry.
彼女はファッション業界でキャリアを築いてきた。
She built her career in the fashion industry.
build は不規則動詞で過去形は built。規則変化の -ed をそのまま付けて builded としてしまう典型的な誤り。
✕ I want to build my English skill.
「英語力を伸ばしたい」と言いたかったが、build 単独では「土台から一気に作り上げる」ニュアンスが強く、能力を少しずつ伸ばす意味では build up の方が自然に響く。skill も単数のままだと不自然。
○ I want to build up my English skills.
英語力を伸ばしていきたい。
I want to build up my English skills.
英語力のような抽象的な能力を「少しずつ積み上げる」場合は build よりも build up の方が自然。また skill は可算名詞として複数形 skills で使うのが普通。
✕ He has a good build body.
「彼は体格が良い」と言いたかったが、build 自体が「体格」を意味するので body を続けると「体格の良い体」と意味が重複し、不自然な英語に聞こえる。
○ He has a good build.
彼は体格が良い。
He has a good build.
名詞 build はそれ自体で「体格・体つき」を意味するため、後ろに body を重ねると意味が重複する。日本語「体格の良い体」を直訳しようとして起きる誤り。
🗾 カタカナ語とのギャップ

カタカナ「ビルド」は「ビ・ル・ド」の3モーラだが、英語は1音節 /bɪld/ で母音は1つだけ。語末の d に母音を付けて「ドゥ」と伸ばさないこと。また日本語で建物を指す「ビル」は英語 building の短縮に由来する外来語で、動詞 build（建てる）とは別物である点にも注意。

🌱 Word family・語源
building（名詞）建物、ビルディングNGSL
builder（名詞）建築業者、建てる人
rebuild（動詞）再建する、建て直す
build-up（名詞）（徐々の）増強、蓄積
outbuilding（名詞）付属建物、離れ
語源: 古英語 byldan「家を建てる」に由来し、さらに古英語の名詞 bold「家、住居」（ゲルマン祖語 *buþlą、印欧祖語で「住む、生える」を表す語根 *bʰew- に遡る）から派生した。もとは「住居を作る」という具体的な意味の語だった。
bower（木陰の東屋、あずまや）— 同じ語根 *bʰew-（住む）に由来し、「人が留まる場所」の意味でつながる
booth（屋台、ブース）— 古ノルド語 bóth「住居」を経由し、build と同じ「住む場所」の語根を共有する
byre（牛小屋）— 同語根から「家畜が住む建物」の意味に発展した語
💡 覚え方: build はもともと「bold（住居）を作る」という意味だった。bower（あずまや）・booth（ブース）・byre（牛小屋）はどれも「人や家畜が住む場所」を表す同じ語根の仲間 — すべて「住む・留まる」イメージで build とつながる。
🔗 句動詞
build on — 〜を土台にしてさらに発展させる
The new policy builds on the success of last year's program.（新しい政策は昨年のプログラムの成功を土台にしている。）
build up — 徐々に築き上げる、増強する
He has built up a large collection of vintage records over the years.（彼は長年かけてビンテージレコードの大きなコレクションを築き上げた。）
build in — （機能などを）組み込む、内蔵させる
The software has several security features built in.（そのソフトウェアにはいくつかのセキュリティ機能が組み込まれている。）
⚖️ 類義語の使い分け
construct — build よりフォーマルで専門的な響きを持ち、建築・土木のニュース記事や公式な文脈でよく使われる。build は日常会話で使う最も一般的な語。
erect — 「（垂直に）建てる、立てる」に限定され、tower や statue、fence のような比較的狭い対象に使われることが多いフォーマルな語。
create — 抽象的なものも含め広く「作り出す」を表せるが、build のような「土台から段階的に積み上げる」物理的・時間的なニュアンスは弱い。
💬 熟語・慣用表現 (4)
build bridges （対立する人・集団の間の）橋渡しをする、関係を築く・修復する中立
The new manager worked hard to build bridges between the two rival departments.（新しいマネージャーは対立する2つの部署の間の関係修復に尽力した。）
🔍 「橋を架ける」という物理的行為の比喩で、対立や隔たりのある者同士の関係を築いたり修復したりすることを表す。
📝 反対の意味の burn bridges（関係を断つ、後戻りできなくする）とセットで覚えると定着しやすい。
build castles in the air 空想にふける、実現性のない計画を立てる口語
Stop building castles in the air and start saving some real money.（夢物語ばかり語っていないで、実際にお金を貯め始めなさい。）
🔍 「空中に城を建てる」という不可能な行為から転じて、根拠のない夢物語や非現実的な計画を指す。
📝 castles in Spain という古い異形もある。行動が伴わない空想的な計画への軽い批判を込めて使われることが多い。
Rome wasn't built in a day ローマは一日にして成らず（大きな成果には時間がかかるものだ）中立
Don't get discouraged—Rome wasn't built in a day, and neither is a successful business.（落ち込まないで。ローマは一日にして成らずと言うし、成功する事業もそれと同じだよ。）
🔍 「ローマは一日で建設されたわけではない」という史実的な比喩で、大事業の達成には長い時間と忍耐が必要だという教訓を表す。
📝 励ましの決まり文句として単独の一文で使われることがほとんどで、過去形・受動態のまま固定されている。
build a nest egg （将来に備えて）貯蓄をする、へそくりを蓄える中立
They've been building a nest egg for their retirement since their twenties.（彼らは20代の頃から老後のための蓄えをこつこつ築いてきた。）
🔍 雌鶏に産卵を促すため巣に残しておく卵（nest egg）の比喩で、将来のために少しずつ蓄えるお金を指す。
📝 nest egg 自体が「蓄え・貯金」を意味する名詞句で、build のほか have/save a nest egg のようにも使われる。
