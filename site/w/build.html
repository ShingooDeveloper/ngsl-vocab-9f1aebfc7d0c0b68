build

動詞・名詞米 /bɪld/ ／ 英 /bɪld/
  1. 【動詞】（建物・道路などを）建てる、建設する（build a house 家を建てる）
  2. 【動詞】（関係・信頼・力などを）築く、育てる（build trust with someone 誰かとの信頼を築く）
  3. 【動詞】（勢い・圧力などが）徐々に高まる、増大する（The pressure began to build. 圧力が高まり始めた）
  4. 【名詞】体格、体つき（He has a muscular build. 彼はがっしりした体格をしている）

例文 — 6つの文法・構文で学ぶ

1They are building a new shopping mall near the station.

駅の近くに新しいショッピングモールが建設中です。

現在進行形（be + -ing）：今まさに進行中の動作を表す

現在進行形（be + -ing）：今まさに進行中の動作を表す。build a mall のように「建物+建てる」の組み合わせが典型。

  • are → 弱形 /ər/（強形 /ɑːr/ ではなく曖昧母音に弱化）
  • the station → 子音の前なので the は弱形 /ðə/
ゼイ ア ビルディング ア ニュー シャピング モール ニア ダ ステイシャン

2This bridge was built more than eighty years ago.

この橋は80年以上前に建てられました。

受動態（be + 過去分詞、過去形）：「建てられた」という受け身の状態・出来事を表す

受動態（be + 過去分詞、過去形）：「建てられた」という受け身の状態・出来事を表す。

/ðɪs ˈbrɪdʒ wəz ˈbɪlt ˈmɔːr ðən ˈeɪɾi ˈjɪrz‿əˈɡoʊ/
  • was → 弱形 /wəz/（強形 /wɑːz/ ではなく曖昧母音に弱化）
  • than → 弱形 /ðən/ に弱化
  • eighty → 母音間のtがフラップ化 [ˈeɪɾi] エイリ（米音）
  • years ago → 語末子音zと母音əが連結 [jɪrzəˈɡoʊ] イアザゴウ

3She has built a successful career in the fashion industry.

彼女はファッション業界で成功したキャリアを築いてきた。

現在完了（have + 過去分詞）：過去から現在までの積み重ねの結果を表す

現在完了（have + 過去分詞）：過去から現在までの積み重ねの結果を表す。build a career「キャリアを築く」の頻出コロケーション。

/ʃi həz ˈbɪlɾə səkˈsesfəl kəˈrɪr‿ɪn ðə ˈfæʃən ˈɪndəstri/
  • has → 弱形 /həz/ に弱化（文中ではさらに /əz/ まで弱まることもある）
  • built a → lの後のtが母音の前でフラップ化し連結 [ˈbɪlɾə] ビルラ
  • career in → 語末のrと後ろの母音がなめらかに連結 [kəˈrɪrɪn] カリアリン
  • the fashion → 子音の前なので the は弱形 /ðə/

他のNGSL語への置き換え

build → construct
元: They are building a new shopping mall near the station.
後: They are constructing a new shopping mall near the station.
訳: 駅の近くに新しいショッピングモールを建設中です。
📝 construct は build よりフォーマルで専門的な響きを持ち、建築・土木のニュース記事や公式な文脈でよく使われる。build は日常会話で使う最も一般的な語。
build → develop
元: She has built a successful career in the fashion industry.
後: She has developed a successful career in the fashion industry.
訳: 彼女はファッション業界で成功したキャリアを発展させてきた。
📝 develop は既にあるものが徐々に成長・向上していく過程に焦点を当てる。build は土台から一つずつ積み上げて作り上げるニュアンスが強い。
build → buy
元: If we had more money, we would build a bigger house.
後: If we had more money, we would buy a bigger house.
訳: もっとお金があれば、もっと大きな家を買うのに。
📝 buy は既存の家を「購入する」ことを意味し、build の「一から建てる」とは行為も費用構造も大きく異なる。
