report

動詞・名詞米 /rɪˈpɔːrt/ ／ 英 /rɪˈpɔːt/re·port（2音節）CEFR B1

🗣 第2音節に強勢がある（rePORT）。強勢のある母音は /ɔːr/ でrの音色を伴う長母音「オー」。第1音節の re は曖昧母音 /rɪ/ で弱く短く発音し、決して「レ」とはっきり伸ばさない

  1. 【動詞】(情報を)報告する、伝える
  2. 【動詞】(事故・犯罪などを)通報する、届け出る
  3. 【動詞】(人に)直属する、報告義務を負う（She reports to the marketing director. 彼女はマーケティング部長の直属だ。）
  4. 【名詞】報告書、レポート
  5. 【名詞】(ニュースなどの)報道、記事

語形変化

形種類発音備考
reports三単現／名詞複数形/rɪˈpɔːrts/規則変化。動詞の三単現と名詞の複数形はつづり・発音とも同じ。語末は無声音の後なので /s/（「リポーツ」であって「リポーヅ」ではない）
reported過去形・過去分詞/rɪˈpɔːrɾɪd/規則変化（-ed）。語末が /t/ の後なので /ɪd/ が加わり音節が1つ増える（「リポーティド」）。母音に挟まれた t はフラップ化しやすい
reportinging形/rɪˈpɔːrɾɪŋ/規則変化。母音に挟まれた t はフラップ化し [rɪˈpɔːrɾɪŋ] のように聞こえることがある

語法マーク

  • 動詞は他動詞用法（report + O）が基本だが、report on ~（~について報告する）、report to + 人（人に直属する／報告する）のように前置詞を伴う用法も頻出
  • report that節を伴い、伝聞・報告内容をそのまま示せる（It is reported that ~ / They reported that ~）。tell とは異なり report + 人 + that節 のSVOO型では使えない
  • 名詞としては可算名詞。a report, reports のように数えられる（不可算の news や information とは異なる）
  • report to + 人 は「(人に)直属する、報告義務を負う」という組織上の上下関係を表す重要表現
  • 他品詞での用法: 名詞用法も頻出。(1)「報告書、レポート」: I need to finish my report by Friday.（金曜までに報告書を仕上げないと）。(2)「(ニュースなどの)報道、記事」: According to a recent report, sales have increased.（最近の報道によれば、売上は伸びている）。カタカナ語「レポート」はこの名詞用法に由来するが、日本語では学校の感想文・宿題エッセイを指すことが多く、英語の report がより客観的・事実重視の文書を指す点とずれがあるので注意。

文型・頻出パターン

  • report + O〜を報告する・伝える（最も基本的な他動詞用法）
    report the results結果を報告する
    report an accident事故を通報する
  • report to + 人（人に）直属する、報告義務を負う（組織上の上下関係）
    She reports to the CEO.彼女はCEOの直属だ。
  • report on + 事柄〜について報告する・取材して伝える
    He will report on the election results tonight.彼は今夜、選挙結果について報告する。
  • It is reported that + S + V〜だと報じられている・伝えられている（伝聞を表す形式主語構文）
    It is reported that the bridge will reopen next month.その橋は来月再開通すると報じられている。
  • report + O + to + 人〜を（人）に報告する・届け出る
    report the theft to the police盗難を警察に届け出る

他のNGSL語への置き換え

report → covered
元: The accident was reported on the local news this morning.
後: The accident was covered on the local news this morning.
訳: その事故は今朝、地元のニュースで取り上げられた。
📝 cover は「(ニュースなどで)取り上げる、特集する」という意味で、report が単に事実を伝えることを指すのに対し、cover は取材・特集としてより手厚く扱うニュアンスを持つ。
report → flag
元: You should report any bugs you find in the game to the server admin.
後: You should flag any bugs you find in the game to the server admin.
訳: ゲーム内で見つけたバグは、サーバー管理者に知らせておくべきだよ。
📝 flag は「目立たせて知らせる、注意を促す」という口語的な動詞で、IT・ゲームの文脈で不具合を伝える際によく使われる。report が中立的・公式な報告動作であるのに対し、flag は「ここを見て」と注意喚起するニュアンスが強い。
report → document
元: It is important to report any errors you find in the data.
後: It is important to document any errors you find in the data.
訳: データの中に見つけた間違いは、必ず記録しておくことが大切です。
📝 document は「正式に記録に残す」という意味で、誰かに伝える report とは異なり、自分の手元やチーム内で後から参照できるよう記録を残す行為に重点がある。

詳細
⚠️ 日本人の典型誤用
✕ He reported me that he would be late.
「彼は私に遅れると報告した」と言いたかったが、tellの感覚でreportを人の前に直接置いてしまっており、文法的に誤り。ネイティブには不自然な文に聞こえる。
○ He reported to me that he would be late.
彼は私に遅れると報告した。
He reported to me that he would be late.
tell は tell + 人 + that節 のSVOO型を取れるが、report は同じ型を取れず、人を示すには前置詞 to が必要（report to + 人 + that節）。
✕ I have to do a report about the trip by Friday.
「金曜までに旅行についてレポートをやらなきゃ」と言いたかったが、do a report は英語として不自然で、文書を作る行為には write を使うのが自然。
○ I have to write a report about the trip by Friday.
金曜までに旅行についてレポートを書かなきゃいけない。
I have to write a report about the trip by Friday.
「レポートをやる」の直訳で do を選んでしまう誤り。report（文書）を作成する動作は write（または submit）と結びつくのが自然で、do a report は子供っぽく不自然に響く。
✕ I reported the police about the accident.
「私は警察に事故のことを報告した」と言いたかったが、語順・前置詞の使い方が英語の型と合っておらず不自然に聞こえる。
○ I reported the accident to the police.
私はその事故を警察に報告した。
I reported the accident to the police.
report の基本語順は report + 物 + to + 人。日本語「警察に事故を報告した」の語順につられて人を先に、物を about で後に置いてしまう誤り。
🗾 カタカナ語とのギャップ

カタカナ「レポート」は各音が均等に近いリズムだが、英語は第2音節に強勢があり /rɪˈpɔːrt/ と「ポー」の部分を強く長めに発音する（頭の re は弱く曖昧な /rɪ/）。また「レポート」は日本語では学校の感想文・宿題エッセイを指すことが多いが、英語の report はニュース報道や公式な報告書など、より客観的・事実重視の文書を指すことが多く、語感にずれがある。

🌱 Word family・語源
reporter（名詞）記者、報道する人NGSL
reportedly（副詞）伝えられるところでは、報道によると
reportage（名詞）ルポルタージュ、現地報道記事
misreport（動詞）誤って報告する、誤報する
underreport（動詞）（実際より少なく）過小に報告する
語源: 古フランス語 reporter を経て、ラテン語 reportare「持ち帰る、報告する」に由来。re-（戻す・再び）+ portare（運ぶ）が原義で、「情報を持ち帰って伝える」という成り立ちを持つ。
reporter（記者、報道する人）— report + -er（〜する人）で「報告する人」
reportedly（伝えられるところでは）— reported + -ly（副詞化）で「報告された形で＝伝聞によれば」
misreport（誤って報告する）— mis-（誤って）+ report で「誤って運び伝える」が原義
💡 覚え方: ラテン語 portare「運ぶ」が語源。re-（戻す・再び）+ port（運ぶ）で「情報を持ち帰って伝える」が report の原義（transport「向こうへ運ぶ」、import「中へ運び入れる」も同じ portare 一族）。reporter は「情報を運んでくる人＝記者」、reportedly は「運ばれてきた情報によれば」、misreport は「運び方を誤る＝誤報する」——すべて『運ぶ』イメージでつながる。
🔗 句動詞
report back — （任務・結果を持って）戻って報告する
Please report back to me as soon as you finish the inspection.（点検が終わったらすぐに私に報告してください。）
report in — （出勤・到着などを）報告する、連絡を入れる
All officers must report in at the start of their shift.（全ての警官は勤務開始時に報告を入れなければならない。）
⚖️ 類義語の使い分け
announce — announce は公式に大勢へ向けて発表するときに使う。report は事実をありのまま伝える「報告」に重点があり、必ずしも公表・発表の意味を含まない。
inform — inform は「（人に）知らせる」ことが中心で、目的語は必ず人（inform + 人）。report は「（出来事・事実）を伝える」ことに焦点があり、対象は情報そのもの。
notify — notify は公式・事務的な通知を表し、法的・制度的な文脈でよく使われる（notify the authorities）。report はより一般的な報告行為全般を指す、幅広い語。
💬 熟語・慣用表現 (3)
report card 成績表、通知表；（比喩的に）実績・成果の評価中立
Her report card showed straight A's this semester.（今学期、彼女の成績表はオールAだった。）
🔍 「報告のカード」という直訳では意味が伝わらない、学校が成績を保護者に伝えるための決まった様式の文書を指す固定表現。転じて、人や組織・政策の実績を評価する文脈でも比喩的に使われる。
📝 アメリカ英語で特に一般的。give someone a report card の形で「（人や組織の）実績に評価を下す」という比喩表現にもなる（例: The magazine gave the new mayor's first year a mixed report card. その雑誌は新市長の初年度の実績に賛否両論の評価を下した）。
on report （主に英・学校や軍隊などで）素行不良のため要注意人物として記録・監視されている口語（主にイギリス英語）
If you get into another fight, you'll be put on report.（もしまた喧嘩をしたら、要注意人物として記録されてしまうよ。）
🔍 「報告の上に」ではなく、素行不良のためたびたび行動が記録・報告される状態を指すイギリス英語の口語的な決まり文句。
📝 主に学校や軍隊などの組織内で使われ、put/be on report の形が典型。アメリカ英語ではあまり使われない。
report for duty 任務に就くために出頭する、着任の報告をするフォーマル／軍隊・組織用語
The new recruits must report for duty at six o'clock sharp tomorrow morning.（新兵たちは明朝6時ちょうどに出頭しなければならない。）
🔍 「任務のことを報告する」ではなく、軍隊や新しい職場で「到着し、任務・勤務を始める準備ができたことを申し出る」という決まり文句。report が「情報を伝える」ではなく「出頭する」という意味に転じる点が丸暗記ポイント。
📝 主に軍隊・警察や新しい職場の初日などで使われる。「Reporting for duty, sir!」のように敬礼を伴う決まり文句としても使われる。単に「出社する」だけなら arrive at work や show up for work の方が一般的。
