really

副詞 (adverb)米 /ˈriːəli/（口語では2音節に縮約されて /ˈrɪli/ となることが多い） ／ 英 /ˈrɪəli/re·al·ly（3音節）CEFR A1

🗣 第1音節に強勢（RE-al-ly）。丁寧な発音では /ˈriːəli/ と3音節だが、日常会話ではeal部分が縮まって2音節の /ˈrɪli/（sillyやhillyと同じ響き）になることが非常に多い。日本人は「リ・ア・リ・ー」と均等な4拍で発音しがちだが、強いのは冒頭のREだけで、後ろは弱く速く流すのがコツ

  1. 【副詞】本当に、とても（形容詞・副詞・動詞を強めて）
  2. 【副詞】実際に、実のところ（事実・現実を述べて）
  3. 【副詞】（否定文で）それほど、あまり（〜ない。例: I don't really know. よく分からない）
  4. 【副詞】本当に？、マジで？（相手の発言への驚き・確認の相づち。Really? だけで一文として使う）

語法マーク

  • very と異なり、really は動詞を直接修飾できる（I really like it. は可。× I very like it. は不可）。動詞を強めたいとき really は最も汎用的な強意語
  • 語順の注意: really は修飾する語（動詞・形容詞・副詞）の直前に置く。目的語の後ろに置いて I like really this song. のようにしない（→ I really like this song.）
  • 否定文での位置が意味の強さを変える: don't really + 動詞は「あまり〜ない」という婉曲的な部分否定（頻出）。really don't + 動詞は「本当に〜ない」という強い直接否定。日本語の「あまり」「全然」の使い分けに近く、位置を逆にすると伝わる強さが変わる
  • 比較変化を持たない絶対的な強意語（more really や reallier のような形は存在しない）
  • Really? / Not really. の形で単独の1語文として会話の相づちに使われる。この用法では動詞を修飾しているのではなく、相手の発言全体に対する反応を表す
  • 他品詞での用法: 副詞のみ。ただし単独で1語文としても使われ、(1) Really? だけで「本当に?／マジで?」と相手の発言への驚き・確認の相づちになる。(2) Not really. だけで「そうでもない／あまり」という控えめな否定の返答になる。この2つの単独用法は普通の様態副詞にはない、談話標識（discourse marker）的な働き。

文型・頻出パターン

  • really + 形容詞／副詞程度を強める最も基本的な用法。「本当に〜、とても〜」
    really good本当にいい
    really quickly本当に速く
  • really + 動詞動詞そのものを強める。very と違いこの用法が可能な点が really の大きな特徴
    I really enjoy this song.この曲、本当に楽しんでる。
    She really wants to go.彼女は本当に行きたがっている。
  • don't really + 動詞「あまり〜ない」という婉曲的な部分否定。会話で非常に頻出
    I don't really like spicy food.辛い食べ物はあまり好きじゃない。
  • really don't + 動詞「本当に〜ない」という強い直接否定。don't really よりも否定の度合いが強い
    I really don't want to go there again.そこにはもう本当に行きたくない。
  • Really?単独で「本当に?／マジで?」と驚き・確認を表す相づち
    Really? I had no idea.本当に? 全然知らなかった。
  • Not really.単独で「そうでもない、あまり」という控えめな否定の返答
    "Are you tired?" "Not really."「疲れた?」「そうでもないよ。」

他のNGSL語への置き換え

really → truly
元: I've really enjoyed all the food at this restaurant.
後: I've truly enjoyed all the food at this restaurant.
訳: このレストランの料理は心から気に入りました。
📝 truly はやや改まった書き言葉的な響きで、深く誠実な気持ちを表す。really は日常会話で最も頻繁に使われる中立的な強調語で、話し言葉ではほぼ常にこちらが好まれる。
really → actually
元: If you really wanted this job, you would prepare more for the interview.
後: If you actually wanted this job, you would prepare more for the interview.
訳: 本当にこの仕事が欲しいのだとしたら、面接にもっと準備するはずだ。
📝 actually は「見かけとは違って実は」という対比・疑念のニュアンスを含みやすい（本当は欲しくないのでは、という含み）。really は単なる強調で、そうした疑いの含みは薄い。
really → quite
元: I don't really understand why he changed his mind so suddenly.
後: I don't quite understand why he changed his mind so suddenly.
訳: 彼がなぜ突然気持ちを変えたのか、私にはいまひとつ分からない。
📝 don't quite は「あと少しで分かりそうだが完全ではない」という部分的な理解を示す。don't really はそれより理解の程度が低く「あまり分からない・大して分からない」に近い。

⚠️ 日本人の典型誤用
✕ I like really this movie.
「この映画、本当に好き」と言いたかったが、really を目的語の後ろに置く語順は英語として不自然で、ネイティブには片言のように聞こえてしまう。
○ I really like this movie.
この映画、本当に好き。
I really like this movie.
really は修飾する動詞の直前（主語のすぐ後ろ、一般動詞の前）に置く。目的語の後ろに置いてしまうのは日本語の「この映画、本当に好き」の語順に引きずられた誤り。
✕ This cake is very really delicious.
「このケーキ、すごくとても美味しい」のつもりで強意語を2つ重ねてしまったが、英語では very と really を並べて使うことはなく、不自然な誤りに聞こえる。
○ This cake is really delicious.
このケーキ、本当に美味しい。
This cake is really delicious.
very と really はどちらも「とても」に当たる強意語で、2つ重ねて使うことはできない。強めたいときはreally単独、あるいはreally very deliciousではなくreally, really delicious のように really自体を繰り返すのが自然。
✕ I don't really know. （本当は「全く分からない」と強く言いたかった場合）
「（本当に）全然分からない」と強く言いたかったが、don't really を使うと「あまりよく分からない」という控えめな響きになり、伝えたかった強さが弱まって伝わってしまう。
○ I really don't know.
本当に（全く）分からない。
I really don't know.
don't really（あまり〜ない）と really don't（本当に〜ない）は位置が違うだけで否定の強さが逆転する。「全然分からない」という強い否定を言いたいのに don't really を使うと「あまりよく分からない」という弱いニュアンスに変わってしまう。
🗾 カタカナ語とのギャップ

カタカナ「リアリー」は「リ・ア・リ・ー」と4拍均等に読みがちだが、英語は強勢が冒頭のREにしかなく、丁寧な発音でも3音節、口語では2音節 /ˈrɪli/（sillyと同じ響き）まで縮む。また綴りが似ている rarely（レアリー、「めったに〜ない」）と混同しないよう注意。母音を伸ばして「リーアリー」と読むと不自然に間延びして聞こえる。

🌱 Word family・語源
real（形容詞）本当の、現実のNGSL
reality（名詞）現実、実態NGSL
realize（動詞）気づく、実現するNGSL
unreal（形容詞）現実離れした（口語では「信じられないほどすごい」の意味でも使う）
realistic（形容詞）現実的な、写実的な
語源: ラテン語 res「もの、実体」から後期ラテン語 realis「事物に関わる、実在の」が生まれ、古フランス語を経て中英語 real「本当の、現実の」に。そこに古英語由来の副詞化接尾辞 -ly（もとは「〜の形をした」の意）が付いて really「現実に即して＝本当に」という副詞ができた。
real（本当の、現実の）— really の元になった形容詞そのもの
reality（現実）— real + -ity（名詞化）＝「現実であること」
realize（気づく、実現する）— real + -ize（〜化する）＝「現実のものにする」→「実現する」「はっと現実に気づく」
💡 覚え方: core は res（もの・実体）。real は「実体として存在する＝本当の」、reality は「実体としてあること＝現実」、realize は「実体化する＝実現する／現実だと気づく」、そして really はそこに -ly が付いて「実体に即して言うと＝本当に」。全部「もの・実体」のイメージでつながる。
⚖️ 類義語の使い分け
truly — truly はやや改まった書き言葉的な響きで、深く誠実な気持ちを表す。really は日常会話で最も頻繁に使われる中立的な強調語で、動詞も直接修飾できる汎用性が高い。
actually — actually は「見かけ・予想とは違って実は」という対比・訂正のニュアンスを含む（It's actually good.＝意外にもいい）。really は単純な程度の強調で、そうした意外性の含みは薄い。
genuinely — genuinely は「見せかけでなく本心から」という誠実さ・純粋さを強調する、ややフォーマルな語。really よりも感情・態度の真正さに焦点がある。
indeed — indeed は前の発言や形容詞を後から強めて肯定する、ややフォーマルな語（very good indeed）。really のように動詞の前に置いて単独で使う口語的な強調とは使われる位置・文体が異なる。
💬 熟語・慣用表現 (2)
really and truly 本当に、正真正銘に（誓って本当だと念を押す）口語（やや古風）
I really and truly didn't know she was coming to the party.（彼女がパーティーに来るなんて、本当に本当に知らなかったんです。）
🔍 really（本当に）と truly（真実に）を重ねた強意句。ほぼ同義の2語をセットにして「本当に本当に」と誓うように強調する決まり文句で、単なる really の繰り返しよりも定型化されている。
📝 驚いた相手への聞き返しとして "Really and truly?"（本当に本当?）という形でも使われる。単独の really だけでは足りないほど強く真実性を訴えたいときの定型表現。
well, really! まったく、もう!（呆れ・憤慨を表す間投詞）イギリス英語寄り／やや古風
Well, really! I've never been spoken to like that before.（まったくもう! こんな口の利き方をされたのは初めてだわ。）
🔍 文字通りは「まあ、本当に」だが、実際には相手の失礼な言動や驚くような出来事に対して「呆れた」「なんて失礼な」と非難・憤慨を表す決まり文句。相づちの Really?（本当に?という驚き）とは異なり、不満・非難のニュアンスが前面に出る点が予測しにくい。
📝 主にイギリス英語で使われる、やや古風な言い回し。失礼な発言をされた後などに、憤慨や呆れを上品に表す際に使う。若い世代のくだけた会話ではあまり使われない。
