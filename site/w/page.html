page

名詞・動詞米 /peɪdʒ/ ／ 英 /peɪdʒ/page（1音節）CEFR

🗣 1音節語なので強勢の位置に迷う余地はない。母音は二重母音 /eɪ/ で「エ」から「イ」へなめらかに滑らせて1拍にする。語末は有声の破擦音 /dʒ/ で、cage・stage・age と同じ音。無声の /tʃ/ で終わる watch などと混同しないよう、喉の振動（有声）を意識する

  1. 【名詞】(本・雑誌・ウェブサイトなどの)ページ（Please turn to page ten. 10ページを開いてください。）
  2. 【動詞】(館内放送・ポケベルなどで)人を呼び出す（Could you page Dr. Smith to the front desk? スミス先生をフロントまで呼び出してもらえますか?）

語法マーク

  • 名詞は可算名詞。単数はa page、複数はpages
  • 「page + 数字」は無冠詞のラベル的用法（turn to page ten / on page five）。room 10、chapter 3 と同じ「番号で指し示す」パターンで、the page ten のようにtheを入れない
  • 動詞pageは他動詞。呼び出す相手を必ず目的語に取る（× Please page. のような単独使用は不可）
  • page through ~ は自動詞的な句動詞で「(本などを)ぱらぱらめくる、ざっと目を通す」を表す
  • 実務的な文脈では受動態 be paged（呼び出される）が頻出する
  • 他品詞での用法: 名詞にはさらに、中世の貴族の館や城で仕えた「小姓・侍童」（貴人に仕え、将来騎士を目指して修行する少年）という古い意味がある。現代でも結婚式で花嫁のドレスの裾を持つ男の子を page boy と呼ぶなど、この意味は儀礼的な場面に残っている。動詞の「(館内放送・ポケベルなどで)人を呼び出す」は、まさにこの小姓が来客の名を呼び上げて取り次いだ役目に由来する用法で、病院・空港・ホテルなど実務的な施設内呼び出しで使われる。また page-turner（ページをめくる手が止まらないほど面白い本）、webpage／homepage（インターネット関連の複合語）のように、新しい複合語を生み出す生産性の高い語根でもある。

文型・頻出パターン

  • page + 数字（無冠詞）ページ番号を無冠詞のラベルとして示す
    Please turn to page five.5ページを開いてください。
    The answer is on page twelve.答えは12ページにあります。
  • page + 人（他動詞）(人)を呼び出す
    Could you page the manager?マネージャーを呼び出してもらえますか?
  • be paged（受動態）(館内放送・ポケベルなどで)呼び出される
    He was paged to the operating room.彼は手術室へ呼び出された。
  • page through + 物~をぱらぱらめくる、ざっと目を通す
    She paged through the catalog.彼女はカタログをぱらぱらめくった。

他のNGSL語への置き換え

page → called
元: The doctor was paged twice during the meeting because of an emergency.
後: The doctor was called twice during the meeting because of an emergency.
訳: 緊急事態のため、会議中にその医師は2回電話で呼ばれた。
📝 page は館内放送・ポケベルなど特定の手段で呼び出すことを指す専門的な表現。call はより一般的な「呼ぶ・電話する」で、呼び出し手段を限定しない中立的な語。
page → chapters
元: I've already read the first hundred pages of this novel.
後: I've already read the first hundred chapters of this novel.
訳: この小説はもう最初の100章を読み終えた。
📝 page(ページ)は紙面上の物理的な単位で読了量の目安になるのに対し、chapter(章)は内容構成上の単位。100 chapters となると小説自体が非常に長編であることを示唆する。
page → words
元: This report has twice as many pages as the one I wrote last year.
後: This report has twice as many words as the one I wrote last year.
訳: この報告書は、去年私が書いたものの2倍の語数がある。
📝 pages はレイアウトやフォントサイズに左右される見た目の分量、words は正確な語数を示す厳密な尺度。分量比較の精密さが異なる。

⚠️ 日本人の典型誤用
✕ Please look at the page of ten.
「10ページを見てください」と言いたかったが、the page of tenという言い方は不自然で、まるで「10という数のページ」を遠回しに指しているように聞こえてしまう。
○ Please look at page ten.
10ページを見てください。
Please look at page ten.
「page + 数字」は無冠詞・ofなしのラベル的表現として使う。日本語「10ページ」の「の」につられてofを挿入したり、theを付けたりしてしまう誤り。
✕ Please page to Dr. Smith.
「スミス先生を呼び出してください」と言いたかったが、page to ~ という言い方は英語として不自然で、余計な前置詞のせいでぎこちなく響く。
○ Please page Dr. Smith.
スミス先生を呼び出してください。
Please page Dr. Smith.
動詞pageは他動詞で前置詞を伴わない。call/speak to のような前置詞を伴う類義表現のイメージにつられてtoを入れてしまう誤り。
✕ I read that in page five.
「それは5ページで読んだ」と言いたかったが、in page fiveという前置詞の使い方は不自然で、ネイティブには違和感を与える。
○ I read that on page five.
それは5ページで読んだ。
I read that on page five.
「~ページに」は前置詞onを使う（inではない）。紙面という「表面」に情報が載っているイメージなので、page・screen・listなど平面的なものには一貫してonを使う。
🗾 カタカナ語とのギャップ

カタカナ「ページ」は「ペ・ー・ジ」の3モーラだが、英語は1音節/peɪdʒ/。母音は二重母音/eɪ/として1拍でつなげ、語末の/dʒ/にはカタカナの「ジ」のような母音を付けない（喉の振動だけで音を止める）。また複数形pagesをカタカナ感覚で「ペイジス」と無声のsで読むと誤り。語幹が有声の/dʒ/で終わるため、正しくは有声の/z/を含む2音節「ペイジェズ」に近い音になる。

🌱 Word family・語源
pager（名詞）ポケットベル、呼び出し用受信機
webpage（名詞）ウェブページ
homepage（名詞）ホームページ、(サイトの)トップページ
page-turner（名詞）(夢中で読んでしまうほど)面白い本
paginate（動詞）(文書に)ページ番号を振る
語源: 現代英語のpageには由来の異なる二つの語源が合流している。(1) 「紙面のページ」の意味は古フランス語page、さらにラテン語pagina（パピルスの一欄、綴じられた紙葉）に遡る。paginaはpangere「打ち込む、固定する」に由来し、もとはブドウ畑の畝のように区切られた列や、綴じ合わされた紙葉を指した。(2) 「小姓・侍童」および動詞「呼び出す」の意味は、別の古フランス語page（若い従者、小姓）に遡り、さらにイタリア語paggio、あるいはギリシャ語paidion（少年）にまで遡るとされる（諸説あり）。中世の館では小姓が来客の名を呼び上げて取り次いだため、そこから20世紀に「(館内放送・ポケベルで)人を呼び出す」という動詞用法が生まれた。
paginate（ページ番号を振る）— 英語pageと同じくラテン語paginaを祖先に持つが、フランス語経由で学術的な語として独立に英語に入った派生語
pager（ポケットベル）— 動詞page「呼び出す」に道具を表す接尾辞-erを付けた派生語。1980〜90年代に普及した呼び出し専用の携帯受信機を指す
💡 覚え方: pageには全く別の起源を持つ二つの顔がある。紙のpage(ページ)はラテン語pagina「綴じられた紙葉」から、動詞のpage(呼び出す)は昔の貴族の館で客の名を呼び上げていた「小姓(page)」から。ページをめくる(page through)のと、人を呼び出す(page someone)のとで、頭の中の「page」を2枚に分けてイメージすると混同しにくい。

⚖️ 類義語の使い分け
leaf — leafは本の「1枚の紙葉」を指し、表裏で2ページ(page)分にあたる。日常会話ではpageの方が圧倒的に使用頻度が高く、leafはやや文語的・専門的な表現(古書や聖書の記述など)にとどまる。
sheet — sheetは綴じられていない1枚の紙全般を指す。本や雑誌に綴じられた「ページ」を指すときは必ずpageを使う。
call — (動詞)callは手段を問わない一般的な「呼ぶ」。pageは館内放送やポケベルなど特定の呼び出しシステムを介して呼ぶ、という限定されたニュアンスが強い。
summon — (動詞)summonはより権威的・公式な「呼び出す、召喚する」(法廷や上司からの呼び出しなど)。pageは病院・空港・ホテルなど施設内の実務的な呼び出しに使われることが多い。
💬 熟語・慣用表現 (2)
on the same page 意見が一致している、同じ理解を共有しているビジネス／中立
Let's make sure everyone is on the same page before we start the project.（プロジェクトを始める前に、全員の認識が一致しているか確認しましょう。）
🔍 「同じページを開いている」状態から転じて、全員が同じ情報・認識・方針を共有していることを表す比喩。文字通り本のページを指す場面ではなく、会議やチームの意思疎通の文脈で使われる決まり文句。
📝 be on the same page as A のように比較対象を明示することもある。認識がずれている時は not on the same page、足並みを揃える動きは get everyone on the same page と言う。
take a page from someone's book 〜を手本にする、〜のやり方を見習う口語
You should take a page from her book and start saving money every month.（彼女を見習って、毎月お金を貯め始めた方がいいよ。）
🔍 「誰かの本から1ページを取る」という比喩から、その人物の方法や成功例を参考にして真似ることを表す。
📝 take a page out of someone's book とも言う。イギリス英語では pageの代わりにtake a leaf out of someone's book という異形も使われる（leafはpageと同義の古い語）。
