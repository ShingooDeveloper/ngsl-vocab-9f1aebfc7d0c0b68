this

限定詞・代名詞米 /ðɪs/ ／ 英 /ðɪs/this（1音節）CEFR A1

🗣 1音節語。語頭の th は有声の歯摩擦音 /ð/（舌先を上下の歯の間に軽く挟んで震わせる）で、無声の th（think の /θ/）と混同しないこと。母音は短く緩んだ /ɪ/ で、「ディース」のように伸ばさない。機能語として使われるときは無強勢で速く発音される

  1. 【限定詞】この、この～（すぐ近くにある物・今話題にしている事柄を指す）（This book is mine. この本は私のです。）
  2. 【限定詞】（時間を表す語の前で）今度の、今の（this morning, this weekなどの形で使う）（I have a meeting this afternoon. 今日の午後、会議がある。）
  3. 【代名詞】これ、この人・こと（近くの物や、今言及したばかりの事柄を指す）（This is my sister. こちらは私の妹です。）
  4. 【代名詞】（これから言うことを指して）これ、こういうこと（Listen to this. ちょっとこれを聞いて。）

語形変化

形種類発音備考
these複数形/ðiːz/this の不規則な複数形。指す名詞が複数のとき this→these、be動詞も is→are に変わる（this book is / these books are）。母音が短い /ɪ/ から長い /iː/ に変わり、綴りも this→these と変化する点に注意

語法マーク

  • 単数専用の指示語。複数の名詞・事柄を指すときは複数形 these を使い、be動詞も合わせて変える（this problem is / these problems are）
  • 物理的に話し手の近くにある物だけでなく、現在進行中の話題やこれから述べる内容を指すときにも使う（前方照応・後方照応の両方が可能）
  • 電話の決まり文句では対面時の近遠の感覚と異なり、話し手・相手のどちらを指す場合も this を使う（Hi, this is Tom speaking. / Who is this?）
  • 限定詞として名詞の前に置く用法（this book）と、名詞なしで単独の代名詞として主語・目的語になる用法（This is mine. / I meant this.）の両方がある
  • 他品詞での用法: 本来は限定詞・代名詞のみだが、口語では形容詞・副詞の前に置いて程度を表す副詞としても使われる（「こんなに〜」）: I didn't think it would be this difficult.（こんなに難しいとは思わなかった）/ We can't stay out this late.（こんなに遅くまで外にはいられない）。that の副詞用法（not that difficult）と対になる口語表現。

文型・頻出パターン

  • This is + 名前（電話・対面での自己紹介）電話や対面で自分や居合わせた人を「こちらは〜です」と紹介する定型表現
    Hi, this is Yuki.もしもし、ユキです。
    This is my colleague, Mr. Sato.こちらは同僚の佐藤です。
  • this + 単数名詞 ⇔ these + 複数名詞指す名詞の数に応じて this と these を使い分ける（be動詞の数も一致させる）
    This idea sounds interesting.この考えは面白そうだね。
    These ideas sound interesting.これらの考えは面白そうだね。
  • like thisこのように（やり方・様子を示す）
    Hold the knife like this.ナイフはこう持って。
  • this + 形容詞・副詞（程度を表す口語の副詞用法）こんなに〜、これほど〜（口語で程度・範囲を強調する）
    I didn't expect it to be this hot in June.6月にこんなに暑くなるとは思わなかった。
    We can't stay out this late.こんなに遅くまで外にいられない。
  • This is + what / how節内容・仕組みを指して「これが〜だ」と強調して導入する頻出構文
    This is what I mean.これが私の言いたいことだ。
    This is how it works.これがその仕組みだ。

他のNGSL語への置き換え

this → that
元: This house was built more than a hundred years ago.
後: That house was built more than a hundred years ago.
訳: あの家は100年以上前に建てられたんだよ。
📝 this は話し手に近い、または今まさに話題にしているものを指し、that は話し手から離れた、または既に話の中心から外れたものを指す。this/that の基本的な対比。
this → these
元: Please sign this form before you leave the office.
後: Please sign these forms before you leave the office.
訳: オフィスを出る前に、これらの用紙にサインしておいてください。
📝 this の複数形は these。名詞も form（単数）から forms（複数）に変わり、限定詞と名詞の数が一致する点に注意。
this → that
元: This is why I decided to quit my job last year.
後: That is why I decided to quit my job last year.
訳: それが、去年私が仕事を辞めることに決めた理由だったんだ。
📝 this is why はこれから理由を詳しく説明する、または直前の内容を臨場感を持って指す響き。that is why は既に述べた事柄を少し距離を置いてまとめて指す響きで、ほぼ同義だがニュアンスがわずかに異なる。

頻出コロケーション

  • this is why + S + V（こういうわけで〜だ、〜する理由はこれだ）
  • in this way（このようにして、こうすることで）
  • this time（今回は、今度こそ）

⚠️ 日本人の典型誤用
✕ This are my new shoes.
「これは私の新しい靴です」と言いたかったが、this のまま複数名詞に続けると数が矛盾した文になり、文法ミスとしてすぐに気づかれてしまう。
○ These are my new shoes.
これらは私の新しい靴です。
These are my new shoes.
this の複数形は these で、be動詞も複数の名詞に合わせて are にする必要がある。単数の this のまま複数名詞 shoes につなげてしまうと数が一致しない誤りになる。
✕ Hello, here is Tanaka speaking.
電話で「こちら田中です」と言いたかったが、here is Tanaka speaking は英語として不自然で、ネイティブには聞き慣れない言い方に響く。
○ Hello, this is Tanaka speaking.
もしもし、田中です。
Hello, this is Tanaka speaking.
電話で名乗るときの「こちら〜です」を here is と直訳してしまう誤り。英語の電話の決まり文句は this is + 名前 + speaking で、here is は使わない。
✕ Hello, who is that?
電話で「どちら様ですか」と尋ねたかったが、that を使うと英語としては不自然に聞こえ、ネイティブは違和感を覚える。
○ Hello, who is this?
もしもし、どちら様ですか。
Hello, who is this?
対面での近遠の感覚（近い→this、遠い→that）を電話にもそのまま当てはめてしまう誤り。電話では相手が物理的に離れていても、電話の向こうの相手を指すのに this を使うのが慣習で、Who is this?（どちら様ですか）が定型。
🗾 カタカナ語とのギャップ

カタカナ「ディス」の「デ」は破裂音の d で置き換えているが、実際の th は舌先を上下の歯の間に軽く挟んで震わせる有声摩擦音 /ð/ であり、d とは調音の仕方が異なる。またカタカナの「イ」は長さが曖昧になりがちだが、this の母音は短く緩んだ /ɪ/ で、「ディース」のように伸ばすと別の語のように聞こえる。無声の th（think, thing など）と混同しないことも重要。

🌱 Word family・語源
these（限定詞・代名詞）これら、これらの（this の複数形）NGSL
that（限定詞・代名詞）あの、あれ（this と対をなし、話し手から離れたものを指す）NGSL
those（限定詞・代名詞）あれら、あれらの（that の複数形）NGSL
thus（副詞）このように、それゆえに（this と同語源のフォーマルな副詞）NGSL
語源: 古英語の指示代名詞 þes（男性形）・þēos（女性形）・þis（中性形）に由来。ゲルマン祖語の指示詞語根 *þa-（あれ・それ）に近さを表す接尾辞 -si が付いて作られた「近くのもの」を指す形。that・those・thus・the・then など th- で始まる指示・関係の語群はすべて同じ語根を共有する。
that（あれ・その）— 同じ指示詞語根から分かれた「遠いもの」を指す対の形
these（これら）— 中英語期に this + 複数語尾が結びついて作られた複数形
thus（このように）— 同じ th- 語根から生まれた副詞形。「この（やり方）で」が原義
💡 覚え方: this・that・these・those・thus・then・there はすべて古いゲルマン語の指示語根 th- の一族。th で始まったら「何かを指し示す・つなぐ」語だと考えると覚えやすい。
⚖️ 類義語の使い分け
that — 距離感の対比。this は話し手に近いもの・現在進行中の話題を指し、that は話し手から離れたもの・既に一歩引いて扱う話題を指す。
it — it は既に話題に出たものを無強勢で淡々と受け直すだけで、指し示す力を持たない。this は近くのもの・今まさに注目させたいものを強く指し示す力を持つ点が異なる。
💬 熟語・慣用表現 (5)
at this rate この調子だと、このままでは口語
At this rate, we'll never finish the project on time.（この調子だと、期限内にプロジェクトを終えられそうにない。）
🔍 「この割合で」が直訳だが、現在の状況やペースがこのまま続けばどうなるかを予測する口語表現。良くない結果を暗に予想する文脈でよく使われる。
📝 望ましくない結果を予測して警告・忠告するときに文頭で使うことが多い。良い結果の予測にも使えなくはないが、否定的な文脈が圧倒的に多い。
this and that あれこれ、いろいろなこと口語
We just talked about this and that over coffee.（コーヒーを飲みながら、あれこれと雑談したよ。）
🔍 「これとあれ」から、特定できない・重要でない雑多な事柄をまとめて指す口語表現。
📝 内容をぼかして軽く述べるときの決まり文句。this, that and the other という三つ並べた拡張形もある。
in this day and age 今日日、今のこの時代では中立
In this day and age, almost everyone has a smartphone.（今日日、ほとんどの人がスマートフォンを持っている。）
🔍 「この日、この時代において」という強調的な言い回しで、現代社会の状況を際立たせて述べるときに使う。
📝 現代の常識や変化を、驚きや皮肉を込めて強調する場面でよく使われる。nowadays よりやや大げさで会話的な響き。
this is it いよいよだ、これが本番だ、これで最後だ口語
This is it—the moment we've been waiting for.（いよいよだ。ずっと待っていた瞬間が来た。）
🔍 「これがそれだ」という直訳だが、待ち望んだ、あるいは恐れていた瞬間がついに来たことを告げる決まり文句。
📝 緊張感の高まる場面での決め台詞として使われる。文脈によっては「これで終わりだ」と結末・限界を告げる意味にもなる。
just this once 今回だけは、これっきり口語
Fine, you can stay up late—just this once.（分かった、今回だけは夜更かししてもいいよ。）
🔍 「まさにこの一回」という強調表現から、例外的に今回だけ特別に認めるという許可・妥協のニュアンスを表す。
📝 普段は許さないことを例外として認める場面で使う。「次はないよ」という警告を暗に伴うことが多い。
