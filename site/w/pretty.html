pretty

形容詞・副詞米 /ˈprɪɾi/ ／ 英 /ˈprɪti/pret·ty（2音節）CEFR A1

🗣 第1音節に強勢（ˈpret·ty）。母音は短母音 /ɪ/。米音では2つのtが母音に挟まれフラップ化し [ɾ] になるため、綴りの「ティ」通りではなく「リ」に近い音（プリティではなくプリリ）

  1. 【形容詞】（見た目が）かわいい、きれいな（女性・子供・物などに使う、繊細で感じの良い美しさ）（She wore a pretty summer dress to the party. 彼女はパーティーにかわいい夏用のワンピースを着ていった。）
  2. 【形容詞】（皮肉的に）ひどい、とんでもない（a pretty mess のように）（You've made a pretty mess of the kitchen while baking those cookies. クッキーを焼いている間に、台所をひどい有様にしてくれたね。）
  3. 【副詞】（口語の強意語で）かなり、まあまあ（pretty good, pretty much など）（The movie was pretty good, though a bit long. その映画はまあまあ良かったけど、少し長かった。）

語形変化

形種類発音備考
prettier比較級/ˈprɪɾiɚ/-y で終わる2音節形容詞のルール通り y→i+er。more pretty は誤り
prettiest最上級/ˈprɪɾiɪst/同じルールで y→i+est。most pretty は誤り
pretties三人称単数現在形（動詞用法）/ˈprɪɾiz/動詞 pretty（くだけた語、通例 pretty up の形）の三単現。語末は有声音なので /z/
prettyinging形（動詞用法）/ˈprɪɾiɪŋ/yはそのまま残し -ing を付ける（prettiing にはしない）

語法マーク

  • 形容詞としては限定用法（a pretty girl）・叙述用法（She looks pretty.）のどちらも可能
  • 比較変化は -y で終わる2音節形容詞のルールに従い prettier / prettiest（more pretty / most pretty は誤り）
  • 副詞の pretty（「かなり」の意の強意語）には比較変化が無い。形容詞・副詞の前にのみ置け、動詞を直接修飾できない（× I pretty like it. は誤り）
  • 皮肉的に「ひどい、とんでもない」を表す用法は必ず a pretty + 名詞 の限定用法のみで使われ、叙述用法（It is pretty.の意で「ひどい」の意味）にはならない
  • 動詞としての pretty はまれで、通例 pretty up の形の句動詞として他動詞的に、または再帰的に pretty oneself up として使われる
  • 他品詞での用法: まれに動詞としても使われる（くだけた言い方、通例 pretty up の形）。意味は「〜をきれいに、見栄え良く整える」: They spent the weekend prettying up the garden before the guests arrived.（客が来る前に週末をかけて庭をきれいに整えた。）この動詞用法は日常会話や軽い文体に限られ、フォーマルな文章では prettify や decorate が好まれる。

文型・頻出パターン

  • pretty + 形容詞／副詞（強意副詞）かなり、まあまあ（口語で程度を強める）

  • a pretty + 名詞（皮肉的用法）「ひどい〜」「とんでもない〜」（皮肉を込めて悪い状態を言う）
    a pretty messひどい有様
    a pretty pickle厄介な状況
  • pretty up（句動詞・くだけた表現）〜をきれいに見せる、飾って整える
    pretty up the yard庭をきれいに整える
    pretty oneself up身なりを整える、おめかしする
  • look／be／seem + pretty見た目がかわいい、きれいだ（連結動詞＋形容詞）
    look prettyきれいに見える
    seem prettyきれいなようだ

他のNGSL語への置き換え

pretty → beautiful
元: The town looks especially pretty when it's covered in snow.
後: The town looks especially beautiful when it's covered in snow.
訳: 雪に覆われると、その町はとりわけ美しく見える。
📝 pretty は繊細で愛らしい、比較的小規模な美しさを表すのに対し、beautiful はより壮大で強い美しさを表すフォーマルな語。雪に覆われた町全体のような雄大な情景には beautiful の方がやや自然に響くこともある。
pretty → fairly
元: I'm pretty sure it's going to rain later, so you should bring an umbrella.
後: I'm fairly sure it's going to rain later, so you should bring an umbrella.
訳: 後で雨が降るとかなり確信しているから、傘を持って行った方がいいよ。
📝 pretty は口語的でカジュアルな強意語なのに対し、fairly はよりフォーマルで書き言葉にも使われる。日常会話では pretty の方が圧倒的に自然に聞こえる。
pretty → nicer
元: If you put some fresh flowers on the table, the room would look a lot prettier.
後: If you put some fresh flowers on the table, the room would look a lot nicer.
訳: テーブルに生花を飾れば、部屋はずっと素敵に見えるだろう。
📝 prettier は見た目の可愛らしさ・華やかさに焦点を当てるのに対し、nicer は見た目に限らず全体的な感じの良さを表す、より広い意味の語。

⚠️ 日本人の典型誤用
✕ This bag is more pretty than that one.
「このバッグはあのバッグよりかわいい」と言いたかったが、more pretty はネイティブには回りくどく、教養の乏しい響きに聞こえてしまう。
○ This bag is prettier than that one.
このバッグはあのバッグよりかわいい。
This bag is prettier than that one.
-y で終わる2音節形容詞は比較級で -ier を取る例外グループに属す（happy→happier、easy→easierと同様）。more pretty は英語として不自然。
✕ He is very pretty and strong.
「彼はとてもかっこよくて力強い」と言いたかったが、男性に pretty を使うと女性的な響きになり、意図しないニュアンスで伝わってしまう。
○ He is very handsome and strong.
彼はとてもハンサムで力強い。
He is very handsome and strong.
pretty は伝統的に女性・子供・繊細な物の見た目に使う語で、男性の外見には通常使わない（意図的な皮肉や特定の文脈を除く）。男性には handsome が自然。
✕ I pretty like this song.
「この曲、かなり好き」と言いたかったが、pretty を動詞の前に単独で置く形は英語として成立しておらず、不自然な直訳に聞こえてしまう。
○ I pretty much like this song.
この曲、かなり好き。
I pretty much like this song.
強意副詞の pretty は形容詞・副詞の前にしか置けず、動詞を直接修飾できない。動詞にかけて「かなり」を表すには pretty much という決まった言い方を使う。
🗾 カタカナ語とのギャップ

カタカナ「プリティ」は「プ・リ・テ・ィ」の4モーラだが、英語は2音節。米音では2つ目のtが母音に挟まれフラップ化して[ɾ]になるため、実際の音は「プリリ」に近く、「ティ」とはっきり発音しない。強勢は第1音節（PRIH-)に置く。

🌱 Word family・語源
prettify（動詞）（表面的に）きれいに見せる、装飾する
prettiness（名詞）かわいらしさ、きれいなこと
prettily（副詞）かわいらしく、きれいに
unpretty（形容詞）きれいでない、見栄えの良くない
語源: 古英語 prættig「ずる賢い、抜け目ない、器用な」に由来し、策略・技巧を意味する prætt から派生した。中英語期を通じて「器用に作られた→上品な、しゃれた」という意味を経て、15世紀頃までに現代の「かわいらしい、きれいな」という意味に転じた。
prank（いたずら）— pretty と同じ語根 *prætt（策略・悪ふざけ）に由来するとされ、こちらは元の『ずる賢さ』側の意味を残している
💡 覚え方: pretty はもともと「ずる賢い、抜け目ない」という意味だった（同じ語根から prank『いたずら』も生まれた）。それが『器用に作られた→上品な→かわいい』とプラスの方向へ意味がずれていき、現在の『かわいい』にたどり着いた。
🔗 句動詞
pretty up — （くだけた言い方で）〜をきれいに、見栄え良く整える
They prettied up the old house before putting it up for sale.（彼らはその古い家を売りに出す前にきれいに手直しした。）
⚖️ 類義語の使い分け
cute — cute は幼さ・愛らしさ・親しみやすい魅力に焦点があり、赤ちゃんや動物、小物によく使う。pretty はより広く、見た目の美しさ全般に使える。
lovely — lovely は見た目だけでなく、雰囲気や体験全体の心地よさまで含む、より包括的で（特に英）好意的な語。pretty は主に視覚的な美しさに限定される。
gorgeous — gorgeous は「息をのむほど美しい」という、pretty よりはるかに強い程度の美しさを表す。pretty は控えめで穏やかな美しさを表す。
quite — 副詞としての pretty（かなり）はくだけた会話調。quite はよりフォーマルで、特にイギリス英語の書き言葉でも自然に使われる。
💬 熟語・慣用表現 (5)
sitting pretty 有利な立場にいる、（苦労せず）楽な状況にある口語
With the house paid off and no debts, they're sitting pretty for retirement.（家のローンを完済し借金もないので、彼らは老後を安泰に迎えられる状況にある。）
🔍 「prettyな状態に座っている」という直訳では意味が通らない定型表現。sit にはここでは物理的な着席の意味はなく、「（努力の結果や幸運で）今は恵まれた立場にある」ことを表す。
📝 経済的・状況的に恵まれていることを表すことが多い。過去の努力や幸運の結果、今は楽な立場にいるというニュアンスを含む。
cost a pretty penny すごく高くつく、大金がかかる口語
That antique clock must have cost a pretty penny.（あのアンティークの置き時計はきっと大金がかかったに違いない。）
🔍 「かわいらしい1ペニー」ではなく、この pretty は古い用法で「かなりの、相当な」の意味を持つ。penny（少額）を使いながら実際には高額であることを表す点が構成語から予測しにくい。
📝 cost のほか pay や spend とともに使われることもある（pay a pretty penny for ~）。少額を表す penny が逆に高額を意味する皮肉な組み合わせ。
pretty as a picture 絵のように美しい、とても魅力的な中立
In her white dress, the bride looked pretty as a picture.（白いドレスを着た花嫁は、絵に描いたように美しかった。）
🔍 「一枚の絵のようにきれい」という直喩が定型化した表現。単に pretty picture（きれいな絵・光景）という名詞句とは異なり、人や景色の美しさを褒める決まり文句として独立した意味を持つ。
📝 やや古めかしい響きの決まり文句で、主に人（特に女性）や風景の美しさを褒める際に使われる。as pretty as a picture の形でも使われる。
pretty much ほとんど、だいたい、事実上口語
I've pretty much finished the report; just a few details left.（レポートはほとんど終わったよ。細かい部分が少し残っているだけ。）
🔍 副詞 pretty（かなり）+ much（多く）が固定化し、単なる「かなり多く」ではなく「ほぼ全部」「事実上」という意味を持つ定型の口語表現になったもの。
📝 話し言葉で非常に高頻度。「pretty much the same as ~」（~とほとんど同じ）のように名詞句の前でも使われる。
not just a pretty face 見た目がいいだけではない、美貌だけでなく実力も兼ね備えている口語
She's not just a pretty face — she also runs a successful tech company.（彼女は見た目がいいだけの人ではなく、成功したテック企業を経営してもいる。）
🔍 文字通りには「単なるきれいな顔ではない」だが、entry.collocationsのpretty face（かわいい顔立ち）という構成的な名詞句とは別に、「外見の魅力だけの人ではなく、能力・知性も備えている」という人物評価を表す独立した決まり文句。
📝 しばしば人物評価で使われ、外見だけでなく能力も認める際の定型句。逆に「見た目だけの人」と評す場合は just a pretty face と言う。
