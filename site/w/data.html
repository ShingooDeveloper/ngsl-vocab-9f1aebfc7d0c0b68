data

名詞 (noun)米 /ˈdeɪɾə/ ／ 英 /ˈdeɪtə/da·ta（2音節）CEFR B1

🗣 第1音節 da に強勢がある（DA-ta）。母音は二重母音 /eɪ/ で「デ」から「イ」へなめらかに滑らせて1拍で発音する。第2音節の a は弱い曖昧母音 /ə/。アメリカ英語では強勢のある母音と弱母音に挟まれた t が有声化し、フラップの [ɾ] となる（/ˈdeɪɾə/、日本語の「ラ行」に近い音）。イギリス英語ではフラップ化せず /ˈdeɪtə/ とはっきり t を発音する

  1. 【名詞】データ、資料（調査・観察・実験などで集められた事実や数値の情報）（We need more data before we can decide. 決める前にもっとデータが必要だ）
  2. 【名詞】（コンピュータで処理・保存される）データ、情報（Make sure you save your data before turning off the computer. パソコンの電源を切る前に、必ずデータを保存してください）

語法マーク

  • 不可算名詞として扱うのが現代英語の主流（the data is, a lot of data）。a/an を直接つけられない（× a data）。数えたいときは a piece of data / a data point / an item of data という単位表現を使う
  • 本来はラテン語 datum の複数形なので、学術・フォーマルな文体では今も複数扱い（the data are, these data suggest ~）にすることがある。同じ文章内で単数扱いと複数扱いを混在させない（It needs のように単数扱いの動詞・代名詞と are を混ぜない）のが原則
  • 複数形 datas は存在しない。data 自体がすでに複数形由来のため、さらに -s を付けることはない
  • 動詞用法を持たない。「データを与える」「データ化する」は provide data, digitize, collect data など別の表現で表す
  • datum という単数形は統計学・測地学など専門的な文脈で使われる程度で、一般の会話ではほぼ使わない
  • 他品詞での用法: data はラテン語 datum の複数形に由来するが、現代英語では主に2つの扱い方が併存する。(1) 現代の日常・IT系の用法では不可算の集合名詞として単数扱い（the data is, this data）される。(2) 学術・科学系のフォーマルな文章では今も本来の複数扱い（the data are, these data support ~）が好まれることが多い。単数形 datum は「個々のデータ1点」を指す専門用語として使われることがあるが、日常会話ではほとんど使われず、代わりに a piece of data / a data point という言い方が好まれる。

文型・頻出パターン

  • a piece of data / a data point不可算名詞 data を「1件」と数えるときの単位表現
    This is a fascinating piece of data.これは実に興味深いデータの1件だ。
    Each data point represents one customer's response.それぞれのデータポイントは、1人の顧客の回答を表している。
  • data on/about + N〜についてのデータ
    We don't have enough data on customer behavior yet.顧客の行動について、まだ十分なデータがない。
    The data about air quality is updated every hour.大気の質についてのデータは、1時間ごとに更新される。
  • based on the dataデータに基づいて
    Based on the data, we decided to change our marketing strategy.データに基づいて、私たちはマーケティング戦略を変更することにした。
  • the data suggest(s)/show(s) that + S + Vデータが〜ということを示唆する・示す（単数扱いか複数扱いかで動詞の形が変わる）
    The data suggests that sales will increase next quarter.そのデータは、来四半期に売り上げが伸びることを示唆している。（単数扱い・口語的）
    The data suggest a strong correlation between the two variables.そのデータは、その2つの変数の間に強い相関があることを示している。（複数扱い・学術的）

他のNGSL語への置き換え

data → information
元: We have collected a large amount of data from the survey over the past three months.
後: We have collected a large amount of information from the survey over the past three months.
訳: 私たちはこの3か月間、調査から大量の情報を集めてきた。
📝 information は一般的な知識・情報を指す広い不可算名詞。data は特に調査・分析のために集められた事実や数値を指し、統計・研究・コンピュータ処理などのより専門的・技術的な文脈で好まれる。information の方が日常会話で使われる中立的な語。
data → results
元: The data was analyzed carefully before the report was published.
後: The results were analyzed carefully before the report was published.
訳: その結果は、報告書が発表される前に注意深く分析された。
📝 results は可算名詞で複数形になるため were を使う。一方 data は不可算の集合名詞として扱われ、口語では is/was のように単数で受けるのが一般的（本来は datum の複数形だが現代英語では単数扱いが主流）。可算・不可算と動詞の一致の違いを示す好例。
data → verify
元: You should always double-check your data before submitting the final report.
後: You should always verify your data before submitting the final report.
訳: 最終報告書を提出する前に、必ずデータを確認しなさい。
📝 verify はより公式でフォーマルな響きを持ち、正確性を保証する検証行為を強調する動詞。double-check は口語的でカジュアルな表現で、日常業務の会話でよく使われる。意味はほぼ同じだが register（文体）が異なる。

🗾 カタカナ語とのギャップ

カタカナ「データ」は「デ・エ・タ」のように母音をはっきり区切って発音しがちだが、英語の第1音節は二重母音 /eɪ/ で「デ」から「イ」へ1拍で滑らせるように発音する。またアメリカ英語では第2音節の t が母音に挟まれてフラップ化し、「タ」よりも軽い「ラ」に近い音 [ɾ] になる（[ˈdeɪɾə] デイラ）。さらに日本語の「データ」は単数・複数を区別しないが、英語ではもともとの単数形 datum（データの1点）と複数形由来の data（データ全体）が区別される点にも注意。

🌱 Word family・語源
datum（名詞）データの1点、単一の観測値・数値（data の本来の単数形。統計学など専門的な文脈以外ではほとんど使われない）
database（名詞）データベース（データを整理して蓄積する仕組み）
dataset（名詞）データセット、ひとまとまりのデータ群
data-driven（形容詞）データ駆動の、データに基づいた（data-driven decisions データに基づく意思決定）
語源: ラテン語 datum（与えられたもの）の複数形。datum はラテン語動詞 dare（与える）の過去分詞にあたる。ラテン語文法に厳密に従えば data は複数形なので本来は複数扱い（data are）が正しいが、英語に取り入れられて久しい今日では、ひとまとまりの情報の塊とみなされ、不可算の集合名詞として単数扱い（data is）されるのが日常・IT分野では一般的になっている。
date（日付）— 同じくラテン語 dare（与える）の過去分詞形に由来。手紙などに「与えられた（記された）日」を意味する littera data という言い回しから「日付」の意味に発展した
mandate（命令、権限の付与）— man-（手）+ datum（与えられたもの）で「手に与えられたもの」→「委任、命令」。data と同じ -datum の部分を共有する
donate（寄付する）— ラテン語 donare（与える）に由来し、dare と同じ「与える」の語根を共有する近縁語
💡 覚え方: data はラテン語で文字通り「与えられたもの」。date（手紙に与えられた日付）や mandate（手に与えられた権限=委任）も同じ dare「与える」の語根から生まれた仲間。「データとは、観察や測定によって私たちに与えられる情報」というイメージで覚えると、date や mandate ともつながる。
⚖️ 類義語の使い分け
information — information はより広く一般的な知識・情報全般を指す語。data は特に調査・観察・実験・コンピュータ処理などによって集められた、体系的で分析対象になりうる事実や数値を指す、より専門的・技術的な語。
statistics — statistics は数値として集計・分析された量的なデータに限定される。data は数値に限らず、テキスト・画像・観測記録なども含むより広い概念で、statistics はその一部と言える。
facts — facts は個々の「事実」そのものを指す一般的な語で、日常会話でも使われる。data は研究・分析・コンピュータ処理などの文脈で体系的に集められた情報のまとまりを指すことが多く、よりフォーマルで専門的な響きを持つ。
