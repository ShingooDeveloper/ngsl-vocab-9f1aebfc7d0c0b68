now

副詞・接続詞米 /naʊ/ ／ 英 /naʊ/now（1音節）CEFR A1

🗣 1音節語なので強勢の位置に迷う余地はない。母音は二重母音 /aʊ/ で、「ア」から「ウ」へ切れ目なく1拍で滑らせて発音する。カタカナ「ナ・ウ」のように2拍に分けない

  1. 【副詞】今、現在（現在の時点を表す。文中で主語と動詞の間や文末に置かれることが多い。She used to live in Osaka, but now she lives in Tokyo. 彼女は昔大阪に住んでいたが、今は東京に住んでいる）
  2. 【副詞】今すぐに、たった今（即時性・緊急性を表す。Do it now! 今すぐそれをやって！）
  3. 【副詞】（文頭に置き、話題転換や注意喚起を示す談話標識として）さて、では（Now, where did I put my keys? さて、鍵をどこに置いたっけ？）
  4. 【接続詞】（now (that) の形で）今や～だから、～するからには（理由を表す。Now that you mention it, I do remember him. 言われてみれば、彼のこと覚えているよ）

語法マーク

  • 副詞としての基本位置は文中（be動詞・助動詞の後、一般動詞の前）または文末。文頭に置くと「さて」という話題転換・注意喚起の談話標識として読まれやすくなる点に注意
  • 形容詞用法は無く、名詞を直接修飾できない（× a now problem）。名詞用法も the now / in the now などごく限られた慣用表現に限られる
  • 接続詞としての now (that) は理由を表す従属節を導く（≒ since / as）。話し言葉では that を省略することが多い
  • 比較級・最上級を作らない（nower / nowest のような形は存在しない）
  • 『今この瞬間』を指す now と、過去と対比した『現代の一般的傾向』を指す nowadays / these days は区別が必要。文頭で一般論を述べるつもりで Now, ... と言うと談話標識に聞こえてしまう
  • 他品詞での用法: まれに名詞用法があり、the now / in the now の形で「今この瞬間」を表す（マインドフルネス系の言い回し: live in the now 今を生きる、というかなりくだけた表現）。また Now, now, ... と2回重ねると、子どもをなだめたり軽くたしなめたりする決まり文句になる（Now, now, don't cry. よしよし、泣かないで）。

文型・頻出パターン

  • S + be/助動詞 + now + 補語現在の状態・変化した状態を示す（most basic な時の副詞用法）
    The museum is now closed for renovations.その美術館は改装のため今は閉まっている。
    She can now speak three languages.彼女は今では3か国語を話せる。
  • Now, ...（文頭の談話標識）さて、では（話題転換・注意喚起のクッション）
    Now, where were we?さて、どこまで話したっけ？
    Now, listen carefully.さて、よく聞いて。
  • Now (that) + S + V今や〜だから、〜するからには（理由を表す接続詞）
    Now that you're here, we can begin.君が来たからには、もう始められるね。
  • from now on今後は、これからは（未来の起点を示す副詞句）
    From now on, please arrive on time.今後は時間通りに来てください。
  • up to now / until now今まで、これまで（現在完了とともに使われることが多い）
    Up to now, everything has gone smoothly.これまでのところ、すべて順調に進んでいる。
  • just nowたった今、つい先ほど（ごく近い過去の出来事）
    I saw her just now in the hallway.たった今、廊下で彼女を見かけた。

他のNGSL語への置き換え

now → right now
元: Please turn off your phone now; the movie is about to start.
後: Please turn off your phone right now; the movie is about to start.
訳: 今すぐ携帯の電源を切ってください。もうすぐ映画が始まりますよ。
📝 now だけでも「今」の意味は伝わるが、right が加わった right now は「まさに今すぐ」という緊急性・即時性を際立たせる。映画がまもなく始まる差し迫った状況をより強く表せる。
now → currently
元: Scientists now believe that regular exercise can prevent many age-related illnesses.
後: Scientists currently believe that regular exercise can prevent many age-related illnesses.
訳: 科学者たちは現時点では、定期的な運動が加齢に伴う多くの病気を防げると考えている。
📝 now は口語的で「以前とは違って今は」という変化のニュアンスを含みやすいのに対し、currently はより硬い書き言葉・学術的な文体で使われ、「現時点では（将来変わる可能性もある）」という中立的な時点を示す。論文や報道文では currently の方が自然に響く。
now → already
元: The road that was closed for repairs yesterday is now open to traffic again.
後: The road that was closed for repairs yesterday is already open to traffic again.
訳: 昨日修理のために閉鎖されていた道路は、もう通行できるようになっている。
📝 now は単に「現在の状態」を中立的に述べるのに対し、already は「予想より早く」「もう」という驚き・完了の早さのニュアンスを加える。早期の復旧に意外感がある場合は already の方が自然。

✕ I did it now.
「たった今それをやった」と言いたかったが、now には『ごく最近』という意味がなく、英語として不自然かつ意味が伝わりにくい。
○ I just did it. / I've just done it.
たった今それをやった。
I just did it.
I've just done it.
「たった今〜した」という近接した過去を表すには just（+過去形または現在完了）を使う。now は『現在の一点』を指す語であって、『ごく最近』という近接性の意味は持たないため、過去の完了とうまく結びつかない。
✕ Now, we'll use Plan A; we can change it later if needed.
『差し当たりプランAを使います』と言いたかったが、文頭の Now, だけでは『さて』と話を切り替える合図に聞こえ、暫定性のニュアンスが消えてしまう。
○ For now, we'll use Plan A; we can change it later if needed.
差し当たり、プランAを使います。後で必要なら変更できます。
For now, we'll use Plan A. We can change it later if needed.
『今のところ・差し当たり』という暫定性を表すには for now を使う。単独の Now, ... を文頭に置くと『さて』という話題転換の談話標識として読まれ、暫定的な措置というニュアンスが伝わらない。
✕ Now, many companies allow remote work.
『今は多くの企業がリモートワークを許可している』という一般論のつもりだったが、文頭の Now, は『さて』と話を切り出す合図に聞こえ、一般的傾向を述べる文には不自然に響く。
○ Nowadays, many companies allow remote work. / These days, many companies allow remote work.
近頃は、多くの企業がリモートワークを許可している。
Nowadays, many companies allow remote work.
These days, many companies allow remote work.
過去と対比した『現代の一般的な傾向』を述べるときは nowadays / these days を使う。文頭の Now, は『さて』という話題転換の合図として読まれやすく、継続的な一般論を述べる意図が伝わりにくい。
🗾 カタカナ語とのギャップ

カタカナ「ナウ」は「ナ・ウ」と2モーラに分けて読みがちだが、英語は1音節の二重母音 /naʊ/。「ア」から「ウ」へ切れ目なく滑らせて1拍で発音する。また『ナウい（今っぽい、流行りの）』という昭和〜平成初期の俗語は now 由来だが、現代英語の now 自体にそのような形容詞用法はない。さらに /naʊ/（now）と /noʊ/（no, know）は日本語ではどちらも『ノウ』に近く聞こえがちだが、二重母音の出発点の母音が異なる別の語なので混同しないよう注意。

🌱 Word family・語源
nowadays（副詞）近頃は、今日び（過去と対比して現在の一般的傾向を述べる、やや書き言葉寄りの語）
語源: 古英語 nū（現在の一瞬・現時点を指す副詞）に由来し、さらに遡るとゲルマン祖語 *nu、印欧祖語の *nu にたどり着く。この語形は驚くほど古く安定しており、ラテン語 nunc（今）、ギリシャ語 nyn（今）、ドイツ語 nun（さて、今や）、オランダ語 nu（今）など、印欧語族の広い範囲でよく似た形のまま『今』を表す語として残っている、英語の中でも最も基礎的な時間副詞の一つ。
nowadays（近頃は）— now + a（前置詞 on の名残）+ days（日々）の複合語。『今の（これらの）日々に』が原義
💡 覚え方: now は英語の中でも最も古い基礎語の一つで、ラテン語 nunc やドイツ語 nun と兄弟のような関係。『いま、この一点』を指す最短の言葉として、大昔からほとんど形を変えずに使われ続けている。
⚖️ 類義語の使い分け
currently — より書き言葉的・フォーマルで、談話標識として文頭で話題転換する用法や now that の接続詞用法は持たない。単に『現時点では』という中立的な事実を述べる。
immediately — 『即座に、直ちに』という緊急性・迅速さに重点があり、do it immediately は間を置かない切迫感を伴う。now は単に『この時点で』を指すだけの場合にも使え、immediately ほど切迫していないことも多い。
nowadays — 過去と対比した『現代の一般的傾向・時代の変化』を述べるときに使う、より継続的・一般的な現在を表す語。now は瞬間的な一時点も表せるが、nowadays はそのような瞬間性の意味では使えない。
💬 熟語・慣用表現 (4)
now or never 今しかない、今がその時だ口語
This is your only chance to apply, so it's now or never.（応募できるのはこれが最後のチャンスだから、今しかないよ。）
🔍 「今」か「決してない（never）」かの二択を突きつける言い方。文字通りには「今、さもなくば二度とない」で、このチャンスを逃せば永遠に失われるという切迫感を表す決まり文句。
📝 It's now or never! のように単独で感嘆的に使われることも多い。二度と巡ってこない機会を強調する場面に限って使う。
now and then 時々、たまに中立
We still meet up for coffee now and then.（私たちは今でも時々コーヒーを飲みに会っている。）
🔍 「今」と「その時」を並べた形だが、字義通りの意味ではなく「時折・たまに」という頻度を表す慣用句。
📝 ほぼ同義の now and again も使われる。every now and then はより強調的な同義表現。sometimes よりもやや口語的で親しみのある響きを持つ。
here and now 今この瞬間に、今すぐ／（名詞的に）今という現実中立
Stop worrying about tomorrow and focus on the here and now.（明日のことを心配するのはやめて、今この瞬間に集中しなよ。）
🔍 「ここ」と「今」を結びつけ、過去や未来ではなく「まさに目の前の現在」そのものを指す慣用句。
📝 the here and now の形で「現在という時」を表す名詞句としても使われる。副詞的に「今すぐ、この場で」の意味でも使われる（I need an answer here and now.）。
any time now もうすぐ、間もなく口語
The bus should be arriving any time now.（バスはもうすぐ来るはずだ。）
🔍 「今からどの時点でも（起こりうる）」という字義から転じて、予想される出来事がまさに差し迫っていることを表す慣用句。
📝 出産や到着など、いつ起きてもおかしくない出来事を待つ場面でよく使われる。any minute now もほぼ同義。
