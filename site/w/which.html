which

限定詞・代名詞米 /wɪtʃ/ ／ 英 /wɪtʃ/which（1音節）CEFR A1

🗣 1音節語。/wɪtʃ/。唇を丸めたまま /w/ で滑り込み、短母音 /ɪ/ を経て破擦音 /tʃ/ で止める。カタカナ「ウィッチ」のように「チ」に母音を足さない（語末は tʃ で息を止めて終わる）

  1. 【限定詞】どの、どちらの（複数の中から特定のものを選ぶ疑問文で使う）（Which bus goes to the station? どのバスが駅に行きますか?）
  2. 【代名詞】どれ、どちら（疑問代名詞。複数の選択肢の中からどれかを尋ねる）（Which do you prefer, tea or coffee? 紅茶とコーヒー、どっちがいい?）
  3. 【代名詞】～という、～するところの（関係代名詞・制限用法。物を表す名詞を先行詞とし、後ろから説明を加える）（The book which is on the table is mine. テーブルの上にあるその本は私のです。）
  4. 【代名詞】そのことは、それは（関係代名詞・非制限用法。前の節や文全体の内容を受けて補足説明を加える）（He was late again, which annoyed everyone. 彼はまた遅刻し、そのことがみんなを苛立たせた。）

語法マーク

  • 先行詞は「物・こと」に限る。人には使えない（× the man which → ○ the man who）
  • 疑問限定詞としては後ろに名詞を伴う（Which bus...?）。疑問代名詞としては単独で使い、後ろに名詞を置かない
  • 非制限用法（, which）は前の節全体を受けられる点が that には無い特徴。that は非制限用法（コンマ付き）では使えない
  • 間接疑問文の中では平叙文の語順（S+V）になる。疑問文の語順（do/does/did+S+V）をそのまま持ち込まない（× tell me which do you like）
  • which one(s) の形で複数の選択肢から1つ（または複数）を選ばせる表現になる
  • 前置詞＋which（in which, of which, on which など）はフォーマルな書き言葉に多く、口語では前置詞を文末に送って that/which を省略することが多い（the hotel we stayed in）
  • 他品詞での用法: 疑問限定詞・疑問代名詞としての用法に加え、関係代名詞としての用法が中心的。関係代名詞には(1)制限用法（先行詞を限定・特定する。コンマなし。The book which is on the table = テーブルの上にある方の本、と特定）と(2)非制限用法（コンマを伴い、先行詞や前の節全体に補足説明を加える。関係詞節が無くても文意は成立する）の2種類があり、非制限用法では節全体（前の文の内容全部）を先行詞として受けられるのが that には無い which 特有の働き。また of which / in which のようにフォーマルな書き言葉では前置詞を直前に置く形が好まれる。

文型・頻出パターン

  • Which + 名詞 〜?どの〜（複数の選択肢の中から特定させる疑問限定詞）
    Which bus goes to the station?どのバスが駅に行きますか?
    Which size do you need?どのサイズが必要ですか?
  • Which (do/is) 〜?どれ、どちら（名詞を伴わない疑問代名詞）
    Which do you prefer, tea or coffee?紅茶とコーヒー、どっちがいい?
  • 先行詞（物）+ which + V/S+V 〜（関係代名詞・制限用法）〜という、〜するところの（先行詞を特定・限定する）
    The book which is on the table is mine.テーブルの上にあるその本は私のです。
    The novel which was translated into ten languages became a bestseller.10か国語に翻訳されたその小説はベストセラーになった。
  • 節, which + V（関係代名詞・非制限用法）そのことは、それは（前の節・文全体の内容を受けて補足する）
    He was late again, which annoyed everyone.彼はまた遅刻し、そのことがみんなを苛立たせた。
  • 前置詞 + which（フォーマルな関係詞節）〜する（ところの）（in which, of which など前置詞を直前に伴う書き言葉的な形）
    This is the hotel in which we stayed during our honeymoon.ここが、私たちが新婚旅行で泊まったホテルです。
  • 動詞 + which + S + V（間接疑問文）どの〜か（を）〜（which節が名詞節として動詞の目的語になる。語順は平叙文のまま）
    Could you tell me which train I should take to get downtown?どの電車に乗ればいいか教えてもらえますか?
    If you can remember which key opens this door, we won't need a locksmith.このドアを開ける鍵がどれか思い出せれば、鍵屋を呼ばなくて済むよ。

他のNGSL語への置き換え

which → that
元: The novel which was translated into ten languages became a bestseller.
後: The novel that was translated into ten languages became a bestseller.
訳: 10か国語に翻訳されたその小説はベストセラーになった。
📝 制限用法の関係詞節では that も置き換え可能で、特にアメリカ英語の口語では which より that の方が好まれる傾向がある。ただしコンマを伴う非制限用法（例: My flight was delayed, which...）では that は使えず which しか使えない点が重要な違い。
which → what
元: Which color do you like better, blue or green?
後: What color do you like better, blue or green?
訳: 青と緑、どっちの色が好き?
📝 which は blue or green のように限られた選択肢の中から選ぶニュアンスが強く、この文脈ではより自然。what はより一般的・無制限な問いかけに使われることが多く、選択肢を挙げていない場面（例: What color do you like?）でよく使われる。
which → what
元: Could you tell me which train I should take to get downtown?
後: Could you tell me what train I should take to get downtown?
訳: どの電車に乗ればいいか教えてもらえますか?
📝 which は路線図上の決まった選択肢の中から特定するニュアンスを持ち、この場面（具体的な路線の中から選ぶ）ではより自然。what はより漠然とした一般的な情報を尋ねる響きになる。

詳細
⚠️ 日本人の典型誤用
✕ I don't know which do you like.
「どっちが好きか分からない」と言いたかったが、which の後にもう一度疑問文の語順（do you）を繰り返してしまい、ネイティブには文法的に破綻した二重疑問文のように聞こえる。
○ I don't know which you like.
どっちが好きか分からない。
I don't know which you like.
which が導く節が動詞 know の目的語になる間接疑問文では、疑問文の語順（do you like）ではなく平叙文の語順（you like）にする必要がある。日本語の疑問文の感覚を引きずって疑問文の語順のまま埋め込んでしまう誤り。
✕ The woman which called you is my sister.
「あなたに電話してきた女性は私の姉です」と言いたかったが、which は人には使えないため、ネイティブには不自然、または非文法的な文に聞こえる。
○ The woman who called you is my sister.
あなたに電話してきた女性は私の姉です。
The woman who called you is my sister.
which の先行詞は物・こと限定で、人には使えない。人を先行詞にする関係代名詞は who（主格）／whom（目的格）。日本語の「〜という人」を機械的に which に当てはめてしまう誤り。
✕ Which one do you want it?
「どっちが欲しい?」と言いたかったが、which one がすでに目的語の役割を果たしているため、文末の it は文法的に不要かつ非文法的で、ネイティブには違和感のある繰り返しに聞こえる。
○ Which one do you want?
どっちが欲しい?
Which one do you want?
which（one）自体が want の目的語なので、後ろにさらに it を置くと目的語が二重になってしまう。日本語の「それ、どれがいい?」の「それ」を引きずって余分な代名詞を足してしまう誤り。
🗾 カタカナ語とのギャップ

カタカナ「ウィッチ」は「ウ・ィ・ッ・チ」の4モーラで捉えがちだが、英語は /wɪtʃ/ の1音節。語頭の /w/ は唇を丸めてすぼめてから素早く開く動きで「ウ」よりも唇の動きが強く、語末の /tʃ/ は「チ」ではなく母音を伴わない破擦音で止める。なお多くのアメリカ英語話者は wh- を /hw/ ではなく /w/ で発音するため、which は魔女を意味する witch と全く同じ発音（同音異義語）になる。

🌱 Word family・語源
whichever（限定詞・代名詞）どちらの〜でも、どれでも（which に -ever が付いた強調・譲歩形）
語源: 古英語 hwilc（「どの、どちらの」）に由来し、さらに遡るとゲルマン祖語 *hwilīkaz（hwi-「どの」+ -līk-「〜の形をした、〜のような」）に行き着く。つまり元々は「どの形の（もの）」というのが文字通りの意味。who, what, when, where, why と同じ wh- で始まる疑問語のグループに属し、これらは全て同じゲルマン祖語の疑問代名詞の語根から枝分かれしている。
what（何）— 同じ wh- 疑問語根から派生。which が「複数の中のどれ」と選択肢を限定するのに対し、what はより無制限に「何」を問う
who（誰）— 同じ wh- 疑問語根から派生。人を尋ねる点が which（物・こと）との使い分けの境界線になっている
💡 覚え方: which は who・what・when・where・why と同じ「wh- 疑問語ファミリー」の一員で、元々は「どの形の（もの）」という意味。この wh- のつながりを意識すると、which が疑問語であると同時に関係代名詞にもなる二刀流の性質が腑に落ちる。
⚖️ 類義語の使い分け
what — which は文脈上すでに挙がっている、または限られた選択肢の中から選ぶニュアンス（Which do you prefer, tea or coffee?）。what は選択肢を提示せず、より一般的・無制限に問う（What do you want to drink?）。
that — 制限用法の関係代名詞としては物を表す先行詞に対して which と that はほぼ置き換え可能で、口語では that の方が好まれる傾向がある。ただしコンマを伴う非制限用法や、前置詞の直後（in which など）では that は使えず which しか使えない。
💬 熟語・慣用表現 (2)
which is which どちらがどちらか（を見分ける）中立
The twins look so alike that even their teacher can't tell which is which.（その双子はあまりに似ていて、担任の先生でさえどちらがどちらか見分けがつかない。）
🔍 似ていて区別がつきにくい2つ（以上）の人・物について「どれがどれなのか」を問う定型表現。tell/know which is which の形で「見分ける／区別できる」という決まり文句になる。
📝 tell which is which / know which is which の形で使われることが多い。3つ以上のものにも使える（見分けがつかない対象の数に制限はない）。
which way the wind is blowing 情勢がどう動いているか、事態がどちらに向かっているか中立
Before deciding whether to invest, she wanted to see which way the wind was blowing in the market.（投資するかどうか決める前に、彼女は市場の情勢がどう動いているか見極めたかった。）
🔍 文字通りは「風がどちら向きに吹いているか」だが、比喩的に「物事の成り行き・世論や状況の傾向」を指す。see/know which way the wind is blowing の形で使う。
📝 政治・ビジネスの文脈で「様子見をする」ニュアンスでよく使われる。blow は必ず進行形 blowing で使う点に注意。
