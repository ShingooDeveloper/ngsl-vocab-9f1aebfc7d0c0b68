task

名詞・動詞米 /tæsk/ ／ 英 /tɑːsk/task（1音節）CEFR B1

🗣 1音節語。母音は前寄りの短母音 /æ/ で、日本語の「ア」より口を横に大きく開けて出す（cat, bag と同じ母音）。語末は子音クラスター /sk/ が母音を挟まず連続する

  1. 【名詞】仕事、作業、任務（課された具体的な一つの仕事を指す。I still have three tasks to finish today. 今日中に終わらせなきゃいけない仕事がまだ3つある）
  2. 【名詞】（コンピュータの）タスク、処理（バックグラウンドで実行される処理の単位。Check Task Manager to see which task is using the most memory. タスクマネージャーで、どのタスクが一番メモリを使っているか確認して）
  3. 【動詞】（人に）任務・仕事を課す、負わせる（多くは受動態 be tasked with の形で使われる。She was tasked with training the new employees. 彼女は新入社員の教育を任された）

語形変化

形種類発音備考
tasks複数形／三人称単数現在形/tæsks/語末が無声音 /k/ のため -s は /s/（「タスクス」であって「タスクズ」ではない）。名詞の複数形（多くの仕事）と動詞の三単現（She tasks him with ~）を兼ねる
tasked過去形・過去分詞/tæskt/規則変化。語末が無声音 /k/ のため -ed は /t/（「タスクト」。「タスクド」ではない）。多くは受動態 be tasked with で使われる
taskinging形/ˈtæskɪŋ/規則変化。子音字の重複なし（taskking ではない）。multitasking のように名詞・形容詞的にも使われる

語法マーク

  • 可算名詞：区切られた具体的な一つの仕事・任務を指すため countable（a task / tasks）。仕事全般を漠然と表す不可算名詞の work とは対照的
  • 動詞のtaskはほぼ常に受動態 be tasked with + 名詞/動名詞 の形で使われる。能動態で直接目的語を2つ続ける用法（× task him a job）は不可
  • task A with B の型では B に to不定詞ではなく前置詞 with + 名詞/動名詞が続く（× task him to do ではなく task him with doing）
  • task force, task manager, task list のように他の名詞の前に置かれ複合名詞を作りやすい
  • 他品詞での用法: 名詞用法が基本で、動詞用法はより形式的な言い方。動詞のtaskはtask + 人 + with + 名詞/動名詞「（人）に～を課す」の型に限定され、たいてい受動態 be tasked with ~ で使われる（能動態で日常会話に使われることは少なく、口語では assign や give の方が一般的）。名詞のtaskは他の名詞の前に置かれて形容詞的に働くこともある（task list「作業リスト」、task-oriented「作業志向の」、task force「特別対策チーム」）。

文型・頻出パターン

  • be tasked with + 名詞/動名詞～という任務を課される（動詞taskの最も基本的な形。受動態が定番）
    She was tasked with finding a solution to the problem.彼女はその問題の解決策を見つける任務を課された。
    The committee is tasked with reviewing the new policy.その委員会はその新方針を見直す任務を課されている。
  • task + 人 + with + 名詞/動名詞（人）に～という仕事を課す（能動態。フォーマルな響き）
    The manager tasked him with training the new staff.マネージャーは彼に新人研修という仕事を課した。
  • assign / complete / finish a task仕事・任務を割り当てる／完了する（taskと結びつく頻出動詞）
    Please complete this task by Friday.金曜までにこの仕事を終わらせてください。
    Who assigned you this task?誰があなたにこの仕事を割り当てたの?
  • It is (人)'s task to do～することが（人）の任務だ（taskを名詞補語として使う形式主語構文）
    It is the teacher's task to grade all the essays.すべての作文を採点するのが教師の仕事だ。
  • task force / task list / multitaskingtaskが他の名詞と結びついて作る複合語（特別対策班、作業リスト、複数作業の同時進行）
    The government set up a task force to deal with the crisis.政府はその危機に対応するための特別対策チームを設置した。
    Multitasking can actually reduce your efficiency.マルチタスクは実は効率を下げることがある。

他のNGSL語への置き換え

task → plan
元: She has been tasked with organizing the entire conference by herself.
後: She has been tasked with planning the entire conference by herself.
訳: 彼女はカンファレンス全体を一人で企画するという任務を課された。
📝 organize は当日の運営・段取りという実行面に重点があるのに対し、plan は事前の計画立案に重点がある。どちらも task と自然に結びつくが、焦点となる作業の段階が異なる。
task → job
元: The task that she assigned to me turned out to be much harder than it looked.
後: The job that she assigned to me turned out to be much harder than it looked.
訳: 彼女が私に割り当てた職務は、見た目よりずっと難しいとわかった。
📝 task は区切られた具体的な一つの作業を指すのに対し、job はより広く「職務・仕事」全般を指す。この文脈では意味は近いが、job の方がやや漠然とした範囲を含意する。
task → project
元: You should break this task into smaller steps so it doesn't feel so overwhelming.
後: You should break this project into smaller steps so it doesn't feel so overwhelming.
訳: このプロジェクトはもっと小さいステップに分けたほうがいい、そうすればそれほど大変に感じなくなる。
📝 project は複数の task から構成される規模の大きな取り組みを指す。task が単一の作業単位であるのに対し、project はそれらをまとめた枠組みというニュアンスの違いがある。

⚠️ 日本人の典型誤用
✕ I have many works to do today.
言いたかったのは「今日やらなきゃいけない仕事がたくさんある」。だが work は数えられない名詞なので works と複数形にすると不自然で、ネイティブには違和感のある英語に聞こえる。
○ I have many tasks to do today.
今日やらなきゃいけない仕事がたくさんある。
I have many tasks to do today.
work「仕事」は不可算名詞なので複数形 works や many work とは言えない（work が可算になるのは「作品」の意味の時）。区切られた具体的な仕事の単位を数えたい時は可算名詞の task を使う。
✕ She was tasked to organize the event.
言いたかったのは「彼女はそのイベントを取り仕切る任務を課された」。意味は伝わるが、task の後に to不定詞を続けるのは標準的でなく、ネイティブにはやや不自然に響く。
○ She was tasked with organizing the event.
彼女はそのイベントを取り仕切る任務を課された。
She was tasked with organizing the event.
ask/tell somebody to do の型からの類推で task の後にも to不定詞を続けてしまう誤り。動詞 task は be tasked with + 名詞/動名詞 の形が標準で、to不定詞は続かない。
✕ I have a task from my teacher due tomorrow.
言いたかったのは「先生から出された課題が明日締め切りだ」。意味は通じるが、学校の宿題を task と呼ぶのはやや不自然で、ネイティブには assignment や homework の方が自然に聞こえる。
○ I have an assignment from my teacher due tomorrow.
先生から出された課題が明日締め切りだ。
I have an assignment from my teacher due tomorrow.
日本語の「課題」を直訳して学校の宿題にも task を使ってしまう誤り。学校で出される宿題・課題は assignment や homework が普通で、task はより一般的に「仕事・作業」全般を指す語なので、学校の文脈で使うとやや不自然に響く。
🗾 カタカナ語とのギャップ

カタカナ「タスク」は「タ・ス・ク」の3モーラだが、英語は1音節。語末は子音クラスター /sk/ を母音なしで一気に閉じ、「スク」と伸ばさない。またアメリカ英語の母音は /æ/ で、日本語の「ア」より口を横に大きく開けた明るい音（cat, bag と同じ）。イギリス英語では /tɑːsk/ と長い「アー」に近い音になり、米英で母音が大きく異なる点にも注意（本教材はIPA_USを採用）。

🌱 Word family・語源
multitask（動詞・名詞）複数の作業を同時にこなす（こと）
multitasking（名詞）マルチタスク、複数作業の同時進行
taskmaster（名詞）（厳しく）仕事を課す人、こき使う監督者
語源: 古フランス語 tasque（tâche の異形）を経て、中世ラテン語 tasca に由来し、さらにラテン語 taxare「評価する、課する」に遡る。もともと「(税や義務として)課されたもの」を意味した語で、tax と同じ語根を共有する。
tax（税）— 共にラテン語 taxare「評価する、課する」に由来。「税を課す」も「仕事を課す」も『上から課す』という同じイメージでつながる
taxing（骨の折れる、負担の大きい）— 同じ taxare 由来。『重荷を課すほど大変な』という意味に発展した形容詞で、a taxing job のように使う
💡 覚え方: task はもともと tax（税）と同じラテン語 taxare「課す」から来ている。税金(tax)も仕事(task)も『上から課されるもの』というイメージで結びつけて覚える。
⚖️ 類義語の使い分け
duty — duty は役職・立場に伴う道徳的・職務上の「義務」に重点がある抽象的な語。task は具体的に区切られた実行可能な作業の単位を指す点で異なる。
assignment — assignment は先生・上司など権威ある立場の人から正式に「割り当てられた」仕事・課題を指すことが多い。task はより一般的で、誰から与えられたかを問わない仕事全般を指せる。
chore — chore は掃除・洗濯など家庭内の地味で反復的な雑用を指すことが多い。task はもっと中立的で、仕事・学業・プロジェクトなど幅広い場面の「仕事」を指せる。
💬 熟語・慣用表現 (3)
take someone to task （人を）厳しく叱る、責任を問いただすややフォーマル／文章語
The manager took him to task for missing the deadline again.（上司は彼が再び締め切りに間に合わなかったことを厳しく叱った。）
🔍 「人を任務の場に引っ張っていく」から転じて「過失や責任を公然と問いただし、こっぴどく叱る」の意味になった。単なる小言ではなく、相手のミスを面と向かって指摘するニュアンスを持つ。
📝 take + 人 + to task (for + 名詞/動名詞) の形が基本。ニュース記事やビジネス文書でもよく使われ、「公の場で責任を追及する」含みが強い。
be up to the task その任務をこなせるだけの力量がある、その仕事に対応できる中立
I wasn't sure if the new intern was up to the task, but she handled it perfectly.（新しいインターンがその仕事をこなせるか不安だったが、彼女は完璧にやり遂げた。）
🔍 「その任務の求める水準まで届いている」が転じて「大変な仕事をやり遂げる能力や気概がある」の意味になった。否定形 not up to the task は「力不足だ」を表す。
📝 能力や覚悟が試される場面で疑問文・否定文とともに使われることが多い。ほぼ同義の言い換えに be equal to the task がある。
the task at hand 今取り組んでいる仕事、目の前の課題中立～ややフォーマル
Let's stop worrying about tomorrow and focus on the task at hand.（明日のことを心配するのはやめて、今目の前にある仕事に集中しよう。）
🔍 「手の届く範囲にある任務」が転じて「今まさに取り組むべき、目の前の課題」を指す定型表現。at hand 自体が「すぐ近くに、現在進行中で」という比喩的な意味を持つため、字面から意味を予測しにくい。
📝 「今集中すべき課題」を指す決まった言い回し。同じパターンで the matter at hand や the job at hand も使われる。
